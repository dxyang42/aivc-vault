# 📚 关于本知识库 (About)

> **AIVC Perturbation 预测知识库** - 单细胞扰动预测领域最全面的中文知识库

---

## 🎯 知识库使命

### 我们的目标

本知识库致力于构建**单细胞扰动预测 (Single-Cell Perturbation Prediction)** 领域最全面、最系统的中文知识资源，帮助研究人员：

1. **快速了解领域全貌** - 通过系统化的分类与导航
2. **找到合适的方法** - 通过详细的比较与选择指南
3. **理解技术原理** - 通过清晰的概念解释与关联
4. **跟踪前沿进展** - 通过持续更新的方法收录

### 核心价值

| 价值 | 说明 |
|-----|------|
| **系统性** | 覆盖从基础概念到前沿方法的完整知识体系 |
| **结构化** | Obsidian 双链结构，概念与方法相互关联 |
| **实用性** | 提供方法选择指南、对比矩阵等决策工具 |
| **时效性** | 持续更新，跟踪 2018-2026 领域发展 |
| **开源性** | 开放协作，欢迎社区贡献 |

## 🏗️ 技术特点

### Obsidian 风格设计

本知识库采用 [Obsidian](https://obsidian.md/) 风格的 Markdown 组织方式：

- **双链语法** `[[文档名]]` - 文档间相互关联
- **嵌入语法** `![[文档名]]` - 嵌入其他文档内容
- **标签系统** `#标签` - 多维度分类
- **Mermaid 图表** - 可视化关系与流程
- **Dataview 查询** - 动态内容聚合

### GitHub 兼容

所有文档保持 GitHub 兼容：

- 标准 Markdown 语法
- 表格、列表、代码块正常渲染
- Mermaid 图表支持
- 无需 Obsidian 也能阅读
