## 使用建议

### 日常习惯

1. **每日开始**: 复制当日日期模板，设定目标
2. **阅读时**: 使用方法研读或概念学习模板
3. **读论文时**: 使用文献阅读模板
4. **跑实验时**: 使用实验记录模板
5. **每日结束**: 更新进度，记录收获

### 标签建议

- `#daily-note` - 日常笔记
- `#method-study` - 方法研读
- `#concept-study` - 概念学习
- `#paper-reading` - 文献阅读
- `#experiment` - 实验记录
- `#idea` - 灵感记录

### 链接建议

- 使用 `[[文档名]]` 链接到知识库中的相关文档
- 使用 `#标签` 标记笔记类型
- 使用 `![[嵌入文档]]` 嵌入相关内容
