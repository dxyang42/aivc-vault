# 技术路线对比

## Perturbation Prediction 技术路线深度对比

### 概述

Perturbation Prediction（扰动效果预测）是单细胞分析的核心任务之一，旨在预测细胞在基因敲除、药物处理等扰动条件下的转录组响应。不同技术路线在这个任务上各有优劣，本节专门对比分析。

```
+---------------------------------------------------------------------+
|                    Perturbation Prediction 技术路线全景              |
+---------------------------------------------------------------------+
|                                                                     |
|   +-------------+  +-------------+  +-------------+  +------------+ |
|   | VAE-based   |  | GNN-based   |  |Flow-based   |  |Transformer | |
|   |             |  |             |  |             |  |  -based    | |
|   | - scGen     |  | - GEARS     |  | - CellFlow  |  | - STATE    | |
|   | - CPA       |  | - CellOracle|  | - scDFM     |  | - X-Pert   | |
|   |             |  |             |  |             |  |            | |
|   | 概率框架    |  | 先验知识    |  | 连续建模    |  | 预训练表示 | |
|   | 不确定性量化|  | 机制可解释  |  | 高质量生成  |  | 零样本能力 | |
|   +-------------+  +-------------+  +-------------+  +------------+ |
|                                                                     |
|   核心问题: 如何准确预测细胞对扰动的转录组响应？                      |
|                                                                     |
+---------------------------------------------------------------------+
```

## 路线1: VAE-based Perturbation Prediction

### 代表方法
scGen (2019), CPA (2023), scPerb (2024)

### 核心思想
VAE-based方法将扰动预测建模为潜在空间中的向量运算。学习一个从对照细胞到扰动后细胞的潜在空间映射向量，新细胞的预测通过潜在空间向量加法完成。

```
对照细胞表达        潜在空间          扰动后细胞表达
     |                 |                  |
     |    +-------+    |     +-------+    |
     +--->|Encoder|----+---->|Decoder|----+
          +---+---+    |     +-------+
              |        |
              v        |
           z_control   |
              +        |
           delta_z     | (扰动向量)
              |        |
              v        |
           z_perturbed |
```

### 在Perturbation Prediction上的优势

| 维度 | 表现 | 说明 |
|------|------|------|
| **不确定性量化** | 优秀 | 概率框架天然支持置信度估计 |
| **计算效率** | 高 | 单次前向传播，推理速度快 |
| **数据效率** | 中等 | 相比GNN需要更多数据，但比Flow-based少 |
| **组合扰动** | CPA支持 | 解耦表示支持组合预测 |
| **剂量响应** | CPA支持 | 连续剂量建模能力 |

### 在Perturbation Prediction上的局限

| 局限 | 说明 |
|------|------|
| **生成质量** | 相比Flow-based方法，生成样本质量较低 |
| **高斯先验约束** | 潜在空间的分布假设可能限制表达能力 |
| **组合泛化** | scGen原生不支持组合扰动 |
| **连续动态** | 不支持细胞状态的连续插值 |

### 适用场景
- 需要不确定性量化的研究
- 计算资源有限的场景
- 批次效应校正
- 剂量响应建模（CPA）

## 路线2: GNN-based Perturbation Prediction

### 代表方法
GEARS (2022), Cell Oracle (2022), Celcomen (2025)

### 核心思想
GNN-based方法显式利用基因关系网络（如PPI网络），通过图神经网络传播扰动信号。扰动基因作为图上的信号源，通过网络传播影响其他基因。

```
        扰动基因A
            |
            v
    +---------------+
    |   基因关系图   |
    |  +---+ +---+  |
    |  | A |<->| B |  |
    |  +-+-+ +---+  |
    |    |   +---+  |
    |    +-->| C |  |
    |        +-+-+  |
    |      +---+    |
    |      v        |
    |    +---+      |
    |    | D |      |
    |    +---+      |
    +---------------+
            |
            v
    预测表达变化
```

### 在Perturbation Prediction上的优势

| 维度 | 表现 | 说明 |
|------|------|------|
| **机制可解释性** | 优秀 | 注意力权重反映基因重要性 |
| **组合效应建模** | GEARS优秀 | 显式建模非加性相互作用 |
| **先验知识整合** | 优秀 | 利用PPI等生物学网络 |
| **样本效率** | 高 | 网络结构减少数据需求 |
| **新基因预测** | 部分支持 | 可预测网络中未见基因的间接效应 |

### 在Perturbation Prediction上的局限

| 局限 | 说明 |
|------|------|
| **网络依赖** | 预测质量受限于输入网络的质量和完整性 |
| **网络偏见** | 现有网络可能偏向研究较多的基因和通路 |
| **动态建模** | 不支持细胞状态的连续动态变化 |
| **空间信息** | 仅Celcomen支持空间转录组 |

### 适用场景
- 需要机制解释的药物靶点发现
- 基因调控网络推断
- 组合扰动效应预测
- 空间转录组分析（Celcomen）

## 路线3: Flow-based Perturbation Prediction

### 代表方法
CellFlow (2025), DC-DSB (2025), scDFM (2026), SCALE (2026), CFM-GP (2025), AlphaCell (2026)

### 核心思想
Flow-based方法学习细胞状态空间中的速度场，将扰动预测建模为从对照状态到扰动状态的连续流动。基于最优传输理论，学习最可能的细胞状态转换路径。

```
细胞状态空间

对照状态        中间状态        扰动后状态
   *------------*------------*
   |            |            |
   |    速度场   |            |
   |     |      |            |
   |     |      |            |
   |     |      |            |
   +---->|------|------>|----+

   流匹配: 学习速度场 v(x,t)
   扩散模型: 学习去噪过程
```

### 各Flow-based方法对比

| 方法 | 核心创新 | 优势 | 局限 |
|------|---------|------|------|
| **CellFlow** | 最优传输流匹配 | 连续插值，可解释路径 | 标准CFM，无特殊约束 |
| **DC-DSB** | 方向约束扩散薛定谔桥 | 生物学可解释路径 | 计算成本高 |
| **scDFM** | 分布级条件流匹配 | 捕捉群体异质性 | 训练复杂 |
| **SCALE** | 大规模可扩展流匹配 | 1亿数据支持 | 需要多卡训练 |
| **CFM-GP** | 跨细胞类型流匹配 | 跨类型迁移 | 需要多类型数据 |
| **AlphaCell** | 世界模型+流匹配 | 全基因组表征 | 训练成本最高 |

### 在Perturbation Prediction上的优势

| 维度 | 表现 | 说明 |
|------|------|------|
| **生成质量** | 优秀 | 当前最高质量的单细胞生成 |
| **连续插值** | 优秀 | 支持观察中间过渡状态 |
| **不确定性量化** | 良好 | 通过多采样估计不确定性 |
| **可解释路径** | 优秀 | 转换路径具有生物学意义 |
| **大规模数据** | SCALE优秀 | 可扩展至亿级数据 |

### 在Perturbation Prediction上的局限

| 局限 | 说明 |
|------|------|
| **计算成本** | 推理需要多步采样，计算开销大 |
| **训练数据需求** | 需要大量配对的扰动-响应对 |
| **推理速度** | 相比单次前向传播方法慢10-100倍 |
| **训练稳定性** | 某些架构可能存在训练不稳定问题 |

### 适用场景
- 需要高质量生成样本的虚拟细胞实验
- 需要观察细胞状态连续转换过程的研究
- 对预测不确定性有要求的应用场景
- 有足够计算资源和训练数据的场景

## 路线4: Transformer-based Perturbation Prediction

### 代表方法
scGPT (2023), STATE (2025), X-Pert (2025), scFoundation (2024), UCE (2024)

### 核心思想
Transformer-based方法通过大规模预训练学习基因的通用表示，将扰动预测转化为条件生成任务。利用自注意力机制捕捉基因间的长程依赖关系。

```
输入序列: [CLS] gene_A:0.5 gene_B:2.3 ... [PERT] KO_gene_X [MASK]...
                |
                v
    +-----------------------+
    |    Transformer Encoder |
    |  +-----------------+  |
    |  |  Self-Attention |  |
    |  |  +---+ +---+     |  |
    |  |  | Q | | K |     |  |
    |  |  +-+-+ +-+-+     |  |
    |  |    +->|+|        |  |
    |  |       |V|        |  |
    |  |       +-+        |  |
    |  +---------+--------+  |
    |            |           |
    |      FFN Layer         |
    +------------+-----------+
                 |
                 v
    预测被掩码基因的表达值
```

### 各Transformer方法对比

| 方法 | 核心创新 | 优势 | 局限 |
|------|---------|------|------|
| **scGPT** | 生成式预训练 | 多组学支持，生态完善 | 原生不支持组合预测 |
| **STATE** | 双模块+零样本 | 1.7亿细胞，零样本预测 | 预训练成本极高 |
| **X-Pert** | 统一扰动框架 | 统一处理遗传/化学/组合 | 架构复杂 |
| **scFoundation** | 大规模预训练 | 1亿参数，读取级建模 | 推理资源需求高 |
| **UCE** | 跨物种通用嵌入 | 36种物种统一表示，ESM-2整合 | 需下游模型进行具体预测 |

### 在Perturbation Prediction上的优势

| 维度 | 表现 | 说明 |
|------|------|------|
| **零样本预测** | STATE/X-Pert优秀 | 无需针对特定扰动训练 |
| **迁移能力** | 优秀 | 预训练模型可迁移到多种任务 |
| **上下文感知** | 优秀 | 自注意力捕捉长程依赖 |
| **推理速度** | 高 | 单次前向传播 |
| **大规模数据利用** | 优秀 | 可利用海量未标注数据 |

### 在Perturbation Prediction上的局限

| 局限 | 说明 |
|------|------|
| **计算资源需求** | 预训练需要大量GPU资源 |
| **可解释性** | 深度Transformer内部机制难解释 |
| **预训练偏见** | 预训练数据的质量和偏见影响表现 |
| **不确定性量化** | 相比VAE和Flow方法，不确定性估计较弱 |

### 适用场景
- 大规模筛选
- 零样本预测
- 需要快速推理
- 有充足计算资源进行预训练或微调

## 路线5: 因果推断 Perturbation Prediction

### 代表方法
CASCADE (2024), scCausalVI (2025), CausalBERT (2025)

### 核心思想
因果推断方法显式建模变量间的因果关系，而不仅仅是相关性。通过利用扰动数据中的干预信息，这些方法可以学习因果图并进行反事实预测。

```
因果图学习:

    基因A ----> 基因B ----> 基因C
      |          |          |
      |          v          |
      +-------> 基因D <-----+

    扰动基因A (do(A=0)):
    
    基因A=0    基因B ----> 基因C
                 |          |
                 v          |
                基因D <-----+

    反事实预测: P(基因C | do(A=0))
```

### 各因果方法对比

| 方法 | 核心创新 | 优势 | 局限 |
|------|---------|------|------|
| **CASCADE** | VAE + 因果发现 | 深度学习+因果发现结合 | 因果发现计算成本高 |
| **scCausalVI** | 因果VAE + do-算子 | 单细胞分辨率反事实 | 变分推断近似误差 |
| **CausalBERT** | Transformer + 因果图 | 预训练+显式因果干预 | 依赖先验因果图质量 |

### 在Perturbation Prediction上的优势

| 维度 | 表现 | 说明 |
|------|------|------|
| **因果可解释性** | 最高 | 输出有向因果图 |
| **反事实预测** | 支持 | "如果...会怎样"分析 |
| **干预设计** | 支持 | 推荐最优干预策略 |
| **机制理解** | 深入 | 理解扰动传播的因果机制 |
| **预训练能力** | CausalBERT优秀 | 结合大规模预训练优势 |

### 在Perturbation Prediction上的局限

| 局限 | 说明 |
|------|------|
| **数据需求** | 需要足够的扰动覆盖来学习因果结构 |
| **计算复杂度** | 因果发现算法计算成本较高 |
| **因果方向** | 确定因果方向可能需要特定扰动设计 |
| **假设依赖** | 依赖因果推断的基本假设 |
| **先验依赖** | CausalBERT依赖先验因果图质量 |

### 适用场景
- 需要因果解释的药物靶点发现
- 基因调控网络推断
- 合成致死筛选
- 重编程因子设计
- 反事实推理分析 (CausalBERT)

## 路线6: 网络干扰/网络流 Perturbation Prediction

### 代表方法
NetPerturb (2025), Cell Oracle (2022), SCENIC (2017)

### 核心思想
网络干扰方法基于基因调控网络(GRN)和网络流理论，显式建模扰动在网络中的传播过程。将基因调控视为信息流网络，扰动视为网络流注入，通过深度学习预测传播效应。

```
网络流传播:

    扰动源: GATA1 (KO)
    
    GATA1 ──[0.95]──→ KLF1 ──[0.72]──→ HBB
      │
      ├──[0.68]──→ TAL1 ──[0.55]──→ GYPA
      │
      └──[0.42]──→ SPI1 ──[0.31]──→ CD14
    
    边权重 = 扰动传播强度 (可学习)
```

### 各网络方法对比

| 方法 | 核心创新 | 优势 | 局限 |
|------|---------|------|------|
| **NetPerturb** | GNN + 可学习网络流 | 传播路径可追踪，流系数可学习 | 依赖GRN质量，计算复杂度高 |
| **Cell Oracle** | GRN推断 + 线性模拟 | 细胞命运预测，生态完善 | 线性传播假设 |
| **SCENIC** | 调控子推断 + 活性评分 | 调控子发现，网络活性量化 | 非直接扰动预测 |

### 在网络扰动预测上的优势

| 维度 | 表现 | 说明 |
|------|------|------|
| **传播路径可追踪** | NetPerturb优秀 | 显式输出扰动传播路径 |
| **网络模块分析** | 优秀 | 识别受影响的通路和模块 |
| **级联效应建模** | NetPerturb优秀 | 捕捉多跳传播效应 |
| **机制可解释性** | 高 | 基于网络结构的解释 |
| **先验知识整合** | 高 | 利用已知网络结构 |

### 在网络扰动预测上的局限

| 局限 | 说明 |
|------|------|
| **GRN依赖** | 预测质量受限于输入网络质量 |
| **计算成本** | 流传播计算复杂度较高 |
| **网络不完整** | 现有网络可能缺失关键边 |
| **动态网络** | 对时变网络建模能力有限 |

### 适用场景
- 通路机制解析
- 药物靶点预测 (级联效应)
- 合成致死分析
- 网络生物学研究
- 实验设计优化

## 路线8: 时空建模 Perturbation Prediction

### 代表方法
Spateo (2024), TopoVelo (2025), Celcomen (2025)

### 核心思想
时空建模方法将空间转录组数据视为动态系统，通过整合空间位置信息和基因表达的时间演化，实现细胞命运转变的时空动态建模。这些方法能够预测扰动效应在组织中的空间传播和时间演化。

### 各方法对比

| 方法 | 核心创新 | 空间建模 | 时间建模 | 适用数据 |
|------|---------|---------|---------|----------|
| **Spateo** | Spatiotemporal GNN | Delaunay三角剖分 | 时间卷积 | Stereo-seq, MERFISH |
| **TopoVelo** | 空间耦合动力系统 | k-NN + 拓扑分析 | 微分方程 | Slide-seq, Xenium |
| **Celcomen** | 因果GNN | 空间邻接图 | 不支持 | 空间转录组 |

### 在Perturbation Prediction上的应用

1. **扰动传播预测**: 预测扰动如何在组织中随时间和空间扩散
2. **命运改变预测**: 预测扰动如何改变细胞的正常发育轨迹
3. **空间效应范围**: 估计扰动影响的空间范围
4. **时间动态**: 建模扰动效应的时间演化过程

### 时空建模方法选择指南

| 需求 | 推荐方法 | 理由 |
|------|---------|------|
| 发育时空图谱构建 | Spateo | 完整3D重建能力 |
| 细胞命运推断 | TopoVelo | RNA velocity空间扩展 |
| 扰动时空传播 | Spateo + TopoVelo | 空间建模 + 时间动态 |
| 空间因果解耦 | Celcomen | 拉格朗日约束优化 |

## 路线9: CRISPR筛选分析

### 代表方法
PS (2025), Mixscape (2021), Augur (2020), MELD (2021), MIX-CRISPR (2021)

### 核心思想
CRISPR筛选分析方法专注于从Perturb-seq等单细胞CRISPR筛选数据中提取生物学洞察。这些方法不直接预测未见扰动的效应，而是量化已观测扰动的响应特征和异质性。

### 各方法对比

| 方法 | 核心功能 | 单细胞分辨率 | 异质性量化 | 组合分析 |
|------|---------|-------------|-----------|----------|
| **PS** | 异质性响应量化 | ✓ | ✓ | ✗ |
| **Mixscape** | 扰动检测与QC | ✗ | 部分 | ✗ |
| **Augur** | 细胞类型优先级 | ✗ | ✗ | ✗ |

## 技术路线发展趋势

### 当前主流
1. **Flow-based**: 生成质量最高，成为虚拟细胞实验首选
2. **Transformer**: 零样本能力突出，适合大规模筛选
3. **GNN**: 机制可解释性强，适合药物靶点发现

### 未来方向
1. **混合架构**: 结合多种技术路线的优势
   - GNN+Flow: 机制可解释+高质量生成
   - Transformer+Flow: 零样本+连续插值
   - VAE+Causal: 概率框架+因果推理

2. **世界模型**: AlphaCell开创的方向
   - 支持任意扰动组合的通用模拟器
   - 全基因组动态建模

3. **多模态整合**: 整合转录组、蛋白质、空间信息
   - 更全面的细胞状态表示
   - 跨模态扰动预测

4. **时空建模**: Spateo, TopoVelo开创的方向
   - 空间转录组扰动预测
   - 组织水平效应建模
