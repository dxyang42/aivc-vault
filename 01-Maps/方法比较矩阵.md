# 方法比较矩阵

## Perturbation Prediction 专项对比

### 单基因扰动支持

| 方法 | 技术路线 | 支持程度 | 特点 |
|------|---------|---------|------|
| **scGen** | VAE-based | 完全支持 | 首个ML扰动预测工具，潜在空间向量运算 |
| **CPA** | VAE-based | 完全支持 | 解耦表示，支持剂量-响应 |
| **GEARS** | GNN-based | 完全支持 | 知识图谱整合，非加性效应建模 |
| **Cell Oracle** | GNN-based | 完全支持 | GRN推断+模拟，机制可解释 |
| **Celcomen** | GNN-based | 完全支持 | 空间因果解耦，拉格朗日约束 |
| **CellFlow** | Flow-based | 完全支持 | 最优传输，连续插值 |
| **DC-DSB** | Flow-based | 完全支持 | 方向约束，生物学可解释路径 |
| **scDFM** | Flow-based | 完全支持 | 分布级建模，PAD-Transformer |
| **SCALE** | Flow-based | 完全支持 | 1亿数据大规模训练 |
| **CFM-GP** | Flow-based | 完全支持 | 跨细胞类型迁移 |
| **scPPDM** | Flow-based | 不支持 | 专用于药物扰动预测 |
| **AlphaCell** | Flow-based | 完全支持 | 世界模型，全基因组表征 |
| **X-Cell** | Flow-based | 完全支持 | 49B参数扩散语言模型 |
| **scGPT** | Transformer | 完全支持 | 生成式Transformer，多组学 |
| **scFoundation** | Transformer | 完全支持 | 1亿参数，5000万细胞预训练 |
| **STATE** | Transformer | 完全支持 | 1.7亿细胞训练，零样本预测 |
| **X-Pert** | Transformer | 完全支持 | 统一扰动框架，剂量感知 |
| **GenePT** | Transformer | 不支持 | 仅支持细胞状态表示 |
| **scLAMBDA** | VAE+LLM | 完全支持 | LLM嵌入+解耦表示 |
