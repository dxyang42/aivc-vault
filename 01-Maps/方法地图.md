# 方法地图

## Perturbation Prediction 专门分类

### 按技术路线细分

#### 1. VAE-based 扰动预测方法

基于变分自编码器的扰动预测方法，提供概率框架和不确定性量化。

```
┌─────────────────────────────────────────────────────────────┐
│                    VAE-based 方法谱系                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   scVI (2018) ──→ scGen (2019) ──→ CPA (2023)              │
│      │                │                │                    │
│      │                │                └── 组合扰动+剂量响应 │
│      │                └── 首个ML扰动预测工具                │
│      └── 单细胞VAE奠基                                     │
│                                                             │
│   totalVI (2020) ──→ MultiVI (2021)                        │
│      │                │                                    │
│      └── RNA+蛋白      └── 马赛克多组学整合                  │
│                                                             │
│   CoupleVAE (2025)                                         │
│      └── 耦合VAE非线性扰动建模                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 方法 | 核心创新 | 扰动类型支持 | 不确定性量化 |
|------|---------|-------------|-------------|
| **scGen** | 潜在空间向量运算 | 单基因 | 是 |
| **CPA** | 解耦表示学习 | 单基因+组合+药物 | 是 |
| **scANVI** | 半监督细胞类型标注 | 无原生支持 | 是 |
| **CoupleVAE** | 耦合VAE非线性建模 | 单基因 | 是 |

**核心能力**: 概率建模、批次校正、端到端训练
**主要局限**: 生成质量相对流模型较低、高斯先验约束表达能力

---

#### 2. GNN-based 扰动预测方法

利用基因关系图和几何深度学习，显式建模基因间相互作用。

```
┌─────────────────────────────────────────────────────────────┐
│                    GNN-based 方法谱系                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   GEARS (2022) ──→ Celcomen (2025) ──→ Cell Oracle (2022)  │
│      │                │                │                    │
│      │                │                └── GRN+计算机模拟   │
│      │                └── 空间因果解耦                      │
│      └── 知识图谱+非加性效应建模                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 方法 | 核心创新 | 图类型 | 组合效应建模 |
|------|---------|-------|-------------|
| **GEARS** | 基因关系图神经网络 | PPI网络 | 是 |
| **Celcomen** | 因果GNN+拉格朗日约束 | 空间邻接图 | 否 |
| **Cell Oracle** | GRN推断+模拟 | 推断GRN | 否 |

**核心能力**: 先验知识整合、机制可解释、组合效应捕捉
**主要局限**: 依赖网络质量、网络不完整问题

---

**主要局限**: 计算成本高、推理速度慢、训练数据需求大

## 技术路线分类

### 1. 流匹配/扩散模型路线

基于流匹配(Flow Matching)和扩散模型(Diffusion Model)的生成式方法，学习细胞状态之间的连续转换。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **CellFlow** | 流匹配 + 最优传输 | 神经最优传输估计器，连续插值 |
| **DC-DSB** | 方向约束扩散薛定谔桥 | 方向约束机制，生物学可解释路径 |
| **scDFM** | 条件流匹配 + MMD | 分布级建模，PAD-Transformer架构 |
| **SCALE** | 流匹配 + LLaMA编码 | 大规模可扩展(1亿数据)，BioNeMo框架 |
| **CFM-GP** | 跨细胞类型流匹配 | 显式细胞类型建模，跨类型迁移 |
| **CellFlux** | 流模型 | 基于流的单细胞预测 |
| **scPPDM** | 扩散模型(DDPM) | 药物反应预测专用 |
| **AlphaCell** | 世界模型 + 流匹配 | 虚拟细胞世界模型，全基因组表征 |
| **X-Cell** | 扩散语言模型 | 49亿参数，多模态生物先验整合 |

### 2. Transformer 基础模型路线

基于Transformer架构的大规模预训练模型，学习单细胞数据的通用表示。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **scGPT** | 生成式Transformer | 单细胞GPT，多组学支持 |
| **scGPT-spatial** | 持续预训练 + 空间编码 | 空间转录组扩展，MoE解码器 |
| **GenePT** | GPT嵌入 + 加权平均 | 极简设计，零样本能力 |
| **scBERT** | BERT + Performer | 全基因组输入，线性复杂度 |
| **scFoundation** | xTrimoGene架构 | 1亿参数，5000万细胞预训练 |
| **UCE** | Transformer + 蛋白质结构 | 跨物种统一表示，3300万细胞 |
| **STATE** | 双模块Transformer | 1.7亿细胞训练，零样本预测 |
| **X-Pert** | Perturbation Perceiver | 统一遗传/化学/组合扰动建模 |
| **CellFM** | RetNet架构 | 8亿参数，1亿细胞，高效推理 |
| **scKGBERT** | Transformer + PPI知识图谱 | 890万相互作用知识增强 |

### 3. 变分自编码器(VAE)路线

基于变分推断的生成模型，提供概率框架和不确定性量化。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **scVI** | VAE + ZINB | 单细胞变分推断奠基工作 |
| **scGen** | VAE + 潜在空间运算 | 首个ML扰动预测工具 |
| **CPA** | 组合扰动VAE | 解耦表示，剂量-响应建模 |
| **scANVI** | 半监督VAE | 细胞类型注释 |
| **totalVI** | 多模态VAE | RNA+蛋白质联合建模 |
| **MultiVI** | 马赛克整合VAE | 非配对多组学整合 |
| **CoupleVAE** | 耦合VAE | 非线性扰动建模，双分支架构 |

### 4. 图神经网络(GNN)路线

利用基因关系图和几何深度学习捕捉基因间相互作用。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **GEARS** | GNN + 知识图谱 | 基因关系图，非加性效应建模 |
| **Celcomen** | 因果GNN | 空间因果解耦，拉格朗日约束 |
| **Cell Oracle** | GRN + 模拟 | 基因调控网络推断，计算机模拟 |

### 5. 因果推断/因果发现路线

显式建模因果关系，支持反事实预测和干预设计。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **CASCADE** | VAE + 因果发现 | 从扰动数据学习因果图，反事实预测 |
| **scCausalVI** | 因果VAE + do-算子 | 单细胞分辨率反事实，效应解耦 |
| **CausalBERT** | Transformer + 因果图 | 预训练语言模型 + 显式因果干预，do-calculus实现 |

### 6. 线性基线/统计方法路线

经典统计方法和线性基线，为深度学习方法提供参照。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **Linear Baseline** | 平均效应向量 | 扰动预测经典基线，简单高效 |
| **MIX-CRISPR** | GI评分计算 | 组合筛选标准分析，统计严谨 |
| **DrugCell** | VNN + GO | 可见神经网络，GO级别可解释 |
| **Dose-Response** | Hill方程 | 剂量响应建模金标准，参数可解释 |
| **Mixscape** | 似然比检验 | CRISPR筛选QC，扰动检测 |
| **Augur** | 分类器AUC | 细胞类型响应排序 |
| **MELD** | 图信号处理 | 连续扰动效应量化 |
| **NicheNet** | 网络传播 | 细胞间通讯推断 |
| **DIALOGUE** | 惩罚CVA | 多细胞程序发现 |
| **Perturb-Viz** | 交互式可视化 | 扰动效应可视化分析平台 |

### 7. 网络干扰/网络流路线

基于基因网络结构和网络流理论，显式建模扰动在网络中的传播。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **NetPerturb** | GNN + 网络流 | 深度学习网络流传播，可学习流系数，路径可追踪 |
| **Cell Oracle** | GRN + 模拟 | 基因调控网络推断，计算机模拟 |
| **SCENIC** | 网络推断 + 活性 | 调控子推断，网络活性评分 |

**核心能力**: 显式传播路径、网络模块分析、级联效应建模
**主要局限**: 依赖GRN质量、计算复杂度高、对动态网络建模有限

---

### 8. 时序扰动建模路线

专门用于分析时序单细胞扰动数据的方法，显式建模时间动态。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **CellDrift** | GLM + 时间一致性约束 | 时序扰动效应分析，基因程序追踪 |
| **PRESCIENT** | VAE + Neural ODE | 生成式时间模型，细胞命运预测，物理时间建模 (Cell Systems 2020) |
| **scVelo** | 动力学模型 | RNA速度分析，剪接动力学 |

**核心能力**: 时间动态建模、细胞命运预测、轨迹推断
**主要局限**: 需要时序数据、计算复杂度高

---

### 9. 时空建模路线

专门用于空间转录组数据的时空动态建模，分析扰动效应的空间传播。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **Spateo** | Spatiotemporal GNN | 多维时空建模，3D重建，细胞-细胞相互作用 (Nature Methods 2024) |
| **TopoVelo** | 空间耦合动力系统 | RNA velocity空间扩展，拓扑数据分析 (Nature Biotechnology 2025) |
| **scGPT-spatial** | Transformer + 空间编码 | 空间转录组扩展，MoE解码器 |
| **Celcomen** | 因果GNN | 空间因果解耦，拉格朗日约束 |

**核心能力**: 空间动态建模、3D重建、扰动传播预测
**主要局限**: 空间数据依赖、计算成本高

---

### 9. CRISPR筛选分析路线

专门用于分析单细胞CRISPR筛选数据的方法。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **PS** | 统计框架 | 单细胞异质性响应量化，Nature Cell Biology 2025 |
| **Mixscape** | 似然比检验 | CRISPR筛选QC，扰动检测 |
| **MIX-CRISPR** | GI评分计算 | 组合筛选标准分析 |
| **Augur** | 分类器AUC | 细胞类型响应排序 |
| **MELD** | 图信号处理 | 连续扰动效应量化 |

**核心能力**: 异质性量化、响应分析、统计检验
**主要局限**: 非预测方法，仅分析已观测数据

---

### 10. 迁移学习/参考映射路线

### 7. 网络干扰/网络流路线

基于基因网络结构和网络流理论，显式建模扰动在网络中的传播。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **NetPerturb** | GNN + 网络流 | 深度学习网络流传播，可学习流系数，路径可追踪 |
| **Cell Oracle** | GRN + 模拟 | 基因调控网络推断，计算机模拟 |
| **SCENIC** | 网络推断 + 活性 | 调控子推断，网络活性评分 |

**核心能力**: 显式传播路径、网络模块分析、级联效应建模
**主要局限**: 依赖GRN质量、计算复杂度高、对动态网络建模有限

---

### 8. 时序扰动建模路线

专门用于分析时序单细胞扰动数据的方法，显式建模时间动态。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **CellDrift** | GLM + 时间一致性约束 | 时序扰动效应分析，基因程序追踪 |
| **PRESCIENT** | 生成式时间模型 | 细胞命运预测，物理时间建模 |

### 9. 可视化分析工具

专门用于扰动效应可视化和交互式分析的平台和工具。

| 方法 | 核心技术 | 特点 |
|------|---------|------|
| **Perturb-Viz** | 交互式可视化平台 | 扰动专用可视化，组合效应分析，动态轨迹动画 |
| **Mixscape** | 统计可视化 | CRISPR筛选QC，扰动检测可视化 |

**核心能力**: 交互式探索、多维度可视化、预测结果验证、报告生成
**主要局限**: 非预测方法，依赖输入数据质量
