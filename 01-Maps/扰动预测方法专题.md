# 扰动预测方法专题

## 概述

Perturbation Prediction（扰动效果预测）是计算生物学和单细胞分析领域的核心任务，旨在预测细胞在基因敲除（KO）、基因过表达（OE）、药物处理等扰动条件下的转录组响应。

## 各方法核心创新点

### VAE-based 方法

#### scGen (2019)
- **核心创新**: 潜在空间向量运算
- **技术细节**: 学习对照到扰动的潜在空间位移向量，新细胞预测通过向量加法完成
- **意义**: 首个端到端ML扰动预测工具

#### CPA (2023)
- **核心创新**: 解耦表示学习 + 剂量响应建模
- **技术细节**: 将扰动效应分解为细胞状态、扰动类型、剂量三个独立因子
- **意义**: 首个同时支持组合扰动和剂量响应的方法

#### scPerb (2024)
- **核心创新**: 风格迁移 + VAE
- **技术细节**: 将扰动视为"风格"，内容-风格解耦编码
- **意义**: 创新的风格迁移思路应用于扰动预测

### GNN-based 方法

#### GEARS (2022)
- **核心创新**: 基因关系图 + 非加性效应建模
- **技术细节**: 利用PPI网络构建基因图，GNN消息传递捕捉基因间相互作用
- **意义**: 首个显式建模组合扰动非加性效应的深度学习方法

#### Cell Oracle (2022)
- **核心创新**: GRN推断 + 计算机模拟
- **技术细节**: 从scRNA-seq数据推断基因调控网络，通过网络传播模拟扰动效应
- **意义**: 提供机制可解释的扰动预测

#### Celcomen (2025)
- **核心创新**: 因果GNN + 空间解耦
- **技术细节**: 拉格朗日约束优化，分离空间效应和扰动效应
- **意义**: 首个支持空间转录组扰动预测的GNN方法

### Flow-based 方法

#### CellFlow (2025)
- **核心创新**: 最优传输流匹配
- **技术细节**: 学习神经最优传输估计器，实现细胞状态的连续插值
- **意义**: 将最优传输理论引入单细胞扰动预测

#### DC-DSB (2025)
- **核心创新**: 方向约束扩散薛定谔桥
- **技术细节**: 在扩散过程中加入生物学方向约束，确保路径合理性
- **意义**: 提高生成路径的生物学可解释性

#### scDFM (2026)
- **核心创新**: 分布级条件流匹配
- **技术细节**: 在分布级别而非样本级别建模，捕捉群体异质性
- **意义**: 更准确地预测细胞群体的多样性响应

#### SCALE (2026)
- **核心创新**: 大规模可扩展流匹配
- **技术细节**: 基于BioNeMo框架，支持1亿数据训练
- **意义**: 展示流匹配方法的可扩展性

#### AlphaCell (2026)
- **核心创新**: 虚拟细胞世界模型
- **技术细节**: 结合世界模型和流匹配，学习全基因组动态
- **意义**: 迈向通用虚拟细胞的重要一步

#### CFM-GP (2025)
- **核心创新**: 跨细胞类型流匹配
- **技术细节**: 统一条件流匹配框架，显式细胞类型编码
- **意义**: 支持跨细胞类型的扰动效应迁移

### Transformer-based 方法

#### scGPT (2023)
- **核心创新**: 生成式Transformer预训练
- **技术细节**: 将基因视为词汇，细胞视为句子，采用GPT架构
- **意义**: 将NLP预训练范式成功迁移到单细胞领域

#### scFoundation (2024)
- **核心创新**: 大规模预训练 + 测序读取增强
- **技术细节**: 1亿参数，5000万细胞，读取级建模
- **意义**: 展示大规模预训练在单细胞领域的潜力

#### STATE (2025)
- **核心创新**: 双模块Transformer + 零样本预测
- **技术细节**: 1.7亿细胞训练，无需针对特定扰动微调
- **意义**: 实现真正的零样本扰动预测

#### X-Pert (2025)
- **核心创新**: 统一扰动框架
- **技术细节**: Perturbation Perceiver架构，统一处理遗传/化学/组合扰动
- **意义**: 首个真正统一的扰动预测框架

### 因果推断方法

#### CASCADE (2024)
- **核心创新**: 深度学习 + 因果发现
- **技术细节**: 从扰动数据学习因果图，支持反事实预测和干预设计
- **意义**: 专为扰动数据设计的因果发现工具

#### scCausalVI (2025)
- **核心创新**: 因果VAE + do-算子
- **技术细节**: 结构因果模型结合变分推断，单细胞分辨率反事实预测
- **意义**: 实现单细胞级别的因果推理和效应解耦

### 统计/分析方法

#### Linear Baseline (2019)
- **核心创新**: 线性基线模型
- **技术细节**: 计算每个扰动的平均效应向量，新细胞预测通过向量加法完成
- **意义**: 扰动预测领域的经典基线，为深度学习方法提供性能参照

#### Mixscape (2021)
- **核心创新**: CRISPR筛选QC和扰动检测
- **技术细节**: 似然比检验区分真实扰动细胞和未受扰动细胞
- **意义**: 解决gRNA脱靶和编辑效率变异问题

#### MIX-CRISPR (2021)
- **核心创新**: 大规模组合CRISPR筛选分析
- **技术细节**: 系统性双基因组合筛选，定量遗传相互作用(GI)评分
- **意义**: 组合扰动筛选的标准分析方法

#### Augur (2020)
- **核心创新**: 细胞类型响应优先级排序
- **技术细节**: 使用分类器AUC量化细胞类型对扰动的可分离性
- **意义**: 识别组织中最响应扰动的细胞类型

#### MELD (2021)
- **核心创新**: 连续扰动效应量化
- **技术细节**: 图信号处理增强实验信号，密度似然比估计效应强度
- **意义**: 捕捉细胞状态的渐变响应

#### Dose-Response Modeling (经典)
- **核心创新**: 药理学剂量响应建模
- **技术细节**: Hill方程拟合剂量-响应曲线，EC50/IC50参数定量
- **意义**: 扰动效应定量化的标准方法

#### NicheNet (2019/2020/2024)
- **核心创新**: 细胞间通讯推断
- **技术细节**: 整合先验知识预测配体-受体-靶基因调控路径
- **意义**: 从表达数据反推活跃的细胞间通讯

#### DIALOGUE (2022)
- **核心创新**: 多细胞程序发现
- **技术细节**: 惩罚典型变量分析识别跨细胞类型共变的基因程序
- **意义**: 揭示组织水平的功能单元

## 应用场景对比

### 药物发现

| 场景 | 推荐方法 | 理由 |
|------|---------|------|
| 靶点发现 | GEARS, Cell Oracle, GNN-based | 机制可解释，整合PPI网络 |

| 场景 | 推荐方法 | 理由 |

| 合成致死预测 | GEARS | 专门优化组合效应建模 |

### 临床转化

| 场景 | 推荐方法 | 理由 |
|------|---------|------|

### 技术路线优劣势对比

#### VAE-based (scGen, CPA, scPerb)

**优势:**
- 概率框架，天然支持不确定性量化
- 计算效率高，单次前向传播
- 理论基础扎实，数学可解释性强
- 生态成熟，scvi-tools支持完善

**劣势:**
- 生成质量不如Flow-based方法
- 高斯先验假设限制表达能力
- 对复杂非线性相互作用建模能力有限

**适用场景:** 需要不确定性量化、计算资源有限、批次效应校正

#### GNN-based (GEARS, Cell Oracle, Celcomen)

**优势:**
- 机制可解释性强，整合先验知识
- 组合效应建模能力强（GEARS）
- 样本效率高，网络结构减少数据需求
- 支持空间转录组（Celcomen）

**劣势:**
- 依赖高质量基因关系网络
- 现有网络可能不完整或有偏见
- 对未见网络结构泛化能力有限

**适用场景:** 需要机制解释、药物靶点发现、组合扰动预测

#### Flow-based (CellFlow, DC-DSB, scDFM, SCALE, CFM-GP, AlphaCell)

**优势:**
- 生成质量最高，当前最佳
- 支持连续状态插值，可解释路径
- 理论基础扎实（最优传输）
- 可扩展至大规模数据（SCALE）

**劣势:**
- 计算成本高，推理需要多步采样
- 训练数据需求大
- 推理速度慢10-100倍

**适用场景:** 需要最高生成质量、虚拟细胞实验、连续动态观察

#### Transformer-based (scGPT, STATE, X-Pert, scFoundation)

**优势:**
- 零样本预测能力（STATE, X-Pert）
- 迁移能力强，预训练模型可复用
- 上下文感知，捕捉长程依赖
- 推理速度快（单次前向）

**劣势:**
- 预训练需要大量GPU资源
- 可解释性相对较弱
- 对罕见细胞类型表现有限

**适用场景:** 大规模筛选、零样本预测、需要快速推理

#### 因果推断 (CASCADE, scCausalVI)

**优势:**
- 因果可解释性最高
- 支持反事实预测
- 可推荐最优干预策略
- 机制理解深入

**劣势:**
- 计算复杂度高
- 需要足够的扰动覆盖
- 因果方向确定有挑战

**适用场景:** 需要因果解释、干预设计、机制研究

## 总结

Perturbation Prediction领域经历了从统计方法到深度学习，再到基础模型和生成式AI的演进。不同技术路线各有优劣：

- **VAE-based**: 概率框架，不确定性量化，计算高效
- **GNN-based**: 机制可解释，先验知识整合，样本高效
- **Flow-based**: 生成质量最高，连续插值，可解释路径
- **Transformer-based**: 零样本预测，迁移能力强，大规模数据利用
- **因果推断**: 显式因果建模，反事实预测，机制理解

选择合适的方法需要考虑扰动类型、预测准确率要求、可解释性需求、计算资源约束等多个因素。随着技术的不断进步，我们正朝着通用虚拟细胞的目标迈进。

## 线性基线与其他方法的对比

### Linear Baseline 的定位

Linear Baseline (Norman 2019) 是扰动预测领域的经典基线方法，虽然简单但具有重要意义：

| 对比维度 | Linear Baseline | 深度学习方法 | 说明 |
|---------|-----------------|-------------|------|
| **模型复杂度** | 极低 (向量加法) | 高 (神经网络) | Linear无需训练，直接计算平均效应 |
| **预测精度** | 中等 (R²: 0.55-0.65) | 高 (R²: 0.75-0.95) | 深度学习方法相对提升30-50% |
| **非加性效应** | 不支持 | GEARS等支持 | 这是Linear最主要的局限 |
| **计算速度** | 极快 (毫秒级) | 中等/慢 | Linear查表式预测 |
| **可解释性** | 高 | 中/高 | Linear效应向量直接可见 |
| **训练需求** | 无 | 需要大量数据 | Linear无需模型训练 |

### 与主要方法的详细对比

#### Linear vs GEARS

| 特性 | Linear | GEARS |
|------|--------|-------|
| 组合效应 | 简单加性 | 显式建模非加性效应 |
| 知识整合 | 无 | PPI网络 |
| GI评分 | 不支持 | 支持预测GI |
| 适用场景 | 基线比较 | 合成致死发现 |

**关键差异**: GEARS的核心优势是捕捉基因间的非加性相互作用，这是Linear无法做到的。

#### Linear vs CPA

| 特性 | Linear | CPA |
|------|--------|-----|
| 组合预测 | 加性假设 | 解耦表示学习 |
| 剂量响应 | 不支持 | 显式支持 |
| 不确定性 | 无 | 概率框架 |
| 药物扰动 | 不支持 | 完全支持 |

**关键差异**: CPA通过解耦表示学习，能够更灵活地建模组合效应和剂量响应。

#### Linear vs Flow-based (CellFlow, scDFM)

| 特性 | Linear | Flow-based |
|------|--------|-----------|
| 生成质量 | 低 | 很高 |
| 连续插值 | 不支持 | 支持 |
| 计算成本 | 极低 | 高 |
| 训练数据 | 无需训练 | 需要大量数据 |

**关键差异**: Flow-based方法在生成质量和连续动态建模上远超Linear，但计算成本也显著更高。

### Linear Baseline 的应用价值

尽管深度学习方法在预测精度上远超Linear，但它仍有重要价值：

1. **性能基准**: 为评估深度学习方法提供参照
2. **快速筛选**: 大规模扰动的初步筛选
3. **效应分析**: 理解单个扰动的平均效应
4. **教学演示**: 理解扰动预测的基本原理

## 组合扰动预测方法对比

### 组合扰动预测的核心挑战

组合扰动预测需要解决以下关键问题：
1. **非加性效应**: 基因间相互作用无法简单相加
2. **组合爆炸**: 可能的基因对数量巨大
3. **数据稀疏**: 实验只能覆盖一小部分组合

### 主要组合扰动预测方法

| 方法 | 技术路线 | 组合建模策略 | 核心优势 | 主要局限 |
|------|---------|-------------|---------|---------|
| **Linear Baseline** | 线性 | 简单加性 | 极简高效 | 无法捕捉非加性效应 |
| **MIX-CRISPR** | 统计分析 | GI评分计算 | 系统性筛选分析 | 需要专门实验设计 |
| **GEARS** | GNN | 知识图谱+非加性解码器 | 显式建模相互作用 | 依赖网络质量 |
| **CPA** | VAE | 解耦表示学习 | 剂量+组合联合建模 | 组合泛化有限 |
| **scDFM** | Flow | 分布级条件流匹配 | 高质量组合预测 | 计算成本高 |
| **X-Pert** | Transformer | Perturbation Perceiver | 统一框架，零样本 | 架构复杂 |
| **STATE** | Transformer | 大规模预训练 | 无需组合训练数据 | 预训练成本极高 |

## 药物响应预测方法对比

### 药物响应预测的特殊性

相比基因扰动预测，药物响应预测有以下特点：
1. **剂量依赖性**: 需要建模剂量-响应关系
2. **化学多样性**: 药物分子结构差异巨大
3. **多靶点效应**: 药物可能同时影响多个通路
4. **细胞特异性**: 不同细胞系响应差异显著

### 主要药物响应预测方法

| 方法 | 输入类型 | 剂量建模 | 机制解释 | 单细胞支持 |
|------|---------|---------|---------|-----------|
| **Dose-Response** | 浓度-响应数据 | Hill方程 | 参数可解释 | 群体水平 |
| **DrugCell** | 突变+药物指纹 | 多剂量训练 | GO级别 | 否 (批量) |
| **CPA** | 转录组+剂量 | 连续嵌入 | 解耦因子 | 是 |
| **scPPDM** | 单细胞+药物 | 条件扩散 | 中等 | 是 |
| **X-Pert** | 转录组+药物ID | 剂量感知模块 | 注意力权重 | 是 |
| **CellFlow** | 转录组 | 不支持 | 速度场 | 是 |

### 剂量响应建模能力对比

| 方法 | 剂量建模方式 | EC50/IC50 | 协同评估 | 适用场景 |
|------|-------------|-----------|---------|---------|
| **Dose-Response** | Hill方程拟合 | 核心输出 | Bliss/Loewe | 标准药理学分析 |
| **DrugCell** | 多剂量训练 | 预测输出 | 扩展支持 | 批量细胞系筛选 |
| **CPA** | 连续条件嵌入 | 间接推断 | 部分支持 | 单细胞剂量实验 |
| **X-Pert** | 剂量感知模块 | 支持 | 支持 | 统一扰动框架 |

### 药物响应预测方法选择

| 应用场景 | 推荐方法 | 理由 |
|---------|---------|------|
| 标准药理学分析 | Dose-Response | 金标准方法，参数可解释 |
| 批量细胞系筛选 | DrugCell | GO级别可解释性 |
| 单细胞药物实验 | CPA, X-Pert | 支持单细胞分辨率 |
| 虚拟筛选 | DrugCell, X-Pert | 可预测新化合物 |
| 组合用药优化 | CPA, X-Pert | 协同效应建模 |
