## 各方法代码

### scGen
```python
import scgen

# 加载模型
model = scgen.VAE.load("model.pt", adata)

# 预测
pred = model.predict(adata, perturbation="gene_name")
```

### GEARS
```python
from gears import PertData, GEARS

# 初始化
pert_data = PertData("./data")
pert_data.load(data_name='norman')

# 训练
gears_model = GEARS(pert_data)
gears_model.train()

# 预测
gears_model.predict([['gene1', 'gene2']])
```

### CPA
```python
import cpa

# 训练
model = cpa.CPA(adata, ...)
model.train()

# 预测
pred = model.predict(adata, perturbation="...")
```

## 常用代码片段

### 数据预处理
```python
import scanpy as sc

# 读取
adata = sc.read_h5ad("data.h5ad")

# 质控
sc.pp.filter_cells(adata, min_genes=200)
sc.pp.filter_genes(adata, min_cells=3)

# 标准化
sc.pp.normalize_total(adata)
sc.pp.log1p(adata)
```

### 可视化
```python
import scanpy as sc

# UMAP
sc.pp.neighbors(adata)
sc.tl.umap(adata)
sc.pl.umap(adata, color='cell_type')
```
