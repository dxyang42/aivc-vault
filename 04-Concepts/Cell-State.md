# Cell State (细胞状态)

> [[概念地图]] | [[Gene-Expression]] | [[Embedding]]

## 定义

**细胞状态 (Cell State)** 是指细胞在特定时刻的分子特征总和，包括基因表达谱、蛋白质水平、代谢物浓度等。它决定了细胞的功能、行为和对环境刺激的响应方式。

### 核心概念

细胞状态是动态的、多层次的：

```
分子层: DNA甲基化 → 染色质状态 → 转录组 → 蛋白质组 → 代谢组
                    ↓
功能层: 信号通路活性 → 细胞周期阶段 → 分化状态 → 应激响应
                    ↓
表型层: 形态特征 → 运动能力 → 分泌功能 → 相互作用
```

在单细胞分析中，**转录组状态**是最常用的状态表征，因为scRNA-seq技术能够高通量地测量单个细胞的基因表达。

## 状态空间

### 连续状态空间 (Continuous State Space)

**定义**: 细胞状态被视为高维空间中的连续点。

```
数学表示:
- 状态空间 S ⊂ R^G (G = 基因数)
- 每个细胞 s ∈ S 是一个点
- 状态变化是空间中的轨迹

示例:
分化过程: s_干细胞 → s_祖细胞 → s_成熟细胞
          (连续渐变，无明确边界)
```

**特点**:
- 状态间平滑过渡
- 适合描述分化、发育等渐进过程
- 常用流形学习（如UMAP）可视化

**适用场景**:
- 胚胎发育
- 组织分化
- 细胞重编程

### 离散状态空间 (Discrete State Space)

**定义**: 细胞状态被划分为离散的类别或类型。

```
数学表示:
- 状态空间 S = {类型1, 类型2, ..., 类型K}
- 每个细胞被分配到某一类型
- 状态变化是类型间的跳转

示例:
免疫细胞类型: {T细胞, B细胞, 单核细胞, NK细胞, ...}
细胞周期: {G1期, S期, G2期, M期}
```

**特点**:
- 明确的类别边界
- 便于分类和注释
- 适合描述稳定细胞类型

**适用场景**:
- 细胞类型鉴定
- 疾病亚型分类
- 功能状态标记

### 混合表示

实际应用中常结合两种表示：

```
层次化状态表示:

第一层 (离散): 细胞大类
├── T细胞
│   ├── CD4+ T细胞 (离散子类型)
│   │   ├── Naive (连续状态)
│   │   ├── Effector (连续状态)
│   │   └── Memory (连续状态)
│   └── CD8+ T细胞
├── B细胞
└── 单核细胞
```

## 状态维度

### 主要状态维度

| 维度 | 描述 | 测量方法 |
|------|------|----------|
| **细胞类型** | 细胞的身份类别 | 标记基因表达、聚类 |
| **细胞周期** | 增殖状态 | 周期相关基因表达 |
| **分化阶段** | 成熟程度 | 分化标记基因 |
| **代谢状态** | 能量代谢模式 | 代谢相关基因 |
| **应激状态** | 对环境压力的响应 | 应激基因表达 |
| **空间位置** | 在组织中的位置 | 空间转录组 |

### 状态向量表示

```
细胞状态可表示为多个维度的组合:

s = [类型编码, 周期阶段, 分化水平, 代谢指标, 应激水平, ...]

示例:
- T细胞 (类型)
- G1期 (周期)
- 效应阶段 (分化)
- 糖酵解活跃 (代谢)
- 无应激 (应激)
```

## 状态转换

### 转换类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **分化** | 干细胞→成熟细胞 | 造血干细胞→红细胞 |
| **去分化** | 成熟细胞→干细胞样 | iPSC重编程 |
| **转分化** | 类型A→类型B | 成纤维细胞→神经元 |
| **激活** | 静息→活跃 | Naive T细胞→Effector T细胞 |
| **应激响应** | 正常→应激 | 热休克响应 |
| **恶性转化** | 正常→癌变 | 癌基因激活 |

### 转换动力学

```
状态转换的数学描述:

确定性模型:
ds/dt = f(s, u, t)
其中 u 是控制输入（如扰动）

随机模型:
ds = f(s,t)dt + σ(s,t)dW
考虑噪声和不确定性
```

### 状态转换图

```
                干细胞
               /   |   \
              /    |    \
           祖细胞A  祖细胞B  祖细胞C
            /  \    /  \    /  \
           /    \  /    \  /    \
        成熟A1  成熟A2  成熟B1  成熟B2

边权重: 转换概率或速率
节点: 稳定状态或亚稳态
```

## 状态与扰动的关系

### 扰动引起的状态变化

```
扰动前状态: s
扰动 p 作用
扰动后状态: s' = f(s, p)

状态变化: Δs = s' - s

其中 f 是状态转换函数（由扰动预测模型学习）
```

### 状态依赖的扰动响应

**关键观察**: 相同扰动对不同状态细胞效应不同。

```
示例: TP53敲除

正常细胞状态:
- 细胞周期阻滞
- DNA修复激活
- 可能凋亡

癌细胞状态 (已失活TP53):
- 无明显变化（已适应）
- 继续增殖

结论: 扰动效应依赖于细胞状态
```

### 状态特异性预测

| 方法 | 策略 |
|------|------|
| **条件模型** | 将细胞状态作为条件输入 |
| **状态感知** | 学习状态-扰动交互 |
| **个性化** | 针对特定状态训练子模型 |

## 状态推断方法

### 1. 聚类方法

```
识别离散细胞类型:
1. 降维 (PCA/UMAP)
2. 聚类 (Louvain/Leiden)
3. 标记基因鉴定
4. 细胞类型注释
```

### 2. 伪时间分析

```
推断连续状态轨迹:
1. 选择起始细胞（如干细胞）
2. 构建细胞相似性图
3. 计算每个细胞的伪时间（距起点的距离）
4. 排序细胞，重建分化路径

代表工具: Monocle, Slingshot, PAGA
```

### 3. RNA速度 (RNA Velocity)

```
基于剪接动力学预测状态变化:

未剪接mRNA (u) → 剪接mRNA (s)
     ↑                ↓
   转录              翻译

速度: v = ds/dt ≈ u - γs

正速度: 基因表达增加（细胞向该基因高表达状态移动）
负速度: 基因表达减少
```

### 4. 深度学习推断

| 方法 | 原理 |
|------|------|
| **scVI** | 概率模型推断潜在状态 |
| **CellRank** | 结合转录相似性和RNA速度 |
| **PRESCIENT** | 生成模型预测未来状态 |

## 状态空间可视化

### 常用方法

| 方法 | 特点 | 适用 |
|------|------|------|
| **UMAP** | 保留局部结构，分离簇 | 细胞类型可视化 |
| **t-SNE** | 强调簇分离 | 小规模数据 |
| **PCA** | 线性，可解释 | 主要变异方向 |
| **扩散图** | 捕捉非线性结构 | 发育轨迹 |
| **ForceAtlas** | 网络布局 | 细胞关系图 |

### 可视化示例

```
UMAP可视化中的状态信息:

颜色1 (细胞类型): 显示离散类型
├── T细胞簇
├── B细胞簇
└── 单核细胞簇

颜色2 (伪时间): 显示连续分化
├── 深色 (早期)
└── 浅色 (晚期)

颜色3 (扰动响应): 显示扰动效应
├── 蓝色 (下调)
├── 白色 (无变化)
└── 红色 (上调)
```
