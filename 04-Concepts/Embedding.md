# Embedding (嵌入)

> [[概念地图]] | [[Latent-Space]] | [[术语统一规范]]

## 定义

**嵌入 (Embedding)** 是将高维离散或稀疏数据映射到低维连续向量空间的技术。在单细胞生物学中，嵌入将高维基因表达谱转换为紧凑的、有意义的低维表示，便于计算分析、可视化和下游任务。

### 核心思想

```
高维空间 (输入)                    低维空间 (嵌入)
┌─────────────────┐              ┌─────────────┐
│ 基因1: 10000维   │              │             │
│ 基因2: 10000维   │   嵌入函数    │  50-512维   │
│ 基因3: 10000维   │ ───────────→ │  连续向量   │
│ ...             │              │             │
│ 细胞: 20000维    │              │             │
└─────────────────┘              └─────────────┘

关键特性:
- 降维: 从高维到低维
- 连续: 离散→连续
- 有意义: 保留语义/功能相似性
```

### 数学定义

给定输入空间 X（如基因表达空间 R^G），嵌入函数 f 将其映射到嵌入空间 Z：

```
f: X → Z

其中:
- X ∈ R^G: 输入空间（G = 20,000 基因）
- Z ∈ R^d: 嵌入空间（d = 50-512 维度）
- d << G: 显著降维
```

## 嵌入类型

### 1. 基因嵌入 (Gene Embedding)

**定义**: 将基因表示为低维向量，捕捉基因间的功能相似性。

| 特征 | 描述 |
|------|------|
| **输入** | 基因ID或基因表达模式 |
| **输出** | d维向量 (通常 128-512维) |
| **语义** | 功能相似、共表达的基因向量接近 |

**应用示例**:
```
输入: "TP53" (肿瘤抑制基因)
嵌入: [0.2, -0.5, 0.8, ..., 0.1] (256维)

相似基因:
- MDM2 (TP53调控因子): 余弦相似度 0.85
- CDKN1A (TP53靶点): 余弦相似度 0.82
- 随机基因: 余弦相似度 ~0.0
```

**典型方法**:
- [[GenePT]]: 基于基因描述文本的嵌入
- [[scBERT]]: 预训练基因表示
- 共表达网络嵌入

### 2. 细胞嵌入 (Cell Embedding)

**定义**: 将单个细胞表示为低维向量，捕捉细胞状态和类型。

| 特征 | 描述 |
|------|------|
| **输入** | 单细胞表达谱 (20,000维) |
| **输出** | d维向量 (通常 10-128维) |
| **语义** | 相同类型的细胞聚类，状态连续变化 |

**应用示例**:
```
细胞类型分离:
- T细胞: 嵌入空间聚类A
- B细胞: 嵌入空间聚类B
- 单核细胞: 嵌入空间聚类C

分化轨迹:
干细胞 → 祖细胞 → 成熟细胞
在嵌入空间中形成连续路径
```

**典型方法**:
- [[scVI]]: VAE学习细胞嵌入
- [[scGPT]]: Transformer学习细胞表示
- UMAP/t-SNE: 可视化嵌入

### 3. 扰动嵌入 (Perturbation Embedding)

**定义**: 将扰动（基因KO/OE/化合物）表示为低维向量。

| 特征 | 描述 |
|------|------|
| **输入** | 扰动标识（基因名或化合物ID） |
| **输出** | d维向量 (通常 64-256维) |
| **语义** | 相似效应的扰动向量接近 |

**应用示例**:
```
基因敲除嵌入:
- TP53 KO: 嵌入向量A
- MDM2 KO: 嵌入向量B (与A相似，因MDM2调控TP53)
- 无关基因KO: 嵌入向量C (与A、B距离远)

化合物嵌入:
- 同靶点药物: 向量聚类
- 不同机制药物: 向量分离
```

**典型方法**:
- [[CPA]]: 学习扰动和细胞状态的联合嵌入
- [[GEARS]]: 图神经网络学习基因扰动表示

## 嵌入性质

### 1. 线性 vs 非线性

| 性质 | 线性嵌入 | 非线性嵌入 |
|------|----------|------------|
| **方法** | PCA, NMF | VAE, t-SNE, UMAP |
| **映射** | z = Wx + b | z = f(x; θ) |
| **优势** | 可解释性强、计算快 | 表达能力更强 |
| **局限** | 无法捕捉复杂结构 | 可能过拟合 |

**选择指南**:
- 需要可解释性 → PCA
- 需要捕捉复杂生物过程 → VAE/深度学习
- 需要可视化 → UMAP/t-SNE

### 2. 解耦性 (Disentanglement)

**定义**: 嵌入的不同维度对应不同的生物学因素。

```
理想解耦嵌入:
维度1: 细胞类型 (T/B/单核)
维度2: 细胞周期 (G1/S/G2M)
维度3: 分化阶段 (干细胞→成熟)
维度4: 应激状态 (正常/应激)
...

非解耦嵌入:
每个维度都是多个因素的混合
```

**解耦方法**:
- β-VAE: 通过KL散度约束
- FactorVAE: 全相关惩罚
- 对比学习: 基于属性的正负样本

### 3. 语义保持性

**定义**: 嵌入空间中的距离反映生物学相似性。

| 性质 | 描述 | 验证方法 |
|------|------|----------|
| **细胞类型保持** | 同类型细胞嵌入接近 | 聚类纯度 |
| **轨迹连续性** | 分化路径在嵌入空间连续 | 伪时间相关性 |
| **扰动相似性** | 相似效应的扰动向量接近 | 效应相关性 |

## 嵌入学习方法

### 1. 自编码器方法

```
架构:
输入 x (20000维)
    ↓
编码器 (神经网络)
    ↓
嵌入 z (50维)
    ↓
解码器 (神经网络)
    ↓
重构 x' (20000维)

损失: L = ||x - x'||² + KL(z || N(0,I))
```

**代表方法**:
- [[scVI]]: 针对scRNA-seq的VAE
- [[DCA]]: 去噪自编码器
- [[scGen]]: 条件VAE用于扰动预测

### 2. 对比学习方法

```
核心思想: 相似样本嵌入接近，不相似样本远离

正样本对:
- 同一细胞的不同增强视图
- 相同类型的细胞
- 相邻时间点的细胞

负样本对:
- 不同类型的细胞
- 随机采样的细胞

损失: InfoNCE = -log(exp(sim(z_i, z_j)/τ) / Σ exp(sim(z_i, z_k)/τ))
```

**代表方法**:
- [[scConGraph]]: 图对比学习
- [[scVI-CL]]: 对比学习变体

### 3. 语言模型方法

```
将基因视为词汇，细胞视为句子:

细胞 "句子":
[GeneA表达高] [GeneB表达低] [GeneC表达高] ...

Transformer编码:
→ 基因嵌入 + 位置编码
→ 自注意力计算
→ 细胞级表示
```

**代表方法**:
- [[scGPT]]: 单细胞GPT
- [[scBERT]]: 单细胞BERT
- [[Geneformer]]: 基因语言模型

### 4. 图神经网络方法

```
输入: 基因-基因关系图
节点: 基因
边: 调控关系或共表达

GNN消息传递:
h_i^(l+1) = UPDATE(h_i^(l), AGGREGATE(h_j^(l) for j in N(i)))

输出: 基因嵌入 (考虑网络结构)
```

**代表方法**:
- [[GEARS]]: 利用PPI网络
- [[Cell Oracle]]: 结合GRN

## 嵌入评估

### 质量指标

| 指标 | 描述 | 目标 |
|------|------|------|
| **重构误差** | 解码后与原数据差异 | 最小化 |
| **聚类纯度** | 细胞类型分离度 | 最大化 |
| **轨迹保持** | 伪时间相关性 | 最大化 |
| **批次校正** | 去除批次效应 | 最小化批次间差异 |
| **生物保守** | 保留生物学信号 | 最大化 |

### 下游任务性能

```
嵌入质量最终由下游任务验证:

1. 细胞类型分类准确率
2. 扰动预测R²
3. 批次整合效果
4. 轨迹推断准确性
```
