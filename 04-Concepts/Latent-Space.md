# Latent Space (潜在空间)

> [[概念地图]] | [[Embedding]] | [[VAE-in-Single-Cell]]

## 数学定义

**潜在空间 (Latent Space)** 是生成模型中学习到的低维表示空间，其中数据的复杂结构被编码为紧凑、连续的向量。它是深度生成模型（如VAE、GAN、扩散模型）的核心组件。

### 基本定义

```
设:
- X: 观测数据空间（如基因表达空间 R^G）
- Z: 潜在空间（低维空间 R^d，d << G）

编码器 (Encoder): q(z|x) 将数据映射到潜在空间
解码器 (Decoder): p(x|z) 从潜在空间重构数据

目标: 学习有效的潜在表示 z，使得:
1. z 捕捉数据的主要变异
2. 从 z 可以高质量重构 x
3. z 具有有意义的结构
```

### VAE中的潜在空间

```
变分自编码器的潜在空间:

编码: q_φ(z|x) = N(z; μ_φ(x), σ²_φ(x))
      输入x → 神经网络 → 均值μ + 方差σ²

采样: z ~ q_φ(z|x)
      从分布中采样潜在向量

解码: p_θ(x|z) = 适当分布（如ZINB对于scRNA-seq）
      潜在向量z → 神经网络 → 重构x

训练目标 (ELBO):
L = E_q[log p_θ(x|z)] - KL(q_φ(z|x) || p(z))
    ↑重构损失              ↑正则化项
```

## 潜在空间的性质

### 1. 连续性 (Continuity)

**定义**: 潜在空间中相邻的点对应相似的数据样本。

```
数学表达:
如果 ||z₁ - z₂|| < ε，则 x₁ = decode(z₁) 和 x₂ = decode(z₂) 应该相似

示例:
在细胞潜在空间中:
z_A ──z_B── z_C
 相似   相似
  ↓      ↓
细胞A  细胞B  细胞C
(T细胞) (过渡) (B细胞)

连续插值产生平滑过渡的细胞状态
```

**意义**:
- 保证生成样本的质量
- 支持平滑的插值和过渡
- 反映生物过程的连续性（如分化）

### 2. 插值 (Interpolation)

**定义**: 可以在两个潜在点之间进行有意义的插值。

```
线性插值:
z_α = (1-α) × z₁ + α × z₂,  α ∈ [0,1]

当α从0到1变化:
- α=0: 完全细胞1
- α=0.5: 中间状态
- α=1: 完全细胞2

应用示例:
干细胞 z_stem ─────────────────→ 成熟细胞 z_mature
      α=0        α=0.25  α=0.5  α=0.75   α=1
      早期分化 → 中期分化 → 晚期分化 → 成熟
```

**球面插值 (Slerp)**:
```
对于单位向量，使用球面插值保持模长:
z_α = sin((1-α)θ)/sin(θ) × z₁ + sin(αθ)/sin(θ) × z₂
其中 θ = arccos(z₁·z₂ / (||z₁|| ||z₂||))
```

### 3. 算术运算 (Arithmetic Operations)

**定义**: 潜在空间中的向量运算对应有意义的语义操作。

```
经典示例 (在图像潜在空间中):
"戴眼镜的男人" - "男人" + "女人" = "戴眼镜的女人"

在单细胞潜在空间中的类似操作:
"激活T细胞" - "静息T细胞" + "静息B细胞" = "激活B细胞"

数学表达:
z_result = z₁ - z₂ + z₃
          ↓
    decode(z_result) = 具有目标特性的细胞
```

**扰动算术**:
```
扰动向量: v_perturb = z_perturbed - z_control

应用:
- 预测: z_predicted = z_new_cell + v_perturb
- 组合: v_combo = v_A + v_B (假设线性)
- 强度: v_strong = 2 × v_weak
```

## 潜在空间的结构

### 流形假设

**假设**: 高维数据实际上位于一个低维流形上。

```
数据空间 R^G (G=20000)
      │
      │  数据点集中在
      │  低维流形 M 上
      │  dim(M) = d << G
      ↓
潜在空间 R^d

流形学习:
- 发现数据的内在结构
- 去除冗余维度
- 揭示变化的主要因素
```

### 潜在空间的组织

```
良好学习的潜在空间通常显示:

1. 聚类结构
   ├── T细胞簇
   ├── B细胞簇
   └── 单核细胞簇
   
2. 连续轨迹
   干细胞 → 祖细胞 → 成熟细胞
   
3. 层次结构
   免疫细胞
   ├── 淋巴细胞
   │   ├── T细胞
   │   └── B细胞
   └── 髓系细胞
       ├── 单核细胞
       └── 粒细胞
```

## 生物学解释

### 潜在维度 = 生物学因素

```
理想情况: 每个潜在维度对应一个生物学因素

维度1: 细胞类型轴
       T细胞 ←────────────────→ B细胞
       
维度2: 细胞周期轴
       G1期 ←────────────────→ M期
       
维度3: 分化阶段轴
       干细胞 ←────────────────→ 成熟细胞
       
维度4: 应激水平轴
       正常 ←────────────────→ 应激
```

### 解耦表示

**目标**: 使不同维度对应不同、独立的生物学因素。

```
解耦评估:
1. 相关性分析: 潜在维度与已知生物学标签的相关
2. 干预实验: 改变一个维度，观察是否只影响对应因素
3. 可视化: 各维度与生物学注释的对应关系

方法:
- β-VAE: 增加KL权重促进解耦
- FactorVAE: 全相关惩罚
- β-TCVAE: 总相关性分解
```

### 扰动在潜在空间中的表示

```
扰动效应的潜在空间视角:

控制细胞: z_control
扰动细胞: z_perturbed

扰动向量: Δz = z_perturbed - z_control

性质:
- 相同扰动在不同细胞产生相似的Δz
- 相似扰动产生相似的Δz
- 组合扰动可能近似于单个扰动的和
```

## 潜在空间操作

### 1. 探索与可视化

```
降维可视化:
潜在空间 (50维) → PCA/UMAP/t-SNE → 2D/3D可视化

应用:
- 识别细胞类型簇
- 发现分化轨迹
- 检测异常细胞
```

### 2. 条件生成

```
在特定位置采样生成细胞:

z ~ N(μ_target, σ²)
↓
x = decode(z)
↓
生成具有目标特性的细胞

应用:
- 生成特定类型的细胞
- 数据增强
- 扰动效应模拟
```

### 3. 异常检测

```
基于潜在空间的异常检测:

正常细胞: 潜在空间中高密度区域
异常细胞: 潜在空间中低密度区域

方法:
- 计算潜在向量的密度
- 远离训练数据分布的视为异常
- 可用于质量控制
```
