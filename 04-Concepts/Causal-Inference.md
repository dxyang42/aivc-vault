# Causal Inference

> [[概念地图]] | [[统计学]] | [[因果性]]

## 概述

**因果推断 (Causal Inference)** 是从数据中识别因果关系的方法论。在单细胞扰动分析中，因果推断帮助区分真正的因果效应和相关性。

## 核心问题

### 因果效应 vs 相关性

```
相关性: X和Y同时变化
因果性: X的变化导致Y的变化

关键区别:
  相关性 ≠ 因果性
  因果性 ⇒ 相关性 (通常)
```

### 因果推断的挑战

```
1. 混杂因素 (Confounding)
   - Z影响X和Y
   - 导致虚假的相关性

2. 选择偏差 (Selection Bias)
   - 样本选择不随机
   - 导致估计偏差

3. 反事实 (Counterfactual)
   - 无法同时观察到干预和不干预
   - 需要估计"如果...会怎样"
```

## 主要方法

### 1. 潜在结果框架 (Potential Outcomes)

```
定义:
  Y(1): 接受干预的结果
  Y(0): 不接受干预的结果
  
个体因果效应: τ_i = Y_i(1) - Y_i(0)
平均因果效应: ATE = E[Y(1) - Y(0)]

问题: 对每个个体，只能观察到Y(1)或Y(0)之一
```

### 2. 结构因果模型 (SCM)

```
有向无环图 (DAG):
  X → Y 表示 X 导致 Y
  
例子:
  基因敲除 → 基因表达变化 → 表型变化
```

### 3. Do-Calculus

```
干预分布: P(Y | do(X=x))
观察分布: P(Y | X=x)

do-calculus规则:
  允许从观察数据推导干预效果
```

## 在单细胞扰动分析中的应用

### 1. 扰动效应估计

```
问题: 估计基因敲除对基因表达的真实因果效应

方法:
  - 使用对照细胞和扰动细胞的对比
  - 控制混杂因素 (批次、细胞周期等)
  - 估计平均因果效应
```

### 2. 基因调控网络推断

```
问题: 从扰动数据中推断因果关系

方法:
  - 干预数据提供因果信息
  - 结合观察数据增强
  - 构建因果图
```

### 3. 机制解耦

```
问题: 区分直接效应和间接效应

方法:
  - 中介分析 (Mediation Analysis)
  - 估计直接效应和中介效应
```

## 评估因果效应

### 随机化实验
```
金标准: 随机分配干预
单细胞中的实现: Perturb-seq
```

### 观察性研究
```
方法:
  - 倾向得分匹配
  - 逆概率加权
  - 双重稳健估计
```

## 优势

1. **因果解释**: 提供机制性理解
2. **干预预测**: 预测未观察干预的效果
3. **鲁棒性**: 对分布变化更鲁棒

## 局限

1. **假设要求**: 需要无未观察混杂等假设
2. **数据需求**: 需要干预数据
3. **计算复杂**: 因果发现是NP难问题
