# Combinatorial Perturbation (组合扰动)

> [[概念地图]] | [[Perturbation-Prediction]] | [[Gene-Regulatory-Network]] | [[术语统一规范]]

## 定义

**组合扰动 (Combinatorial Perturbation)** 是指同时对多个基因或靶点进行扰动（敲除、敲低或过表达），以研究基因间的相互作用和协同效应。与单基因扰动相比，组合扰动能揭示更复杂的生物学机制和潜在的治疗策略。

### 核心概念

```
单基因扰动 vs 组合扰动:

单基因:
基因A KO → 观测效应
基因B KO → 观测效应

组合扰动:
基因A KO + 基因B KO → 观测联合效应
                    ↓
        可能 ≠ 单基因效应的简单加和
                    ↓
        揭示基因间相互作用
```

### 组合扰动的生物学意义

| 价值 | 描述 |
|------|------|
| **揭示相互作用** | 发现基因间的调控关系 |
| **识别冗余性** | 找到功能备份机制 |
| **发现合成致死** | 识别联合治疗靶点 |
| **理解通路结构** | 解析信号通路的逻辑 |
| **预测联合用药** | 指导药物组合设计 |

## 组合扰动的挑战

### 1. 组合爆炸

```
组合数量随基因数指数增长:

基因数n=20,000

单基因扰动: C(n,1) = 20,000
双基因组合: C(n,2) ≈ 2亿
三基因组合: C(n,3) ≈ 1.3万亿

实验不可行性:
- 即使每天测试1万个组合
- 测试所有双基因组合需要50年
- 测试所有三基因组合需要数百万年

解决方案:
- 智能采样策略
- 计算预测优先排序
- 自适应实验设计
```

### 2. 非线性效应

```
组合效应 ≠ 单基因效应之和

数学表示:
设 f(A) = 基因A KO的效应
    f(B) = 基因B KO的效应
    f(A,B) = 联合KO的效应

简单加和假设: f(A,B) = f(A) + f(B)

实际可能:
- 协同效应: f(A,B) > f(A) + f(B)
- 拮抗效应: f(A,B) < f(A) + f(B)
- 掩蔽效应: f(A,B) ≈ f(A) 或 f(B)
- 新表型: f(A,B) 呈现全新特征

非线性来源:
- 基因调控网络的级联效应
- 反馈回路的补偿机制
- 细胞状态的系统性变化
```

### 3. 协同与拮抗作用

#### 协同作用 (Synergy)

```
定义: 联合效应大于单效应之和

类型:
1. 增强型协同:
   f(A,B) = f(A) + f(B) + α (α > 0)

2. 超协同:
   f(A,B) >> f(A) + f(B)
   (效应放大数倍)

3. 合成致死:
   f(A) ≈ 0, f(B) ≈ 0
   f(A,B) = 致死

生物学示例:
- 平行通路的双重阻断
- 冗余基因的同时失活
- 正反馈回路的激活
```

#### 拮抗作用 (Antagonism)

```
定义: 联合效应小于单效应之和

类型:
1. 部分拮抗:
   f(A,B) = f(A) + f(B) - α (α > 0)

2. 完全拮抗:
   f(A,B) ≈ 0 (互相抵消)

3. 掩蔽:
   f(A,B) ≈ f(A) (B的效应被掩盖)

生物学示例:
- 负反馈回路的补偿
- 功能冗余基因的备份
- 拮抗信号通路的平衡
```

## 组合效应模型

### 1. 加法模型 (Additive Model)

```
最简单假设: 效应可线性叠加

模型:
f(A,B) = β₀ + β₁f(A) + β₂f(B)

其中:
- β₀: 截距
- β₁, β₂: 单基因效应权重

评估交互:
残差 = f(A,B)_observed - f(A,B)_predicted

大残差 → 存在非线性交互

适用场景:
- 初步筛选
- 效应近似加性的基因对
- 计算资源有限时

局限:
- 无法捕捉协同/拮抗
- 忽略基因间相互作用
```

### 2. 非线性模型

#### 基于神经网络的模型

```
架构:
输入: [f(A), f(B), 特征A, 特征B]
      ↓
   MLP / Transformer
      ↓
输出: f(A,B)预测

优势:
- 自动学习非线性交互
- 可整合多种特征
- 表达能力强

代表方法:
- [[GEARS]]: 图神经网络预测组合效应
- [[CPA]]: 条件VAE建模多因子扰动
```

#### 基于统计的交互模型

```
模型:
f(A,B) = β₀ + β₁f(A) + β₂f(B) + β₁₂f(A)f(B)

交互项 β₁₂:
- β₁₂ > 0: 正向交互（协同）
- β₁₂ < 0: 负向交互（拮抗）
- β₁₂ ≈ 0: 无交互（加性）

扩展:
- 高阶交互项
- 非线性基函数
- 正则化防止过拟合
```

### 3. 基于图的方法

```
利用基因调控网络预测组合效应:

输入:
- 基因A和B的PPI网络邻域
- 网络距离和拓扑特征
- 单基因扰动数据

图神经网络处理:
1. 基因节点嵌入
2. 消息传递聚合邻居信息
3. 组合节点表示学习
4. 效应预测

代表方法: [[GEARS]]

优势:
- 利用已知的生物学知识
- 可解释性强
- 数据效率高
```

## 实验设计

### 1. 因子设计 (Factorial Design)

```
全因子设计:

对于k个基因，每个基因2个水平(KO/WT):
组合数 = 2^k

示例 (k=3):
基因A, B, C

组合:
000: WT, WT, WT (对照)
001: WT, WT, KO_C
010: WT, KO_B, WT
011: WT, KO_B, KO_C
100: KO_A, WT, WT
101: KO_A, WT, KO_C
110: KO_A, KO_B, WT
111: KO_A, KO_B, KO_C

优点:
- 可估计所有交互效应
- 统计效率高

缺点:
- 组合数指数增长
- 仅适用于少量基因
```

### 2. 自适应设计 (Adaptive Design)

```
主动学习策略:

迭代流程:
1. 初始: 测试少量组合
2. 建模: 训练预测模型
3. 选择: 选择信息量最大的下一批组合
4. 实验: 测试选中的组合
5. 更新: 重新训练模型
6. 重复直到收敛

选择策略:
- 不确定性采样: 选择模型最不确定的
- 期望改进: 选择可能改进最大的
- 多样性采样: 覆盖不同区域

优势:
- 减少实验次数
- 聚焦信息丰富的组合
- 动态调整策略
```

### 3. 分层设计

```
分阶段实验策略:

阶段1: 单基因筛选
- 测试所有单基因KO
- 识别有显著效应的基因

阶段2: 配对筛选
- 仅测试阶段1阳性基因的配对
- 组合数: C(k,2)，k为阳性基因数

阶段3: 高阶组合
- 测试阶段2协同配对的组合
- 添加第三、第四个基因

优势:
- 大幅减少实验量
- 聚焦有希望的候选
- 资源分配优化
```

## 交互效应量化

### 交互分数计算

```
基于Bliss独立模型的交互分数:

S = f(A,B) - [f(A) + f(B) - f(A)×f(B)]

解释:
- S > 0: 协同
- S < 0: 拮抗
- S ≈ 0: 加性

基于Loewe加和模型:

计算等效剂量组合
判断实际效应是否偏离预期

基于ZIP模型:

考虑剂量-效应关系
量化协同强度
```

### 可视化

```
交互热图:

       基因B1  基因B2  基因B3  ...
基因A1   +0.2   -0.5    +0.8
基因A2   -0.3   +0.1    -0.2
基因A3   +0.6   +0.4    -0.1
...

颜色:
- 红色: 协同 (正交互)
- 蓝色: 拮抗 (负交互)
- 白色: 加性 (无交互)

网络图:
节点: 基因
边: 交互强度
边颜色: 协同(红) / 拮抗(蓝)
```

## 相关方法

| 方法 | 类型 | 特点 |
|------|------|------|
| **Norman 2021** | 实验 | 大规模组合扰动筛选 |
| **CPA** | 深度生成 | 条件VAE建模多因子 |
| **GEARS** | 图网络 | 利用PPI网络预测组合 |
| **MIX-CRISPR** | 实验 | 混合筛选策略 |
| **CombPert** | 预测 | 组合效应预测框架 |

### Norman 2021

**方法类型**: 大规模组合扰动实验

```
研究设计:
- 系统测试数千个基因对
- 单细胞测序读出
- 识别协同和拮抗效应

贡献:
- 提供了大规模组合数据集
- 揭示了基因相互作用模式
- 验证了计算预测方法
```

### CPA (Conditional Perturbation Autoencoder)

**方法类型**: 条件深度生成模型

```
模型:
编码器: q(z|x, c₁, c₂, ...)
解码器: p(x|z, c₁, c₂, ...)

条件c: 扰动标识
- 单基因: c = [1, 0, 0, ...]
- 组合: c = [1, 1, 0, ...]

优势:
- 显式建模多因子效应
- 可分离各因子贡献
- 生成组合扰动后的细胞
```

### GEARS

**方法类型**: 图神经网络

```
核心思想:
1. 基因嵌入:
   - 基于PPI网络学习基因表示
   - 功能相关基因嵌入接近

2. 组合预测:
   - 输入: 两个基因的嵌入
   - GNN消息传递
   - 输出: 组合效应预测

3. 非线性建模:
   - 自动学习协同/拮抗模式
   - 考虑网络拓扑

优势:
- 利用先验知识
- 零样本预测新组合
- 可解释性强
```

## 应用案例

### 合成致死靶点发现

```
目标: 找到癌细胞特异性的合成致死基因对

流程:
1. 选择候选基因集
   - 癌基因
   - DNA修复基因
   - 代谢基因

2. 计算预测所有配对效应
   - 正常细胞模型
   - 癌细胞模型

3. 筛选合成致死对
   - 癌细胞: 联合KO致死
   - 正常细胞: 联合KO可存活

4. 实验验证Top候选

价值:
- 发现新的治疗靶点组合
- 减少脱靶毒性
- 克服单靶点耐药
```

### 联合用药设计

```
目标: 预测药物组合效应

流程:
1. 药物靶点映射
   - 化合物 → 靶点基因

2. 零样本预测组合KO效应
   - 使用组合扰动模型

3. 效应评估
   - 协同分数
   - 毒性预测
   - 耐药风险

4. 优先级排序
   - 高协同 + 低毒性

价值:
- 虚拟筛选化合物组合
- 指导临床试验设计
- 个性化用药方案
```
