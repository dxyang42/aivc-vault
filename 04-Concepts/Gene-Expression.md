# Gene Expression (基因表达)

> [[概念地图]] | [[scRNA-seq]] | [[术语统一规范]]

## 定义

**基因表达 (Gene Expression)** 是指基因中的遗传信息被转录成RNA分子，进而翻译成蛋白质的过程。它是细胞功能执行的基础，决定了细胞的类型、状态和行为。

### 核心概念

基因表达水平量化了特定基因在特定时间和特定细胞中的活跃程度。高表达意味着该基因产物（RNA或蛋白质）在细胞中丰富存在，通常与特定生物学功能相关。

---

## 中心法则回顾

### 生物学信息流

```
DNA (基因组)
    ↓ 转录 (Transcription)
mRNA (信使RNA)
    ↓ 翻译 (Translation)
蛋白质 (Protein)
    ↓ 功能执行
细胞表型
```

### 各阶段详解

| 阶段 | 过程 | 产物 | 调控层次 |
|------|------|------|----------|
| **转录** | DNA → RNA | pre-mRNA | 转录因子、染色质状态 |
| **剪接** | pre-mRNA → mRNA | 成熟mRNA | 可变剪接调控 |
| **转运** | 核内 → 细胞质 | 定位mRNA | 核输出信号 |
| **翻译** | mRNA → 蛋白质 | 多肽链 | miRNA、翻译效率 |
| **修饰** | 蛋白质加工 | 功能蛋白 | 磷酸化、糖基化等 |

### 中心法则的扩展

现代分子生物学发现中心法则存在多种例外和扩展：

- **逆转录**: RNA → DNA（逆转录病毒）
- **RNA编辑**: RNA序列在转录后被修改
- **非编码RNA**: 不翻译为蛋白质的功能性RNA（miRNA、lncRNA等）
- **表观遗传**: DNA甲基化、组蛋白修饰等不改变序列的调控

## 基因表达测量方法

### 1. 批量RNA测序 (Bulk RNA-seq)

| 特征 | 描述 |
|------|------|
| **样本** | 数千至数百万细胞的混合物 |
| **输出** | 细胞群体的平均表达谱 |
| **分辨率** | 无法区分细胞异质性 |
| **应用** | 组织水平表达分析、大样本比较 |

**局限性**: 掩盖了细胞间的个体差异，无法研究稀有细胞类型。

### 2. 单细胞RNA测序 (scRNA-seq)

| 特征 | 描述 |
|------|------|
| **样本** | 单个细胞 |
| **输出** | 每个细胞的独立表达谱 |
| **分辨率** | 细胞级别，可识别细胞亚型 |
| **应用** | 细胞异质性、发育轨迹、扰动响应 |

**主要平台**: 详见 [[scRNA-seq]] 文档

### 3. 空间转录组学 (Spatial Transcriptomics)

| 特征 | 描述 |
|------|------|
| **样本** | 组织切片，保留空间位置 |
| **输出** | 基因表达 + 空间坐标 |
| **分辨率** | 从单细胞到多细胞点 |
| **应用** | 组织结构、细胞微环境、肿瘤异质性 |

### 4. 其他测量技术

| 技术 | 测量对象 | 特点 |
|------|----------|------|
| **ATAC-seq** | 染色质可及性 | 表观遗传状态 |
| **ChIP-seq** | 蛋白质-DNA结合 | 转录因子结合位点 |
| **蛋白质组学** | 蛋白质丰度 | 翻译后修饰 |
| **代谢组学** | 代谢物 | 代谢状态 |

## 表达矩阵结构

### 基本定义

scRNA-seq 数据通常表示为一个**基因-细胞表达矩阵**：

```
           细胞1    细胞2    细胞3    ...    细胞N
基因1      0        5        2        ...    1
基因2      10       0        8        ...    12
基因3      3        7        0        ...    4
...        ...      ...      ...      ...    ...
基因G      1        0        6        ...    0
```

### 矩阵维度

| 维度 | 典型规模 | 说明 |
|------|----------|------|
| **基因数 (G)** | 20,000-30,000 | 人类蛋白质编码基因 |
| **细胞数 (N)** | 1,000-1,000,000+ | 取决于实验规模 |
| **数据点** | G × N | 稀疏矩阵，大部分为零 |

### 数据特征

```
表达矩阵 X ∈ N^(G×N)
- X_ij: 基因i在细胞j中的表达计数
- 非负整数（原始UMI计数）
- 极度稀疏（通常 > 90% 为零）
```

### 元数据

除表达矩阵外，每个细胞通常还关联以下元数据：

| 元数据类型 | 示例 |
|------------|------|
| **细胞标识** | 细胞条形码、样本ID |
| **实验条件** | 扰动类型、药物浓度、时间点 |
| **细胞注释** | 细胞类型、细胞周期阶段 |
| **质量指标** | 检测基因数、总UMI数、线粒体比例 |

## 标准化方法

### 为什么需要标准化

1. **测序深度差异**: 不同细胞的总RNA量不同
2. **批次效应**: 不同实验批次的技术差异
3. **细胞大小差异**: 大细胞通常有更多RNA

### 常用标准化方法

| 方法 | 描述 | 适用场景 |
|------|------|----------|
| **CPM** | Counts Per Million | 简单的深度校正 |
| **TPM** | Transcripts Per Million | 考虑基因长度 |
| **FPKM** | Fragments Per Kilobase Million | 批量RNA-seq |
| **Log1p** | log(1 + x) | 压缩高表达值 |
| **scTransform** | 方差稳定变换 | 单细胞专用 |
| **SCRAN** | 大小因子估计 | 考虑细胞类型差异 |

### 标准化流程示例

```python
# 典型scRNA-seq标准化流程
1. 质控过滤 (QC)
   - 过滤低质量细胞
   - 去除双细胞

2. 原始计数 → CPM
   - 每个细胞总计数归一化到1e6

3. Log1p变换
   - 降低高表达基因的影响
   - 使数据更接近正态分布

4. 特征选择
   - 选择高变异基因 (HVG)
   - 通常选择 2,000-5,000 个基因

5. 缩放
   - 均值为0，方差为1
```

## 生物学变异 vs 技术噪声

### 变异来源分解

```
观测到的表达变异 = 生物学变异 + 技术噪声 + 批次效应
```

### 生物学变异

| 类型 | 描述 | 示例 |
|------|------|------|
| **细胞类型差异** | 不同细胞类型的表达谱 | 肝细胞 vs 神经元 |
| **细胞状态差异** | 同一类型的不同状态 | G1期 vs S期细胞 |
| **个体差异** | 不同供体的遗传背景 | 个体A vs 个体B |
| **环境响应** | 对外部刺激的响应 | 药物处理 vs 对照 |

### 技术噪声

| 类型 | 描述 | 来源 |
|------|------|------|
| **Dropout** | 真实表达但未被检测 | 捕获效率低、mRNA降解 |
| **扩增偏差** | PCR扩增引入的偏差 | 不同转录本扩增效率不同 |
| **测序错误** | 碱基识别错误 | 测序仪误差 |
| **批次效应** | 不同实验批次间的差异 | 操作者、试剂、时间 |

### Dropout 现象

**定义**: 由于捕获效率限制，中低表达的基因在部分细胞中检测不到表达。

```
真实表达: 基因A在细胞X中表达水平为5
检测概率: 取决于捕获效率（通常5-10%）
观测结果: 50-90%概率检测为0（dropout）
```

**数学模型**: 零膨胀负二项分布 (ZINB)

```
P(X = k) = π × δ(k) + (1-π) × NB(k; μ, θ)

其中:
- π: dropout概率
- δ(k): 狄拉克函数（k=0时为1，否则为0）
- NB: 负二项分布
- μ: 均值
- θ: 离散参数
```

## 基因表达与扰动预测

### 扰动响应的表达变化

当细胞受到扰动时，基因表达会发生系统性变化：

```
扰动前: x = [x_1, x_2, ..., x_G]  (G个基因的表达)
扰动后: x' = [x'_1, x'_2, ..., x'_G]

差异表达基因 (DEG):
DEG = {i : |x'_i - x_i| > threshold}
```

### 扰动预测的表达建模

| 建模层次 | 描述 | 方法示例 |
|----------|------|----------|
| **基因级** | 预测每个基因的表达变化 | [[GEARS]]、[[scGen]] |
| **通路级** | 预测通路活性变化 | [[DrugCell]] |
| **细胞级** | 预测整体细胞状态变化 | [[CellFlow]] |
| **群体级** | 预测细胞群体分布变化 | [[scVI]] |
