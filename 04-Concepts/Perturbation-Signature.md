# Perturbation Signature (扰动签名)

> [[概念地图]] | [[Perturbation-Prediction]] | [[Gene-Expression]]

## 定义

**扰动签名 (Perturbation Signature)** 是指特定扰动引起的基因表达变化的特征模式。它描述了细胞响应特定干预时，哪些基因上调、哪些基因下调，以及变化的幅度。扰动签名是扰动预测的核心目标——预测未见扰动的签名。

### 核心概念

```
扰动签名 = 扰动后表达谱 - 扰动前表达谱
         = Δ表达 = 差异表达基因集合

数学表示:
- 控制组: x_control ∈ R^G
- 扰动组: x_perturb ∈ R^G
- 签名: s = x_perturb - x_control ∈ R^G

其中 G 是基因数，s_i > 0 表示基因i上调，s_i < 0 表示下调
```

### 签名的层次

| 层次 | 描述 | 示例 |
|------|------|------|
| **基因级** | 单个基因的表达变化 | TP53上调2倍 |
| **通路级** | 功能通路的整体变化 | 凋亡通路激活 |
| **表型级** | 细胞水平的功能变化 | 增殖减缓 |

## 差异表达分析

### 计算流程

```
1. 数据准备
   ├── 控制组细胞: {x₁, x₂, ..., x_n}
   └── 扰动组细胞: {y₁, y₂, ..., y_m}

2. 统计检验 (对每个基因)
   ├── 零假设 H₀: μ_control = μ_perturb
   └── 备择假设 H₁: μ_control ≠ μ_perturb

3. 多重检验校正
   ├── 原始p值 → 校正后p值 (FDR/Benjamini-Hochberg)
   └── 控制假阳性率

4. 差异表达基因筛选
   ├── |log2FC| > 阈值 (如0.5或1)
   ├── FDR < 阈值 (如0.05)
   └── 同时满足以上条件
```

### 常用方法

| 方法 | 适用场景 | 特点 |
|------|----------|------|
| **t-test/Wilcoxon** | 两组比较 | 简单，假设正态/非参 |
| **edgeR** | 计数数据 | 针对RNA-seq优化 |
| **DESeq2** | 计数数据 | 负二项模型，方差稳定 |
| **MAST** | scRNA-seq | 考虑dropout |
| **Wilcoxon rank-sum** | scRNA-seq | 非参，稳健 |

### 差异表达统计量

| 统计量 | 定义 | 解释 |
|--------|------|------|
| **log2FC** | log2(扰动组均值/对照组均值) | 变化倍数 |
| **p-value** | 统计显著性 | 变化是否随机 |
| **FDR** | 校正后的p值 | 控制假阳性 |
| **AUC** | ROC曲线下面积 | 效应大小 |

### 示例

```
基因表达变化分析结果:

基因      log2FC    p-value    FDR      结论
------------------------------------------------
BAX       2.3       1e-10      1e-8     显著上调
(促凋亡)

BCL2      -1.8      1e-8       1e-6     显著下调
(抗凋亡)

GAPDH     0.1       0.5        0.7      无变化
(内参)

MYC       0.05      0.8        0.9      无变化
```

## 签名特性

### 1. 特异性 (Specificity)

**定义**: 不同扰动产生不同的、可区分的签名。

```
理想情况:
扰动A → 签名A
扰动B → 签名B
签名A ≠ 签名B (当A ≠ B)

评估方法:
- 签名间的相似性矩阵
- 分类准确率: 能否从签名识别扰动
- 聚类纯度: 相同扰动的签名是否聚类
```

**特异性挑战**:
- 功能相关基因可能产生相似签名
- 补偿机制可能掩盖特异性
- 细胞状态影响签名表现

### 2. 鲁棒性 (Robustness)

**定义**: 签名在不同条件下保持一致性。

```
鲁棒性维度:

1. 细胞类型鲁棒性
   同一扰动在不同细胞类型产生相似签名
   
2. 批次鲁棒性
   不同实验批次产生一致签名
   
3. 时间鲁棒性
   扰动后不同时间点的签名稳定
   
4. 剂量鲁棒性
   不同扰动强度产生比例一致的签名
```

**评估指标**:
- 跨条件相关性
- 重测信度
- 预测一致性

### 3. 可组合性 (Composability)

**定义**: 组合扰动的签名可以从单个扰动签名预测。

```
线性假设:
s(A+B) ≈ s(A) + s(B)

其中:
s(A): 扰动A的签名
s(B): 扰动B的签名
s(A+B): 组合扰动的签名

实际中常存在非线性相互作用:
s(A+B) = s(A) + s(B) + interaction(A,B)
```

**相互作用类型**:

| 类型 | 定义 | 示例 |
|------|------|------|
| **协同 (Synergy)** | 效应 > 加和 | 1+1 > 2 |
| **拮抗 (Antagonism)** | 效应 < 加和 | 1+1 < 2 |
| **掩蔽 (Masking)** | 一个扰动主导 | 1+1 ≈ 1 |
| **新颖 (Novel)** | 产生全新效应 | 出现非预期变化 |

## 签名应用

### 1. 扰动预测验证

```
预测签名 vs 真实签名

评估指标:
- 基因级: MSE, Pearson r
- DE基因级: Precision@K, Recall@K
- 通路级: 通路富集一致性
```

### 2. 机制推断

```
从签名推断扰动机制:

上调基因 → 富集分析 → 激活通路
                              ↓
下调基因 → 富集分析 → 抑制通路
                              ↓
                         推断作用机制

示例:
TP53 KO签名:
- 上调: 细胞周期基因、DNA复制基因
- 下调: 凋亡基因、DNA修复基因
- 推断: TP53缺失导致细胞周期失控
```

### 3. 药物靶点识别

```
药物签名匹配:

药物A签名 ≈ 基因B KO签名
→ 推断: 药物A可能通过抑制基因B发挥作用

应用:
- 药物重定位
- 脱靶效应预测
- 联合用药设计
```

### 4. 疾病标志物

```
疾病相关签名:
- 比较疾病细胞 vs 正常细胞
- 识别疾病特异性表达变化
- 作为诊断标志物或治疗靶点
```

## 签名数据库

### 主要资源

| 数据库 | 内容 | 规模 |
|--------|------|------|
| **CMap (Connectivity Map)** | 药物签名 | 数千种化合物 |
| **LINCS L1000** | 基因表达签名 | 百万级签名 |
| **scPerturb** | 单细胞扰动数据库 | 400万+细胞 |
| **GEO** | 公共扰动数据集 | 大量研究 |

### 签名格式

```
标准签名格式:
{
  "perturbation_id": "TP53_KO",
  "cell_type": "K562",
  "condition": "knockout",
  "up_genes": ["geneA", "geneB", ...],
  "down_genes": ["geneC", "geneD", ...],
  "log2fc": {"geneA": 2.3, "geneB": 1.8, ...},
  "pvalues": {"geneA": 1e-10, ...},
  "fdr": {"geneA": 1e-8, ...}
}
```

## 签名预测方法

### 1. 直接预测

```
输入: 细胞状态 + 扰动信息
输出: 预测签名 (每个基因的变化)

方法:
- 神经网络回归
- 图神经网络 (利用基因关系)
- 条件生成模型
```

### 2. 隐式预测

```
不直接预测签名，而是预测扰动后状态:

输入: 控制细胞状态 z_control
输出: 扰动后状态 z_perturbed

签名计算: s = decode(z_perturbed) - decode(z_control)

方法:
- 潜在空间插值
- 条件VAE
- 流匹配
```
