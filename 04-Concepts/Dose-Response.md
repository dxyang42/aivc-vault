# Dose-Response (剂量反应)

> [[概念地图]] | [[药物筛选]] | [[Perturbation-Prediction]] | [[术语统一规范]]

## 定义

**剂量反应关系 (Dose-Response Relationship)** 描述生物系统对扰动（药物、化合物或基因表达变化）的强度响应随剂量变化的规律。它是药理学、毒理学和细胞生物学中的核心概念，用于量化扰动的效力和安全性。

### 核心概念

```
剂量反应的基本框架:

剂量 (Dose)
    ↓
[药代动力学] → 体内浓度
    ↓
[受体结合] → 占据率
    ↓
[信号转导] → 细胞响应
    ↓
[表型效应] → 观测响应

剂量-反应曲线:
剂量 (x轴, 对数尺度) → 响应 (y轴, 0-100%)
    ↓
S型曲线 (Sigmoid)
```

### 为什么剂量反应重要

| 价值 | 描述 |
|------|------|
| **效力评估** | 比较不同化合物的活性强度 |
| **安全性评估** | 确定治疗窗口和毒性阈值 |
| **机制理解** | 揭示分子相互作用特性 |
| **剂量优化** | 指导临床剂量选择 |
| **质量控制** | 验证实验和模型的可靠性 |

## 剂量反应模型

### 1. Hill方程 (Hill Equation)

**定义**: 描述配体-受体结合和协同效应的经典模型。

```
数学形式:

E = Emin + (Emax - Emin) × [D]^n / (EC50^n + [D]^n)

其中:
- E: 观测响应
- [D]: 药物浓度（剂量）
- Emin: 最小响应（基线）
- Emax: 最大响应（平台期）
- EC50: 半最大效应浓度
- n: Hill系数（斜率因子）

简化形式（Emin=0）:
E = Emax × [D]^n / (EC50^n + [D]^n)
```

**参数解释**:

| 参数 | 含义 | 生物学意义 |
|------|------|------------|
| **EC50** | 产生50%最大效应的浓度 | 效力指标，越小越强 |
| **Emax** | 最大可达到的效应 | 效能指标 |
| **n (Hill系数)** | 曲线的陡峭程度 | 协同性指标 |
| **Emin** | 无药物时的基线响应 | 背景活性 |

**Hill系数的解释**:

```
n = 1: 简单Michaelis-Menten动力学
       无协同性，单一位点结合

n > 1: 正协同性
       结合一个配体促进后续结合
       曲线更陡峭
       示例: 血红蛋白与氧气

n < 1: 负协同性
       结合一个配体抑制后续结合
       曲线更平缓
       示例: 某些异构酶
```

### 2. Emax模型

**定义**: Hill方程的简化形式，假设无协同性。

```
数学形式:

E = Emax × [D] / (EC50 + [D])

这是Hill方程在n=1时的特例

特点:
- 最简单的饱和动力学模型
- 仅两个参数: Emax, EC50
- 适用于无协同性的系统
```

### 3. S型曲线 (Sigmoid Curve)

**定义**: 剂量反应关系的通用形态描述。

```
曲线特征:

响应(%)
100% |                    ________  平台期 (Emax)
     |                 /
 50% |--------------●--------------  EC50点
     |           /
     |        /
  0% |______/                        基线 (Emin)
     +-------------------------------
     低剂量    EC50    高剂量

四个特征阶段:
1. 基线期: 低剂量无效应
2. 上升期: 剂量增加，响应快速上升
3. 转折期: EC50附近，响应变化最快
4. 平台期: 高剂量达到最大效应
```

**对数剂量转换**:

```
剂量通常用对数尺度表示:

原因:
1. 剂量范围可能跨越多个数量级
2. 对数转换后S型曲线对称
3. 便于线性回归分析

log[D] vs E 的关系:
在EC50附近近似线性
便于参数估计
```

## 关键参数详解

### EC50 (Half Maximal Effective Concentration)

```
定义: 产生50%最大效应的药物浓度

意义:
- 效力的标准度量
- 越小表示化合物越强效
- 用于化合物间比较

计算:
从拟合的Hill方程解出:
EC50 = [D] 当 E = (Emax + Emin) / 2

注意事项:
- 依赖于实验条件
- 不同细胞类型可能不同
- 需要报告测定方法
```

### Emax (Maximum Effect)

```
定义: 药物能达到的最大效应

意义:
- 效能的度量
- 反映药物的内在活性
- 与效力(EC50)不同

示例:
化合物A: EC50 = 1nM, Emax = 80%
化合物B: EC50 = 10nM, Emax = 100%

A更强效(lower EC50)
B更高效(higher Emax)
选择取决于治疗需求
```

### Hill系数 (Hill Coefficient)

```
定义: 剂量反应曲线的斜率因子

意义:
- 反映结合协同性
- 指示作用机制
- 质量控制指标

典型值:
- n ≈ 1: 单一位点，无协同
- n > 1: 正协同，多位点合作
- n < 1: 负协同，位点竞争

异常值可能指示:
- 实验噪声
- 多个靶点
- 复杂的下游信号
```

### IC50 (Half Maximal Inhibitory Concentration)

```
定义: 抑制50%生物过程的浓度

与EC50的关系:
- EC50: 一般效应
- IC50: 特定抑制效应

计算:
对于抑制实验:
%Inhibition = 100% - %Activity
IC50是产生50%抑制的浓度

应用:
- 酶抑制实验
- 细胞增殖抑制
- 受体拮抗剂筛选

注意:
IC50依赖于实验条件
需要标准化报告
```

## 剂量反应的应用

### 1. 药物筛选

```
高通量筛选流程:

1. 化合物处理
   - 多浓度梯度 (通常8-12个浓度)
   - 对数稀释 (如: 10μM → 3.16μM → 1μM ...)

2. 响应测量
   - 细胞活力
   - 报告基因活性
   - 特定通路标记

3. 曲线拟合
   - 非线性回归
   - 估计EC50/IC50

4. 化合物分级
   - 强效: pEC50 > 8 (EC50 < 10nM)
   - 中等: pEC50 6-8
   - 弱效: pEC50 < 6

5. 优先级排序
   - 效力 + 选择性 + 理化性质
```

### 2. 毒性评估

```
关键参数:

LD50 (Lethal Dose 50):
- 致死50%实验动物的剂量
- 急性毒性指标

TD50 (Toxic Dose 50):
- 产生毒性效应的剂量
- 亚急性/慢性毒性

治疗指数 (Therapeutic Index):
TI = TD50 / ED50

安全范围:
- TI > 100: 安全
- TI 10-100: 中等
- TI < 10: 窄治疗窗，需监测
```

### 3. 机制研究

```
剂量反应揭示机制信息:

饱和动力学:
- 提示特异性结合
- 符合酶动力学

非饱和效应:
- 可能非特异性作用
- 膜损伤等

双相曲线:
- U型: 低剂量刺激，高剂量抑制
- 激素效应常见

平台期低于100%:
- 部分激动剂
- 受体储备
```

## 曲线拟合方法

### 非线性回归

```
拟合目标:
最小化残差平方和:
RSS = Σ(E_observed - E_predicted)²

优化算法:
1. Levenberg-Marquardt
2. 单纯形法
3. 遗传算法（复杂模型）

拟合质量评估:
- R²: 拟合优度
- 参数置信区间
- 残差分析
```

### 数据预处理

```
1. 背景校正:
   响应 = 原始值 - 基线值

2. 归一化:
   %响应 = (响应 - Emin) / (Emax - Emin) × 100%

3. 异常值处理:
   - 识别技术异常
   - 重复测量平均

4. 权重分配:
   - 高剂量点方差大，权重低
   - 中剂量点更重要
```

### 模型选择

| 场景 | 推荐模型 | 理由 |
|------|----------|------|
| 简单抑制/激活 | Emax | 参数少，稳定 |
| 有协同性 | Hill | 捕捉斜率变化 |
| 双相效应 | 双Hill | 两个S型叠加 |
| 时间依赖 | 动力学模型 | 考虑时间因素 |

## 可视化

```
剂量反应图的标准元素:

1. 坐标轴:
   X轴: 剂量 (对数尺度, μM或nM)
   Y轴: 响应 (% of control)

2. 数据点:
   - 均值 ± SEM/SD
   - 重复数标注

3. 拟合曲线:
   - 平滑S型曲线
   - 置信区间阴影

4. 关键参数标注:
   - EC50值
   - Hill系数
   - R²

5. 图例:
   - 化合物名称
   - 细胞类型
   - 处理时间
```
