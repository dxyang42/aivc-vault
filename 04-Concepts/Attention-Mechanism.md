# Attention Mechanism

> [[概念地图]] | [[Transformer]] | [[Deep-Learning]]

## 概述

**注意力机制 (Attention Mechanism)** 是深度学习中的一种技术，允许模型动态地关注输入的不同部分。在单细胞分析中，注意力机制被用于捕获基因间的长程依赖关系。

## 核心思想

### 基本公式

```
Attention(Q, K, V) = softmax(QK^T / sqrt(d_k)) V

其中:
  Q (Query): 查询向量
  K (Key): 键向量  
  V (Value): 值向量
  d_k: 键向量的维度
```

### 直观理解

```
输入序列: [gene1, gene2, gene3, ..., gene_n]
                ↓
         计算注意力权重
                ↓
输出: 每个基因的表达是其他基因的加权组合
```

## 在单细胞分析中的应用

### 1. 基因-基因相互作用

```
输入: 单细胞基因表达向量
输出: 基因间注意力权重矩阵

应用:
  - 识别基因调控网络
  - 发现共表达模块
  - 理解通路关系
```

### 2. 细胞-细胞相互作用

```
输入: 细胞特征向量
输出: 细胞间注意力权重

应用:
  - 细胞通讯分析
  - 微环境建模
  - 空间转录组
```

### 3. 扰动响应建模

```
输入: 对照细胞状态 + 扰动标识
输出: 扰动后细胞状态

注意力作用:
  - 关注扰动目标基因
  - 捕获下游效应
  - 建模级联反应
```

## 变体

### Self-Attention
```
Q = K = V (来自同一输入)
应用: 捕获输入内部的依赖关系
```

### Cross-Attention
```
Q来自一个输入，K,V来自另一个输入
应用: 整合不同模态的信息
```

### Multi-Head Attention
```
并行计算多组注意力
每组关注不同的特征子空间
```

## 优势

1. **长程依赖**: 捕获远距离基因间关系
2. **可解释性**: 注意力权重提供生物学洞察
3. **灵活性**: 适用于多种任务
4. **并行化**: 计算效率高

## 局限

1. **计算复杂度**: O(n²) 序列长度
2. **数据需求**: 需要大量数据训练
3. **过拟合风险**: 参数较多
