# Gene Regulatory Network (基因调控网络, GRN)

> [[概念地图]] | [[Perturbation-Prediction]] | [[术语统一规范]]

## 定义

**基因调控网络 (Gene Regulatory Network, GRN)** 是描述基因之间调控关系的网络模型。它捕捉了转录因子 (Transcription Factor, TF) 如何结合到靶基因的调控区域，从而激活或抑制其表达的分子机制。

### 核心概念

```
转录因子 (TF) ──调控──→ 靶基因 (Target Gene)
     ↑                      ↓
   结合DNA               表达变化
  调控区域              功能执行
```

GRN 是理解细胞如何响应扰动、执行功能和维持稳态的关键框架。在扰动预测中，GRN 提供了基因间相互作用的先验知识，帮助模型推断间接效应。

## TF-靶基因关系

### 转录因子 (TF)

| 特征 | 描述 |
|------|------|
| **定义** | 能够结合DNA并调控转录的蛋白质 |
| **数量** | 人类约 1,600-1,800 个序列特异性TF |
| **结构域** | DNA结合域 (DBD) + 调控域 |
| **功能** | 激活或抑制靶基因表达 |

### 调控关系类型

| 关系类型 | 机制 | 效应 |
|----------|------|------|
| **激活 (Activation)** | TF结合启动子/增强子 → 招募转录机器 | 靶基因表达↑ |
| **抑制 (Repression)** | TF结合抑制子区域 → 阻碍转录 | 靶基因表达↓ |
| **间接调控** | TF调控其他TF → 级联效应 | 多级传播 |
| **组合调控** | 多个TF共同作用 | 协同/拮抗效应 |

### 调控层级

```
主调控因子 (Master Regulators)
         ↓
    中间调控因子
         ↓
      效应基因
         ↓
    细胞功能/表型
```

## 推断方法

### 1. 基于表达数据的推断

#### 相关性方法

| 方法 | 原理 | 优缺点 |
|------|------|--------|
| **Pearson相关** | 线性相关 | 简单，但无法区分因果 |
| **互信息** | 非线性依赖 | 捕捉非线性关系 |
| **偏相关** | 控制混杂变量 | 减少间接关联 |

#### 回归方法

```
模型: 靶基因表达 = f(TF表达, 其他基因)

示例 (LASSO回归):
min ||X_target - W·X_TF||² + λ||W||₁

其中:
- X_target: 靶基因表达向量
- X_TF: TF表达矩阵
- W: 调控权重（稀疏解）
```

#### 基于树的方法

| 方法 | 特点 |
|------|------|
| **GENIE3** | 随机森林特征重要性 |
| **GRNBoost2** | 梯度提升，适合大规模数据 |

### 2. 基于先验知识的方法

#### 数据库整合

| 数据库 | 内容 | 规模 |
|--------|------|------|
| **TRRUST** | 文献整理的TF-靶基因关系 | ~8,000 关系 |
| **ENCODE** | ChIP-seq实验数据 | 大量TF结合位点 |
| **ChEA** | ChIP-X实验数据库 | 多物种 |
| **JASPAR** | TF结合基序 (motif) | ~6,000 基序 |

#### 基于Motif的方法

```
工作流程:
1. 获取靶基因启动子/增强子序列
2. 扫描TF结合基序 (PWM匹配)
3. 结合表达相关性筛选
4. 构建调控边
```

### 3. 基于扰动数据的方法

#### 因果推断

| 方法 | 原理 | 代表工具 |
|------|------|----------|
| **干预数据** | 基因KO/OE后的表达变化 | [[GEARS]] |
| **因果发现** | 基于条件独立性 | [[CASCADE]] |
| **do-calculus** | 因果推理框架 | [[scCausalVI]] |

**原理**:
```
如果敲除TF A导致基因B表达变化
→ A 可能调控 B

数学表达:
P(B | do(A=0)) ≠ P(B)  →  A → B 的因果边
```

### 4. 深度学习方法

| 方法 | 架构 | 特点 |
|------|------|------|
| **SCENIC** | 随机森林 + 调控子分析 | 识别活性调控模块 |
| **Cell Oracle** | 图神经网络 | 结合先验GRN进行扰动模拟 |
| **PerturbNet** | VAE + 注意力 | 端到端学习调控关系 |

## 应用

### 1. 扰动效应预测

**机制**:
```
扰动基因A
    ↓
直接影响: A的靶基因表达变化
    ↓
级联效应: 靶基因的靶基因变化
    ↓
网络传播: 多步传播后的全局效应
```

**应用方法**:
- [[GEARS]]: 利用GRN进行组合扰动预测
- [[Cell Oracle]]: 基于GRN模拟KO效应
- [[NetPerturb]]: 网络流模型预测扰动传播

### 2. 药物靶点发现

| 应用 | 方法 |
|------|------|
| **靶点识别** | 药物响应与GRN关联 |
| **脱靶预测** | 网络分析预测副作用 |
| **组合用药** | 网络拓扑优化 |

### 3. 疾病机制研究

```
疾病相关变异
    ↓
影响TF结合/活性
    ↓
GRN结构改变
    ↓
基因表达重编程
    ↓
疾病表型
```

### 4. 细胞命运工程

- **重编程优化**: 识别关键调控节点
- **分化控制**: 设计分化路径
- **合成生物学**: 构建人工调控回路

## 网络拓扑特性

### 常见网络结构

```
1. 前馈环 (Feed-forward Loop)
   A → B → C
   ↓______↑
   (A直接调控C)
   
2. 反馈环 (Feedback Loop)
   A → B → C
   ↑________↓
   (C调控A)
   
3. 多输入模块
      A → C
      B → C
      (A和B共同调控C)
      
4. 单输入模块
   A → B → C → D
   (A主导级联通路)
```

### 拓扑指标

| 指标 | 定义 | 生物学意义 |
|------|------|------------|
| **度分布** | 连接数分布 | 枢纽基因识别 |
| **聚类系数** | 邻居间的连接密度 | 功能模块 |
| **路径长度** | 节点间最短距离 | 信号传播效率 |
| **中心性** | 节点重要性 | 关键调控因子 |
