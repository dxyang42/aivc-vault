# Zero-Shot Learning (零样本学习)

> [[概念地图]] | [[Perturbation-Prediction]] | [[Generalization]] | [[术语统一规范]]

## 定义

**零样本学习 (Zero-Shot Learning, ZSL)** 是指模型在训练阶段未见过某些类别或样本的情况下，仍能对这些未见过的目标进行准确预测的能力。在扰动预测中，零样本学习意味着预测从未在训练数据中出现过的基因扰动或化合物处理的效果。

### 核心概念

```
传统监督学习:
训练集: {扰动A, 扰动B, 扰动C} → 学习映射
测试集: {扰动A, 扰动B}        → 预测已知扰动

零样本学习:
训练集: {扰动A, 扰动B, 扰动C} → 学习泛化能力
测试集: {扰动D, 扰动E}        → 预测未见扰动
        ↑
   训练时从未出现
```

### 为什么需要零样本学习

| 挑战 | 描述 |
|------|------|
| **组合爆炸** | 人类基因组约2万个基因，两两组合就有2亿种可能 |
| **实验成本** | 测量所有扰动组合成本极高 |
| **新靶点发现** | 需要预测全新靶点的效应 |
| **化合物筛选** | 虚拟化合物库远超已测试数量 |

## 零样本扰动预测方法类型

### 1. 基于嵌入的方法 (Embedding-Based)

**核心思想**: 学习基因或扰动的低维嵌入表示，利用嵌入空间的结构进行泛化。

```
方法框架:
1. 学习基因嵌入空间
   - 每个基因表示为d维向量
   - 功能相似的基因向量接近

2. 扰动效应建模:
   效应(基因i) = f(嵌入_i, 细胞状态)

3. 零样本预测:
   - 对于未见基因j，获取其嵌入
   - 基于嵌入相似性推断效应
   - 或使用学习到的映射函数

示例:
已知: TP53 KO效应
      TP53嵌入 ≈ MDM2嵌入 (功能相关)
预测: MDM2 KO效应 ≈ TP53 KO效应 (调整后)
```

**代表方法**:
- **scGPT**: 预训练细胞嵌入，迁移到新扰动
- **Geneformer**: 基因语言模型学习基因嵌入
- **PertAdapt**: 适应未见扰动的元学习方法

### 2. 基于描述的方法 (Description-Based)

**核心思想**: 利用基因或化合物的文本描述、功能注释等辅助信息进行预测。

```
方法框架:
1. 获取扰动的文本描述
   - 基因: 功能描述、通路注释、文献摘要
   - 化合物: 分子结构、靶点信息、作用机制

2. 文本编码:
   描述 → 语言模型 → 语义嵌入

3. 跨模态映射:
   语义嵌入 → 预测头 → 扰动效应

示例:
基因描述: "TP53是肿瘤抑制基因，调控细胞周期检查点"
    ↓
文本编码器 (BERT/GPT)
    ↓
语义向量 → 预测网络 → 预测表达变化
```

**代表方法**:
- **GenePT**: 使用ChatGPT生成的基因描述
- **BioBERT**: 生物医学文本编码
- **分子描述符**: 化合物结构信息

### 3. 基于知识图谱的方法 (Knowledge Graph-Based)

**核心思想**: 利用生物知识图谱中的关系进行推理和预测。

```
知识图谱结构:
节点: 基因、蛋白质、化合物、疾病、通路
边: 调控关系、相互作用、所属关系

零样本推理:
已知: A → B (A调控B)
      B KO的效应
预测: A KO的效应 (通过B间接推断)

或:
已知: 化合物X靶向基因Y
      基因Y KO的效应
预测: 化合物X的处理效应
```

**代表方法**:
- **PPI网络**: 蛋白质相互作用图
- **GRN**: 基因调控网络
- **知识图谱嵌入**: TransE, RotatE等

## 零样本学习的挑战

### 1. 泛化能力

```
问题: 训练分布 vs 测试分布差异

场景:
训练扰动: 常见基因 ( housekeeping基因 )
测试扰动: 罕见基因 ( 组织特异性基因 )

挑战:
- 嵌入空间中距离远
- 功能模式差异大
- 预测可靠性低

解决方案:
- 多样化训练数据
- 领域自适应
- 不确定性估计
```

### 2. 分布偏移 (Distribution Shift)

| 类型 | 描述 | 示例 |
|------|------|------|
| **协变量偏移** | 输入分布变化 | 新细胞类型 |
| **概念偏移** | 输入-输出关系变化 | 不同遗传背景 |
| **标签偏移** | 输出分布变化 | 不同疾病状态 |

```
分布偏移的影响:

训练数据分布: P_train(X, Y)
测试数据分布: P_test(X, Y) ≠ P_train(X, Y)

模型在P_train上优化
在P_test上性能下降

缓解策略:
- 领域自适应 (Domain Adaptation)
- 不变风险最小化 (IRM)
- 因果推断方法
```

### 3. 评估困难

```
零样本评估的特殊性:

传统评估:
- 划分训练/测试集
- 随机抽样保证同分布

零样本评估:
- 测试集必须是训练时完全未见
- 需要明确定义"未见"
- 评估指标需要反映泛化能力

陷阱:
- 信息泄露 (测试基因出现在训练文本中)
- 近零样本而非真零样本
- 评估集与训练集过于相似
```

## 评估协议

### 1. 严格零样本设置

```
数据划分原则:

1. 基因水平划分:
   训练基因集: G_train
   测试基因集: G_test
   G_train ∩ G_test = ∅

2. 确保无信息泄露:
   - 测试基因不出现在训练文本中
   - 测试基因不在预训练语料中
   - 测试基因的邻居也不在训练中

3. 生物学合理性:
   - 按功能家族划分
   - 按通路归属划分
   - 按保守性划分
```

### 2. 评估指标

| 指标 | 描述 | 用途 |
|------|------|------|
| **R²** | 决定系数 | 预测准确性 |
| **RMSE** | 均方根误差 | 预测误差大小 |
| **Top-K Recall** | 前K个差异基因召回率 | 关键基因预测 |
| **通路富集一致性** | 预测与真实通路一致性 | 功能层面评估 |
| **细胞状态转移准确性** | 状态变化方向正确性 | 生物学意义 |

### 3. 基线方法

```
零样本评估需要强基线:

1. 随机基线:
   随机猜测效应

2. 均值基线:
   预测所有扰动的平均效应

3. 最近邻基线:
   找到训练集中最相似的扰动
   复制其效应作为预测

4. 线性插值基线:
   基于功能相似性加权组合

5. 消融实验:
   移除关键组件验证贡献
```

## 相关方法

### scGPT

**方法类型**: 基于嵌入的零样本学习

```
核心思想:
1. 大规模预训练:
   - 数百万细胞的无监督预训练
   - 学习细胞状态的一般表示

2. 扰动标记:
   - 将扰动作为特殊token
   - 学习扰动嵌入

3. 零样本迁移:
   - 新扰动使用预训练嵌入初始化
   - 少量微调或完全零样本

零样本能力来源:
- 预训练覆盖广泛的基因表达模式
- Transformer的上下文学习能力
- 基因嵌入的语义结构
```

### Geneformer

**方法类型**: 基于描述的零样本学习

```
核心思想:
1. 基因排序:
   - 按表达水平对基因排序
   - 形成"基因句子"

2. 语言模型预训练:
   - 在大量单细胞数据上训练
   - 学习基因间的上下文关系

3. 上下文学习:
   - 新扰动通过上下文示例学习
   - 无需参数更新

零样本能力来源:
- 基因顺序编码功能信息
- 注意力机制捕捉基因关系
- 大规模预训练的知识迁移
```

### PertAdapt

**方法类型**: 元学习零样本适应

```
核心思想:
1. 元学习训练:
   - 学习如何快速适应新扰动
   - 训练任务: 从少量样本学习新扰动

2. 适应策略:
   - 给定新扰动，快速推断效应模式
   - 利用相关扰动的先验知识

3. 零样本推断:
   - 即使没有样本也能预测
   - 基于扰动特征和元知识

零样本能力来源:
- 学习扰动间的共享结构
- 快速适应机制
- 不确定性感知的预测
```

## 方法对比

| 方法 | 类型 | 优势 | 局限 |
|------|------|------|------|
| **scGPT** | 嵌入 | 上下文学习强 | 需要大量预训练 |
| **Geneformer** | 描述 | 利用基因顺序 | 对dropout敏感 |
| **PertAdapt** | 元学习 | 快速适应 | 元训练成本高 |
| **GEARS** | 图网络 | 利用PPI网络 | 需要网络信息 |
| **CPA** | 条件VAE | 显式分离因子 | 需要条件标注 |

## 应用案例

### 新靶点优先级排序

```
场景: 筛选潜在的药物靶点

流程:
1. 收集候选基因列表 (1000个)
2. 对每个基因进行零样本KO预测
3. 评估预测的表型效应
4. 按治疗相关性排序
5. 选择Top 10进行实验验证

优势:
- 大幅减少实验筛选成本
- 优先验证高潜力靶点
- 加速药物发现流程
```

### 化合物效应预测

```
场景: 预测新化合物的细胞效应

流程:
1. 获取化合物结构信息
2. 预测靶点基因
3. 零样本预测靶点KO效应
4. 推断化合物处理效应
5. 预测毒性和脱靶效应

优势:
- 虚拟筛选化合物库
- 早期识别毒性风险
- 指导化合物优化
```
