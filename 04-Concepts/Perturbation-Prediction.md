# Perturbation Prediction (扰动预测)

> [[概念地图]] | [[方法地图]] | [[术语统一规范]]

## 定义

**扰动预测 (Perturbation Prediction)** 是指利用计算模型预测细胞在受到外部干预后基因表达变化的能力。这是单细胞计算生物学中的核心任务，旨在回答"如果改变某个基因或施加某种处理，细胞会如何响应"这一反事实问题。

### 核心问题

给定一个细胞的初始状态（基因表达谱）和扰动信息（如敲除基因A），预测扰动后的细胞状态：

```
输入: 初始细胞状态 x ∈ R^G, 扰动信息 p
输出: 扰动后细胞状态 x' ∈ R^G
```

其中 G 是基因数量，通常为 20,000-30,000 个蛋白质编码基因。

---

## 扰动类型

### 1. 基因敲除 (Gene Knockout, KO)

| 特征 | 描述 |
|------|------|
| **机制** | 完全消除目标基因的功能，表达量接近零 |
| **技术** | CRISPR-Cas9 (KO)、shRNA、siRNA |
| **数据特征** | 目标基因表达量显著降低（通常 < 0.1 原始值） |
| **典型方法** | [[GEARS]]、[[scGen]]、[[CPA]] |

**示例**: 敲除肿瘤抑制基因 TP53，预测细胞增殖相关基因的表达变化。

### 2. 基因过表达 (Gene Overexpression, OE)

| 特征 | 描述 |
|------|------|
| **机制** | 人为增加目标基因的表达水平 |
| **技术** | cDNA过表达载体、转座子系统 |
| **数据特征** | 目标基因表达量显著升高（通常 > 5-10 倍） |
| **典型方法** | [[GEARS]]、[[CPA]] |

**示例**: 过表达转录因子 OCT4，预测干细胞重编程相关基因的变化。

### 3. 化合物扰动 (Chemical Perturbation)

| 特征 | 描述 |
|------|------|
| **机制** | 小分子药物或化合物对细胞的处理 |
| **技术** | 药物筛选、化合物库处理 |
| **数据特征** | 多靶点效应，系统性表达变化 |
| **典型数据集** | [[sci-Plex]]、[[L1000]] |

**示例**: 用化疗药物处理癌细胞，预测细胞凋亡通路的激活情况。

### 4. 组合扰动 (Combinatorial Perturbation)

| 特征 | 描述 |
|------|------|
| **机制** | 同时施加两个或多个扰动 |
| **技术** | 多重CRISPR (Perturb-seq)、药物组合 |
| **数据特征** | 非线性相互作用（协同/拮抗） |
| **典型方法** | [[CPA]]、[[GEARS]]、[[scPerturb-Benchmark]] |

**示例**: 同时敲除基因A和基因B，预测组合效应是否为简单的加和或存在相互作用。

## 核心挑战

### 1. 反事实推断 (Counterfactual Inference)

**问题**: 无法同时观测同一细胞在扰动前和扰动后的状态。

```
同一细胞无法既被扰动又不被扰动
↓
需要从未观测的状态推断反事实结果
↓
依赖统计学习和因果推断
```

**解决方案**:
- 使用对照细胞作为基线
- 因果推断方法（如 do-calculus）
- 参考 [[scCausalVI]]、[[CASCADE]] 等因果方法

### 2. 分布偏移 (Distribution Shift)

**问题**: 训练时见过的扰动与测试时的新扰动分布不同。

| 偏移类型 | 描述 | 示例 |
|----------|------|------|
| **扰动偏移** | 训练集与测试集扰动不同 | 训练时敲除基因A，测试时敲除基因B |
| **细胞类型偏移** | 不同细胞类型的响应不同 | 训练用K562，测试用RPE1 |
| **批次偏移** | 实验批次间的技术差异 | 不同实验室的数据 |

**解决方案**:
- 域适应技术（[[scArches]]）
- 零样本学习（[[AlphaCell]]、[[CellHermes]]）
- 不变性约束

### 3. 组合爆炸 (Combinatorial Explosion)

**问题**: 可能的扰动组合数量随基因数指数增长。

```
N 个基因 → 2^N 种可能的组合扰动
人类基因组 ~20,000 基因 → 不可能穷举
```

**数学表达**:
- 单基因扰动: O(N)
- 双基因组合: O(N²)
- 三基因组合: O(N³)

**解决方案**:
- 组合泛化方法（[[CPA]]、[[GEARS]]）
- 基因相互作用建模
- 图神经网络利用基因关系（[[GEARS]]）

## 评估指标

### 1. 表达谱重建指标

| 指标 | 描述 | 公式 |
|------|------|------|
| **MSE** | 均方误差 | (1/N) Σ(x_pred - x_true)² |
| **MAE** | 平均绝对误差 | (1/N) Σ\|x_pred - x_true\| |
| **R²** | 决定系数 | 1 - Var(x_pred - x_true)/Var(x_true) |
| **Pearson r** | 皮尔逊相关系数 | Cov(x_pred, x_true)/(σ_pred × σ_true) |

### 2. 差异表达基因预测

| 指标 | 描述 |
|------|------|
| **DE Recall@K** | 前K个预测DE基因中真实DE基因的比例 |
| **DE Precision@K** | 真实DE基因中被正确预测在前K个的比例 |
| **AUPRC** | 精确率-召回率曲线下面积 |

### 3. 下游任务指标

| 指标 | 描述 |
|------|------|
| **细胞状态分类准确率** | 扰动后细胞类型预测准确率 |
| **通路富集一致性** | 预测通路与实际通路的一致性 |
| **药物响应相关性** | 预测与实验测量的药物响应相关性 |

## 应用场景

### 1. 药物发现与重定位

- **虚拟筛选**: 预测化合物对特定细胞类型的效应
- **药物重定位**: 发现现有药物的新适应症
- **毒性预测**: 预测候选药物的脱靶效应

**相关方法**: [[DrugCell]]、[[sci-Plex]]

### 2. 基因功能注释

- **功能推断**: 通过扰动效应推断未知基因功能
- **通路发现**: 识别新的信号通路成员
- **调控关系**: 构建基因调控网络

**相关方法**: [[GEARS]]、[[Cell Oracle]]

### 3. 疾病建模

- **疾病机制**: 模拟疾病相关基因突变的影响
- **治疗靶点**: 识别潜在的治疗干预靶点
- **个性化医疗**: 预测患者特异性药物响应

**相关方法**: [[Perturb-Cancer]]、[[CELLOT]]

### 4. 细胞工程

- **重编程优化**: 优化细胞命运转换条件
- **合成生物学**: 设计合成基因回路
- **细胞治疗**: 工程化改造治疗性细胞
