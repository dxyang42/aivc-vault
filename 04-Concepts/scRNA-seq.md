# scRNA-seq (单细胞RNA测序)

> [[概念地图]] | [[Gene-Expression]] | [[术语统一规范]]

## 定义

**单细胞RNA测序 (single-cell RNA sequencing, scRNA-seq)** 是一种高通量分子生物学技术，能够在单个细胞分辨率水平上定量分析基因表达。它 revolutionized 了我们对细胞异质性的理解，是扰动预测领域最重要的数据来源。

### 核心突破

| 传统方法 | scRNA-seq |
|----------|-----------|
| 群体平均表达 | 单个细胞表达 |
| 掩盖细胞异质性 | 揭示细胞亚型 |
| 无法追踪稀有细胞 | 识别稀有细胞类型 |
| 静态快照 | 动态发育轨迹 |

---

## 技术原理

### 基本流程

```
样本制备
    ↓
单细胞分离
    ↓
细胞裂解 + mRNA捕获
    ↓
逆转录 → cDNA
    ↓
文库构建
    ↓
高通量测序
    ↓
数据分析
```

### 关键技术挑战

| 挑战 | 解决方案 |
|------|----------|
| **微量RNA** | 全转录组扩增 (WTA) |
| **细胞通量** | 微流控、组合索引 |
| **成本控制** | 组合条形码、混合测序 |
| **双细胞** | 计算识别、实验优化 |

---

## 主要技术平台

### 1. 10x Genomics Chromium

| 特征 | 描述 |
|------|------|
| **原理** | 微流控液滴 + 凝胶珠条形码 |
| **通量** | 最高 100,000+ 细胞/运行 |
| **捕获率** | ~50-60% |
| **读长** | 3'端或5'端捕获 |
| **成本** | 中等 (~$1-2/细胞) |

**工作流程**:
```
1. 单细胞悬液 + 凝胶珠(GEM) + 油 → 液滴
2. 细胞裂解，mRNA与凝胶珠上的引物杂交
3. 逆转录，每个cDNA获得细胞特异性条形码
4. 破乳，合并cDNA进行文库构建
5. 测序
```

**优势**:
- 高通量，适合大规模研究
- 商业化成熟，标准化程度高
- 配套分析软件完善 (Cell Ranger)

**局限**:
- 捕获率低，部分细胞丢失
- 3'端偏倚（丢失剪接信息）
- 对大型细胞效果较差

### 2. Smart-seq2

| 特征 | 描述 |
|------|------|
| **原理** | 模板切换逆转录 + PCR扩增 |
| **通量** | 96-384 细胞/运行 |
| **捕获率** | ~20-30% |
| **读长** | 全长度覆盖 |
| **成本** | 较高 (~$10-50/细胞) |

**工作流程**:
```
1. 单细胞分选到96孔板
2. 寡聚dT引物结合polyA尾
3. 模板切换逆转录
4. PCR预扩增
5. 片段化、接头连接
6. 测序
```

**优势**:
- 全转录本覆盖（可检测可变剪接）
- 灵敏度高，适合低表达基因
- 可检测等位基因特异性表达

**局限**:
- 通量低
- 成本高
- 需要分选设备

### 3. 其他平台对比

| 平台 | 通量 | 覆盖度 | 主要应用 |
|------|------|--------|----------|
| **Drop-seq** | 高 | 3'端 | 低成本大规模 |
| **inDrop** | 高 | 3'端 | 类似Drop-seq |
| **Seq-Well** | 高 | 3'端 | 便携式，适合野外 |
| **SPLIT-seq** | 极高 | 3'端 | 组合索引，无微流控 |
| **sci-RNA-seq** | 极高 | 3'端 | 组合索引，核测序 |

## 数据特征

### 1. 稀疏性 (Sparsity)

**现象**: 表达矩阵中大部分元素为零。

```
典型scRNA-seq数据:
- 零值比例: 90-95%
- 原因: 捕获效率低 + 生物学低表达

示例矩阵 (5细胞 × 5基因):
       G1    G2    G3    G4    G5
C1     0     5     0     2     0
C2     3     0     0     0     7
C3     0     0     1     0     0
C4     8     2     0     4     0
C5     0     0     0     0     3

零值比例: 13/25 = 52% (实际数据通常更高)
```

**影响**:
- 增加数据存储和计算复杂度
- 需要专门的统计模型（如ZINB）
- 降维和聚类需考虑稀疏性

### 2. Dropout

**定义**: 由于技术限制，真实表达的基因未被检测到。

```
真实表达水平 vs 检测概率:

高表达基因: 几乎100%被检测
中表达基因: 50-80%被检测
低表达基因: < 50%被检测（大量dropout）
```

**数学建模**:
```
观测值 X_ij 的生成过程:

1. 真实表达 λ_ij ~ Gamma(α, β)
2. 捕获 u_ij ~ Poisson(λ_ij)
3. 测序采样 X_ij ~ Binomial(u_ij, ρ)

综合: X_ij ~ NegativeBinomial(μ, θ)
```

### 3. 高维性 (High Dimensionality)

| 维度 | 规模 | 挑战 |
|------|------|------|
| **基因维度** | 20,000-30,000 | 维度灾难、计算复杂度 |
| **细胞维度** | 1,000-1,000,000+ | 可扩展性、存储需求 |
| **数据点** | 10^9+ | 需要降维和特征选择 |

**降维策略**:
- 选择高变异基因 (HVG): 2,000-5,000个
- PCA降维: 通常50-100个主成分
- 非线性降维: UMAP, t-SNE用于可视化

## 预处理流程

### 标准分析流程

```
原始数据 (FASTQ)
    ↓
比对/定量 → 表达矩阵
    ↓
质控 (QC) → 过滤低质量细胞
    ↓
标准化 → 校正测序深度
    ↓
特征选择 → 高变异基因
    ↓
降维 → PCA/UMAP
    ↓
聚类 → 细胞类型识别
    ↓
注释 → 生物学解释
    ↓
下游分析 → 差异表达、轨迹等
```

### 1. 质控 (Quality Control)

**过滤标准**:

| 指标 | 下限 | 上限 | 说明 |
|------|------|------|------|
| **检测基因数** | 200-500 | 通常不设 | 去除低质量/死细胞 |
| **总UMI数** | 500-1000 | 通常不设 | 去除空液滴 |
| **线粒体基因比例** | 0% | 5-20% | 高比例提示细胞损伤 |
| **核糖体基因比例** | - | - | 辅助指标 |

**示例代码逻辑**:
```python
# 质控过滤
filtered_cells = cells[
    (cells.n_genes > 200) &      # 至少检测200个基因
    (cells.n_genes < 8000) &     # 排除双细胞
    (cells.total_counts > 500) & # 至少500个UMI
    (cells.pct_mt < 10)          # 线粒体基因<10%
]
```

### 2. 标准化

详见 [[Gene-Expression]] 文档的标准化方法部分。

常用流程:
```
原始计数
    ↓
CPM标准化 (每细胞总计数归一化)
    ↓
Log1p变换 (log(1 + CPM))
    ↓
缩放 (均值为0，方差为1)
```

### 3. 特征选择

**高变异基因 (Highly Variable Genes, HVG)**:

```
选择标准:
1. 计算每个基因的均值和方差
2. 拟合均值-方差关系
3. 选择偏离拟合线的高变异基因
4. 通常选择 2,000-5,000 个基因
```

**目的**:
- 降低计算复杂度
- 去除管家基因（无信息量的稳定表达基因）
- 保留生物学变异的主要信号

### 4. 批次校正

**问题**: 不同实验批次间的技术差异。

**常用方法**:

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| **Harmony** | 迭代软聚类 | 大规模整合 |
| **Seurat CCA** | 典型相关分析 | 小样本整合 |
| **scVI** | 深度生成模型 | 复杂批次结构 |
| **Scanorama** | 互近邻匹配 | 快速整合 |

## 质量控制指标

### 实验层面

| 指标 | 目标值 | 说明 |
|------|--------|------|
| **测序饱和度** | > 50% | 新读数发现新分子的概率 |
| **中位基因数/细胞** | > 1000 | 每个细胞检测到的基因数 |
| ** reads 比对率** | > 70% | 比对到基因组的读数比例 |
| **细胞回收率** | > 50% | 实际测到的细胞/上样细胞 |

### 数据分析层面

| 指标 | 检查点 | 异常表现 |
|------|--------|----------|
| **线粒体比例** | 质控 | 双峰分布，大量高比例细胞 |
| **总UMI分布** | 质控 | 大量零或极低UMI细胞 |
| **批次效应** | 降维可视化 | 细胞按批次聚类 |
| **双细胞** | 聚类后 | 异常大的细胞簇 |

## 扰动实验中的scRNA-seq

### Perturb-seq

**定义**: 将CRISPR扰动与scRNA-seq结合的技术。

```
工作流程:
1. 构建sgRNA文库（每个sgRNA带条形码）
2. 感染细胞池
3. scRNA-seq捕获细胞转录组 + sgRNA条形码
4. 每个细胞的转录组与其扰动关联
```

**应用**:
- 大规模基因功能筛选
- 基因调控网络推断
- 药物靶点发现

### 代表性数据集

| 数据集 | 规模 | 扰动类型 | 应用 |
|--------|------|----------|------|
| **Norman 2021** | ~100万细胞 | 组合KO/OE | 组合扰动预测基准 |
| **Replogle 2022** | ~250万细胞 | 单基因KO | 大规模KO筛选 |
| **sci-Plex** | ~65万细胞 | 化合物 | 药物筛选 |
