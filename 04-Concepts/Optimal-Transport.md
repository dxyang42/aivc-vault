# Optimal Transport (最优传输)

> [[概念地图]] | [[Batch-Effect]] | [[Trajectory-Inference]] | [[术语统一规范]]

## 定义

**最优传输 (Optimal Transport, OT)** 是数学中研究如何以最小成本将一个概率分布转换为另一个概率分布的理论。在单细胞生物学中，最优传输用于建模细胞状态转换、批次校正、发育轨迹推断和扰动效应预测。

### 核心概念

```
直观理解:

想象有两个沙堆:
- 沙堆A (源分布): 细胞状态分布1
- 沙堆B (目标分布): 细胞状态分布2

问题: 如何以最小工作量将A变成B?

最优传输回答:
- 找到最优的"运输方案"
- 最小化总运输成本
- 成本 = 距离 × 质量

在单细胞中的应用:
- 细胞分化: 干细胞分布 → 成熟细胞分布
- 扰动响应: 对照组分布 → 处理组分布
- 批次校正: 批次A分布 → 批次B分布
```

### 为什么最优传输重要

| 价值 | 描述 |
|------|------|
| **几何结构** | 在概率分布空间定义距离 |
| **可解释性** | 提供细胞级别的对应关系 |
| **灵活性** | 适用于各种分布形状 |
| **理论基础** | 有坚实的数学基础 |
| **多领域应用** | 跨批次、轨迹、扰动预测 |

## 数学定义

### Monge问题 (Monge Problem)

**原始形式** (1781年):

```
问题设定:
- 源分布: μ (定义在空间X上)
- 目标分布: ν (定义在空间Y上)
- 成本函数: c(x, y) = 从x运输到y的成本

寻找确定性映射 T: X → Y

最小化:
∫ c(x, T(x)) dμ(x)

约束:
T#μ = ν  (T将μ推前到ν)

解释:
每个源点x必须映射到一个目标点T(x)
且整体分布要匹配
```

**局限性**:

```
Monge问题的限制:

1. 可能无解:
   当μ是离散点，ν是连续分布时
   无法找到确定性映射

2. 非凸优化:
   难以高效求解

3. 刚性约束:
   不允许质量分割
```

### Kantorovich问题 (Kantorovich Problem)

**松弛形式** (1942年):

```
问题设定:
寻找耦合分布 γ ∈ Π(μ, ν)

Π(μ, ν) = {γ: 边缘分布为μ和ν的所有联合分布}

最小化:
∫∫ c(x, y) dγ(x, y)

解释:
γ(x, y) 表示从x运输到y的质量
允许质量分割: x可以部分运到多个y

优势:
- 总是存在解
- 线性规划问题
- 更灵活的运输方案
```

**与Monge问题的关系**:

```
Kantorovich是Monge的推广:

Monge: 确定性映射 T(x)
Kantorovich: 概率性耦合 γ(x,y)

当最优解是确定性时:
γ(x,y) = μ(x) × δ(y - T(x))

Kantorovich解总是存在
Monge解可能不存在
```

## 距离度量

### Wasserstein距离 (Earth Mover's Distance)

**定义**: 最优传输成本定义的概率分布距离。

```
数学定义:

W_p(μ, ν) = (inf_{γ∈Π(μ,ν)} ∫∫ d(x,y)^p dγ(x,y))^(1/p)

其中:
- p ≥ 1: 阶数
- d(x,y): 底层空间的距离
- 通常p=1或p=2

W_1 (Earth Mover's Distance):
W_1(μ, ν) = inf_γ ∫∫ ||x - y|| dγ(x,y)

W_2:
W_2(μ, ν) = (inf_γ ∫∫ ||x - y||² dγ(x,y))^(1/2)
```

**性质**:

| 性质 | 描述 |
|------|------|
| **度量性质** | 满足距离公理（非负、对称、三角不等式） |
| **弱收敛** | 与分布的弱收敛相容 |
| **几何意义** | 反映分布的几何结构 |
| **鲁棒性** | 对分布偏移比KL散度更鲁棒 |

**与KL散度的比较**:

```
KL散度:
- 不对称
- 当支撑集不重叠时无穷大
- 对分布形状不敏感

Wasserstein:
- 对称
- 即使支撑集不重叠也有有限值
- 反映分布的几何距离

示例:
μ = δ(0)      (集中在0的点质量)
ν = δ(ε)      (集中在ε的点质量)

KL(μ||ν) = ∞  (支撑集不重叠)
W_2(μ, ν) = ε  (反映实际距离)
```

### Sinkhorn距离

**定义**: 熵正则化的最优传输距离。

```
正则化形式:

W_ε(μ, ν) = inf_γ {∫∫ c dγ + ε × H(γ)}

其中:
- H(γ) = ∫∫ γ log γ: 熵
- ε > 0: 正则化强度

熵的作用:
- 使问题严格凸
- 保证唯一解
- 加速计算

Sinkhorn距离:
S_ε(μ, ν) = W_ε(μ, ν) - (W_ε(μ, μ) + W_ε(ν, ν))/2

确保 S_ε(μ, μ) = 0
```

## 计算方法

### 熵正则化 (Entropic Regularization)

```
正则化问题的优势:

原始OT:
- 线性规划
- 复杂度 O(n³) 或更高
- 对于大规模问题不可行

熵正则化OT:
- 严格凸优化
- 可用Sinkhorn迭代高效求解
- 复杂度 O(n²) 每迭代
- 可并行化

权衡:
- ε越大，计算越快，但近似越粗糙
- ε越小，越接近真实OT，但计算越慢
- 实践中ε需要调参
```

### Sinkhorn算法

**迭代过程**:

```
算法: Sinkhorn-Knopp迭代

输入: 成本矩阵 C, 正则化参数 ε
      源分布 μ, 目标分布 ν

初始化: K = exp(-C/ε), u = 1

重复直到收敛:
    v = ν / (K^T u)
    u = μ / (Kv)

输出: 传输计划 γ = diag(u) K diag(v)

解释:
- K: Gibbs核矩阵
- u, v: 对偶变量（缩放因子）
- 迭代更新直到满足边缘约束

收敛性:
- 线性收敛
- 通常几十到几百次迭代
```

**矩阵形式**:

```
离散情况:

μ, ν: 离散概率分布 (n维向量)
C: 成本矩阵 (n×n)
γ: 传输计划矩阵 (n×n)

优化问题:
min_γ <C, γ> + ε H(γ)
s.t. γ1 = μ, γ^T 1 = ν

Sinkhorn解:
γ = diag(u) exp(-C/ε) diag(v)

其中u, v通过迭代求解
```

## 在单细胞中的应用

### 1. 批次校正

```
应用场景:
- 整合多个实验批次的数据
- 消除技术变异，保留生物学信号

方法: 最优传输整合

步骤:
1. 计算批次间的最优传输计划
2. 根据传输计划映射细胞
3. 获得批次校正的嵌入

优势:
- 保持细胞类型的局部结构
- 可解释的细胞对应关系
- 处理非线性批次效应

代表方法: [[scOTM]]
```

### 2. 轨迹推断

```
应用场景:
- 重建细胞分化或发育轨迹
- 预测细胞命运

方法: 动态最优传输

模型:
- 将每个时间点的细胞视为分布
- 学习连续时间的最优传输
- 获得细胞状态转换路径

数学形式:
μ_t: 时间t的细胞分布
最小化: ∫ W_2(μ_t, μ_{t+Δt}) dt

代表方法: [[TrajectoryNet]]

优势:
- 连续轨迹建模
- 考虑细胞增殖和死亡
- 概率性命运预测
```

### 3. 扰动预测

```
应用场景:
- 预测扰动后的细胞状态分布
- 建模对照组到处理组的转换

方法: 条件最优传输

模型:
学习映射: T(x, p): 细胞x在扰动p下的新状态

最优传输视角:
- 源: 对照组细胞分布
- 目标: 扰动组细胞分布
- 学习最优传输映射

代表方法: [[CellOT]]

优势:
- 分布级别的预测
- 捕捉细胞异质性响应
- 可解释的状态转换
```

## 相关方法

| 方法 | 类型 | 应用 |
|------|------|------|
| **CellOT** | 条件OT | 扰动预测 |
| **TrajectoryNet** | 动态OT | 轨迹推断 |
| **scOTM** | OT映射 | 批次校正 |
| **Waddington-OT** | OT | 发育轨迹 |
| **SCOT** | 耦合OT | 多组学整合 |

### CellOT

**方法类型**: 条件最优传输

```
核心思想:
学习条件传输映射 T(x, p)

输入:
- 细胞状态 x
- 扰动标识 p

输出:
- 扰动后的细胞状态 x'

训练:
1. 收集对照组和扰动组数据
2. 对每个扰动p，计算OT映射
3. 学习神经网络近似这些映射
4. 泛化到新扰动

优势:
- 分布级别的预测
- 考虑细胞间异质性
- 可解释性强
```

### TrajectoryNet

**方法类型**: 动态最优传输

```
核心思想:
使用连续时间OT建模细胞动态

模型:
ρ(t, x): 时间t的细胞密度
最小化: ∫∫ (1/2)|v|² ρ dx dt
约束: 连续性方程 ∂ρ/∂t + ∇·(ρv) = 0

其中v是速度场

求解:
- 神经网络参数化速度场
- 端到端训练
- 获得连续轨迹

应用:
- 发育生物学
- 重编程研究
- 疾病进展
```

### scOTM (Single-Cell Optimal Transport Mapping)

**方法类型**: 最优传输批次整合

```
核心思想:
使用OT进行批次校正和整合

步骤:
1. 计算批次间的OT计划
2. 基于传输计划对齐细胞
3. 在共享空间中整合

特点:
- 保持局部细胞邻域
- 处理大规模数据
- 可扩展的算法

与Harmony/Scanorama比较:
- OT提供更强的数学保证
- 计算成本较高
- 适合精细分析
```

## 可视化

```
最优传输计划可视化:

源分布 μ          传输计划 γ         目标分布 ν
   ●                    ╱╲                ◆
  /|\                  ╱  ╲              /|\
 / | \    ─────→      ●────◆            / | \
●  ●  ●              / \  / \          ◆  ◆  ◆

边宽度: 传输质量
边颜色: 传输成本

在单细胞中的解释:
●: 对照组细胞
◆: 扰动组细胞
边: 细胞间的对应关系
```
