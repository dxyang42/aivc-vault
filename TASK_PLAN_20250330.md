  - 02-Methods/GNN/ (图神经网络)

## Subagent 分工

### Agent A: 搜索与扩充专家
**任务**: 使用 minimax mcp 搜索新模型方法
- 每轮搜索 3-5 个高质量方法
- 覆盖不同技术路线
- 输出标准格式的方法文档

### Agent B: 结构重构专家
**任务**: 重构现有方法文档
- 移除 LaTeX 公式
- 用自然语言描述
- 完善 ASCII 结构图

### Agent C: 比较分析专家
**任务**: 深化方法间比较
- 维护方法对比矩阵
- 创建技术路线对比
- 添加使用场景推荐

### Agent D: 链接与文档同步专家
**任务**: 维护文档一致性
- 更新 README.md
- 更新地图文档
- 修复重构后的链接

### Agent E: 质量审查专家
**任务**: 每轮质量把关
- 审查输出质量
- 提出改进意见
- 检查事实性错误

## 风险与注意事项

1. **链接失效**: 文件夹重构会导致大量内部链接失效，需要系统修复
2. **内容一致性**: 多 agent 并行可能导致内容风格不一致
3. **搜索质量**: minimax mcp 搜索结果需要人工筛选
4. **时间控制**: 10轮迭代需要合理分配时间
