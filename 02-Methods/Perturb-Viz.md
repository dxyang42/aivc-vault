## 核心功能

### 1. 扰动效应矩阵可视化

```
┌─────────────────────────────────────────────────────────────┐
│              扰动效应热图矩阵示例                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│        Gene1  Gene2  Gene3  Gene4  Gene5  ...  PathwayN    │
│  Pert1   ■      ■      □      ■      □   ...     ■        │
│  Pert2   □      ■      ■      □      ■   ...     □        │
│  Pert3   ■      □      □      ■      ■   ...     ■        │
│  ...                                                    ... │
│  PertM   □      ■      ■      ■      □   ...     □        │
│                                                             │
│  图例: ■ 上调 (logFC > 1)   □ 下调 (logFC < -1)           │
│                                                             │
│  交互功能:                                                  │
│  • 行列聚类 (层次聚类)                                      │
│  • 动态筛选 (按效应大小)                                    │
│  • 悬停详情 (统计值)                                        │
│  • 导出图片/数据                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. 组合效应分析

| 分析类型 | 描述 | 可视化 |
|---------|------|--------|
| **加性检验** | 比较观测vs预期加性效应 | 散点图 + 回归线 |
| **协同检测** | 识别超加性效应 | 火山图 |
| **拮抗检测** | 识别亚加性效应 | 火山图 |
| **上位性分类** | 遗传相互作用类型 | 网络图 |

### 3. 动态轨迹可视化

```
┌─────────────────────────────────────────────────────────────┐
│              扰动响应轨迹动画                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│    UMAP1                                                    │
│      │                                                      │
│      │    ○ → → → ●                                        │
│      │   /         ↑                                       │
│      │  ○    ○ → → ●                                       │
│      │ /    /                                                │
│      │○ → → ●                                                │
│      └────────────────────── UMAP2                          │
│                                                             │
│   ○ = 对照细胞    ● = 扰动后细胞                            │
│   → = 轨迹方向                                               │
│                                                             │
│   控制面板:                                                 │
│   [▶播放] [⏸暂停] [⏪重置] 速度: [慢-快]                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 支持的输入格式

| 格式 | 描述 | 示例工具输出 |
|------|------|-------------|
| **AnnData (.h5ad)** | 标准单细胞格式 | Scanpy, scVI |
| **Seurat对象** | R单细胞格式 | Seurat |
| **CSV/TSV矩阵** | 表达矩阵 | 自定义分析 |
| **预测结果** | 方法输出 | GEARS, CPA, scGen |
| **Perturb-seq** | 带sgRNA标识 | Mixscape |

## 输出格式

| 输出类型 | 格式 | 用途 |
|---------|------|------|
| **静态图** | PNG, PDF, SVG | 发表级图片 |
| **交互图** | HTML (D3.js) | 网页展示 |
| **数据表** | CSV, Excel | 下游分析 |
| **报告** | PDF/Markdown | 完整分析 |
| **Session** | JSON | 保存状态 |

## 应用场景

1. **实验数据探索**: 快速浏览大规模扰动数据集
2. **预测结果验证**: 可视化比较预测与实验
3. **组合设计优化**: 识别协同扰动对
4. **机制假设生成**: 从可视化中发现新模式
5. **报告生成**: 自动生成分析报告
6. **教学演示**: 交互式展示扰动效应

## 优势与局限

### 优势
- 专门针对扰动数据设计
- 丰富的交互式可视化
- 支持多种输入格式
- 与主流预测方法集成
- 用户友好的界面

### 局限
- 主要关注可视化，非预测
- 大数据集可能需要预聚合
- 部分高级功能需要编程
