## 核心思想

CellFlux 是一种基于**流模型（Flow-based Model）**的单细胞扰动预测方法。流模型通过**可逆神经网络**学习从简单分布（如高斯分布）到复杂数据分布的映射，实现精确的密度估计和高质量的样本生成。

核心创新：
1. **可逆变换**: 通过可逆神经网络实现双向映射
2. **精确密度估计**: 可计算精确的对数似然
3. **条件生成**: 以扰动条件为输入生成预测
4. **连续时间建模**: 可建模细胞状态的连续变化

### 耦合层详细结构

```
┌─────────────────────────────────────────────────────────────────────┐
│              仿射耦合层 (Affine Coupling Layer)                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   输入 x 被分成两部分: x = [x_a, x_b]                               │
│                                                                     │
│   正向变换:                                                         │
│   ───────────                                                       │
│   y_a = x_a (保持不变)                                              │
│                                                                     │
│   y_b = x_b * exp(s(x_a, c)) + t(x_a, c)                           │
│                                                                     │
│   其中:                                                             │
│   - s(x_a, c): 缩放网络输出 (由x_a和条件c决定)                      │
│   - t(x_a, c): 平移网络输出 (由x_a和条件c决定)                      │
│                                                                     │
│   逆向变换:                                                         │
│   ───────────                                                       │
│   x_a = y_a (保持不变)                                              │
│                                                                     │
│   x_b = (y_b - t(y_a, c)) * exp(-s(y_a, c))                        │
│                                                                     │
│   网络结构:                                                         │
│   ┌─────────────────────────────────────────────────────┐          │
│   │                                                     │          │
│   │   x_a ──┐                                           │          │
│   │         ├──→ MLP(s) ──→ 缩放系数 s                  │          │
│   │   c ────┤                                           │          │
│   │         ├──→ MLP(t) ──→ 平移系数 t                  │          │
│   │                                                     │          │
│   │   y_b = x_b * exp(s) + t                           │          │
│   │                                                     │          │
│   └─────────────────────────────────────────────────────┘          │
│                                                                     │
│   雅可比行列式: 对角矩阵，行列式易计算                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Perturbation 预测能力

### 核心功能

| 功能 | 描述 |
|------|------|
| **精确密度估计** | 可计算预测的概率密度 |
| **高质量生成** | 流模型生成高质量样本 |
| **条件控制** | 以扰动条件控制生成过程 |
| **不确定性量化** | 通过潜在空间采样估计不确定性 |

### 适用场景

- **单细胞扰动预测**: 预测基因敲除/过表达效应
- **细胞状态转换**: 建模细胞状态的连续变化
- **虚拟细胞实验**: 模拟未见扰动的效应
- **密度估计**: 估计细胞群体的概率分布

## 与其他方法的对比

| 方法 | 主要区别 | 适用场景 |
|------|----------|----------|
| **CellFlux** | 流模型，精确密度 | 需要概率建模 |
| **scDFM** | 流匹配 | 分布级扰动预测 |
| **CFM-GP** | 条件流匹配 | 基因扰动预测 |
| **SCALE** | 最优传输 | 大规模扰动预测 |
| **scGen** | VAE | 潜在空间运算 |

## 优势与局限

### 优势
- 精确的密度估计和对数似然计算
- 可逆性允许双向映射
- 生成样本质量高
- 可量化预测不确定性

### 局限
- 可逆架构限制网络设计
- 计算成本较高
- 需要配对的训练数据
- 对复杂分布建模能力有限

## 典型应用

1. **单细胞扰动预测**: 预测基因扰动后的细胞状态
2. **细胞状态建模**: 建模细胞状态的连续演化
3. **虚拟实验**: 模拟未见扰动的效应
4. **密度估计**: 估计细胞群体的分布
