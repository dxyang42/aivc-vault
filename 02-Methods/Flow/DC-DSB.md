# DC-DSB

> [[方法地图]] | [[团队地图]]

## 1. 基本信息

| 属性 | 内容 |
|------|------|
| **名称** | DC-DSB (Direction-Constrained Diffusion Schrödinger Bridge) |
| **发布时间** | 2025年10月 |
| **机构** | Luo Y, Liu L, Fu Y, Deng Y, Tang L (中国团队) |
| **论文** | J Biomed Inform. 2025 Oct;170:104915 |
| **PMID** | 40987372 |
| **模型类型** | 扩散模型 (Diffusion Schrödinger Bridge) |
| **应用领域** | 单细胞扰动响应预测 |

### 核心创新

DC-DSB 是一种基于**方向约束扩散薛定谔桥**的单细胞扰动响应预测方法。该方法通过引入方向约束机制，确保生成的扰动路径符合生物学机制，同时提供生物学上可解释的中间状态。

## 2. 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 预测基因敲除后的细胞状态 |
| **单基因过表达 (OE)** | 完全支持 | 预测基因过表达效应 |
| **药物处理** | 完全支持 | 预测药物扰动响应 |
| **多扰动建模** | 支持 | 支持多种扰动类型的联合建模 |
| **未见扰动** | 有限 | 对训练未见的扰动泛化有限 |

### 预测准确率与评估指标

| 指标 | 描述 | 典型值 |
|------|------|--------|
| **基因表达相关性** | Pearson/Spearman相关 | r > 0.80 |
| **MSE/RMSE** | 基因表达均方误差 | 低 |
| **R² 分数** | 决定系数 | 0.75-0.85 |
| **差异表达基因重叠** | 预测与真实DE基因一致性 | Jaccard > 0.6 |
| **通路富集一致性** | 富集通路的Jaccard相似度 | > 0.5 |

## 3. 与其他扰动预测方法的差异

| 特性 | DC-DSB | CellFlow | scDFM | GEARS |
|------|--------|----------|-------|-------|
| **核心框架** | 薛定谔桥 | 流匹配 | 流匹配 | GNN |
| **方向约束** | 有 | 无 | 无 | 无 |
| **中间状态** | 可解释 | 连续 | 连续 | 无 |
| **概率建模** | 强 | 中等 | 中等 | 弱 |
| **路径平滑度** | 高 | 中等 | 中等 | - |
| **生物学可解释性** | 高 | 中等 | 中等 | 高 |

## 4. 输入/输出

### 输入

| 输入类型 | 描述 |
|----------|------|
| **对照细胞状态** | 未受扰动的单细胞基因表达谱（scRNA-seq） |
| **扰动信息** | 扰动类型（药物、基因敲除/过表达等） |
| **条件编码** | 扰动条件的向量表示 |

### 输出

| 输出类型 | 描述 |
|----------|------|
| **扰动后细胞状态** | 预测的扰动后单细胞基因表达谱 |
| **中间路径** | 从对照状态到扰动状态的生物学可解释中间状态序列 |
| **方向向量** | 描述细胞状态演化的方向约束信息 |

## 5. 网络架构

### 5.1 整体框架

DC-DSB 基于扩散薛定谔桥（Diffusion Schrödinger Bridge, DSB）框架，引入方向约束机制：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DC-DSB 架构概览                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  对照状态 x_0                                                      │
│       ↓                                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              前向扩散过程 (Forward Process)                  │   │
│  │  ├── 从对照状态向噪声状态添加噪声                            │   │
│  │  └── 时间步: t = 0 → T                                       │   │
│  └─────────────────────────────────────────────────────────────┘   │
│       ↓                                                             │
│  噪声状态 x_T                                                      │
│       ↓                                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              反向去噪过程 (Reverse Process)                  │   │
│  │  ├── 得分网络估计数据分布梯度                                │   │
│  │  ├── 方向约束模块: 确保演化符合生物学方向                    │   │
│  │  │   ┌─────────────────────────────────────────┐            │   │
│  │  │   │ 方向约束机制:                           │            │   │
│  │  │   │ - 生物学方向编码                        │            │   │
│  │  │   │ - 路径正则化                            │            │   │
│  │  │   │ - 中间状态可解释性                      │            │   │
│  │  │   └─────────────────────────────────────────┘            │   │
│  │  └── 时间步: t = T → 0                                     │   │
│  └─────────────────────────────────────────────────────────────┘   │
│       ↓                                                             │
│  扰动状态 x_1                                                      │
│                                                                     │
│  输出: 扰动后预测 + 中间状态序列 + 方向向量                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 5.2 核心组件

| 组件 | 功能描述 |
|------|----------|
| **前向过程** | 从对照状态向扰动状态添加噪声 |
| **反向过程** | 从噪声状态恢复扰动状态 |
| **方向约束模块** | 确保演化路径符合生物学方向 |
| **得分网络** | 估计数据分布的梯度 |

### 5.3 方向约束机制

- **生物学方向编码**：将已知的生物学先验知识编码为方向约束
- **路径正则化**：在训练过程中约束中间状态的演化方向
- **可解释性增强**：确保生成的中间状态具有生物学意义

## 6. 训练数据

### 6.1 数据来源

| 数据类型 | 描述 |
|----------|------|
| **scRNA-seq数据** | 单细胞RNA测序数据，包含对照和扰动样本 |
| **扰动-响应对** | 配对的扰动前后细胞状态 |
| **多批次数据** | 不同实验条件下的数据整合 |

### 6.2 数据预处理

- **质量控制**：过滤低质量细胞和基因
- **归一化**：对基因表达进行标准化处理
- **批次校正**：消除批次效应
- **降维**：使用PCA或变分自编码器进行特征提取

### 6.3 训练集构建

- **配对样本**：对照-扰动细胞对
- **条件标签**：扰动类型和剂量信息
- **验证策略**：按细胞类型或扰动类型划分验证集

## 7. 训练策略

### 7.1 损失函数组成

| 损失项 | 描述 |
|--------|------|
| **DSB损失** | 薛定谔桥的边际约束损失 |
| **方向约束损失** | 确保演化方向符合生物学先验 |
| **重构损失** | 保证输入输出的一致性 |
| **正则化项** | 防止过拟合 |

### 7.2 训练流程

```
1. 初始化前向和反向扩散过程
2. 编码方向约束信息
3. 迭代优化：
   - 采样时间步 t
   - 计算前向扩散状态
   - 估计得分函数
   - 应用方向约束
   - 反向传播更新参数
4. 验证和早停
```

### 7.3 超参数设置

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 扩散时间步数 T | 100-1000 | 离散化步数 |
| 学习率 | 1e-4 ~ 1e-3 | Adam优化器 |
| 批次大小 | 256-1024 | 根据显存调整 |
| 方向约束权重 | 0.1-1.0 | 平衡约束强度 |

## 12. 优势与局限

### 12.1 优势

| 优势 | 说明 |
|------|------|
| **生物学可解释性** | 方向约束机制提供可理解的演化路径 |
| **机制一致性** | 确保预测符合已知的生物学原理 |
| **概率建模能力** | 薛定谔桥框架天然支持不确定性量化 |
| **端到端学习** | 直接从原始数据学习，无需手工特征工程 |

### 12.2 局限

| 局限 | 说明 |
|------|------|
| **数据需求** | 需要大量配对的扰动-响应对进行训练 |
| **计算成本** | 扩散模型推理需要多步采样，计算开销较大 |
| **方向约束设计** | 需要领域知识设计合适的方向约束 |
| **泛化能力** | 对未见过的扰动类型泛化能力有限 |

### 12.3 改进方向

- 结合自监督学习减少对配对数据的依赖
- 开发快速采样算法降低推理成本
- 引入更灵活的约束机制适应不同生物学场景
- 探索跨物种/跨细胞类型的迁移学习
