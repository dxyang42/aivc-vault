# scDCA

> Efficient Fine-Tuning of Single-Cell Foundation Models Enables Zero-Shot Molecular Perturbation Prediction

## 核心思想

scDCA通过**药物条件适配器微调scGPT**，实现了对未见分子扰动的零样本预测。该方法在保持基础模型通用能力的同时，注入药物特异性知识，使模型能够预测新药物的细胞效应。

### 关键创新

1. **药物条件适配器**: 为每种药物学习条件化表示
2. **scGPT微调**: 在大型预训练模型上高效适配
3. **零样本泛化**: 预测训练时未见过的药物效应
4. **参数高效**: 使用LoRA等技术减少可训练参数

## 药物条件化机制

### 药物嵌入学习

```
药物表示学习:
├── 分子结构编码 (Morgan指纹, SMILES)
├── 靶点信息编码 (作用靶点基因集合)
├── 机制类别编码 (通路分类)
└── 端到端学习 (直接从数据学习)

条件融合方式:
├── 早期融合: 药物嵌入作为额外输入token
├── 中期融合: 在Transformer层中交叉注意力
└── 晚期融合: 在输出层结合药物信息
```

### 零样本预测策略

1. **相似性迁移**: 基于分子/靶点相似性组合已知药物效应
2. **元学习**: 学习"如何学习药物效应"的元知识
3. **结构化预测**: 利用药物-靶点-通路关系结构化预测

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| scGPT权重 | 预训练的单细胞基础模型 |
| 训练数据 | 已知药物的扰动-响应对 |
| 药物信息 | 分子结构、靶点、机制等 |
| 新药物 | 需要预测的未见药物 |

### 输出
| 类型 | 描述 |
|------|------|
| 微调模型 | 适配后的scDCA模型 |
| 零样本预测 | 新药物的预测细胞效应 |
| 相似性矩阵 | 药物间相似性评分 |

## 优势与局限

### 优势
- 零样本预测未见药物，减少实验成本
- 参数高效微调，计算成本低
- 保留scGPT的通用单细胞理解能力
- 可解释的药物相似性分析

### 局限
- 依赖scGPT的基础能力
- 对机制全新药物预测可能不准
- 需要药物的先验信息(结构/靶点)
- 药物间复杂相互作用难以建模

## 应用场景

1. **药物重定位**: 预测老药新用
2. **毒性预测**: 早期评估药物安全性
3. **联合用药**: 预测药物组合效应
4. **个性化用药**: 基于细胞类型预测响应
