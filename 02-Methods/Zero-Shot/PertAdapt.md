## 核心思想

PertAdapt提出了一种**即插即用(Plug-and-Play)的扰动适配器**，使单细胞基础模型能够在有限数据下实现遗传扰动预测。通过自适应损失函数设计，该方法实现了在少样本甚至零样本场景下的有效泛化。

### 关键创新

1. **轻量适配器**: 冻结预训练基础模型，只训练小型适配器模块
2. **自适应损失**: 根据数据量动态调整损失函数权重
3. **即插即用**: 可应用于任何单细胞基础模型(scGPT, Geneformer等)
4. **数据高效**: 在极少标注数据下实现有效学习

## 自适应损失函数

### 动态权重策略

```
数据量评估:
├── N < 100 (极少数据)
│   └── 权重: 重建=0.3, 正则化=0.5, 先验=0.2
│
├── 100 <= N < 1000 (少量数据)
│   └── 权重: 重建=0.5, 正则化=0.3, 对比=0.2
│
└── N >= 1000 (充足数据)
    └── 权重: 重建=0.7, 对比=0.2, 正则化=0.1
```

### 损失组成

1. **重建损失**: 均方误差衡量预测精度
2. **对比损失**: 拉近相同扰动样本，推远不同扰动样本
3. **先验约束**: 利用基础模型的先验知识约束预测
4. **稀疏正则化**: L2正则化防止适配器过拟合

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 预训练模型 | 单细胞基础模型权重 |
| 训练数据 | 少量配对的控制-扰动样本 |
| 扰动编码 | 扰动基因的one-hot或嵌入表示 |
| 目标数据 | 需要预测的细胞和扰动组合 |

### 输出
| 类型 | 描述 |
|------|------|
| 适配器权重 | 训练好的轻量适配器参数 |
| 预测表达 | 目标扰动后的基因表达 |
| 不确定性 | 预测置信度估计 |

## 优势与局限

### 优势
- 极少量数据即可微调 (少样本学习)
- 计算成本低，只训练少量参数
- 通用性强，适配多种基础模型
- 防止灾难性遗忘，保留预训练知识

### 局限
- 依赖基础模型的质量
- 对未见扰动的泛化仍有限
- 适配器容量限制表达能力
- 需要仔细选择学习率

## 应用场景

1. **新细胞类型快速适配**: 已有模型迁移到新细胞类型
2. **罕见疾病研究**: 数据稀缺情况下的扰动预测
3. **实验设计优化**: 指导有限实验预算的分配
4. **基础模型增强**: 为通用模型添加扰动预测能力
