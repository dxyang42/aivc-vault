## 核心技术

### 问题背景

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Systema 要解决的问题                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   当前扰动预测方法评估的问题                                             │
│                                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │  系统性变异 (Systematic Variation)                               │  │
│   │                                                                 │  │
│   │  定义: 扰动组与对照组之间的一致转录差异                           │  │
│   │        (由选择偏差或混杂因素引起)                                 │  │
│   │                                                                 │  │
│   │  来源:                                                          │  │
│   │  ├── CRISPR筛选中的选择压力                                      │  │
│   │  ├── 细胞培养条件的系统性差异                                    │  │
│   │  ├── 批次效应                                                    │  │
│   │  └── 技术平台差异                                                │  │
│   │                                                                 │  │
│   │  影响: 模型可以"作弊" - 只学习系统性变异就能在                   │  │
│   │        标准指标上获得高分，而不真正学习扰动特异性效应             │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │  现有评估指标的缺陷                                              │  │
│   │                                                                 │  │
│   │  常用指标 (MSE, Pearson, Spearman) 对系统性变异敏感              │  │
│   │  → 导致性能被高估                                                │  │
│   │  → 无法区分"真正学习" vs "记忆偏差"                              │  │
│   │                                                                 │  │
│   │  示例: 预测所有扰动都有相同的"平均扰动效应"                      │  │
│   │        可能获得合理的MSE分数                                     │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 架构设计

Systema采用**系统性变异分解**的评估框架：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    Systema 评估框架架构                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   步骤1: 系统性变异量化                                                  │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │                                                                 │  │
│   │  在10个数据集上量化系统性变异 (跨越3种技术，5种细胞系)            │  │
│   │                                                                 │  │
│   │  计算方法:                                                      │  │
│   │  ├── 对照组 vs 扰动组的整体分布差异                              │  │
│   │  ├── 跨扰动的一致性响应模式                                      │  │
│   │  └── 主成分分析识别主导变异方向                                  │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              ↓                                          │
│   步骤2: 扰动特异性效应提取                                              │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │                                                                 │  │
│   │  数学分解:                                                      │  │
│   │                                                                 │  │
│   │  观测响应 = 系统性变异 + 扰动特异性效应 + 噪声                   │  │
│   │                                                                 │  │
│   │  目标: 隔离并评估扰动特异性部分                                  │  │
│   │                                                                 │  │
│   │  方法:                                                          │  │
│   │  ├── 去均值化 (减去平均扰动效应)                                 │  │
│   │  ├── 残差分析                                                   │  │
│   │  └── 扰动特异性基因集关注                                       │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              ↓                                          │
│   步骤3: 纠偏评估指标                                                    │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │                                                                 │  │
│   │  Systema指标:                                                   │  │
│   │                                                                 │  │
│   │  1. 扰动景观重建能力                                             │  │
│   │     - 评估预测是否正确重建扰动间的相对关系                        │  │
│   │     - 使用秩相关而非绝对值相关                                   │  │
│   │                                                                 │  │
│   │  2. 扰动特异性预测精度                                           │  │
│   │     - 在去除系统性变异后的预测准确性                             │  │
│   │     - 交叉验证: 留一扰动-out                                     │  │
│   │                                                                 │  │
│   │  3. 泛化能力评估                                                 │  │
│   │     - 跨细胞类型                                                 │  │
│   │     - 跨技术平台                                                 │  │
│   │     - 跨基因家族                                                 │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              ↓                                          │
│   步骤4: 方法比较与排名                                                  │
│   ├── 使用纠偏指标重新评估现有方法                                      │
│   └── 提供公平的性能排名                                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 关键发现

### 数据集分析结果

| 数据集特征 | 覆盖范围 |
|-----------|---------|
| **数据集数量** | 10个 |
| **技术平台** | 3种 (Perturb-seq, CROP-seq, ECCITE-seq) |
| **细胞系** | 5种 (K562, RPE1, A549, Jurkat, THP1) |
| **扰动类型** | CRISPR敲除、过表达 |

### 主要发现

1. **系统性变异普遍存在**: 在所有数据集中都观察到显著的系统性变异

2. **性能高估现象**: 常用指标(MSE, Pearson)可能高估模型性能达30-50%

3. **方法排名变化**: 使用Systema指标后，方法排名发生显著变化

4. **简单基线问题**: 一些简单基线(如平均扰动效应)在标准指标上表现意外好

## 技术优势

### 评估框架对比

| 特性 | Systema | 传统评估 | PerturBench | CausalBench |
|------|---------|---------|-------------|-------------|
| **系统性变异处理** | 显式建模 | 忽略 | 部分考虑 | 部分考虑 |
| **扰动特异性关注** | 是 | 否 | 是 | 是 |
| **跨数据集验证** | 支持 | 有限 | 支持 | 支持 |
| **生物学可解释性** | 高 | 中 | 中 | 高 |

### 核心创新

1. **系统性变异形式化**: 首次形式化定义并量化扰动数据中的系统性变异

2. **效应分解框架**: 将观测响应分解为系统性变异和扰动特异性效应

3. **纠偏指标**: 设计对系统性变异鲁棒的评估指标

4. **大规模实证**: 在10个真实数据集上验证框架有效性

## 对领域的影响

### 实践建议

1. **评估指标选择**: 建议使用Systema提供的纠偏指标

2. **基线比较**: 必须包含简单基线(如平均扰动效应)作为对照

3. **交叉验证策略**: 采用留一扰动-out验证而非随机分割

4. **数据报告**: 报告系统性变异的程度

### 对方法开发的启示

- 需要开发显式建模或去除系统性变异的方法
- 关注扰动特异性效应的学习
- 提高跨扰动的泛化能力
