## 数据资源

### 数据集统计

| 特征 | 数量 |
|------|------|
| **数据集总数** | 44个 |
| **转录组数据** | 主要 |
| **蛋白质组数据** | 包含 |
| **表观遗传数据** | 包含 |
| **扰动类型** | 基因KO/OE、药物处理 |

### 数据特点

- **统一质量控制**: 所有数据集经过统一QC流程
- **特征注释统一**: 基因ID、通路注释等标准化
- **元数据丰富**: 实验条件、细胞类型、技术平台等信息完整

## E-statistics 技术优势

### 与传统方法对比

| 特性 | E-statistics | t-SNE/UMAP | Differential Expression |
|------|-------------|------------|------------------------|
| **分布整体比较** | ✓ | 可视化 | 基因级 |
| **统计显著性** | ✓ | ✗ | ✓ |
| **计算效率** | 高 | 中 | 中 |
| **高维数据** | 原生支持 | 需降维 | 需多重检验校正 |
| **可解释性** | 高 | 中 | 高 |

### 核心创新

1. **E-distance**: 作为单细胞表达谱集合间的通用距离度量

2. **扰动效应量化**: 无需预设差异表达基因，整体评估扰动效应

3. **显著性检验**: 基于置换检验的统计显著性评估

4. **多模态适用**: 适用于转录组、蛋白质组、表观遗传数据

## 应用场景

### 1. 方法开发

- **基准测试**: 使用统一数据集评估新方法
- **数据探索**: 快速了解不同扰动的效应大小
- **特征工程**: 基于E-distance构建特征

### 2. 生物学发现

- **扰动聚类**: 基于效应相似性聚类扰动
- **剂量分析**: 量化剂量-效应关系
- **通路分析**: 识别功能相关的扰动组

### 3. 实验设计

- **效应大小预估**: 预测所需细胞数
- **扰动选择**: 选择信息量大的扰动组合
