## 核心思想

LPM 是一种利用**大型语言模型（LLM）**进行单细胞扰动预测的方法。其核心创新在于将基因表达数据和扰动信息**文本化**，利用语言模型的强大表示能力进行扰动效应预测。

核心概念：
1. **文本化基因表达**: 将数值型基因表达转换为文本格式
2. **序列到序列学习**: 使用 seq2seq 范式学习扰动映射
3. **语言模型表示**: 利用 LLM 的预训练知识理解生物学关系

## 算法流程

1. **文本化**: 将基因表达向量转换为文本描述
2. **分词**: 使用 LLM 的 tokenizer 将文本转为 token IDs
3. **编码**: LLM 处理输入序列，生成上下文表示
4. **解码**: 自回归生成预测的扰动后表达文本
5. **解析**: 将生成的文本转回数值型表达向量

## Perturbation 预测能力

### 核心功能

| 功能 | 描述 |
|------|------|
| **文本化预测** | 将扰动预测作为文本生成任务 |
| **知识迁移** | 利用 LLM 预训练的生物学知识 |
| **可解释性** | 文本输出具有更好的可解释性 |
| **多模态** | 可整合文本描述的扰动信息 |

### 适用场景

- **扰动效应预测**: 预测基因敲除/过表达效应
- **药物机制解释**: 利用文本描述解释药物作用
- **基因功能注释**: 结合文本知识进行功能预测
- **假设生成**: 利用 LLM 生成生物学假设

## 与其他方法的对比

| 方法 | 主要区别 | 适用场景 |

## 优势与局限

### 优势
- 利用 LLM 的预训练知识
- 文本输出具有可解释性
- 可整合多种文本信息
- 无需专门训练即可使用（零样本）

### 局限
- 信息可能丢失（数值转文本）
- 依赖 LLM 的文本理解能力
- 对未见基因的泛化有限
- 难以捕捉复杂的基因-基因交互

## 典型应用

1. **扰动效应预测**: 预测基因扰动后的转录响应
2. **基因功能注释**: 结合文本知识预测基因功能
3. **药物机制研究**: 解释药物作用的分子机制
4. **假设生成**: 生成可验证的生物学假设
