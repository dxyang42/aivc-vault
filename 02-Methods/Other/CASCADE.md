# CASCADE

## 基本信息

- **全称**: CASCADE: Causal Discovery for single-cell perturbation analysis
- **发表时间**: 2024年
- **发表期刊**: Nature Communications
- **机构**: 芝加哥大学 Gao Lab
- **作者**: Jingtao Wang, Ge Gao, et al.

## 核心技术

CASCADE 是一个基于深度学习的因果发现工具，专为单细胞扰动数据设计：

1. **因果发现**: 从单细胞扰动数据中推断基因调控网络
2. **反事实预测**: 预测未观察到的扰动效应
3. **干预设计**: 推荐最优干预策略
4. **高内容筛选**: 支持大规模扰动筛选数据

### 因果发现算法

```
┌─────────────────────────────────────────────────────────────┐
│              Causal Discovery Algorithm                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   输入: 单细胞扰动数据                                      │
│         D = {(x_i, p_i)}_{i=1}^n                         │
│         x_i: 基因表达向量                                   │
│         p_i: 扰动目标 (哪个基因被敲除/过表达)               │
│                                                             │
│   算法流程:                                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   步骤1: 初始化完全图                               │  │
│   │   G = 完全无向图                                    │  │
│   │                                                     │  │
│   │   步骤2: 骨架学习 (Skeleton Learning)               │  │
│   │   for 每对基因 (i, j):                              │  │
│   │       for 每个条件集 S:                             │  │
│   │           if 条件独立检验(X_i ⊥ X_j | S, P):        │  │
│   │               移除边 i-j                            │  │
│   │               记录分离集                            │  │
│   │                                                     │  │
│   │   步骤3: 方向确定 (Orientation)                     │  │
│   │   for 每个 v-结构 i-k-j:                            │  │
│   │       if k 不在分离集中:                            │  │
│   │           定向为 i→k←j                              │  │
│   │                                                     │  │
│   │   步骤4: 方向传播                                   │  │
│   │   应用Meeks规则传播方向约束                         │  │
│   │                                                     │  │
│   │   步骤5: 深度学习优化                               │  │
│   │   使用VAE进一步精调因果强度                         │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   条件独立性检验:                                           │
│   - 利用扰动作为自然干预                                    │
│   - 基于核的条件独立性检验                                  │
│   - 考虑单细胞数据的高维特性                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 训练数据

| 数据属性 | 规模 |
|---------|------|
| 扰动实验 | 1000+ 基因 |
| 细胞数量 | 数百万 |
| 数据来源 | Perturb-seq, CROP-seq |
| 扰动类型 | CRISPR KO, CRISPRa, CRISPRi |

## 核心能力

| 能力 | 描述 |
|------|------|
| **因果网络推断** | 从扰动数据学习有向调控网络 |
| **反事实预测** | 预测未观察扰动的效应 |
| **干预设计** | 推荐最优干预组合 |
| **机制解析** | 理解扰动传播机制 |

## 技术特点

1. **扰动感知**: 专门设计用于扰动数据
2. **可扩展**: 支持大规模基因筛选
3. **因果可解释**: 输出有向因果图
4. **不确定性量化**: 提供因果关系的置信度

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 表达矩阵 | 扰动后单细胞表达 |
| 扰动标签 | 每个细胞的扰动目标 |
| 对照数据 | 未扰动细胞作为对照 |

### 输出
| 类型 | 描述 |
|------|------|
| 因果网络 | 基因间有向调控边 |
| 因果效应 | 每条边的因果效应大小 |
| 反事实预测 | 新扰动的预测效应 |
| 干预推荐 | 最优干预策略 |

## 应用场景

1. **药物靶点发现**
   - 识别关键调控基因
   - 预测药物组合效应

2. **基因功能注释**
   - 推断基因调控关系
   - 发现新通路

3. **合成致死筛选**
   - 预测基因对组合效应
   - 识别合成致死对

4. **重编程优化**
   - 设计最优重编程因子
   - 预测重编程轨迹

## 优势与局限

### 优势
- 专为扰动数据设计的因果发现
- 支持反事实预测
- 可解释的有向网络输出
- 大规模数据支持

### 局限
- 需要足够的扰动覆盖
- 因果方向依赖扰动设计
- 计算复杂度较高
