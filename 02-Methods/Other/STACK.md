## 核心技术

### 架构设计
STACK是一种基于上下文学习的单细胞基础模型，采用新颖的表格注意力架构：

```
┌─────────────────────────────────────────────────────────────────┐
│                    STACK 技术架构                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   1. 输入编码层                                                   │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 基因表达向量编码                                       │   │
│   │  • 细胞元数据编码 (批次、条件、供体等)                     │   │
│   │  • 扰动条件编码                                           │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ↓                                       │
│   2. 表格注意力层 (核心创新)                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 细胞内注意力 (Intra-cellular attention)               │   │
│   │    - 基因-基因相互作用建模                               │   │
│   │                                                         │   │
│   │  • 细胞间注意力 (Inter-cellular attention)               │   │
│   │    - 细胞-细胞关系建模                                   │   │
│   │                                                         │   │
│   │  • 表格结构处理                                          │   │
│   │    - 处理异质单细胞数据                                  │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ↓                                       │
│   3. 上下文学习层                                                 │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 支持示例学习 (In-context learning)                    │   │
│   │  • 零样本预测 (Zero-shot prediction)                     │   │
│   │  • 条件生成                                              │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ↓                                       │
│   4. 输出解码层                                                   │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 基因表达预测                                          │   │
│   │  • 细胞状态预测                                          │   │
│   │  • 扰动效应预测                                          │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 技术特点
- **表格注意力架构**: 新颖的注意力机制，同时建模细胞内和细胞间关系
- **大规模预训练**: 在1.5亿个统一预处理的单细胞上训练
- **上下文学习**: 无需微调即可从支持示例学习
- **零样本预测**: 可以从未标记的参考细胞进行预测
- **编码器-解码器结构**: 灵活处理各种单细胞任务

## 输入/输出

### 输入格式
| 输入类型 | 描述 | 维度 |
|---------|------|------|
| **目标细胞** | 需要预测的细胞基因表达 | 20,000+ 基因 |
| **支持细胞** | 参考/示例细胞 (上下文) | N × 20,000+ 基因 |
| **条件标签** | 批次、供体、扰动等元数据 | 可变 |

### 输出格式
| 输出类型 | 描述 | 维度 |
|---------|------|------|
| **预测表达** | 目标条件下的基因表达 | 20,000+ 基因 |
| **细胞嵌入** | 低维细胞表示 | 模型依赖 |
| **注意力权重** | 可解释的注意力图 | 细胞 × 基因 |

## 核心能力

### 1. 上下文学习 (In-context Learning)
```
支持示例 (Reference cells)          目标细胞 (Target cell)
┌─────────────────────┐            ┌─────────────────────┐
│ 细胞A: 条件X         │            │ 细胞B: 条件?         │
│ 基因表达向量         │    ───→    │ 预测基因表达         │
└─────────────────────┘   学习     └─────────────────────┘
         ↓                              ↓
   学习条件X的效应                  应用学到的映射
```

### 2. 零样本预测 (Zero-shot Prediction)
- 无需针对特定任务微调
- 从参考细胞直接预测目标条件
- 支持未见过的细胞类型和条件

### 3. Perturb Sapiens
STACK 生成了首个AI预测的人类全生物体扰动细胞图谱：
- 预测各种化学扰动对细胞的影响
- 预测不同供体背景下的细胞响应
- 覆盖多种细胞类型和组织

## 功能特点

- ✅ **上下文学习**: 无需微调，从示例学习
- ✅ **零样本预测**: 对未见条件进行预测
- ✅ **扰动效应预测**: 预测化学/遗传扰动效应
- ✅ **大规模预训练**: 1.5亿细胞训练数据
- ✅ **表格注意力**: 新颖的架构设计
- ✅ **开源可用**: GitHub + PyPI + Hugging Face
- ⚠️ **预印本状态**: 尚未经过同行评审

## 应用场景

1. **虚拟细胞模拟**
   - 预测细胞对各种扰动的响应
   - 模拟不同供体背景下的细胞行为

2. **药物发现**
   - 预测药物对细胞的影响
   - 筛选潜在药物靶点

3. **疾病建模**
   - 模拟疾病条件下的细胞状态
   - 预测治疗响应

4. **Perturb Sapiens 图谱**
   - 探索人类全生物体扰动效应
   - 生成假设和实验设计
