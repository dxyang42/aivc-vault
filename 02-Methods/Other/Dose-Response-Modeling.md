# Dose-Response Modeling (剂量响应建模)

> [[方法地图]] | [[团队地图]]

## 1. 基本信息

| 属性 | 详情 |
|------|------|
| **模型名称** | Dose-Response Modeling (剂量响应建模) |
| **发布时间** | 经典方法，持续演进 |
| **应用领域** | 药理学、毒理学、细胞生物学 |
| **数学基础** | Hill方程、Logistic模型 |
| **类型** | 药理学建模方法 |
| **定位** | 扰动效应定量化的标准方法 |

剂量响应建模是药理学和细胞生物学中的核心方法，用于描述扰动强度（如药物浓度、sgRNA效率）与生物响应之间的关系。该方法为单细胞扰动预测提供了重要的定量框架。

## 2. 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **药物浓度** | 核心功能 | 剂量-响应曲线拟合 |
| **sgRNA效率** | 支持 | 不同效率的扰动建模 |
| **时间梯度** | 支持 | 时程响应建模 |
| **组合剂量** | 支持 | 多药物组合剂量效应 |
| **阈值效应** | 支持 | 识别有效浓度阈值 |

### 核心数学模型

```
┌─────────────────────────────────────────────────────────────────────┐
│              Hill 方程 (剂量响应建模核心)                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   基本形式:                                                         │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │                    Emax * C^n                               │  │
│   │   E = E0 + ─────────────────────                            │  │
│   │            EC50^n + C^n                                     │  │
│   │                                                             │  │
│   │   参数:                                                     │  │
│   │   - E0: 基线响应 (无药物时)                                 │  │
│   │   - Emax: 最大效应                                          │  │
│   │   - EC50: 半最大效应浓度                                    │  │
│   │   - n: Hill系数 (斜率因子)                                  │  │
│   │   - C: 药物浓度                                             │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   四参数Logistic模型 (更一般形式):                                  │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │                    Top - Bottom                             │  │
│   │   Y = Bottom + ─────────────────────                        │  │
│   │                1 + 10^((LogEC50 - X) * HillSlope)           │  │
│   │                                                             │  │
│   │   参数:                                                     │  │
│   │   - Bottom: 最小响应 (基线)                                 │  │
│   │   - Top: 最大响应                                           │  │
│   │   - LogEC50: 半最大效应浓度的对数                           │  │
│   │   - HillSlope: 斜率参数                                     │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   简化形式 (抑制/激活):                                             │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   抑制:  Response = 100 / (1 + (C/IC50)^n)                  │  │
│   │                                                             │  │
│   │   激活:  Response = 100 * C^n / (EC50^n + C^n)              │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 关键参数解释

| 参数 | 符号 | 生物学意义 |
|------|------|-----------|
| **EC50** | 半最大效应浓度 | 药物效力指标，越小越有效 |
| **IC50** | 半抑制浓度 | 抑制50%效应的浓度 |
| **Emax** | 最大效应 | 药物的最大生物活性 |
| **Hill系数** | n | 协同性指标，>1正协同，<1负协同 |
| **AUC** | 曲线下面积 | 综合药效指标 |

## 3. 建模流程

### 整体架构

```
┌─────────────────────────────────────────────────────────────────────┐
│              剂量响应建模流程                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   阶段1: 数据准备                                                   │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   输入数据:                                                 │  │
│   │   - 不同浓度的扰动实验数据                                  │  │
│   │   - 对应的生物响应测量值                                    │  │
│   │   - 技术重复和生物学重复                                    │  │
│   │                                                             │  │
│   │   数据格式示例:                                             │  │
│   │   ┌──────────┬────────────────┐                             │  │
│   │   │ 浓度 (M) │ 响应 (%Control)│                             │  │
│   │   ├──────────┼────────────────┤                             │  │
│   │   │ 1e-10    │ 98.5           │                             │  │
│   │   │ 1e-9     │ 95.2           │                             │  │
│   │   │ 1e-8     │ 78.3           │                             │  │
│   │   │ 1e-7     │ 45.6           │                             │  │
│   │   │ 1e-6     │ 23.1           │                             │  │
│   │   │ 1e-5     │ 12.4           │                             │  │
│   │   └──────────┴────────────────┘                             │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   阶段2: 曲线拟合                                                   │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   拟合方法:                                                 │  │
│   │   - 非线性最小二乘法 (Levenberg-Marquardt)                  │  │
│   │   - 最大似然估计                                            │  │
│   │   - 贝叶斯推断                                              │  │
│   │                                                             │  │
│   │   约束条件:                                                 │  │
│   │   - EC50/IC50 > 0                                           │  │
│   │   - Hill系数通常在 0.5-4 之间                               │  │
│   │   - 响应值在生物学合理范围内                                │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   阶段3: 模型评估                                                   │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   评估指标:                                                 │  │
│   │   - R²: 拟合优度                                            │  │
│   │   - RMSE: 均方根误差                                        │  │
│   │   - 参数置信区间                                            │  │
│   │   - 残差分析                                                │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   阶段4: 参数解释与应用                                             │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   应用:                                                     │  │
│   │   - 比较不同药物的效力 (EC50)                               │  │
│   │   - 评估协同效应 (Hill系数)                                 │  │
│   │   - 确定有效剂量范围                                        │  │
│   │   - 预测未测试剂量的响应                                    │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 模型变体

| 模型 | 适用场景 | 特点 |
|------|---------|------|
| **Hill方程** | 标准剂量响应 | 最常用，参数少 |
| **四参数Logistic** | S型曲线 | 更灵活，拟合更好 |
| **五参数Logistic** | 非对称曲线 | 允许不对称性 |
| **Gompertz** | 生长曲线 | 用于细胞增殖 |
| **Weibull** | 毒性评估 | 用于毒理学 |

## 4. 组合剂量响应

### 协同效应评估

```
┌─────────────────────────────────────────────────────────────────────┐
│              组合剂量响应模型                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   参考模型 (期望效应):                                              │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   1. Bliss 独立模型:                                        │  │
│   │      E_expected = E_A + E_B - E_A * E_B                     │  │
│   │      (假设两个药物独立作用)                                 │  │
│   │                                                             │  │
│   │   2. Loewe 加性模型:                                        │  │
│   │      C_A/EC50_A + C_B/EC50_B = 1                            │  │
│   │      (假设共同靶点)                                         │  │
│   │                                                             │  │
│   │   3. Highest Single Agent (HSA):                            │  │
│   │      E_expected = max(E_A, E_B)                             │  │
│   │      (假设取最强单药效应)                                   │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   协同评分:                                                         │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   协同指数 (Combination Index, CI):                         │  │
│   │   CI = (C_A/EC50_A) + (C_B/EC50_B)                          │  │
│   │                                                             │  │
│   │   CI < 1: 协同 (Synergistic)                                │  │
│   │   CI = 1: 加性 (Additive)                                   │  │
│   │   CI > 1: 拮抗 (Antagonistic)                               │  │
│   │                                                             │  │
│   │   超额抑制 (Excess over Bliss):                             │  │
│   │   Excess = E_observed - E_Bliss_expected                    │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 5. 在单细胞扰动预测中的应用

### 整合到深度学习方法

| 方法 | 剂量建模方式 |
|------|-------------|
| **CPA** | 剂量作为连续条件嵌入 |
| **X-Pert** | 剂量感知模块显式建模 |
| **scPPDM** | 通过条件扩散建模 |
| **DrugCell** | 多剂量水平训练 |

### 单细胞特异性考量

```
┌─────────────────────────────────────────────────────────────────────┐
│              单细胞剂量响应建模特点                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   挑战:                                                             │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   1. 细胞异质性                                             │  │
│   │      - 不同细胞对相同剂量响应不同                           │  │
│   │      - 需要拟合细胞群体响应分布                             │  │
│   │                                                             │  │
│   │   2. 稀疏数据                                               │  │
│   │      - 每个剂量点的细胞数有限                               │  │
│   │      - 需要聚合或建模不确定性                               │  │
│   │                                                             │  │
│   │   3. 技术噪声                                               │  │
│   │      - Dropout和捕获效率影响                                │  │
│   │      - 需要校正或鲁棒拟合                                   │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   解决方案:                                                         │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   1. 群体水平拟合                                           │  │
│   │      - 按细胞类型或状态分组                                 │  │
│   │      - 拟合每组的平均响应曲线                               │  │
│   │                                                             │  │
│   │   2. 分层模型                                               │  │
│   │      - 混合效应模型                                         │  │
│   │      - 同时建模群体和个体效应                               │  │
│   │                                                             │  │
│   │   3. 深度学习方法                                           │  │
│   │      - 神经网络学习剂量-响应映射                            │  │
│   │      - 隐式建模异质性                                       │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
```

## 6. 功能特点

### 核心能力

| 功能 | 描述 |
|------|------|
| **效力排序** | 比较不同扰动的效力 (EC50) |
| **剂量选择** | 确定最佳实验剂量 |
| **效应预测** | 预测未测试剂量的响应 |
| **协同评估** | 定量评估组合效应 |
| **安全评估** | 确定治疗窗口 |

### 技术特色

- **标准化**: 药理学标准方法，结果可比较
- **可解释**: 参数具有明确的生物学意义
- **灵活**: 多种模型适应不同数据特征
- **广泛适用**: 适用于药物、基因扰动等多种场景
