## 核心技术

### 架构设计

scVelo基于**RNA速度(RNA velocity)**概念，利用剪接动力学推断细胞状态变化方向：

```
┌─────────────────────────────────────────────────────────────────────┐
│                  scVelo RNA速度分析架构                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  核心生物学概念:                                                    │
│  未剪接mRNA (unspliced) ──→ 剪接mRNA (spliced)                     │
│         ↑                                              ↓            │
│    转录中                                          翻译/降解        │
│                                                                     │
│  速度 = 转录速率 - 降解速率                                          │
│       = du/dt = alpha - beta * u                                    │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              步骤1: 数据预处理                               │   │
│  │                                                             │   │
│  │  输入: scRNA-seq (10x Genomics等)                           │   │
│  │       ├── 剪接计数矩阵 (spliced counts)                     │   │
│  │       └── 未剪接计数矩阵 (unspliced counts)                 │   │
│  │                                                             │   │
│  │  预处理:                                                    │   │
│  │  - 质量控制 (QC)                                            │   │
│  │  - 归一化                                                   │   │
│  │  - 高变基因选择                                             │   │
│  │  - 邻居图构建 (kNN)                                         │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              步骤2: 速度估计模式                             │   │
│  │                                                             │   │
│  │  模式1: 稳态模型 (Steady-State)                             │   │
│  │  ├── 假设: 系统处于稳态                                      │   │
│  │  ├── 速度 = 未剪接 - 稳态未剪接                             │   │
│  │  └── 适用: 快速变化的系统                                   │   │
│  │                                                             │   │
│  │  模式2: 动态模型 (Dynamical) ★推荐                          │   │
│  │  ├── 求解微分方程: du/dt = alpha - beta * u                │   │
│  │  ├── 求解微分方程: ds/dt = beta * u - gamma * s            │   │
│  │  ├── 估计参数: alpha(转录), beta(剪接), gamma(降解)        │   │
│  │  └── 适用: 复杂发育过程                                     │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              步骤3: 速度向量计算                             │   │
│  │                                                             │   │
│  │  对每个细胞:                                                │   │
│  │  - 计算基因层面的速度                                       │   │
│  │  - 投影到低维嵌入 (UMAP/t-SNE)                              │   │
│  │  - 平滑处理 (基于邻居图)                                    │   │
│  │                                                             │   │
│  │  输出: 速度向量场 (velocity vector field)                   │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              步骤4: 轨迹推断                                 │   │
│  │                                                             │   │
│  │  - 识别根细胞 (root cells)                                  │   │
│  │  - 构建马尔可夫链转移矩阵                                   │   │
│  │  - 计算细胞命运概率                                         │   │
│  │  - 识别终末状态 (terminal states)                           │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 关键创新

1. **动态模型求解**
   - 求解转录-剪接-降解的微分方程系统
   - 估计基因特异性参数 (alpha, beta, gamma)
   - 比稳态模型更准确

2. **速度向量场可视化**
   - 在低维嵌入上绘制速度向量
   - 直观显示细胞分化方向
   - 支持流线图(stream plot)

3. **轨迹推断**
   - 从速度场推断细胞命运
   - 识别发育起点和终点
   - 计算细胞到达各终末状态的概率

## 扰动预测能力

### 核心功能

| 功能 | 描述 | 适用场景 |
|------|------|----------|
| **分化方向预测** | 推断细胞将分化成什么类型 | 发育生物学 |
| **轨迹推断** | 重建细胞状态转移路径 | 重编程研究 |
| **根细胞识别** | 识别发育起始细胞 | 干细胞研究 |
| **终末状态预测** | 预测细胞最终命运 | 疾病进展 |
| **伪时间推断** | 估计细胞在轨迹上的位置 | 时序分析 |

### 与扰动预测的结合

scVelo本身不直接预测扰动响应，但可以：
- 分析扰动前后细胞轨迹变化
- 识别扰动导致的命运偏移
- 与扰动预测方法(如CellFlow)结合使用

## 技术细节

### 动态模型方程

```
┌─────────────────────────────────────────────────────────────────────┐
│              动态模型微分方程                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  对每个基因:                                                        │
│                                                                     │
│  du/dt = alpha(t) - beta * u(t)                                     │
│  ds/dt = beta * u(t) - gamma * s(t)                                 │
│                                                                     │
│  其中:                                                              │
│  - u(t): 未剪接mRNA丰度                                             │
│  - s(t): 剪接mRNA丰度                                               │
│  - alpha(t): 转录速率 (可随时间变化)                                │
│  - beta: 剪接速率                                                   │
│  - gamma: 降解速率                                                  │
│                                                                     │
│  求解方法:                                                          │
│  - 解析解 (alpha为常数时)                                           │
│  - 数值积分 (alpha变化时)                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 参数估计

```
┌─────────────────────────────────────────────────────────────────────┐
│              参数估计流程                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. 识别相空间中的动态                                              │
│     - 绘制未剪接 vs 剪接的相图                                      │
│     - 拟合稳态线: u = (gamma/beta) * s                              │
│                                                                     │
│  2. 基因分类                                                        │
│     - 诱导型 (induction): 转录上调                                  │
│     - 抑制型 (repression): 转录下调                                 │
│     - 稳态型 (steady-state): 转录恒定                               │
│                                                                     │
│  3. 最大似然估计                                                    │
│     - 对每个基因独立估计参数                                        │
│     - 使用EM算法优化                                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 与其他方法的对比

| 方法 | 类型 | 输入数据 | 核心优势 | 局限性 |
|------|------|----------|----------|--------|
| **scVelo** | RNA速度 | 剪接/未剪接计数 | 物理模型基础，方向性明确 | 需要剪接信息 |
| **Monocle** | 伪时间 | 表达矩阵 | 无需剪接信息 | 假设树状结构 |
| **PAGA** | 抽象图 | 表达矩阵 | 处理复杂拓扑 | 无方向性 |
| **CellRank** | 概率模型 | 速度+表达 | 结合速度与相似性 | 计算复杂 |
| **CASCADE** | 深度学习 | 时间序列 | 直接预测未来状态 | 需要时间标签 |
| **CellFlow** | 流匹配 | 扰动-响应对 | 预测扰动效果 | 需要训练数据 |

## 代码示例

```python
import scvelo as scv

# 读取数据 (包含剪接/未剪接计数)
adata = scv.datasets.pancreas()

# 预处理
scv.pp.filter_and_normalize(adata)
scv.pp.moments(adata)

# 估计RNA速度
scv.tl.velocity(adata, mode='dynamical')  # 推荐动态模型
scv.tl.velocity_graph(adata)

# 可视化
scv.pl.velocity_embedding_stream(adata, basis='umap')

# 轨迹推断
scv.tl.recover_dynamics(adata)
scv.tl.latent_time(adata)
```
