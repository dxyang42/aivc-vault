# SPACEL

## 基本信息

- **全称**: SPACEL: deep learning-based characterization of spatial transcriptomics
- **发表时间**: 2023年
- **发表期刊**: Nature Communications
- **机构**: 浙江大学、西湖大学
- **作者**: Jianyang Zeng, et al.

## 核心技术

SPACEL 是一个基于深度学习的空间转录组分析方法，包含三个核心模块：

1. **Spoint**: 细胞类型反卷积 (Cell Type Deconvolution)
2. **Splane**: 空间域识别 (Spatial Domain Identification)
3. **Scube**: 3D重建与对齐 (3D Reconstruction and Alignment)

### Spoint 详细架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Spoint Architecture                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   训练阶段:                                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   scRNA-seq 参考数据                                │  │
│   │        ↓                                            │  │
│   │   ┌─────────────────────────────────────────────┐  │  │
│   │   │  编码器 (Encoder)                           │  │  │
│   │   │  - 输入: 基因表达 (20,000维)                │  │  │
│   │   │  - 层: MLP (2048 → 1024 → 512)              │  │  │
│   │   │  - 输出: 细胞嵌入 (256-dim)                 │  │  │
│   │   │  - 激活: ReLU + BatchNorm                   │  │  │
│   │   └─────────────────────────────────────────────┘  │  │
│   │        ↓                                            │  │
│   │   ┌─────────────────────────────────────────────┐  │  │
│   │   │  分类器 (Classifier)                        │  │  │
│   │   │  - 输入: 256-dim 嵌入                       │  │  │
│   │   │  - 输出: 细胞类型概率分布                   │  │  │
│   │   │  - 损失: 交叉熵                             │  │  │
│   │   └─────────────────────────────────────────────┘  │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   预测阶段:                                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   空间转录组 spot                                   │  │
│   │        ↓                                            │  │
│   │   预训练编码器 → 嵌入                               │  │
│   │        ↓                                            │  │
│   │   分类器 → 初始细胞类型分布                         │  │
│   │        ↓                                            │  │
│   │   ┌─────────────────────────────────────────────┐  │  │
│   │   │  空间平滑优化                               │  │  │
│   │   │                                             │  │  │
│   │   │  目标: 最小化总变差                         │  │  │
│   │   │  min ||C_i - C_j||^2  (i,j为邻居)          │  │  │
│   │   │                                             │  │  │
│   │   │  约束: 重建误差 < epsilon                   │  │  │
│   │   │  ||X - C @ G||^2 < epsilon                 │  │  │
│   │   │  (X: spot表达, C: 细胞比例, G: 细胞表达)    │  │  │
│   │   │                                             │  │  │
│   │   └─────────────────────────────────────────────┘  │  │
│   │        ↓                                            │  │
│   │   最终细胞类型比例                                  │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 模型配置

| 参数 | Spoint | Splane | Scube |
|-----|--------|--------|-------|
| 架构 | MLP | GCN | OT + 插值 |
| 层数 | 3 | 2-3 | - |
| 隐藏维度 | 256-2048 | 512 | - |
| 激活函数 | ReLU | ReLU | - |
| 输出 | 细胞比例 | 域标签 | 3D坐标 |

## 输入/输出

- **输入**:
  - 空间转录组数据 (Visium, Slide-seq, etc.)
  - 参考 scRNA-seq 数据 (用于Spoint)
  - 多个连续切片 (用于Scube)
- **输出**:
  - 细胞类型反卷积结果
  - 空间域标签
  - 3D重建结构

## 性能指标

SPACEL 在多个空间转录组数据集上达到 SOTA：

| 任务 | 数据集 | SPACEL | 次优方法 |
|-----|--------|--------|----------|
| 细胞反卷积 | Human DLPFC | PCC 0.89 | RCTD (0.82) |
| 细胞反卷积 | Mouse Brain | PCC 0.85 | SPOTlight (0.78) |
| 空间域识别 | Human DLPFC | ARI 0.72 | BayesSpace (0.65) |
| 空间域识别 | Breast Cancer | NMI 0.68 | stLearn (0.61) |
| 3D重建 | Mouse Embryo | 对齐精度 0.91 | PASTE (0.84) |

## 功能特点

1. **一体化流程**: 从原始数据到3D重建的完整分析
2. **深度学习**: 基于CNN和GNN的先进架构
3. **空间感知**: 显式建模空间关系
4. **多切片支持**: 支持连续切片3D重建
5. **开源工具**: 易于使用的Python包

## 应用场景

- 肿瘤微环境解析
- 脑组织空间结构
- 胚胎发育3D重建
- 器官图谱构建
- 空间异质性分析

## 优势与局限

**优势**:
- 端到端空间转录组分析
- 深度学习架构
- 3D重建能力
- 良好的可扩展性

**局限**:
- 需要参考scRNA-seq数据
- 计算资源需求较高
- 对低质量数据敏感
