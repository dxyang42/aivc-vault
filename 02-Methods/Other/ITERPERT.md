## 核心技术

### 架构设计

IterPert采用**主动学习+多模态先验**的迭代实验设计框架：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    IterPert 迭代实验设计框架                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   初始化阶段                                                             │
│   ├── 种子扰动数据集 (初始Perturb-seq实验)                              │
│   └── 多模态先验知识整合                                                 │
│       ├── 基因功能注释 (GO, KEGG)                                       │
│       ├── 蛋白相互作用网络 (PPI)                                        │
│       ├── 已知基因关系数据库                                             │
│       └── 预训练语言模型嵌入                                             │
│                              ↓                                          │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │              迭代循环 (直到预算耗尽)                             │  │
│   │                                                                 │  │
│   │  步骤1: 训练预测模型                                             │  │
│   │  ├── 输入: 当前已观测的扰动-响应数据                              │  │
│   │  └── 输出: 训练好的扰动响应预测模型                               │  │
│   │                      ↓                                          │  │
│   │  步骤2: 不确定性量化                                             │  │
│   │  ├── 对候选扰动集合计算预测不确定性                               │  │
│   │  ├── 方法: 贝叶斯神经网络 / 集成方法 / MC Dropout                │  │
│   │  └── 输出: 每个候选扰动的不确定性分数                             │  │
│   │                      ↓                                          │  │
│   │  步骤3: 信息增益计算                                             │  │
│   │  ┌─────────────────────────────────────────────────────────┐   │  │
│   │  │  信息增益 = f(不确定性, 先验知识, 实验成本)              │   │  │
│   │  │                                                         │   │  │
│   │  │  多模态先验整合:                                         │   │  │
│   │  │  - 基因功能相似性: 相似基因可能产生相似响应               │   │  │
│   │  │  - 通路共成员: 同通路基因扰动信息互补                     │   │  │
│   │  │  - 网络拓扑: PPI网络中的中心性/模块性                     │   │  │
│   │  │  - 序列嵌入: 预训练基因语言模型编码                       │   │  │
│   │  └─────────────────────────────────────────────────────────┘   │  │
│   │                      ↓                                          │  │
│   │  步骤4: 最优扰动选择                                             │  │
│   │  ├── 选择信息增益最高的k个扰动进行下一轮实验                      │  │
│   │  └── 考虑实验批次约束和成本控制                                   │  │
│   │                      ↓                                          │  │
│   │  步骤5: 执行Perturb-seq实验                                      │  │
│   │  └── 获取新扰动的单细胞转录组响应                                 │  │
│   │                      ↓                                          │  │
│   │  步骤6: 数据整合                                                 │  │
│   │  └── 将新数据加入训练集，更新模型                                │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              ↓                                          │
│   最终输出: 优化后的扰动响应预测模型 + 完整的扰动-响应数据集              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 技术特点

- **主动学习 (Active Learning)**: 智能选择最有信息量的扰动进行实验
- **多模态先验整合**: 融合基因注释、网络拓扑、序列信息等多种先验知识
- **顺序最优实验设计**: 迭代优化实验设计，最大化信息获取效率
- **不确定性引导采样**: 优先探索模型预测不确定的区域

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 主要应用场景 |
| **单基因过表达 (OE)** | 支持 | 可扩展 |
| **双基因组合** | 支持 | 通过组合策略 |
| **化学扰动** | 部分支持 | 需要适配 |

### 核心优势

| 能力 | 描述 |
|------|------|
| **实验效率提升** | 相比随机采样，用更少实验获得同等信息量 |
| **成本优化** | 在固定预算下最大化数据质量 |
| **探索-利用平衡** | 智能平衡已知区域细化和新区域探索 |
| **先验知识融合** | 有效整合多源生物学先验知识 |

## 技术优势

### 与传统方法的差异

| 特性 | IterPert | 随机实验设计 | 网格搜索 |
|------|----------|-------------|---------|
| **实验效率** | 高 (主动学习) | 低 | 中 |
| **成本适应性** | 强 (预算感知) | 弱 | 弱 |
| **先验利用** | 充分 | 无 | 无 |
| **迭代优化** | 支持 | 不支持 | 不支持 |
| **不确定性量化** | 显式 | 无 | 无 |

### 创新点

1. **多模态先验整合**: 首次系统性地将基因功能注释、PPI网络、序列嵌入等多种先验整合到主动学习框架中

2. **扰动特异性信息增益**: 针对Perturb-seq实验特点设计的信息增益函数，考虑单细胞数据的特殊性

3. **批次感知选择策略**: 考虑实际实验中的批次效应和成本约束

## 应用场景

1. **药物靶点发现**: 高效筛选潜在药物靶点组合
2. **基因功能注释**: 优先探索功能未知的基因
3. **合成致死筛选**: 智能选择基因对进行组合扰动
4. **通路解析**: 优化实验设计以解析复杂调控网络
