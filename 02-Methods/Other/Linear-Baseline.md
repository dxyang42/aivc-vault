# Linear Baseline (Norman 2019)

> [[方法地图]] | [[团队地图]]

## 1. 基本信息

| 属性 | 详情 |
|------|------|
| **模型名称** | Linear Baseline (线性基线) |
| **发布时间** | 2019年 |
| **机构** | Broad Institute |
| **论文** | Norman et al., Science 2019 |
| **类型** | 线性回归基线方法 |
| **定位** | 扰动预测领域的经典基线 |

Linear Baseline 是单细胞扰动预测领域最经典的基线方法，由 Norman 等人在 2019 年的 Science 论文中提出。该方法基于简单的线性假设，为后续深度学习方法提供了重要的性能参照。

## 2. 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 线性预测单基因效应 |
| **单基因过表达 (OE)** | 完全支持 | 线性预测过表达效应 |
| **双基因组合** | 加性假设 | 假设效应可线性叠加 |
| **多基因组合** | 加性假设 | 扩展至多基因组合 |
| **非加性效应** | 不支持 | 无法捕捉协同/拮抗效应 |

### 预测原理

```
┌─────────────────────────────────────────────────────────────────────┐
│              Linear Baseline 预测原理                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   单基因扰动预测:                                                   │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   y_pred = x_control + delta_gene                           │  │
│   │                                                             │  │
│   │   其中:                                                     │  │
│   │   - x_control: 对照细胞基因表达向量                         │  │
│   │   - delta_gene: 该基因扰动的平均效应向量                    │  │
│   │   - y_pred: 预测的扰动后表达                                │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   组合扰动预测 (加性假设):                                          │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   y_combo = x_control + delta_A + delta_B                   │  │
│   │                                                             │  │
│   │   假设: 两个基因扰动的效应可以简单相加                      │  │
│   │   局限: 无法捕捉基因间的非加性相互作用                      │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 评估指标

| 指标 | 结果 | 说明 |
|------|------|------|
| **整体预测精度** | 中等 | 作为基线参考 |
| **单基因预测** | 可接受 | 线性假设基本成立 |
| **组合预测** | 较差 | 非加性效应被忽略 |
| **计算速度** | 极快 | 简单向量加法 |

## 3. 网络架构

### 整体架构

```
┌─────────────────────────────────────────────────────────────────────┐
│              Linear Baseline 架构                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   训练阶段:                                                         │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   输入: 配对的对照-扰动样本                                 │  │
│   │        ↓                                                    │  │
│   │   计算每个扰动的平均效应:                                   │  │
│   │                                                             │  │
│   │   delta_gene = mean(y_perturbed - x_control)                │  │
│   │                                                             │  │
│   │   存储: 扰动效应库 {gene_id: delta_vector}                  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   预测阶段:                                                         │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   输入: 对照细胞表达 + 目标扰动ID                           │  │
│   │        ↓                                                    │  │
│   │   查表获取效应向量: delta = lookup(perturbation_id)         │  │
│   │        ↓                                                    │  │
│   │   预测: y_pred = x_control + delta                          │  │
│   │                                                             │  │
│   │   组合预测: y_combo = x_control + delta_A + delta_B + ...   │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 核心特点

| 特点 | 说明 |
|------|------|
| **简单高效** | 无需训练复杂模型，计算极快 |
| **可解释性强** | 每个扰动的效应直接可见 |
| **基线作用** | 为深度学习方法提供参照基准 |
| **局限明显** | 无法建模非线性相互作用 |

## 4. 训练数据

### 数据来源

| 数据类型 | 描述 | 规模 |
|----------|------|------|
| **Perturb-seq** | 单细胞CRISPR筛选数据 | 大规模 |
| **CROP-seq** | 单细胞水平扰动数据 | 中等规模 |
| **批量RNA-seq** | 传统扰动实验数据 | 多样化 |

### 数据要求

- 需要配对的对照-扰动样本
- 每个扰动需要足够样本计算平均效应
- 基因表达维度: 20,000+

## 5. 与其他扰动预测方法的比较

| 特性 | Linear | GEARS | CPA | scGen |
|------|--------|-------|-----|-------|
| **模型复杂度** | 极低 | 中等 | 中等 | 中等 |
| **非加性效应** | 不支持 | 核心优势 | 部分支持 | 不支持 |
| **计算速度** | 极快 | 中等 | 中等 | 中等 |
| **训练需求** | 无 | 需要 | 需要 | 需要 |
| **泛化能力** | 无 | 强 | 中等 | 中等 |
| **可解释性** | 高 | 高 | 高 | 中等 |

## 6. 功能特点

### 核心能力

| 功能 | 描述 |
|------|------|
| **快速预测** | 查表式预测，毫秒级响应 |
| **基线评估** | 评估复杂模型的相对提升 |
| **效应可视化** | 直接查看每个扰动的效应向量 |
| **简单扩展** | 易于添加新的扰动效应 |

### 技术局限

| 局限 | 说明 |
|------|------|
| **非加性效应** | 无法捕捉协同/拮抗相互作用 |
| **细胞类型特异性** | 假设效应在不同细胞类型间一致 |
| **未见扰动** | 无法预测训练未见的扰动 |
| **剂量效应** | 无法建模剂量-响应关系 |

## 7. 应用场景

### 适用场景

| 应用 | 描述 |
|------|------|
| **基线比较** | 作为深度学习方法的性能参照 |
| **快速筛选** | 大规模扰动的初步筛选 |
| **效应分析** | 理解单个扰动的平均效应 |
| **教学演示** | 理解扰动预测的基本原理 |

### 不适用场景

| 场景 | 原因 |
|------|------|
| **组合优化** | 无法捕捉非加性效应 |
| **精准预测** | 精度低于深度学习方法 |
| **机制研究** | 无法揭示基因相互作用 |

## 8. 优势与局限

### 核心优势

| 优势 | 说明 |
|------|------|
| **极简实现** | 几行代码即可完成 |
| **无需训练** | 直接计算平均效应 |
| **计算高效** | 适合大规模预测 |
| **结果稳定** | 无随机性，结果可复现 |
| **可解释** | 效应向量直接可解读 |

### 当前局限

| 局限 | 说明 | 潜在改进方向 |
|------|------|-------------|
| **线性假设** | 生物系统往往非线性 | 引入非线性变换 |
| **组合效应** | 忽略基因相互作用 | 结合知识图谱 |
| **细胞特异性** | 假设跨细胞一致 | 按细胞类型分组 |
| **未见泛化** | 无法预测新扰动 | 结合基因嵌入 |

## 9. 历史意义

Linear Baseline 虽然简单，但在扰动预测领域具有重要意义：

1. **开创性工作**: Norman 2019 论文奠定了 Perturb-seq 数据分析的基础
2. **性能基准**: 为后续 GEARS、CPA 等方法提供了比较基准
3. **方法启示**: 揭示了组合扰动中非加性效应的重要性
4. **数据资源**: 发布的 Norman 数据集成为领域标准基准

## 10. 参考信息

- **论文**: Norman et al. (2019). Exploring genetic interaction manifolds constructed from rich single-cell phenotypes. *Science*.
- **数据集**: Norman dataset (105单基因 + 131双基因扰动)
