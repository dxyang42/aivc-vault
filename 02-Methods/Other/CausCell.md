## 核心技术

### 架构设计
CausCell是一种因果解耦的单细胞表示学习方法，通过预定义因果结构分离生物过程中的语义概念：

```
┌─────────────────────────────────────────────────────────────────┐
│                    CausCell 技术架构                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   1. 因果图定义层                                                 │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 预定义因果结构 (Causal Graph)                          │   │
│   │    - 细胞类型 → 基因表达                                  │   │
│   │    - 批次效应 → 技术变异                                  │   │
│   │    - 扰动条件 → 细胞响应                                  │   │
│   │                                                         │   │
│   │  • 概念定义 (Concepts)                                   │   │
│   │    - 生物概念: 细胞类型、分化状态、通路活性               │   │
│   │    - 技术概念: 批次、测序深度、捕获效率                   │   │
│   │    - 扰动概念: 药物、基因敲除、环境刺激                   │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ↓                                       │
│   2. 因果解耦编码器                                               │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 变分自编码器 (VAE) 框架                                │   │
│   │                                                         │   │
│   │  • 概念分离编码                                          │   │
│   │    - 每个概念对应独立的隐变量子空间                       │   │
│   │    - 因果约束确保概念间的独立性                           │   │
│   │                                                         │   │
│   │  • 因果干预建模                                          │   │
│   │    - do-算子实现因果干预                                  │   │
│   │    - 反事实样本生成                                       │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ↓                                       │
│   3. 可控生成层                                                   │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 概念组合生成                                          │   │
│   │    - 任意组合不同概念生成细胞                             │   │
│   │                                                         │   │
│   │  • 反事实预测                                            │   │
│   │    - "如果细胞类型是X，表达会如何变化?"                   │   │
│   │                                                         │   │
│   │  • 干预模拟                                              │   │
│   │    - 模拟基因敲除/药物处理的效果                          │   │
│   └─────────────────────────────────────────────────────────┘   │
│                          ↓                                       │
│   4. 验证与解释层                                                 │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │  • 因果一致性验证                                        │   │
│   │  • 概念可解释性分析                                      │   │
│   │  • 生成样本真实性评估                                    │   │
│   └─────────────────────────────────────────────────────────┘   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 技术特点
- **因果解耦**: 将单细胞数据分离为独立的因果概念
- **预定义因果结构**: 基于生物学知识构建因果图
- **可控反事实生成**: 生成符合因果结构的反事实细胞
- **可解释表示**: 每个隐维度对应明确的生物概念
- **概念组合**: 支持任意概念组合生成新细胞

## 输入/输出

### 输入格式
| 输入类型 | 描述 | 维度 |
|---------|------|------|
| **基因表达** | 单细胞RNA-seq数据 | 20,000+ 基因 |
| **元数据** | 细胞类型、批次、条件等 | 可变 |
| **因果图** | 预定义的因果结构 | 图结构 |
| **干预目标** | 想要干预的概念 | 概念索引 |

### 输出格式
| 输出类型 | 描述 | 维度 |
|---------|------|------|
| **解耦表示** | 分离的概念嵌入 | 概念数 × 维度 |
| **反事实细胞** | 干预后的基因表达 | 20,000+ 基因 |
| **因果效应** | 干预的因果效应估计 | 效应大小 |
| **概念重要性** | 各概念对预测的贡献 | 概念数 |

## 网络结构 (ASCII图)

```
┌─────────────────────────────────────────────────────────────────┐
│                   CausCell 模型结构                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   输入: 基因表达 X                                                │
│          ↓                                                       │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │              因果解耦编码器 (Causal Encoder)              │   │
│   │                                                          │   │
│   │   输入 X                                                  │   │
│   │     ↓                                                     │   │
│   │   ┌─────────────────────────────────────────────────┐    │   │
│   │   │  概念1编码器 (如: 细胞类型)                        │    │   │
│   │   │  z_1 = f_1(X)  ~ N(mu_1, sigma_1)                │    │   │
│   │   └─────────────────────────────────────────────────┘    │   │
│   │     ↓                                                     │   │
│   │   ┌─────────────────────────────────────────────────┐    │   │
│   │   │  概念2编码器 (如: 批次效应)                        │    │   │
│   │   │  z_2 = f_2(X)  ~ N(mu_2, sigma_2)                │    │   │
│   │   └─────────────────────────────────────────────────┘    │   │
│   │     ↓                                                     │   │
│   │   ┌─────────────────────────────────────────────────┐    │   │
│   │   │  概念3编码器 (如: 扰动响应)                        │    │   │
│   │   │  z_3 = f_3(X)  ~ N(mu_3, sigma_3)                │    │   │
│   │   └─────────────────────────────────────────────────┘    │   │
│   │     ↓                                                     │   │
│   │   ... (更多概念)                                          │   │
│   │                                                          │   │
│   │   因果约束: z_i 独立于 z_j (给定父节点)                   │   │
│   │                                                          │   │
│   └────────────────────────┬────────────────────────────────┘   │
│                            ↓                                     │
│   解耦隐变量: [z_1, z_2, z_3, ...]                               │
│                            ↓                                     │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │              因果干预模块 (Intervention)                  │   │
│   │                                                          │   │
│   │   do(z_i = a)  // 干预概念i的值为a                       │   │
│   │     ↓                                                     │   │
│   │   保持其他概念 z_j (j≠i) 不变                            │   │
│   │     ↓                                                     │   │
│   │   生成反事实隐变量 z' = [z_1, ..., a, ..., z_n]          │   │
│   │                                                          │   │
│   └────────────────────────┬────────────────────────────────┘   │
│                            ↓                                     │
│   ┌─────────────────────────────────────────────────────────┐   │
│   │              概念组合解码器 (Decoder)                     │   │
│   │                                                          │   │
│   │   X' = g(z')  // 从隐变量重建基因表达                     │   │
│   │                                                          │   │
│   │   支持:                                                   │   │
│   │   • 原始重建: z = [z_1, z_2, z_3, ...]                  │   │
│   │   • 反事实生成: z' = [z_1, a, z_3, ...]                 │   │
│   │   • 概念组合: z'' = [a, b, c, ...]                      │   │
│   │                                                          │   │
│   └────────────────────────┬────────────────────────────────┘   │
│                            ↓                                     │
│   输出: 重建/反事实/组合细胞                                      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 核心能力

### 1. 因果解耦表示学习
```
传统VAE                    CausCell
─────────────────────────────────────────────────
纠缠的隐变量      →         分离的概念子空间
z = [z1,z2,z3]              z_cell_type ⊥ z_batch ⊥ z_perturbation
                              (因果独立)
```

### 2. 反事实生成
```
原始细胞                    反事实细胞
┌─────────────┐            ┌─────────────┐
│ 细胞类型: A  │   do(      │ 细胞类型: B  │
│ 批次: 1     │  细胞类型=B) │ 批次: 1     │
│ 表达: X     │    →       │ 表达: X'    │
└─────────────┘            └─────────────┘
        ↓                        ↓
  "A型细胞在批次1     "如果这是B型细胞
   的表达谱"          在批次1的表达谱"
```

### 3. 概念组合生成
```
概念1: 细胞类型 = T细胞
概念2: 扰动条件 = 药物X
概念3: 批次 = 批次1
         ↓
    组合生成
         ↓
"T细胞在批次1中
 经药物X处理后的
 预测表达谱"
```

## 功能特点

- ✅ **因果解耦**: 分离混淆因素和真实生物信号
- ✅ **反事实预测**: 回答"如果...会怎样"的问题
- ✅ **概念组合**: 生成未见过的概念组合
- ✅ **可解释性**: 明确的生物概念对应
- ✅ **因果验证**: 生成的反事实符合因果结构
- ✅ **批次校正**: 作为副产品实现批次效应去除
- ⚠️ **需要因果图**: 依赖预定义或学习的因果结构

## 应用场景

1. **因果发现**
   - 验证基因调控关系
   - 识别细胞命运决定因素

2. **批次效应分析**
   - 分离技术变异和生物变异
   - 跨数据集整合

3. **扰动效应预测**
   - 预测未见扰动的效果
   - 组合扰动预测

4. **虚拟实验**
   - 模拟基因敲除实验
   - 预测药物组合效应

5. **数据增强**
   - 生成稀有细胞类型
   - 平衡不平衡数据集

## 优势与局限

### 优势
- 高可解释性的解耦表示
- 符合因果结构的反事实生成
- 支持概念组合和虚拟实验
- 开源且文档完善
- 经过同行评审(Nature Communications)

### 局限
- 依赖因果图的准确性
- 复杂因果结构学习困难
- 计算成本较高
- 需要领域知识定义概念
