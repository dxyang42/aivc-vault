## 核心技术

### 架构设计
scPerb 将风格迁移概念与变分自编码器结合，用于单细胞扰动预测：

```
┌─────────────────────────────────────────────────────────────────────┐
│                     scPerb 架构概览                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  输入层                                                             │
│  ├── 输入1: 未扰动细胞表达 (Query Cell)                            │
│  └── 输入2: 参考扰动细胞 (Reference Cell)                          │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                    双编码器结构                              │   │
│  │                                                             │   │
│  │  ┌─────────────────────┐    ┌─────────────────────┐        │   │
│  │  │   内容编码器         │    │    风格编码器        │        │   │
│  │  │   (Content Encoder) │    │    (Style Encoder)  │        │   │
│  │  │                     │    │                     │        │   │
│  │  │  输入: 未扰动细胞    │    │  输入: 参考扰动细胞  │        │   │
│  │  │       ↓             │    │       ↓             │        │   │
│  │  │  全连接网络         │    │  全连接网络         │        │   │
│  │  │       ↓             │    │       ↓             │        │   │
│  │  │  内容潜变量 z_c     │    │  风格潜变量 z_s     │        │   │
│  │  │  (128-dim)          │    │  (64-dim)           │        │   │
│  │  │                     │    │                     │        │   │
│  │  │  捕获: 细胞固有特性  │    │  捕获: 扰动相关变异  │        │   │
│  │  │       (内容)        │    │       (风格)        │        │   │
│  │  └─────────────────────┘    └─────────────────────┘        │   │
│  │                              ↓                              │   │
│  │                    潜变量拼接 [z_c || z_s]                  │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                    解码器 (Decoder)                          │   │
│  │  ├── 输入: 拼接潜变量 (128+64=192 dim)                      │   │
│  │  ├── 结构: 全连接网络 (对称于编码器)                        │   │
│  │  └── 输出: 预测的扰动后细胞表达 (20,000+ 基因)              │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  输出: 扰动后细胞预测 + 差异表达基因                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 技术特点
- **风格迁移**: 将扰动视为"风格"，从参考细胞提取
- **解耦表示**: 内容和风格解耦，分别编码
- **VAE架构**: 变分自编码器提供概率建模
- **显式扰动分离**: 显式提取扰动相关变异

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 需要该扰动的参考细胞 |
| **单基因过表达 (OE)** | 完全支持 | 需要该扰动的参考细胞 |
| **药物处理** | 支持 | 需要药物处理的参考细胞 |
| **双基因组合** | 部分支持 | 需要组合扰动的参考细胞 |
| **零样本预测** | 不支持 | 必须有参考细胞 |

### 预测准确率与评估指标

| 指标 | 结果 | 对比基线 |
|------|------|---------|
| **扰动预测精度** | Pearson r > 0.85 | 优于GEARS |
| **差异表达预测** | AUPRC > 0.45 | 优于其他VAE方法 |
| **Top基因预测** | Spearman r > 0.80 | - |
| **计算效率** | 快速推理 | 比GEARS更快 |
| **未见扰动泛化** | 有限 | 需要参考细胞 |

### 评估数据集
- **Perturb-seq数据集**: 多种细胞类型的CRISPR筛选
- **CROP-seq数据集**: 单细胞水平的基因扰动
- **药物扰动数据集**: 小分子药物处理数据

## 与其他扰动预测方法的差异

| 特性 | scPerb | GEARS | CPA | Linear Model |
|------|--------|-------|-----|--------------|
| **核心思想** | 风格迁移 | 图神经网络 | 解耦VAE | 线性回归 |
| **需要参考细胞** | 是 | 否 | 否 | 否 |
| **零样本预测** | 不支持 | 支持 | 支持 | 支持 |
| **内容-风格解耦** | 显式 | 隐式 | 部分 | 无 |
| **计算速度** | 快 | 中等 | 中等 | 最快 |
| **可解释性** | 高 | 高 | 中等 | 高 |

## 训练策略

### 训练损失组成

1. **重建损失**: 预测与真实扰动后细胞的均方误差 (MSE)
2. **KL散度**: 正则化VAE潜变量分布
3. **风格一致性损失**: 相同扰动的风格嵌入相似
4. **内容保持损失**: 确保内容编码器捕获细胞固有特性

### 训练数据
- 扰动数据集 (Perturb-seq, CROP-seq等)
- 配对的控制-扰动细胞
- 约20,000基因维度
- 人类和小鼠数据

## 功能特点

- 单基因扰动预测
- 多基因组合扰动预测（需参考）
- 差异表达基因预测
- 扰动风格解耦（可解释性强）
- 快速推理
- 不支持零样本扰动预测（需要参考细胞）

## 应用场景

1. **药物筛选**: 预测药物扰动效应
2. **基因功能研究**: 预测基因敲除/过表达效果
3. **组合扰动**: 预测多基因同时扰动的效应
4. **机制研究**: 通过风格嵌入理解扰动机制
5. **实验设计**: 优先选择最有信息量的扰动实验

## 优势与局限

### 优势
- 创新的风格迁移思路
- 显式分离扰动效应，可解释性强
- 高精度的扰动预测
- 计算效率高

### 局限
- 需要参考扰动细胞（不支持零样本）
- 主要针对扰动预测任务
- 对其他单细胞任务支持有限
