### 技术特点

- **原位gRNA扩增**: T7启动子介导的原位转录解决短序列检测难题
- **汉明码解码**: 高效准确的gRNA身份解码方案
- **单细胞分辨率**: 真正的单细胞水平空间转录组
- **高复杂度支持**: 支持140个gRNA的高复杂度扰动库

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **CRISPR-KO** | 完全支持 | 基因敲除 |
| **CRISPRi** | 完全支持 | 基因抑制 |
| **碱基编辑** | 兼容 | 可扩展 |
| **CRISPR-off** | 兼容 | 可扩展 |

### 独特能力

| 能力 | 描述 |
|------|------|
| **细胞内回路** | 解析单个细胞内的基因调控网络 |
| **细胞间回路** | 研究细胞间相互作用和信号传递 |
| **功能表型关联** | 将转录变化与功能表型(如钙信号)关联 |
| **空间上下文** | 考虑细胞微环境对扰动响应的影响 |

### 核心创新

1. **技术整合**: 首次成功整合CRISPR筛选、空间转录组和成像技术

2. **原位扩增策略**: T7启动子介导的gRNA原位转录扩增是技术突破

3. **高保真解码**: 汉明码方案实现<1%的低错误率

4. **多尺度分析**: 同时支持细胞内、细胞间和组织尺度的分析

## 应用案例

### 案例1: THP1巨噬细胞LPS响应
- 验证Perturb-FISH与Perturb-seq的一致性 (Pearson > 70%)
- 仅需20-50个细胞即可获得可靠结果
- 分析细胞密度和邻近细胞对基因表达的影响

### 案例2: hiPSC星形胶质细胞
- 结合钙成像研究ASD风险基因
- 揭示自闭症相关基因对细胞活性的影响
- 分析钙信号活动及相关分子通路

### 案例3: 肿瘤异种移植模型
- 解析肿瘤细胞与浸润T细胞间的遗传互作
- 发现276种显著的细胞间肿瘤免疫效应
- 揭示免疫调控机制

## 对领域的影响

### 新方法学范式

Perturb-FISH代表了扰动筛选技术的新方向：
- 从"bulk"到"spatial"
- 从"分子"到"功能"
- 从"细胞内"到"细胞间"

### 未来应用

1. **发育生物学**: 研究发育过程中的基因调控网络
2. **肿瘤微环境**: 解析肿瘤-免疫相互作用
3. **神经科学**: 研究神经元回路的功能基因组学
4. **药物筛选**: 在组织背景下评估药物效应
