## 核心技术

### 架构设计
scLAMBDA是**深度生成学习框架**，利用**大语言模型嵌入**：

```
输入: 基因表达 + 扰动信息
      ↓
基因嵌入: 使用LLM嵌入基因
      ↓
解耦编码器: 分离基础细胞表示和扰动状态表示
      ↓
生成器: 生成扰动后细胞状态
      ↓
输出: 单细胞转录响应
```

### 技术特点
- **大语言模型嵌入**: 利用LLM的语义理解能力
- **解耦表示学习**: 分离细胞和扰动表示
- **单细胞级别生成**: 预测单细胞水平的转录响应

## 输入/输出

### 输入格式
| 输入类型 | 描述 | 维度 |
|---------|------|------|
| **基因表达** | 基因表达值 | 20,000+ 基因 |
| **扰动基因** | 被扰动基因ID | 标量/列表 |
| **细胞类型** | 细胞类型标签 | One-hot |

### 输出格式
| 输出类型 | 描述 | 维度 |
|---------|------|------|
| **预测表达** | 扰动后基因表达 | 20,000+ 基因 |
| **细胞嵌入** | 基础细胞表示 | 768维 |
| **扰动嵌入** | 扰动状态表示 | 768维 |
| **不确定性** | 预测不确定性 | 20,000 |

## 网络结构

### 基因嵌入
```python
基因嵌入:
  - 方法: 使用预训练LLM (如GenePT)
  - 输入: 基因ID/描述
  - 输出: 基因嵌入向量 (768维)
```

### 解耦编码器
```python
解耦编码器:
  - 输入: 基因表达
  - 分支1: 基础细胞表示 z_c
  - 分支2: 扰动状态表示 z_p
  - 输出: [z_c, z_p]
```

### 生成器
```python
生成器:
  - 输入: z_c + z_p + 扰动嵌入
  - 架构: Transformer/MLP
  - 输出: 预测基因表达
```

1. **基因功能预测**
