### 核心创新

CoupleVAE 是一种基于**耦合变分自编码器**的单细胞扰动响应预测方法。该方法通过设计两个耦合的VAE网络，分别建模对照状态和扰动状态，实现非线性扰动效应的准确预测。

## 2. 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 预测基因敲除后的细胞状态 |
| **单基因过表达 (OE)** | 完全支持 | 预测基因过表达效应 |
| **药物处理** | 支持 | 预测药物扰动响应 |
| **组合扰动** | 有限支持 | 多基因联合扰动预测 |

### 预测准确率与评估指标

| 指标 | 描述 | 典型值 |
|------|------|--------|
| **基因表达相关性** | Pearson/Spearman相关 | r > 0.75 |
| **MSE/RMSE** | 基因表达均方误差 | 低 |
| **差异表达基因重叠** | 预测与真实DE基因一致性 | Jaccard > 0.55 |

## 4. 核心技术

### 架构设计

CoupleVAE 采用双分支耦合架构：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CoupleVAE 架构概览                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   对照细胞 x_c           扰动细胞 x_p                              │
│        ↓                      ↓                                    │
│   ┌──────────┐           ┌──────────┐                             │
│   │ Encoder  │           │ Encoder  │                             │
│   │    E_c   │           │    E_p   │                             │
│   └────┬─────┘           └────┬─────┘                             │
│        ↓                      ↓                                    │
│     z_c (对照潜变量)      z_p (扰动潜变量)                         │
│        ↓                      ↓                                    │
│        └──────────┬──────────┘                                    │
│                   ↓                                                │
│           ┌───────────────┐                                       │
│           │  耦合模块      │                                       │
│           │ Coupling Unit │                                       │
│           │               │                                       │
│           │ • 扰动编码器   │                                       │
│           │ • 差异建模     │                                       │
│           │ • 共享约束     │                                       │
│           └───────┬───────┘                                       │
│                   ↓                                                │
│        ┌──────────┴──────────┐                                    │
│        ↓                      ↓                                    │
│   ┌──────────┐           ┌──────────┐                             │
│   │ Decoder  │           │ Decoder  │                             │
│   │    D_c   │           │    D_p   │                             │
│   └────┬─────┘           └────┬─────┘                             │
│        ↓                      ↓                                    │
│   x̂_c (重构对照)         x̂_p (重构扰动)                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 技术特点

1. **双分支编码器**：分别编码对照和扰动状态
2. **耦合潜空间**：通过共享约束建立两个潜空间的联系
3. **扰动编码器**：显式建模扰动效应
4. **非线性变换**：捕捉复杂的基因调控关系

## 6. 网络结构

### 编码器架构

```python
# 伪代码示意
class CoupleVAE(nn.Module):
    def __init__(self, input_dim, latent_dim):
        # 对照分支编码器
        self.encoder_control = MLP(input_dim, [512, 256], latent_dim*2)
        # 扰动分支编码器
        self.encoder_perturb = MLP(input_dim, [512, 256], latent_dim*2)
        # 耦合模块
        self.coupling_unit = CouplingModule(latent_dim)
        # 解码器
        self.decoder_control = MLP(latent_dim, [256, 512], input_dim)
        self.decoder_perturb = MLP(latent_dim, [256, 512], input_dim)
    
    def forward(self, x_c, x_p):
        # 编码
        z_c_mean, z_c_logvar = self.encoder_control(x_c)
        z_p_mean, z_p_logvar = self.encoder_perturb(x_p)
        
        # 重参数化采样
        z_c = self.reparameterize(z_c_mean, z_c_logvar)
        z_p = self.reparameterize(z_p_mean, z_p_logvar)
        
        # 耦合
        z_c_coupled, z_p_coupled, delta = self.coupling_unit(z_c, z_p)
        
        # 解码
        x_c_recon = self.decoder_control(z_c_coupled)
        x_p_recon = self.decoder_perturb(z_p_coupled)
        
        return x_c_recon, x_p_recon, delta
```

### 耦合模块设计

耦合模块是 CoupleVAE 的核心创新，包含：

| 组件 | 功能 |
|------|------|
| **扰动编码器** | 学习从对照到扰动的变换 |
| **差异约束** | 确保两个潜空间的一致性 |
| **共享信息** | 提取跨条件的共同特征 |

- **归一化**：log1p转换和缩放

- **批次校正**：消除技术批次效应

## 8. 训练策略

### 损失函数

CoupleVAE 的损失函数包含多个组件：

```
L_total = L_recon + L_KL + L_coupling + L_perturbation

其中：
- L_recon: 重构损失 (MSE)
- L_KL: KL散度 (VAE正则化)
- L_coupling: 耦合约束损失
- L_perturbation: 扰动一致性损失
```

### 训练流程

```
1. 初始化双分支VAE网络
2. 对每个批次:
   a. 编码对照和扰动样本
   b. 通过耦合模块
   c. 计算重构和正则化损失
   d. 反向传播更新参数
3. 验证和早停
4. 保存最佳模型
```
