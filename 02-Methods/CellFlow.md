## 核心技术

### 架构设计

CellFlow 基于**条件流匹配(Conditional Flow Matching)**和**最优传输(Optimal Transport)**理论：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CellFlow 架构概览                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   核心思想: 将扰动预测建模为从对照分布到扰动分布的最优传输           │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              问题建模: 最优传输视角                          │  │
│   │                                                             │  │
│   │   传统方法: 学习映射 f: X_ctrl → X_pert                    │  │
│   │                                                             │  │
│   │   CellFlow: 学习速度场 v(x, t, c) 使得:                    │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   dx/dt = v(x, t, c)                               │  │  │
│   │   │                                                     │  │  │
│   │   │   其中:                                             │  │  │
│   │   │   - x: 细胞状态 (基因表达)                          │  │  │
│   │   │   - t: 时间/传输进度 (0→1)                          │  │  │
│   │   │   - c: 条件 (扰动类型、剂量、细胞类型)              │  │  │
│   │   │                                                     │  │  │
│   │   │   边界条件:                                         │  │  │
│   │   │   - t=0: x ~ p_ctrl (对照分布)                      │  │  │
│   │   │   - t=1: x ~ p_pert (扰动分布)                      │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              条件流匹配网络 (核心创新)                       │  │
│   │                                                             │  │
│   │   输入:                                                     │  │
│   │   ├── 当前状态 x_t (在传输路径上的某点)                     │  │
│   │   ├── 时间 t ∈ [0, 1]                                       │  │
│   │   └── 条件 c = (扰动类型, 剂量, 细胞类型)                   │  │
│   │                                                             │  │
│   │   网络结构:                                                 │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   x_t ──┐                                           │  │  │
│   │   │         ├──→ 特征融合 ──→ MLP/Transformer ──→ v     │  │  │
│   │   │   t ────┤                                           │  │  │
│   │   │         │   (速度场，预测状态变化方向)              │  │  │
│   │   │   c ────┘                                           │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   关键: 学习的是速度(变化率)而非直接映射                    │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              采样过程: ODE求解                               │  │
│   │                                                             │  │
│   │   预测时，从对照细胞出发，沿学习到的速度场积分:              │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   x_0 ~ p_ctrl                                      │  │  │
│   │   │                                                     │  │  │
│   │   │   x_1 = x_0 + ∫₀¹ v(x_t, t, c) dt                  │  │  │
│   │   │                                                     │  │  │
│   │   │   使用ODE求解器 (如Euler, RK4)                      │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   输出: 预测的扰动后细胞状态分布                            │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 流匹配 vs 扩散模型

```
┌─────────────────────────────────────────────────────────────┐
│              流匹配 vs 扩散模型对比                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   扩散模型 (如 scPPDM):                                     │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   前向: 逐步加噪 (随机过程)                         │  │
│   │   x_0 → x_1 → x_2 → ... → x_T (纯噪声)             │  │
│   │                                                     │  │
│   │   反向: 学习去噪                                    │  │
│   │   x_T → x_{T-1} → ... → x_0 (预测)                 │  │
│   │                                                     │  │
│   │   需要: 1000+ 步采样                                │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   流匹配 (CellFlow):                                        │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   直接学习: 对照 → 扰动的直线路径                   │  │
│   │                                                     │  │
│   │   x(t) = (1-t) * x_0 + t * x_1   (线性插值)         │  │
│   │                                                     │  │
│   │   速度场: v = x_1 - x_0  (常数速度)                 │  │
│   │                                                     │  │
│   │   需要: 10-50 步采样 (更快!)                        │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   关键优势:                                                 │
│   • 训练更稳定 (无需噪声调度)                               │
│   • 采样更快 (更少积分步数)                                 │
│   • 路径更直 (最优传输理论保证)                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 训练策略

### 条件流匹配目标

```
┌─────────────────────────────────────────────────────────────┐
│              CellFlow 训练目标                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   训练数据: 配对的对照-扰动细胞 {(x_0, x_1)}               │
│                                                             │
│   对于每对样本:                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   1. 采样时间 t ~ Uniform(0, 1)                   │  │
│   │                                                     │  │
│   │   2. 计算插值点:                                    │  │
│   │      x_t = (1-t) * x_0 + t * x_1                  │  │
│   │                                                     │  │
│   │   3. 计算目标速度:                                  │  │
│   │      u_t = x_1 - x_0  (从x_0到x_1的方向)          │  │
│   │                                                     │  │
│   │   4. 网络预测:                                      │  │
│   │      v_θ(x_t, t, c)  (参数化速度场)                │  │
│   │                                                     │  │
│   │   5. 损失函数:                                      │  │
│   │      L = ||v_θ(x_t, t, c) - u_t||²                │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   直观理解: 网络学习"从当前点指向终点的方向"                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 网络架构细节

| 组件 | 配置 |
|------|------|
| **主干网络** | MLP 或 Transformer |
| **隐藏层** | 3-5层 |
| **隐藏维度** | 512-1024 |
| **时间编码** | 正弦位置编码 |
| **条件编码** | 嵌入层 + MLP |
| **激活函数** | Swish/SiLU |

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因KO** | 完全支持 | 基因敲除效应 |
| **单基因OE** | 完全支持 | 基因过表达 |
| **药物处理** | 完全支持 | 化学扰动 |
| **组合扰动** | 支持 | 多条件组合 |
| **剂量变化** | 核心优势 | 连续剂量建模 |
| **时间序列** | 支持 | 动态响应预测 |

### 预测性能

| 任务 | 数据集 | CellFlow | scGen | CPA |
|-----|--------|----------|-------|-----|
| 单基因KO | Norman 2021 | R² 0.79 | 0.71 | 0.74 |
| 药物响应 | sci-Plex | R² 0.76 | 0.68 | 0.72 |
| 跨细胞类型 | 迁移任务 | R² 0.72 | 0.65 | 0.69 |
| 组合预测 | 双基因 | R² 0.68 | - | 0.65 |

## 核心技术特点

### 1. 连续剂量-响应建模

```
┌─────────────────────────────────────────────────────────────┐
│              连续剂量-响应建模                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   传统方法: 离散剂量水平 (如 0, 0.1, 1, 10 μM)              │
│                                                             │
│   CellFlow: 连续剂量空间                                    │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   条件 c = (drug_id, dose)                          │  │
│   │                                                     │  │
│   │   dose ∈ [0, ∞) 连续值                              │  │
│   │                                                     │  │
│   │   预测:                                             │  │
│   │   • 任意剂量的响应                                  │  │
│   │   • 剂量-响应曲线                                   │  │
│   │   • IC50估计                                        │  │
│   │                                                     │  │
│   │   优势: 无需插值，直接预测未测试剂量                │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. 概率预测与不确定性

- 生成多个样本估计预测分布
- 提供预测置信区间
- 识别预测不确定的细胞

### 3. 可逆变换

- 支持正向预测 (对照→扰动)
- 支持逆向预测 (扰动→对照)
- 支持扰动间转换

## 应用场景

```
工作流程:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   候选药物库 ──┬──→ CellFlow预测 ──→ 响应排序 ──→  top候选  │
```

### 2. 剂量优化

- 预测完整剂量-响应曲线
- 识别最佳治疗窗口
- 预测毒性阈值

### 3. 组合疗法设计

- 预测药物组合的协同效应
- 优化给药顺序
- 避免拮抗组合

## 优势与局限

### 优势
- 采样速度快 (比扩散模型快20-100倍)
- 训练稳定
- 连续条件建模能力强
- 概率预测提供不确定性估计
- 基于最优传输理论，数学基础扎实

### 局限
- 需要配对的对照-扰动数据
- 对复杂多模态分布建模能力有限
- 路径假设为直线可能过于简化
- 大规模数据训练成本较高
