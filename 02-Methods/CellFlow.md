## 核心技术

### 架构设计

CellFlow基于**流匹配(Flow Matching)**和**最优传输(Optimal Transport)**：

```
┌─────────────────────────────────────────────────────────────────────┐
│                  CellFlow 流匹配架构                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   核心概念: 学习速度场 v(t, x, c)，描述细胞状态随时间的演化         │
│                                                                     │
│   输入层:                                                           │
│   - 细胞状态A (源): 基因表达向量 (20,000+维)                        │
│   - 扰动条件 c: 化学/遗传扰动标识 (One-hot编码)                     │
│                                                                     │
│   速度场网络 v(t, x, c) (核心):                                     │
│   - 输入: 时间t + 细胞状态x + 扰动条件c                             │
│   - 条件编码器: 时间嵌入(正弦位置编码) + 扰动条件(MLP编码)          │
│   - 神经网络主干: MLP或Transformer架构                              │
│   - 输出: 速度向量 (与细胞状态同维)                                 │
│   - 训练目标: 速度场逼近目标速度场                                  │
│                                                                     │
│   ODE求解器 (推理阶段):                                             │
│   - 从 t=0 到 t=1 积分速度场                                        │
│   - 初始状态 x(0) = 源细胞状态                                      │
│   - 迭代: 速度 = v(t, x(t), c); x(t+dt) = x(t) + 速度 * dt          │
│   - 求解方法: Euler方法 或 RK4 (Runge-Kutta)                        │
│   - 最终输出: x(1) = 预测的目标细胞状态                             │
│   - 特点: 连续轨迹可解释，支持中间状态插值                          │
│                                                                     │
│   输出层:                                                           │
│   - 预测细胞状态B (扰动后基因表达)                                  │
│   - 演化轨迹 (中间状态序列)                                         │
│   - 预测不确定性 (方差估计)                                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 技术特点
- **神经最优传输估计器**: 学习最优传输映射
- **条件流匹配**: 基于扰动条件生成细胞状态
- **连续插值**: 支持细胞状态之间的连续插值

## 输入/输出

### 输入格式
| 输入类型 | 描述 | 维度 |
|---------|------|------|
| **细胞状态** | 基因表达向量 | 20,000+ 基因 |
| **扰动条件** | 化学/遗传扰动标识 | One-hot编码 |
| **时间参数** | 扰动时间 | 标量 |

### 输出格式
| 输出类型 | 描述 | 维度 |
|---------|------|------|
| **预测细胞状态** | 扰动后基因表达 | 20,000+ 基因 |
| **轨迹** | 中间状态序列 | T × 20,000 |
| **不确定性** | 预测方差 | 20,000 |

## 训练数据

| 数据属性 | 规模 |
|---------|------|
| **化学扰动** | BBBC021数据集 |

| **组合扰动** | JUMP数据集 |
| **细胞类型** | 多种 |

## 训练策略

### 预训练
- **数据集**: 大规模单细胞数据
- **任务**: 学习细胞状态流形

### 微调
- **数据集**: 特定扰动数据
- **任务**: 适应特定扰动类型

### 优化器
- **类型**: Adam
- **学习率**: 1e-4
- **批次大小**: 256

## 功能特点

- 单药预测: 预测单药扰动响应
- 组合药物: 预测药物组合效应
- 基因扰动: 预测基因扰动表型
- 发育建模: 模拟发育过程
- 连续插值: 细胞状态连续转换

## 应用场景

1. **药物发现**
   - 预测新药效果
   - 优化药物组合

2. **疾病建模**
   - 模拟疾病进展
   - 预测治疗响应

3. **细胞工程**
   - 优化分化协议
   - 预测重编程结果

## 优势与局限

### 优势
- 生成式建模，可解释性强
- 连续插值能力
- 支持多种扰动类型
- 开源，社区支持

### 局限
- 计算成本较高
- 需要大量训练数据
- ODE求解时间较长
