## 核心思想

CellFlux 是一种基于**流模型（Flow-based Model）**的单细胞扰动预测方法。流模型通过**可逆神经网络**学习从简单分布（如高斯分布）到复杂数据分布的映射，实现精确的密度估计和高质量的样本生成。

核心创新：
1. **可逆变换**: 通过可逆神经网络实现双向映射
2. **精确密度估计**: 可计算精确的对数似然
3. **条件生成**: 以扰动条件为输入生成预测
4. **连续时间建模**: 可建模细胞状态的连续变化

## 算法流程

### 训练阶段

1. **数据准备**: 收集对照-扰动细胞对
2. **流层构建**: 堆叠多个可逆耦合层
3. **前向传播**: 将数据映射到潜在空间
4. **密度计算**: 利用变量变换公式计算对数似然
5. **优化**: 最大化数据的对数似然

### 推理阶段

1. **采样**: 从标准高斯分布采样潜在向量 z
2. **条件输入**: 提供扰动条件
3. **逆向传播**: 通过逆变换生成预测细胞
4. **输出**: 扰动后的基因表达预测

## Perturbation 预测能力

### 核心功能

| 功能 | 描述 |
|------|------|
| **精确密度估计** | 可计算预测的概率密度 |
| **高质量生成** | 流模型生成高质量样本 |
| **条件控制** | 以扰动条件控制生成过程 |
| **不确定性量化** | 通过潜在空间采样估计不确定性 |

### 适用场景

- **单细胞扰动预测**: 预测基因敲除/过表达效应
- **细胞状态转换**: 建模细胞状态的连续变化
- **虚拟细胞实验**: 模拟未见扰动的效应
- **密度估计**: 估计细胞群体的概率分布

## 优势与局限

### 优势
- 精确的密度估计和对数似然计算
- 可逆性允许双向映射
- 生成样本质量高
- 可量化预测不确定性

### 局限
- 可逆架构限制网络设计
- 计算成本较高
- 需要配对的训练数据
- 对复杂分布建模能力有限

## 典型应用

1. **单细胞扰动预测**: 预测基因扰动后的细胞状态
2. **细胞状态建模**: 建模细胞状态的连续演化
3. **虚拟实验**: 模拟未见扰动的效应
4. **密度估计**: 估计细胞群体的分布
