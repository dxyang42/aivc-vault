# CINEMA-OT

> Causal identification of single-cell experimental perturbation effects

## 基本信息

| 属性 | 内容 |
|------|------|
| **方法名称** | CINEMA-OT (Causal INference with EMBedding and Optimal Transport) |
| **发表年份** | 2023-2024 |
| **期刊** | Nature Methods |
| **核心算法** | Causal Inference + Optimal Transport |
| **任务类型** | 单细胞水平因果推断框架 |
| **论文链接** | https://www.nature.com/articles/s41592-023-02040-5 |

## 核心思想

CINEMA-OT是一个**单细胞水平的因果推断框架**，结合嵌入学习和最优传输理论，从观测数据中识别实验扰动的因果效应。该方法能够在单细胞分辨率下估计扰动效应，同时处理细胞异质性和技术噪声。

### 关键创新

1. **单细胞因果推断**: 在单细胞水平而非群体水平估计效应
2. **最优传输匹配**: 使用OT找到最佳控制-扰动细胞匹配
3. **嵌入空间学习**: 学习低维表示空间进行因果推断
4. **异质性建模**: 显式处理细胞间异质性

## 架构设计

```
┌─────────────────────────────────────────────────────────────────────┐
│                   CINEMA-OT 架构概览                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  数据输入层                                                          │
│  ├── 控制组细胞: X_control = {x_1, x_2, ..., x_n}                 │
│  └── 扰动组细胞: X_treated = {y_1, y_2, ..., y_m}                 │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              嵌入学习层 (Embedding Learning)                 │    │
│  │  ├── 目标: 学习低维表示 z = f(x)                            │    │
│  │  ├── 约束1: 保留生物学变异 (细胞类型、状态)                 │    │
│  │  ├── 约束2: 匹配空间平滑 (相似细胞有相似表示)               │    │
│  │  └── 方法: VAE / 对比学习 / 自编码器                        │    │
│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  嵌入学习策略:                      │               │    │
│  │       │  • 预训练: 在大规模数据上学习通用   │               │    │
│  │       │    细胞表示                         │               │    │
│  │       │  • 微调: 在目标数据集上适配         │               │    │
│  │       │  • 约束: 保持细胞类型分离的同时     │               │    │
│  │       │    允许扰动信号通过                 │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              最优传输匹配层 (Optimal Transport)              │    │
│  │  ├── 目标: 为每个扰动细胞找到最相似的控制细胞               │    │
│  │  ├── 成本矩阵: C_ij = distance(z_control_i, z_treated_j)    │    │
│  │  ├── 优化: min sum(T_ij * C_ij)                             │    │
│  │  │   s.t. sum_j T_ij = a_i (控制细胞权重)                  │    │
│  │  │        sum_i T_ij = b_j (扰动细胞权重)                  │    │
│  │  └── 输出: 最优传输计划 T*                                  │    │
│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  OT匹配优势:                        │               │    │
│  │       │  • 全局最优: 考虑所有细胞间的匹配   │               │    │
│  │       │  • 软匹配: 允许部分匹配处理异质性   │               │    │
│  │       │  • 可解释: 匹配权重表示相似度       │               │    │
│  │       │  • 平衡: 自动处理样本量不平衡       │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              因果效应估计层                                  │    │
│  │  ├── 细胞水平效应:                                          │    │
│  │  │   delta_j = y_j - sum_i T*_ij * x_i / sum_i T*_ij       │    │
│  │  │   (扰动细胞与其匹配控制细胞的加权平均的差异)            │    │
│  │  ├── 基因水平效应: 每个基因的响应                           │    │
│  │  ├── 通路水平效应: 功能通路的富集变化                       │    │
│  │  └── 不确定性: 基于匹配质量的效应置信度                     │    │
│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  效应分解:                          │               │    │
│  │       │  总效应 = 直接效应 + 间接效应       │               │    │
│  │       │  • 直接: 扰动靶点的即时变化         │               │    │
│  │       │  • 间接: 级联调控网络的变化         │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              统计推断层                                      │    │
│  │  ├── 显著性检验: 哪些基因的效应显著不为零                   │    │
│  │  ├── 多重检验校正: FDR控制                                  │    │
│  │  ├── 置信区间: 效应大小的置信范围                           │    │
│  │  └── 敏感性分析: 对嵌入选择和OT参数的稳健性                 │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  输出层: 单细胞因果效应图谱 + 统计显著性报告                           │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## 最优传输匹配详解

### 为什么使用OT?

```
传统匹配方法的问题:
├── 最近邻匹配: 只考虑单个最佳匹配，忽略全局结构
├── 倾向得分匹配: 需要预先定义匹配变量
└── 精确匹配: 难以处理高维数据

最优传输的优势:
├── 全局最优: 同时优化所有细胞的匹配
├── 几何感知: 考虑数据流形的几何结构
├── 软分配: 允许概率性匹配
└── 计算高效: 可使用熵正则化加速
```

### OT求解

```
熵正则化OT:
min_T <T, C> - epsilon * H(T)

其中:
- T: 传输计划 (匹配矩阵)
- C: 成本矩阵 (细胞间距离)
- epsilon: 熵正则化系数
- H(T): 传输计划的熵

求解: 使用Sinkhorn迭代算法
```

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 控制组 | 未扰动细胞的基因表达 |
| 扰动组 | 扰动后细胞的基因表达 |
| 嵌入模型 | 预训练或自定义的细胞嵌入模型 |

### 输出
| 类型 | 描述 |
|------|------|
| 效应矩阵 | 每个细胞每个基因的因果效应 |
| DE基因 | 差异表达基因列表及显著性 |
| 匹配矩阵 | 控制-扰动细胞的OT匹配 |
| 可视化 | UMAP/t-SNE上的效应展示 |

## 优势与局限

### 优势
- 单细胞分辨率，捕捉异质性
- 全局最优匹配，减少选择偏差
- 不依赖强参数假设
- 可解释性强，匹配结果可视化

### 局限
- 需要足够的细胞数量进行可靠匹配
- 嵌入质量直接影响结果
- 计算成本随细胞数增加
- 难以处理极端异质性

## 应用场景

1. **CRISPR筛选分析**: 估计每个扰动的单细胞效应
2. **药物响应研究**: 识别响应/不响应细胞亚群
3. **发育轨迹分析**: 追踪扰动对分化路径的影响
4. **批次效应校正**: 作为因果推断的批次校正方法
