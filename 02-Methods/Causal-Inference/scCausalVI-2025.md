> scCausalVI disentangles single-cell perturbation responses with causality-aware generative modeling

## 核心思想

scCausalVI-2025是一种**因果感知生成模型**，能够分离单细胞数据中的固有生物变异和治疗诱导的扰动效应。通过引入因果推断框架，该方法可以准确估计扰动的因果效应，同时控制混淆因素。

### 关键创新

1. **因果感知编码**: 在VAE中引入因果结构约束
2. **变异分解**: 分离固有变异和扰动效应
3. **混淆控制**: 识别并控制潜在的混淆变量
4. **效应量化**: 精确估计治疗的因果效应

```

│                   scCausalVI-2025 架构概览                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  因果模型定义层                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              因果变量识别                                    │    │
│  │  ├── Z_inherent: 固有细胞变异 (与扰动无关)                  │    │
│  │  │   └── 细胞周期、分化状态、微环境等                      │    │
│  │  ├── Z_treatment: 治疗诱导变异 (因果效应)                   │    │

│  │  ├── Z_confounder: 混淆变量 (影响两者)                      │    │
│  │  │   └── 批次效应、技术变异、细胞类型等                   │    │

│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  因果结构:                          │               │    │
│  │       │   Z_confounder                      │               │    │
│  │       │      /    \                         │               │    │
│  │       │     ↓      ↓                        │               │    │
│  │       │  Z_inherent  Z_treatment            │               │    │
│  │       │     \      /                        │               │    │
│  │       │      ↓    ↓                         │               │    │
│  │       │        X (观测)                     │               │    │
│  │       │                                     │               │    │
│  │       │  目标: 分离Z_inherent和Z_treatment  │               │    │
│  │       │       控制Z_confounder              │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              因果感知编码器                                  │    │
│  │  ├── 输入: 基因表达 X + 扰动标签 T                          │    │
│  │  ├── 推断:                                                │    │
│  │  │   q(Z_inherent | X)      - 固有变异后验                 │    │
│  │  │   q(Z_treatment | X, T)  - 治疗效应后验                 │    │
│  │  │   q(Z_confounder | X)    - 混淆变量后验                 │    │
│  │  └── 因果约束:                                            │    │
│  │       Z_inherent ⊥ Z_treatment | Z_confounder             │    │

│  │       │  • 因果正则化: 强制满足独立性约束 │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              因果解码器                                      │    │
│  │  ├── 输入: Z_inherent, Z_treatment, Z_confounder            │    │

│  │  └── 反事实生成:                                           │    │
│  │       X_cf = f(Z_inherent, Z_treatment', Z_confounder)     │    │

│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  解码器特点:                        │               │    │
│  │       │  • 显式建模三种变量的贡献           │               │    │
│  │       │  • 可解释: 知道每个基因变化的原因   │               │    │
│  │       │  • 可控: 可单独操控某类变异         │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │

│  │  ├── 平均处理效应 (ATE):                                   │    │

│  │  ├── 条件平均处理效应 (CATE):                              │    │
│  │  │   CATE(z) = E[X | do(T=1), Z_inherent=z] -             │    │
│  │  │             E[X | do(T=0), Z_inherent=z]               │    │
```

## 因果效应估计

### 三种效应指标

```
总体效应 (ATE):
├── 计算: 所有细胞的平均处理效应
├── 解释: 扰动对整体群体的平均影响
└── 用途: 评估扰动的整体效果

条件效应 (CATE):
├── 计算: 特定细胞亚群的平均处理效应
├── 解释: 不同细胞类型/状态下的差异化响应
└── 用途: 识别响应特异性，指导精准治疗

个体效应 (ITE):
├── 计算: 单个细胞的反事实差异
├── 解释: 该细胞如果接受/不接受扰动的差异
└── 用途: 个性化预测，患者分层
```

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 表达数据 | 单细胞基因表达矩阵 |
| 扰动标签 | 每个细胞的扰动条件 |
| 协变量 | 批次、细胞类型等 |

### 输出
| 类型 | 描述 |
|------|------|
| 固有表示 | Z_inherent: 细胞固有状态 |
| 治疗表示 | Z_treatment: 扰动诱导变异 |
| 因果效应 | ATE/CATE/ITE估计 |
| 反事实 | 反事实条件下的预测 |

## 优势与局限

### 优势
- 精确分离固有变异和扰动效应
- 提供可解释的因果效应估计
- 控制混淆因素，减少偏差
- 支持反事实推理

### 局限
- 需要足够的样本量学习因果结构
- 未观测混淆变量仍可能影响结果
- 因果假设(如条件独立性)可能不完全成立
- 计算复杂度较高

## 应用场景

1. **药物机制研究**: 区分药物直接效应和细胞固有响应
2. **异质性分析**: 理解细胞间响应差异的原因
3. **生物标志物发现**: 识别与响应相关的固有特征
4. **实验设计优化**: 识别和控制关键混淆因素
