# CausCell-2025

> Causal disentanglement for single-cell representations and controllable counterfactual generation

## 核心思想

CausCell-2025将**因果关系整合到扩散模型**中，实现了可控的反事实细胞生成。通过因果解耦表示学习，该方法能够分离混淆因素和真实生物信号，生成符合因果结构的反事实样本。

### 关键创新

1. **因果扩散模型**: 在扩散过程中注入因果约束
2. **结构因果模型(SCM)**: 显式建模变量间的因果关系
3. **反事实生成**: 回答"如果...会怎样"的因果问题
4. **可控干预**: 精确控制生成过程中的干预类型

## 架构设计

```
┌─────────────────────────────────────────────────────────────────────┐
│                   CausCell-2025 架构概览                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  结构因果模型 (SCM) 层                                               │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              因果图定义                                      │    │
│  │  ├── 外生变量 (噪声): U_cell, U_perturb, U_expr             │    │
│  │  ├── 内生变量: Cell Type, Perturbation, Gene Expression     │    │
│  │  └── 因果函数:                                              │    │
│  │       Cell Type = f_C(U_cell)                               │    │
│  │       Perturbation = f_P(U_perturb)                         │    │
│  │       Expression = f_E(Cell Type, Perturbation, U_expr)     │    │
│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  因果结构:                          │               │    │
│  │       │    U_cell → Cell Type ─┐            │               │    │
│  │       │                        ├──→ Expression│             │    │
│  │       │  U_perturb → Perturbation─┘          │               │    │
│  │       │       ↑                               │               │    │
│  │       │   (干预点)                            │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              因果编码器 (Causal Encoder)                     │    │
│  │  ├── 将观测数据映射到因果变量空间                           │    │
│  │  ├── 分离: 细胞类型表示 z_c                                 │    │
│  │  ├── 分离: 扰动表示 z_p                                     │    │
│  │  └── 分离: 噪声/混杂表示 z_n                                │    │
│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  解耦约束:                          │               │    │
│  │       │  • z_c ⊥ z_p (条件独立)             │               │    │
│  │       │  • z_c ⊥ z_n                        │               │    │
│  │       │  • z_p ⊥ z_n                        │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              因果扩散模型 (Causal Diffusion)                 │    │
│  │  ├── 前向过程: 逐步添加噪声 (保持因果结构)                  │    │
│  │  ├── 反向过程: 去噪生成样本 (遵循因果约束)                  │    │
│  │  └── 干预机制: do-算子实现硬干预                            │    │
│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  因果扩散特点:                      │               │    │
│  │       │  • 每个时间步保持变量间因果关系     │               │    │
│  │       │  • 干预通过修改中间表示实现         │               │    │
│  │       │  • 生成样本符合因果一致性           │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │              反事实生成器                                    │    │
│  │  ├── 观测事实: 细胞A经扰动X后的表达                         │    │
│  │  ├── 干预: do(Perturbation = Y)                             │    │
│  │  └── 反事实: "如果扰动是Y而不是X，表达会怎样"               │    │
│  │       ┌─────────────────────────────────────┐               │    │
│  │       │  三种因果推理:                      │               │    │
│  │       │  1. 关联: P(Y|X) 看到X时Y的概率     │               │    │
│  │       │  2. 干预: P(Y|do(X)) 做X后Y的概率   │               │    │
│  │       │  3. 反事实: P(Y_x|X',Y') 如果做过X  │               │    │
│  │       │     Y会是多少 (已知实际X',Y')       │               │    │
│  │       └─────────────────────────────────────┘               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              ↓                                       │
│  输出层: 反事实细胞表达谱 + 因果效应估计                               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## 因果干预机制

### do-算子实现

```
干预类型:
├── 单基因干预: do(Gene_i = knocked_out)
├── 多基因干预: do(Genes = {ko1, ko2, ...})
├── 药物干预: do(Drug = active)
└── 组合干预: do({Gene: ko, Drug: active})

干预实现:
1. 在因果编码器中固定干预变量的表示
2. 保持其他因果变量的分布不变
3. 通过扩散模型生成符合干预条件的样本
4. 确保生成样本遵循因果图的约束
```

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 观测数据 | 单细胞基因表达矩阵 |
| 因果图 | 预定义或学习的因果结构 |
| 干预目标 | 要执行的干预操作 |
| 参考细胞 | 反事实推理的参考点 |

### 输出
| 类型 | 描述 |
|------|------|
| 反事实样本 | 干预后的预测细胞状态 |
| 因果效应 | 干预的因果效应估计 |
| 解耦表示 | 分离的因果变量表示 |
| 不确定性 | 反事实预测的置信度 |

## 优势与局限

### 优势
- 生成符合因果结构的反事实样本
- 高可解释性的解耦表示
- 支持多种干预类型的灵活组合
- 不确定性量化提供可靠性评估

### 局限
- 依赖因果图的正确性
- 复杂因果结构学习困难
- 扩散模型计算成本较高
- 需要大量数据进行因果发现

## 应用场景

1. **虚拟筛选**: 预测未测试扰动的效应
2. **机制研究**: 理解基因调控的因果关系
3. **实验设计**: 指导优先验证的假设
4. **个性化治疗**: 预测患者特异性响应
