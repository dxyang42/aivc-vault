# DrugCell

> [[方法地图]] | [[团队地图]]

## 1. 基本信息

| 属性 | 详情 |
|------|------|
| **模型名称** | DrugCell |
| **发布时间** | 2021年 |
| **机构** | Stanford University (Ideker Lab) |
| **GitHub** | https://github.com/idekerlab/DrugCell |
| **论文** | Kuenzi et al., Nature Cancer 2021 |
| **类型** | Visible Neural Network (VNN) |
| **定位** | 可解释的药物响应预测模型 |

DrugCell 是一个基于可见神经网络 (Visible Neural Network) 的深度学习模型，专门用于预测癌细胞对药物的反应。该模型的核心创新在于将神经网络结构与基因本体 (GO) 的层次结构对齐，使模型具有高度可解释性。

## 2. 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单药物处理** | 完全支持 | 预测单一药物响应 |
| **药物组合** | 扩展支持 | 预测协同/拮抗效应 |
| **基因突变** | 支持 | 整合基因型信息 |
| **细胞系特异性** | 核心优势 | 不同癌症类型响应 |
| **剂量响应** | 支持 | 多剂量水平预测 |

### 评估指标

| 指标 | 结果 | 说明 |
|------|------|------|
| **AUC预测** | Pearson r > 0.85 | 药物敏感性预测 |
| **分类准确率** | > 80% | 敏感/耐药分类 |
| **可解释性** | 高 | GO term级别的解释 |
| **泛化能力** | 良好 | 跨细胞系预测 |

### 可见神经网络特点

| 特点 | 说明 |
|------|------|
| **结构可见** | 网络节点对应生物学实体 (GO terms) |
| **层次传播** | 信号从基因层向细胞过程层传播 |
| **可解释性** | 每个节点的激活值有生物学意义 |
| **知识引导** | 网络结构由GO知识图谱决定 |

## 5. 与其他药物预测方法的比较

| 特性 | DrugCell | DeepDR | scPPDM | X-Pert |
|------|----------|--------|--------|--------|
| **架构** | VNN | CNN | Diffusion | Transformer |
| **可解释性** | 极高 | 低 | 中等 | 中等 |
| **GO整合** | 显式 | 无 | 无 | 无 |
| **突变输入** | 支持 | 支持 | 不支持 | 支持 |
| **单细胞** | 不支持 | 不支持 | 支持 | 支持 |
| **机制洞察** | 强 | 弱 | 弱 | 中等 |

## 7. 应用场景

### 药物发现

| 应用 | 描述 |
|------|------|
| **虚拟筛选** | 预测化合物库中潜在活性分子 |
| **药物组合** | 发现协同作用的药物对 |
| **耐药预测** | 预测和解释耐药性发展 |
| **生物标志物** | 识别响应预测标志物 |

### 精准医学

| 应用 | 描述 |
|------|------|
| **患者分层** | 基于分子特征预测治疗响应 |
| **方案选择** | 为特定患者推荐最佳治疗方案 |
| **新适应症** | 发现现有药物的新用途 |

## 8. 优势与局限

### 核心优势

| 优势 | 说明 |
|------|------|
| **高可解释性** | GO级别的机制解释 |
| **知识引导** | 整合先验生物学知识 |
| **结构约束** | GO层次约束减少过拟合 |
| **多组学整合** | 支持突变、表达等多模态输入 |
| **开源** | 完整开源实现 |

### 当前局限

| 局限 | 说明 | 潜在改进方向 |
|------|------|-------------|
| **批量数据** | 主要基于批量细胞系数据 | 扩展至单细胞 |
| **静态预测** | 不建模时间动态 | 引入时序组件 |
| **GO覆盖** | 依赖GO注释完整性 | 整合其他知识库 |
| **泛化能力** | 对未见药物泛化有限 | 改进分子编码 |
