# STACK

## 核心思想

STACK是一种基于表格注意力架构的大规模编码器-解码器基础模型，专门设计用于模拟细胞条件。该模型在1.5亿个统一预处理的单细胞数据上进行训练，支持上下文学习（in-context learning），能够通过无标注细胞预测任意条件（如化学扰动或不同供体）对目标细胞群体的影响。

STACK的核心创新在于将单细胞数据视为表格形式，利用表格注意力机制捕获细胞间的复杂关系。这种架构允许模型在推理时接收任意细胞群体作为上下文，并预测该群体在不同条件下的响应，无需针对特定任务进行微调。

## 训练数据与规模

- **训练数据量**: 1.5亿个统一预处理的单细胞
- **架构类型**: 编码器-解码器Transformer
- **核心机制**: 表格注意力（Table Attention）
- **关键能力**: 上下文学习（In-context Learning）

## 关键特性

1. **上下文学习**: 无需微调即可预测新条件下的细胞响应
2. **表格注意力**: 专门设计用于处理单细胞数据的表格结构
3. **条件模拟**: 支持化学扰动、遗传扰动、供体差异等多种条件预测
4. **统一预处理**: 所有训练数据经过标准化预处理，确保一致性

## 优缺点分析

### 优点

- **强大的泛化能力**: 上下文学习能力使其能够快速适应新任务
- **无需微调**: 对于新条件预测，无需额外的模型训练
- **大规模训练**: 1.5亿细胞的训练数据提供了丰富的生物学知识
- **灵活的输入**: 可以接受任意细胞群体作为上下文

### 缺点

- **计算资源需求**: 大规模模型需要显著的计算资源进行推理
- **数据预处理依赖**: 需要严格的统一预处理流程
- **黑盒特性**: 注意力机制的内部工作机制可解释性有限
- **训练成本**: 大规模预训练需要昂贵的计算资源
