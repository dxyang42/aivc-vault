# STATE

## 核心思想

STATE（State Transformer for perturbation prediction）是Arc Institute开发的单细胞扰动预测模型，基于Transformer架构，在2.7亿个细胞上进行训练。它是首个在虚拟细胞领域持续超越简单线性方法的深度学习模型，通过细胞集合操作捕获群体水平和细胞水平的扰动效应。

STATE的核心创新在于将扰动预测任务建模为细胞状态的转移问题。模型学习从控制细胞状态到扰动后细胞状态的映射，同时考虑细胞类型、扰动类型等上下文信息。通过大规模预训练，STATE能够捕获复杂的非线性扰动响应模式。

## 训练数据与规模

- **训练数据量**: 2.7亿个单细胞
- **架构类型**: Transformer
- **关键突破**: 首个持续超越线性方法的深度学习模型
- **预测粒度**: 群体水平 + 细胞水平双重预测

## 关键特性

1. **大规模预训练**: 2.7亿细胞的训练规模提供强大的先验知识
2. **双重预测**: 同时预测群体平均响应和单个细胞响应
3. **上下文感知**: 考虑细胞类型、状态等上下文信息
4. **非线性建模**: 捕获复杂的基因调控关系

## 优缺点分析

### 优点

- **性能领先**: 首个在虚拟细胞领域持续超越线性方法的模型
- **大规模训练**: 2.7亿细胞的训练数据提供丰富的生物学知识
- **双重粒度**: 同时支持群体水平和细胞水平预测
- **上下文感知**: 能够考虑细胞类型和状态信息

### 缺点

- **计算成本高**: 大规模Transformer模型训练和推理成本较高
- **数据需求大**: 需要大量标注的扰动数据进行训练
- **可解释性有限**: 注意力机制的内部决策过程难以解释
- **领域局限**: 主要在特定细胞类型上验证，泛化能力待验证
