# CellFM

## 基本信息

- **全称**: CellFM: a large-scale foundation model pre-trained on transcriptomics data
- **发表时间**: 2025年
- **发表期刊**: Nature Communications
- **机构**: 华为诺亚方舟实验室、清华大学
- **作者**: Xiangzhe Zeng, Xiangyu Yue, et al.

## 核心技术

CellFM 是目前最大的单细胞基础模型之一，具有以下核心特点：

1. **超大规模参数**: 8亿参数，比现有最大单物种模型大8倍
2. **海量训练数据**: 基于1亿个人类单细胞数据进行预训练
3. **创新架构**: 采用改进的 RetNet 框架，在 MindSpore 上训练
4. **卓越性能**: 在细胞注释、扰动预测、基因功能预测等任务上超越现有方法

### RetNet 机制详解

CellFM 采用 RetNet (Retention Network) 作为核心架构：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Retention Mechanism                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   标准 Transformer 注意力:                                          │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │  计算过程:                                                   │  │
│   │  1. 对每对位置计算Query和Key的点积                          │  │
│   │  2. 除以维度平方根缩放                                       │  │
│   │  3. Softmax归一化                                            │  │
│   │  4. 与Value矩阵相乘                                          │  │
│   │  复杂度: 序列长度平方，随长度增加呈二次增长                 │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   Retention 机制:                                                   │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │  双形式计算:                                                │  │
│   │                                                             │  │
│   │  1. 并行形式 (训练):                                        │  │
│   │     - 使用矩阵乘法一次性计算整个序列                        │  │
│   │     - 引入衰减矩阵: 位置越远衰减越大                        │  │
│   │     - 保持GPU并行训练效率                                   │  │
│   │                                                             │  │
│   │  2. 递归形式 (推理):                                        │  │
│   │     - 维护一个状态矩阵                                      │  │
│   │     - 每步更新: 新状态 = 衰减因子 × 旧状态 + 新信息        │  │
│   │     - 计算复杂度与序列长度成线性关系                        │  │
│   │                                                             │  │
│   │  衰减因子是可学习的参数，控制历史信息的保留程度             │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   优势总结:                                                         │
│   - 训练效率: 保持并行化训练                                        │
│   - 推理效率: 线性复杂度，适合长序列                                │
│   - 性能: 在单细胞数据上表现优异                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 预训练任务

### 任务1: 掩码基因建模 (Masked Gene Modeling)

```
输入:  原始表达 [0.2, 0.0, 0.5, MASK, 0.1, 0.0, 0.8]
                                      ↑
预测:  被掩码位置的表达值
损失:  预测值与真实值的均方误差
```

### 任务2: 细胞类型预测 (Cell Type Classification)

```
输入:  基因表达序列
输出:  细胞类型概率分布
损失:  预测分布与真实标签的交叉熵
```

### 任务3: 组织来源预测 (Tissue Origin Prediction)

```
输入:  基因表达序列
输出:  组织类型概率分布
损失:  预测分布与真实标签的交叉熵
```

## 模型配置

| 参数 | CellFM | scFoundation | scGPT |
|-----|--------|--------------|-------|
| 参数量 | 800M | 100M | 51M-330M |
| 训练数据 | 100M 细胞 | 50M 细胞 | 33M 细胞 |
| 架构 | RetNet | Transformer | Transformer |
| 层数 | 48 | 24 | 12-24 |
| 隐藏维度 | 2048 | 1024 | 512-1024 |
| 注意力头数 | 32 | 16 | 8-16 |
| 输出维度 | 2048 | 1024 | 512 |

## 输入/输出

- **输入**:
  - 单细胞基因表达数据
  - 基因ID序列
  - 可选: 细胞/组织类型标记
- **输出**:
  - 2048 维细胞嵌入
  - 基因表达预测
  - 细胞类型预测
  - 基因功能预测

## 性能指标

CellFM 在多个下游任务上达到 SOTA：

| 任务 | 数据集 | CellFM | scFoundation | scGPT |
|-----|--------|--------|--------------|-------|
| 细胞注释 | Zheng68K | 96.5% | 94.2% | 95.1% |
| 细胞注释 | Human Liver | 95.8% | 93.2% | 94.5% |
| 扰动预测 | Norman 2021 | AUPRC 0.48 | 0.42 | 0.45 |
| 基因功能 | GO 预测 | AUC 0.91 | 0.87 | 0.89 |
| 批次校正 | kBET | 0.94 | 0.91 | 0.93 |

## 功能特点

1. **超大规模预训练**: 1亿细胞数据
2. **高效推理**: RetNet 架构支持线性复杂度推理
3. **多任务能力**: 细胞注释、扰动预测、基因功能预测
4. **跨组织泛化**: 支持多种组织类型
5. **开源模型**: 提供预训练权重

## 应用场景

- 大规模细胞图谱注释
- 药物扰动效应预测
- 基因功能预测
- 细胞状态分类
- 跨数据集整合

## 优势与局限

**优势**:
- 目前最大的单细胞基础模型
- RetNet 架构兼顾训练和推理效率
- 海量训练数据带来更强泛化能力
- 多任务性能全面领先

**局限**:
- 模型体积大，推理需要较大内存
- 主要针对人类数据
- 训练成本极高
