# Systema

> A framework for evaluating genetic perturbation response prediction beyond systematic variation

## 核心思想

Systema是一个**强调扰动特异性效应的评估框架**，超越了传统的系统性变异评估。该框架引入了可解释的能力读数(interpretable capability readouts)，能够区分模型的真实扰动预测能力与系统性偏差，提供更准确的模型性能评估。

### 关键创新

1. **扰动特异性评估**: 分离扰动特异性效应与系统性偏差
2. **能力读数**: 可解释的模型能力指标
3. **偏差校正**: 识别并校正评估中的系统性偏差
4. **细粒度分析**: 按扰动类型、细胞状态等维度细分评估

## 系统性偏差 vs 扰动特异性

### 概念区分

```
系统性变异 (Systematic Variation):
├── 与扰动无关的变异
├── 例子:
│   ├── 细胞类型A vs 细胞类型B的基础差异
│   ├── 批次1 vs 批次2的技术差异
│   └── 细胞周期G1 vs S期的固有差异
└── 问题: 模型可能只学习这些系统性模式
         而不是真正的扰动效应

扰动特异性变异 (Perturbation-Specific):
├── 扰动直接引起的变异
├── 例子:
│   ├── 敲除基因X导致的下游基因变化
│   ├── 药物Y作用靶点的特异性响应
│   └── 特定通路激活引起的级联效应
└── 目标: 模型应该主要捕捉这类变异
```

### 评估示例

```
场景: 评估模型在Norman数据集上的性能

传统评估:
├── 总体R2 = 0.85 (看起来很好)
└── 结论: 模型性能优秀

Systema评估:
├── 总体R2 = 0.85
├── 系统性R2 = 0.80 (解释了大部分变异)
├── 特异性R2 = 0.05 (只解释了少量特异性变异)
└── 结论: 模型主要学习系统性模式，
         对扰动特异性效应预测能力弱
```

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 预测值 | 模型预测的扰动后表达 |
| 真实值 | 实验观测的扰动后表达 |
| 元数据 | 细胞类型、批次、扰动类型等 |
| 控制值 | 未扰动对照表达 |

### 输出
| 类型 | 描述 |
|------|------|
| 能力读数 | 特异性、偏差、泛化、鲁棒性 |
| 分层报告 | 各子集的性能分析 |
| 可视化 | 能力雷达图、分层热力图 |
| 建议 | 模型改进方向 |

## 优势与局限

### 优势
- 区分真实能力与系统性偏差
- 可解释的能力读数
- 细粒度的分层分析
- 指导模型改进

### 局限
- 需要详细的元数据
- 变异分解依赖统计假设
- 计算成本较高
- 读数解释需要领域知识

## 应用场景

1. **模型开发**: 指导新模型的设计和训练
2. **模型选择**: 选择真正理解生物机制的方法
3. **失败分析**: 诊断模型预测失败的原因
4. **基准测试**: 更公平的模型比较
