## 核心技术

### 架构设计

scSHAP将**博弈论中的Shapley值**应用于单细胞深度学习模型的解释：

```
┌─────────────────────────────────────────────────────────────────────┐
│                  scSHAP 可解释性分析架构                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  核心概念: Shapley值 = 特征对模型预测的贡献度                        │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              背景: SHAP理论基础                              │   │
│  │                                                             │   │
│  │  Shapley值来自合作博弈论:                                   │   │
│  │  - 玩家 = 特征 (基因)                                       │   │
│  │  - 联盟 = 特征子集                                          │   │
│  │  - 收益 = 模型预测                                          │   │
│  │                                                             │   │
│  │  Shapley值公理:                                             │   │
│  │  1. 效率性: 所有特征贡献之和 = 预测值 - 基线值              │   │
│  │  2. 对称性: 相同贡献的特征有相同Shapley值                   │   │
│  │  3. 虚拟性: 不改变预测的特征贡献为0                         │   │
│  │  4. 可加性: 独立游戏的Shapley值可相加                       │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              步骤1: 模型准备                                 │   │
│  │                                                             │   │
│  │  输入: 训练好的单细胞深度学习模型                           │   │
│  │  ┌─────────────────────────────────────────────────────┐   │   │
│  │  │                                                     │   │   │
│  │  │  支持模型类型:                                      │   │   │
│  │  │  - scGPT (Transformer)                              │   │   │
│  │  │  - scBERT (Transformer)                             │   │   │
│  │  │  - scFoundation (Transformer)                       │   │   │
│  │  │  - Geneformer (Transformer)                         │   │   │
│  │  │  - scVI (VAE)                                       │   │   │
│  │  │  - 自定义神经网络                                   │   │   │
│  │  │                                                     │   │   │
│  │  └─────────────────────────────────────────────────────┘   │   │
│  │                                                             │   │
│  │  背景数据:                                                  │   │
│  │  - 用于计算基线预测的参考细胞集                             │   │
│  │  - 通常是训练集的随机样本                                   │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              步骤2: Shapley值计算                            │   │
│  │                                                             │   │
│  │  对目标细胞的每个基因特征:                                  │   │
│  │                                                             │   │
│  │  ┌─────────────────────────────────────────────────────┐   │   │
│  │  │                                                     │   │   │
│  │  │  Shapley值公式:                                     │   │   │
│  │  │                                                     │   │   │
│  │  │  φ(i) = Σ |S|!(n-|S|-1)!/n! × [v(S∪{i}) - v(S)]    │   │   │
│  │  │                                                     │   │   │
│  │  │  其中:                                              │   │   │
│  │  │  - S: 不包含特征i的特征子集                         │   │   │
│  │  │  - n: 总特征数                                      │   │   │
│  │  │  - v(S): 使用特征集S的模型预测                      │   │   │
│  │  │                                                     │   │   │
│  │  │  近似算法 (KernelSHAP):                             │   │   │
│  │  │  - 线性模型: 精确解                                 │   │   │
│  │  │  - 树模型: TreeSHAP (高效)                          │   │   │
│  │  │  - 深度学习: DeepSHAP / GradientSHAP                │   │   │
│  │  │                                                     │   │   │
│  │  └─────────────────────────────────────────────────────┘   │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              步骤3: 解释性分析                               │   │
│  │                                                             │   │
│  │  全局解释 (Global Interpretation):                          │   │
│  │  - 平均SHAP值: 识别最重要的特征                             │   │
│  │  - SHAP摘要图: 特征重要性与取值的关系                       │   │
│  │  - 特征依赖图: 特征间的交互效应                             │   │
│  │                                                             │   │
│  │  局部解释 (Local Interpretation):                           │   │
│  │  - 单个细胞的预测解释                                       │   │
│  │  - 瀑布图: 从基线到预测的贡献分解                           │   │
│  │  - 力图: 正负贡献的可视化                                   │   │
│  │                                                             │   │
│  │  生物学解释:                                                │   │
│  │  - 关键基因识别                                             │   │
│  │  - 通路富集分析                                             │   │
│  │  - 细胞类型特异性标记                                       │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 关键创新

1. **理论基础扎实**
   - 基于博弈论的Shapley值
   - 满足可加性、对称性等公理
   - 解释结果具有数学保证

2. **模型无关性**
   - 适用于任何机器学习模型
   - 支持深度学习、树模型、线性模型
   - 灵活应用于各种单细胞方法

3. **全局+局部解释**
   - 全局: 理解模型整体行为
   - 局部: 解释单个预测
   - 多层次解释能力

## 扰动预测能力

### 核心功能

| 功能 | 描述 | 适用场景 |
|------|------|----------|
| **关键基因识别** | 找出驱动预测的关键基因 | 靶点发现 |
| **预测解释** | 解释为什么做出特定预测 | 结果验证 |
| **模型调试** | 识别模型的偏见和错误 | 模型改进 |
| **特征工程** | 指导特征选择和设计 | 模型优化 |
| **生物学发现** | 从模型中提取生物学知识 | 机制研究 |

## 技术细节

### SHAP计算算法

```
┌─────────────────────────────────────────────────────────────────────┐
│              SHAP计算算法                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. KernelSHAP (模型无关)                                           │
│  ┌─────────────────────────────────────────────────────┐           │
│  │                                                     │           │
│  │  思想: 使用加权线性回归近似Shapley值                │           │
│  │                                                     │           │
│  │  步骤:                                              │           │
│  │  - 采样特征子集                                     │           │
│  │  - 对每个子集，计算模型预测                         │           │
│  │  - 加权线性回归拟合Shapley值                        │           │
│  │                                                     │           │
│  │  复杂度: O(2^n) → 使用采样近似                      │           │
│  │                                                     │           │
│  └─────────────────────────────────────────────────────┘           │
│                                                                     │
│  2. DeepSHAP (深度学习)                                             │
│  ┌─────────────────────────────────────────────────────┐           │
│  │                                                     │           │
│  │  思想: 使用DeepLIFT近似Shapley值                    │           │
│  │                                                     │           │
│  │  特点:                                              │           │
│  │  - 利用神经网络结构                                 │           │
│  │  - 传播贡献度从输出到输入                           │           │
│  │  - 比KernelSHAP更快                                 │           │
│  │                                                     │           │
│  └─────────────────────────────────────────────────────┘           │
│                                                                     │
│  3. GradientSHAP (深度学习)                                         │
│  ┌─────────────────────────────────────────────────────┐           │
│  │                                                     │           │
│  │  思想: 结合梯度信息和Shapley值                      │           │
│  │                                                     │           │
│  │  特点:                                              │           │
│  │  - 使用随机基线样本                                 │           │
│  │  - 计算期望梯度                                     │           │
│  │  - 适用于大规模模型                                 │           │
│  │                                                     │           │
│  └─────────────────────────────────────────────────────┘           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 单细胞特异性考虑

```
┌─────────────────────────────────────────────────────────────────────┐
│              单细胞数据的SHAP应用注意事项                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. 高维性处理                                                      │
│  - 单细胞数据通常有20000+基因                                       │
│  - 使用高变基因子集 (2000-5000)                                     │
│  - 或使用基因模块/通路作为特征                                      │
│                                                                     │
│  2. 稀疏性处理                                                      │
│  - 单细胞数据有大量零值                                             │
│  - 零值可能代表未表达或dropout                                      │
│  - 使用特定基线 (如平均表达)                                        │
│                                                                     │
│  3. 批次效应                                                        │
│  - 确保基线数据与目标细胞来自同一批次                               │
│  - 或使用批次校正后的数据                                           │
│                                                                     │
│  4. 细胞类型特异性                                                  │
│  - 不同细胞类型可能有不同关键基因                                   │
│  - 按细胞类型分别计算SHAP值                                         │
│  - 比较不同细胞类型的解释                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 与其他方法的对比

| 方法 | 理论基础 | 计算效率 | 适用模型 | 特点 |
|------|----------|----------|----------|------|
| **SHAP** | 博弈论 | 中等 | 通用 | 理论基础最扎实 |
| **LIME** | 局部线性 | 快 | 通用 | 局部近似 |
| **Integrated Gradients** | 积分梯度 | 快 | 深度学习 | 满足敏感性公理 |
| **Attention Visualization** | 注意力权重 | 快 | Transformer | 模型特定 |
| **Saliency Maps** | 梯度 | 快 | 深度学习 | 最简单 |
| **Permutation Importance** | 置换检验 | 慢 | 通用 | 全局特征重要性 |

## 代码示例

```python
import shap
import torch

# 加载预训练模型
model = load_scgpt_model()

# 准备数据
background = torch.tensor(background_cells)  # 基线细胞
X = torch.tensor(target_cells)  # 目标细胞

# 创建DeepExplainer
explainer = shap.DeepExplainer(model, background)

# 计算SHAP值
shap_values = explainer.shap_values(X)

# 可视化
# 1. 摘要图 (全局)
shap.summary_plot(shap_values, X, feature_names=gene_names)

# 2. 力图 (局部)
shap.force_plot(
    explainer.expected_value,
    shap_values[0],
    X[0],
    feature_names=gene_names
)

# 3. 依赖图
shap.dependence_plot("GeneA", shap_values, X, feature_names=gene_names)
```
