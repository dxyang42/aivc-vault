## 核心技术

### 架构设计

scGen基于**变分自编码器 (VAE)**：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    scGen VAE 架构                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                      编码器网络 (Encoder)                    │  │
│   │                                                             │  │
│   │   输入: 基因表达向量 x (高维, ~20000维)                      │  │
│   │        ↓                                                    │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │              全连接神经网络层                        │  │  │
│   │   │                                                     │  │  │
│   │   │   Linear ──→ ReLU ──→ Linear ──→ ReLU ──→ ...      │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                            ↓                               │  │
│   │              潜在空间表示 z (低维, ~10-100维)               │  │
│   │                                                             │  │
│   │   变分推断: 学习均值 μ 和对数方差 log(σ²)                   │  │
│   │   重参数化: z = μ + σ * ε,  ε ~ N(0, I)                    │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                    潜在空间向量运算 (核心创新)                │  │
│   │                                                             │  │
│   │   对照细胞潜在向量: z_control                               │  │
│   │   扰动细胞潜在向量: z_perturbed                             │  │
│   │                                                             │  │
│   │   计算扰动向量 δ:                                           │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   δ = mean(z_perturbed - z_control)                 │  │  │
│   │   │                                                     │  │  │
│   │   │   假设: 扰动效应在潜在空间中是线性的                  │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   预测: z_pred = z_new_control + δ                          │  │
│   │                                                             │  │
│   │   支持跨细胞类型/批次/物种的线性外推                        │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                      解码器网络 (Decoder)                    │  │
│   │                                                             │  │
│   │   输入: 潜在空间预测 z_pred                                 │  │
│   │        ↓                                                    │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │              全连接神经网络层 (与编码器对称)          │  │  │
│   │   │                                                     │  │  │
│   │   │   Linear ──→ ReLU ──→ Linear ──→ ReLU ──→ ...      │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                            ↓                               │  │
│   │   输出: 预测基因表达 x_pred (重建到原始维度)                │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   训练目标: 重构损失 + KL散度正则化                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 核心能力

| 能力 | 描述 |
|------|------|
| **跨细胞类型预测** | 预测未见细胞类型的扰动效应 |
| **跨研究预测** | 跨不同实验批次预测 |
| **跨物种预测** | 跨物种迁移预测 |
| **批次效应去除** | 内置批次校正 |

### 技术特点

- **生成式建模**: 基于VAE的生成能力
- **潜在空间运算**: 在潜在空间中进行向量运算
- **线性外推**: 假设扰动效应在潜在空间中是线性的

## 训练与应用

### 训练数据
- 多细胞类型和条件的数据集
- 支持在一种条件下训练，预测另一种条件下的细胞类型

### 应用场景
1. **多细胞类型场景**
   - 训练于多细胞类型数据集
   - 预测仅在一种条件下存在的细胞类型的扰动效应

2. **多物种场景**
   - 用一种或所有物种预测特定物种的效应

3. **跨数据集预测**
   - 在两个条件的数据集上训练
   - 在具有相似基因的第二个数据集上预测

## 性能表现

- **准确性**: 在多个基准数据集上验证
- **泛化能力**: 跨细胞类型、跨研究、跨物种
- **鲁棒性**: 对批次效应具有鲁棒性

- **历史地位**: 首个机器学习单细胞扰动预测工具
- **引用**: 被广泛引用，成为领域基准
- **后续发展**: 启发了CPA、CellFlow等后续方法

## 优势与局限

### 优势
- 首个ML工具，具有开创性
- 跨细胞类型/研究/物种预测能力
- 开源，社区支持
- 内置批次校正

### 局限
- 线性假设可能不适用于所有扰动
- 对复杂非线性相互作用的建模能力有限
- 需要足够的训练数据
