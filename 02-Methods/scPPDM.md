# scPPDM

## 基本信息

- **全称**: A Diffusion Model for Single-Cell Drug-Response Prediction
- **发表时间**: 2025年10月
- **机构**: 浙江大学
- **作者**: Zhaokang Liang, Shuyang Zhuang, Xiaoran Jiao, Weian Mao, Hao Chen, Chunhua Shen

## 核心技术

scPPDM 是一个专门用于单细胞药物反应预测的扩散模型：

1. **扩散模型架构**: 基于扩散模型的生成框架，用于预测药物扰动后的细胞状态
2. **药物反应预测**: 专注于药物-细胞反应建模
3. **单细胞分辨率**: 在单细胞水平预测药物效应

## 扩散模型架构

### 整体架构

scPPDM 采用去噪扩散概率模型（DDPM）框架：

```
┌─────────────────────────────────────────────────────────────────────┐
│              DDPM 扩散过程                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   前向过程 (加噪):                                                  │
│   x_0 → x_1 → x_2 → ... → x_T                                       │
│     ↓     ↓     ↓         ↓                                         │
│    加噪  加噪   加噪      纯噪声                                      │
│                                                                     │
│   反向过程 (去噪):                                                  │
│   x_T → x_{T-1} → ... → x_0                                         │
│     ↓       ↓           ↓                                           │
│   噪声    去噪         预测                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 网络结构

```
┌─────────────────────────────────────────────────────────────────────┐
│                    scPPDM Architecture                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  输入层                                                             │
│  ├── 基因表达: x_t ∈ R^n_genes                                      │
│  ├── 时间步: t ∈ {1, 2, ..., T}                                     │
│  ├── 药物嵌入: e_drug ∈ R^d_drug                                    │
│  └── 细胞特征: h_cell ∈ R^d_cell                                    │
│       ↓                                                             │
│  编码器 (U-Net)                                                     │
│  ├── 下采样路径 (Encoder)                                           │
│  │   ├── Conv1D(ngenes → 512) + SiLU                                │
│  │   ├── Conv1D(512 → 256) + SiLU                                   │
│  │   ├── Conv1D(256 → 128) + SiLU                                   │
│  │   └── Conv1D(128 → 64) + SiLU                                    │
│  │                                                                   │
│  ├── 瓶颈层 (Bottleneck)                                            │
│  │   ├── Self-Attention × 2                                         │
│  │   ├── Cross-Attention (Drug Condition)                           │
│  │   └── Residual Blocks × 2                                        │
│  │                                                                   │
│  └── 上采样路径 (Decoder)                                           │
│      ├── ConvTranspose1D(64 → 128) + SiLU                           │
│      ├── ConvTranspose1D(128 → 256) + SiLU                          │
│      ├── ConvTranspose1D(256 → 512) + SiLU                          │
│      └── ConvTranspose1D(512 → ngenes)                              │
│       ↓                                                             │
│  输出层                                                             │
│  └── 预测噪声: ε_θ(x_t, t, c_drug) ∈ R^n_genes                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 条件注入机制

药物条件通过多种方式注入：

```
┌─────────────────────────────────────────────────────────────────────┐
│              条件注入机制                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   1. 时间嵌入 (正弦位置编码):                                       │
│                                                                     │
│      t_emb = [sin(t/10000^(2i/d)), cos(t/10000^(2i/d))]            │
│                                                                     │
│   2. 药物嵌入:                                                      │
│                                                                     │
│      d_emb = MLP(MorganFingerprint(drug))                           │
│                                                                     │
│   3. 交叉注意力 (药物-基因交互):                                    │
│                                                                     │
│      CrossAttn(Q_genes, K_drug, V_drug) =                           │
│         softmax(Q_genes·K_drug^T/√d_k)·V_drug                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 去噪过程公式

### 前向过程（加噪）

给定初始数据 x_0，前向过程逐步添加高斯噪声：

```
┌─────────────────────────────────────────────────────────────────────┐
│              前向加噪过程                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   q(x_t | x_{t-1}) = N(x_t; √(1-β_t)·x_{t-1}, β_t·I)               │
│                                                                     │
│   通过重参数化，可直接采样任意时间步:                               │
│                                                                     │
│   x_t = √(ᾱ_t)·x_0 + √(1-ᾱ_t)·ε,   ε ~ N(0, I)                     │
│                                                                     │
│   其中:                                                             │
│   - α_t = 1 - β_t                                                   │
│   - ᾱ_t = Π_{s=1}^t α_s  (累积乘积)                                 │
│   - β_t: 噪声调度 (线性或余弦)                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 反向过程（去噪）

学习反向去噪过程：

```
┌─────────────────────────────────────────────────────────────────────┐
│              反向去噪过程                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   p_θ(x_{t-1} | x_t) = N(x_{t-1}; μ_θ(x_t, t), Σ_θ(x_t, t))        │
│                                                                     │
│   均值预测 (DDPM):                                                  │
│                                                                     │
│   μ_θ(x_t, t) = 1/√α_t · (x_t - β_t/√(1-ᾱ_t) · ε_θ(x_t, t))        │
│                                                                     │
│   方差固定为:                                                       │
│                                                                     │
│   Σ_θ(x_t, t) = (1-ᾱ_{t-1})/(1-ᾱ_t) · β_t · I                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 训练目标

简化训练目标（预测噪声）：

```
┌─────────────────────────────────────────────────────────────────────┐
│              训练目标                                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   L_simple = E_{x_0, t, ε}[ ||ε - ε_θ(x_t, t, c_drug)||² ]          │
│                                                                     │
│   也可选择预测 x_0:                                                 │
│                                                                     │
│   L_x0 = E_{x_0, t, ε}[ ||x_0 - x_θ(x_t, t, c_drug)||² ]            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 采样过程

```
┌─────────────────────────────────────────────────────────────────────┐
│              DDPM 采样过程                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   算法:                                                             │
│   1. 从标准高斯采样: x_T ~ N(0, I)                                  │
│                                                                     │
│   2. 对于 t = T, T-1, ..., 1:                                       │
│                                                                     │
│      a) 预测噪声: ε_pred = model(x_t, t, drug_condition)            │
│                                                                     │
│      b) 计算均值:                                                   │
│         mean = (x_t - β_t/√(1-ᾱ_t) · ε_pred) / √α_t                 │
│                                                                     │
│      c) 如果 t > 0:                                                 │
│         variance = (1-ᾱ_{t-1})/(1-ᾱ_t) · β_t                        │
│         x_{t-1} = mean + √variance · noise                          │
│         否则:                                                       │
│         x_{t-1} = mean                                              │
│                                                                     │
│   3. 返回 x_0                                                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 药物反应预测机制

### 药物表示学习

scPPDM 支持多种药物表示方式：

| 表示方式 | 描述 |
|---------|------|
| **分子指纹** | Morgan/ECFP 编码分子结构 |
| **SMILES 编码** | RNN/Transformer 编码 SMILES 字符串 |
| **图神经网络** | 将分子视为图，GNN 提取特征 |

### 药物-基因交互建模

通过交叉注意力机制建模药物与基因的交互：

```
┌─────────────────────────────────────────────────────────────────────┐
│              药物-基因交互注意力                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   Attention_ij = exp(Q_i·K_j^T/√d) / Σ_k exp(Q_i·K_k^T/√d)          │
│                                                                     │
│   其中:                                                             │
│   - Q_i: 第 i 个基因的查询向量                                      │
│   - K_j: 第 j 个药物特征的关键向量                                  │
│                                                                     │
│   效果: 学习药物对特定基因的影响模式                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 反应强度预测

除生成扰动后表达谱外，scPPDM 还可预测反应强度：

```
Response = MLP(Pool(x_pred - x_ctrl))
```

输出包括：
- IC50 估计
- 细胞活力预测
- 毒性评分

### 多药物组合预测

对于药物组合，scPPDM 采用加法条件：

```
c_combo = c_drug1 + c_drug2 + c_interaction
```

其中 c_interaction 为可学习的交互项。

## 输入/输出

- **输入**:
  - 对照单细胞基因表达 x_0 ∈ R^{n_genes}
  - 药物信息（化合物结构/ID/SMILES）
  - 可选：细胞类型、剂量、处理时间
- **输出**:
  - 预测的药物扰动后单细胞表达谱
  - 可选：反应强度评分

## 训练数据

- 药物扰动单细胞数据（如 LINCS L1000、Sci-Plex）
- 化合物结构信息
- 剂量-反应关系
- 处理时间序列

## 性能指标

- 药物反应预测准确性（MSE、MAE）
- 单细胞水平表达变化预测
- DE 基因恢复率
- 通路富集一致性

## 功能特点

1. **药物专用**: 针对药物扰动优化
2. **扩散生成**: 利用扩散模型的高质量生成能力
3. **单细胞精度**: 单细胞水平的预测
4. **多模态输入**: 支持多种药物表示方式
5. **剂量-反应**: 可建模剂量依赖效应

## 优势与局限

**优势**:
- 针对药物扰动专门优化
- 扩散模型的高质量生成
- 单细胞分辨率
- 支持多种药物表示
- 可解释的药物-基因交互

**局限**:
- 主要针对药物扰动，对其他扰动类型可能不适用
- 需要大量药物-细胞配对数据
- 扩散采样速度较慢（需多步迭代）

## 技术细节补充

### 噪声调度

scPPDM 采用余弦噪声调度：

```
ᾱ_t = f(t)/f(0)

f(t) = cos((t/T + s)/(1+s) · π/2)²
```

其中 s=0.008 为偏移量。

### 分类器自由引导（CFG）

为增强条件控制，scPPDM 支持 CFG：

```
ε̂_θ = ε_θ(x_t, t, ∅) + w · (ε_θ(x_t, t, c) - ε_θ(x_t, t, ∅))
```

其中 w 为引导强度（通常 w ∈ [1, 5]）。
