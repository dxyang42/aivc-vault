# GEARS-2024

> Predicting transcriptional outcomes of novel multigene perturbations

## 基本信息

| 属性 | 内容 |

| **代码链接** | https://github.com/snap-stanford/GEARS |

## 核心思想

GEARS-2024是GEARS方法的扩展版本，专注于**多基因组合扰动的转录结果预测**。该方法整合基因本体论(GO)和基因共表达网络，利用图神经网络(GNN)建模基因间的非线性相互作用，从而预测未见过的基因组合扰动效应。

### 关键创新

1. **知识图谱增强**: 整合Gene Ontology和共表达网络构建基因关系图
2. **非线性相互作用建模**: GNN捕捉基因间的复杂调控关系
3. **组合扰动泛化**: 预测训练时未见过的基因组合效应
4. **不确定性量化**: 提供预测置信度估计

## 技术细节

### 知识图谱构建

GEARS-2024使用多层基因关系图:

```
Layer 1: Gene Ontology
├── 生物过程 (Biological Process)
├── 分子功能 (Molecular Function)
└── 细胞组分 (Cellular Component)

Layer 2: Co-expression Network
├── 基于大规模表达数据计算相关性
├── 阈值过滤保留显著相关对
└── 边权重表示共表达强度

Layer 3: Protein-Protein Interaction
├── 从STRING等数据库导入
├── 整合实验验证和预测相互作用
└── 过滤高置信度相互作用
```

### 消息传递机制

```
对于图中每个基因节点:
    1. 收集邻居节点的特征表示
    2. 计算注意力权重 (基因间相关性)
    3. 加权聚合邻居信息
    4. 更新当前基因的状态表示
    5. 迭代多轮传播全局信息
```

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 控制表达谱 | 未扰动细胞的基因表达 |
| 扰动基因集 | 要敲除/过表达的基因列表 |
| 基因关系图 | GO + 共表达 + PPI网络 |

### 输出
| 类型 | 描述 |
|------|------|
| 预测表达谱 | 扰动后的基因表达预测 |
| 差异表达基因 | 显著变化的基因列表 |
| 置信度分数 | 每个基因的预测不确定性 |

## 优势与局限

### 优势
- 整合多源先验知识，提高泛化能力
- 显式建模非线性基因相互作用
- 支持未见基因组合的零样本预测
- 可解释性强，可追踪预测依据

### 局限
- 依赖知识图谱的完整性和准确性
- 对未见细胞类型的泛化有限
- 计算成本随基因数量增加
- 难以捕捉动态时间效应

## 应用场景

1. **药物组合筛选**: 预测多靶点药物的协同效应
2. **合成致死研究**: 识别基因组合致死对
3. **通路分析**: 理解复杂通路间的交叉调控
4. **基因治疗设计**: 优化多基因编辑策略
