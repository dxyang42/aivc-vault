## 核心思想

PDGrapher是一种**因果启发的图神经网络**，专门设计用于预测能够逆转疾病表型的组合治疗扰动。该方法通过引入因果推断原理，显著提升了训练速度(快25倍)，同时保持高预测精度。

### 关键创新

1. **因果启发架构**: 将因果推断原理融入GNN设计
2. **疾病表型逆转**: 专注于预测治疗性扰动而非任意扰动
3. **高效训练**: 比传统方法快25倍的训练速度
4. **组合优化**: 自动搜索最优药物/基因组合

## 因果启发机制

### 因果图构建

```
观察数据中的相关性        因果干预后的效应
        ↓                        ↓
   P(Y|X) 关联性           P(Y|do(X)) 因果效应
        ↓                        ↓
   可能受混杂因素影响        纯净的因果效应
        ↓                        ↓
   使用因果发现算法          构建干预图
   (PC, GES等)              (Intervention Graph)
```

### 训练加速原理

PDGrapher通过以下策略实现25倍训练加速:

1. **因果剪枝**: 只学习因果相关的边，减少参数
2. **分层传播**: 按因果层级顺序传播，避免冗余计算
3. **干预缓存**: 缓存常见干预的效应，避免重复计算
4. **批量因果推理**: 并行处理多个因果查询

## 输入/输出

### 输入
| 类型 | 描述 |
|------|------|
| 疾病表达谱 | 疾病状态下的单细胞表达 |

| 类型 | 描述 |
|------|------|
| 最优组合 | 推荐的治疗扰动组合 |
| 逆转评分 | 疾病向健康状态转化的概率 |
| 预测表达 | 扰动后的预期基因表达 |
| 机制解释 | 关键调控通路和基因 |

## 优势与局限

### 优势
- 因果启发的架构设计，可解释性强
- 训练速度极快，适合大规模应用
- 专注于治疗性扰动，实用性强
- 自动组合优化，减少实验成本

### 局限
- 依赖因果图的质量
- 对复杂疾病可能需要多轮干预
- 因果发现本身存在计算挑战
- 需要配对的疾病-健康数据

## 应用场景

1. **药物重定位**: 发现现有药物的新适应症
2. **组合疗法设计**: 优化多药联合治疗方案
3. **基因治疗**: 设计多靶点基因编辑策略
4. **个性化医疗**: 基于患者数据定制治疗方案
