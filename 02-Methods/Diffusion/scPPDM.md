# scPPDM

## 基本信息

- **全称**: A Diffusion Model for Single-Cell Drug-Response Prediction
- **发表时间**: 2025年10月
- **机构**: 浙江大学
- **作者**: Zhaokang Liang, Shuyang Zhuang, Xiaoran Jiao, Weian Mao, Hao Chen, Chunhua Shen

## 核心技术

scPPDM 是一个专门用于单细胞药物反应预测的扩散模型：

1. **扩散模型架构**: 基于扩散模型的生成框架，用于预测药物扰动后的细胞状态
2. **药物反应预测**: 专注于药物-细胞反应建模
3. **单细胞分辨率**: 在单细胞水平预测药物效应

### 扩散过程概览

```
┌─────────────────────────────────────────────────────────────────────┐
│              DDPM 扩散过程                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   前向过程 (加噪):                                                  │
│   ─────────────────                                                 │
│                                                                     │
│   x_0 (原始数据)                                                    │
│     ↓ 加少量噪声                                                    │
│   x_1                                                               │
│     ↓ 加更多噪声                                                    │
│   x_2                                                               │
│     ↓ ...                                                           │
│   x_T (接近纯噪声)                                                  │
│                                                                     │
│   特点: 每一步添加高斯噪声，逐步破坏数据结构                        │
│                                                                     │
│   反向过程 (去噪):                                                  │
│   ─────────────────                                                 │
│                                                                     │
│   x_T (纯噪声)                                                      │
│     ↓ 预测并去除噪声                                                │
│   x_{T-1}                                                           │
│     ↓ 继续去噪                                                      │
│   x_{T-2}                                                           │
│     ↓ ...                                                           │
│   x_0 (预测数据)                                                    │
│                                                                     │
│   特点: 神经网络学习逐步恢复数据结构                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 条件注入机制

药物条件通过多种方式注入：

```
┌─────────────────────────────────────────────────────────────────────┐
│              条件注入机制                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   1. 时间嵌入 (正弦位置编码):                                       │
│   ─────────────────────────────                                     │
│                                                                     │
│   将离散时间步t编码为连续向量:                                      │
│   t_emb[i] = sin(t / 10000^(2i/d))  或  cos(...)                   │
│                                                                     │
│   作用: 让网络知道当前去噪进度                                      │
│                                                                     │
│   2. 药物嵌入:                                                      │
│   ─────────────                                                     │
│                                                                     │
│   药物分子指纹 → MLP → 药物嵌入向量                                 │
│                                                                     │
│   MorganFingerprint(drug) → MLP → e_drug                           │
│                                                                     │
│   作用: 编码药物分子结构信息                                        │
│                                                                     │
│   3. 交叉注意力 (药物-基因交互):                                    │
│   ────────────────────────────────                                  │
│                                                                     │
│   基因特征作为Query，药物特征作为Key和Value                         │
│                                                                     │
│   Attention = softmax(Q_genes · K_drug^T / sqrt(d)) · V_drug       │
│                                                                     │
│   作用: 学习药物对特定基因的影响模式                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 扩散过程详解

### 前向过程（加噪）

```
┌─────────────────────────────────────────────────────────────────────┐
│              前向加噪过程                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   给定初始数据 x_0，前向过程逐步添加高斯噪声                        │
│                                                                     │
│   单步加噪:                                                         │
│   x_t = sqrt(1-beta_t) * x_{t-1} + sqrt(beta_t) * noise            │
│                                                                     │
│   其中 beta_t 是噪声调度参数 (随t增大而增大)                        │
│                                                                     │
│   直接采样任意时间步 (重参数化技巧):                                │
│                                                                     │
│   x_t = sqrt(alpha_bar_t) * x_0 + sqrt(1-alpha_bar_t) * noise      │
│                                                                     │
│   其中:                                                             │
│   - alpha_t = 1 - beta_t                                            │
│   - alpha_bar_t = alpha_1 * alpha_2 * ... * alpha_t (累积乘积)      │
│   - noise: 标准高斯噪声                                             │
│                                                                     │
│   效果: 可以直接从x_0计算任意x_t，无需逐步迭代                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 反向过程（去噪）

```
┌─────────────────────────────────────────────────────────────────────┐
│              反向去噪过程                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   学习反向去噪过程，从噪声恢复数据                                  │
│                                                                     │
│   单步去噪:                                                         │
│                                                                     │
│   1. 网络预测噪声: noise_pred = network(x_t, t, condition)         │
│                                                                     │
│   2. 计算去噪后的均值:                                              │
│      mean = (x_t - beta_t/sqrt(1-alpha_bar_t) * noise_pred)        │
│              / sqrt(alpha_t)                                        │
│                                                                     │
│   3. 添加随机性 (t>0时):                                            │
│      variance = (1-alpha_bar_{t-1})/(1-alpha_bar_t) * beta_t       │
│      x_{t-1} = mean + sqrt(variance) * random_noise                │
│                                                                     │
│   4. 最后一步 (t=0) 不加随机性:                                     │
│      x_{-1} = mean                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 训练目标

```
┌─────────────────────────────────────────────────────────────────────┐
│              训练目标                                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   简化训练目标 (预测噪声):                                          │
│   ─────────────────────────                                         │
│                                                                     │
│   对于每个训练样本:                                                 │
│   1. 随机采样时间步 t                                               │
│   2. 采样噪声 epsilon ~ N(0, I)                                     │
│   3. 计算加噪样本 x_t                                               │
│   4. 网络预测噪声 noise_pred                                        │
│   5. 损失 = 预测噪声与真实噪声的均方误差                            │
│                                                                     │
│   也可选择预测 x_0:                                                 │
│   损失 = 预测x_0与真实x_0的均方误差                                 │
│                                                                     │
│   两种目标等价，预测噪声通常更稳定                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 采样过程

```
┌─────────────────────────────────────────────────────────────────────┐
│              DDPM 采样过程                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   算法步骤:                                                         │
│                                                                     │
│   1. 从标准高斯采样: x_T ~ N(0, I)                                  │
│                                                                     │
│   2. 对于 t = T, T-1, ..., 1 (循环T次):                             │
│                                                                     │
│      a) 预测噪声:                                                   │
│         noise_pred = model(x_t, t, drug_condition)                 │
│                                                                     │
│      b) 计算均值:                                                   │
│         mean = (x_t - coeff * noise_pred) / sqrt(alpha_t)          │
│                                                                     │
│      c) 如果 t > 0:                                                 │
│         添加随机噪声: x_{t-1} = mean + sqrt(var) * noise           │
│         否则:                                                       │
│         x_{t-1} = mean (最后一步确定性地输出)                       │
│                                                                     │
│   3. 返回 x_0 (预测的扰动后细胞表达)                                │
│                                                                     │
│   注意: T通常为1000，需要1000次网络前向传播                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 药物反应预测机制

### 药物表示学习

scPPDM 支持多种药物表示方式：

| 表示方式 | 描述 |
|---------|------|
| **分子指纹** | Morgan/ECFP 编码分子结构 |
| **SMILES 编码** | RNN/Transformer 编码 SMILES 字符串 |
| **图神经网络** | 将分子视为图，GNN 提取特征 |

### 药物-基因交互建模

通过交叉注意力机制建模药物与基因的交互：

```
┌─────────────────────────────────────────────────────────────────────┐
│              药物-基因交互注意力                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   注意力权重计算:                                                   │
│                                                                     │
│   Attention_ij = exp(Q_i·K_j^T/sqrt(d)) / Sum_k exp(Q_i·K_k^T/sqrt(d))
│                                                                     │
│   其中:                                                             │
│   - Q_i: 第 i 个基因的查询向量                                      │
│   - K_j: 第 j 个药物特征的关键向量                                  │
│                                                                     │
│   效果: 学习药物对特定基因的影响模式                                │
│                                                                     │
│   可视化理解:                                                       │
│                                                                     │
│        药物特征                                                     │
│           ↓                                                         │
│   基因A ←──┼──→ 高注意力权重 (药物强烈影响基因A)                    │
│   基因B ←──┼──→ 低注意力权重 (药物对基因B影响小)                    │
│   基因C ←──┼──→ 中等注意力权重                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 反应强度预测

除生成扰动后表达谱外，scPPDM 还可预测反应强度：

```
反应强度 = MLP(池化(预测表达 - 对照表达))
```

输出包括：
- IC50 估计
- 细胞活力预测
- 毒性评分

### 多药物组合预测

对于药物组合，scPPDM 采用加法条件：

```
组合条件 = 药物1条件 + 药物2条件 + 交互项

其中交互项为可学习的参数，捕捉药物间的协同/拮抗效应
```

## 输入/输出

- **输入**:
  - 对照单细胞基因表达 x_0 (n_genes维)
  - 药物信息（化合物结构/ID/SMILES）
  - 可选：细胞类型、剂量、处理时间
- **输出**:
  - 预测的药物扰动后单细胞表达谱
  - 可选：反应强度评分

## 训练数据

- 药物扰动单细胞数据（如 LINCS L1000、Sci-Plex）
- 化合物结构信息
- 剂量-反应关系
- 处理时间序列

## 优势与局限

**优势**:
- 针对药物扰动专门优化
- 扩散模型的高质量生成
- 单细胞分辨率
- 支持多种药物表示
- 可解释的药物-基因交互

**局限**:
- 主要针对药物扰动，对其他扰动类型可能不适用
- 需要大量药物-细胞配对数据
- 扩散采样速度较慢（需多步迭代）

## 技术细节补充

### 噪声调度

scPPDM 采用余弦噪声调度：

```
alpha_bar_t = f(t)/f(0)

f(t) = cos((t/T + s)/(1+s) * pi/2)^2
```

其中 s=0.008 为偏移量。

### 分类器自由引导（CFG）

为增强条件控制，scPPDM 支持 CFG：

```
噪声_pred_引导 = 噪声_pred(无条件) + w * (噪声_pred(有条件) - 噪声_pred(无条件))
```

其中 w 为引导强度（通常 w 在1到5之间）。
