## 核心技术

### 架构设计

NetPerturb 将基因调控网络建模为流网络，使用深度学习预测扰动传播：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NetPerturb 架构                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   核心思想: 基因调控 = 信息流网络，扰动 = 网络流注入                │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              输入层: 网络构建                                │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   1. 基因调控网络 (GRN) 推断:                       │  │  │
│   │   │                                                     │  │  │
│   │   │   输入: scRNA-seq 数据                              │  │  │
│   │   │        ↓                                            │  │  │
│   │   │   方法:                                             │  │  │
│   │   │   • 相关性网络 (Pearson/Spearman)                   │  │  │
│   │   │   • 互信息网络                                      │  │  │
│ │   │   │   • 因果推断 (PC算法, GES)                        │  │  │
│   │   │   • 先验知识 (STRING, RegNetwork)                   │  │  │
│   │   │        ↓                                            │  │  │
│   │   │   输出: 加权有向图 G = (V, E, W)                    │  │  │
│   │   │                                                     │  │  │
│   │   │   2. 网络增强:                                      │  │  │
│   │   │   • 添加TF-靶基因先验                               │  │  │
│   │   │   • 蛋白互作网络整合                                │  │  │
│   │   │   • 信号通路注释                                    │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              图神经网络编码器                                │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   节点特征初始化:                                   │  │  │
│   │   │   h_i^0 = [expression_i,                           │  │  │
│   │   │              functional_embedding_i,                │  │  │
│   │   │              network_position_i]                    │  │  │
│   │   │                                                     │  │  │
│   │   │   其中:                                             │  │  │
│   │   │   - expression: 基因表达值                          │  │  │
│   │   │   - functional: 基因功能嵌入 (来自Gene Ontology)    │  │  │
│   │   │   - position: 网络位置编码 (PageRank等)             │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                              ↓                              │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   图注意力层 (k = 1 to K):                          │  │  │
│   │   │                                                     │  │  │
│   │   │   h_i^(k) = sigma(sum_j (alpha_ij^(k) * W^(k) * h_j^(k-1))) │  │
│   │   │                                                     │  │  │
│   │   │   注意力系数:                                       │  │  │
│   │   │   alpha_ij = softmax(LeakyReLU(a^T [Wh_i || Wh_j])) │  │  │
│   │   │                                                     │  │  │
│   │   │   边权重调制:                                       │  │  │
│   │   │   alpha_ij' = alpha_ij * w_ij (来自GRN权重)         │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              网络流传播模块 (核心创新)                       │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   扰动注入:                                         │  │  │
│   │   │                                                     │  │  │
│   │   │   对于目标基因 g:                                   │  │  │
│   │   │   • KO: h_g = 0 (节点特征置零)                      │  │  │
│   │   │   • OE: h_g = h_g + delta (增加特征值)              │  │  │
│   │   │   • KD: h_g = h_g * (1 - alpha) (部分抑制)          │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                              ↓                              │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   流传播算法:                                       │  │  │
│   │   │                                                     │  │  │
│   │   │   迭代传播 (t = 1 to T):                            │  │  │
│   │   │                                                     │  │  │
│   │   │   f_i^(t) = f_i^(t-1) +                             │  │  │
│   │   │           sum_j (w_ji * g(f_j^(t-1)) * flow_ji)     │  │  │
│   │   │                                                     │  │  │
│   │   │   其中:                                             │  │  │
│   │   │   - f_i: 节点i的扰动流强度                          │  │  │
│   │   │   - w_ji: 边j→i的权重                               │  │  │
│   │   │   - g(): 非线性激活 (ReLU/LeakyReLU)                │  │  │
│   │   │   - flow_ji: 学习到的流系数                         │  │  │
│   │   │                                                     │  │  │
│   │   │   物理意义: 模拟信号在网络中的级联传播              │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                              ↌                              │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   收敛检测:                                         │  │  │
│   │   │   ||f^(t) - f^(t-1)|| < epsilon                     │  │  │
│   │   │                                                     │  │  │
│   │   │   或达到最大迭代次数 T_max                          │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              扰动效应解码器                                  │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   最终扰动强度:                                     │  │  │
│   │   │   delta_i = MLP([h_i^K, f_i^(T), h_i^0])            │  │  │
│   │   │                                                     │  │  │
│   │   │   预测表达:                                         │  │  │
│   │   │   x_i^pred = x_i^0 + delta_i                        │  │  │
│   │   │                                                     │  │  │
│   │   │   差异表达概率:                                     │  │  │
│   │   │   p_DE_i = sigmoid(MLP(delta_i))                    │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              输出层                                        │  │
│   │                                                             │  │
│   │   - 预测基因表达变化                                      │  │
│   │   - 扰动传播路径                                          │  │
│   │   - 关键节点识别                                          │  │
│   │   - 网络模块效应                                          │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   训练目标: 重建损失 + 流一致性 + 网络正则化                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 核心创新

| 创新点 | 描述 |
|--------|------|
| **网络流建模** | 将扰动传播建模为网络流问题 |
| **可学习流系数** | 神经网络学习最优流传播系数 |
| **级联效应捕捉** | 显式建模多跳扰动传播 |
| **网络位置编码** | 利用基因在网络中的位置信息 |

## 训练策略

### 损失函数

```
┌─────────────────────────────────────────────────────────────┐
│                NetPerturb 训练损失                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   L_total = L_recon + lambda_1 * L_flow + lambda_2 * L_reg  │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   1. 重建损失 (Reconstruction Loss):                │  │
│   │                                                     │  │
│   │   L_recon = MSE(X_pred, X_true)                     │  │
│   │                                                     │  │
│   │   确保预测表达与观测一致                            │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   2. 流一致性损失 (Flow Consistency):               │  │
│   │                                                     │  │
│   │   L_flow = sum_i (f_i - sum_j w_ji * f_j)^2         │  │
│   │                                                     │  │
│   │   确保流传播符合网络结构                            │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   3. 网络正则化 (Network Regularization):           │  │
│   │                                                     │  │
│   │   L_reg = L2(网络权重) +                           │  │
│   │           稀疏性约束(GRN边) +                      │  │
│   │           连通性约束                                │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 节点特征置零 |
| **单基因过表达 (OE)** | 完全支持 | 节点特征增加 |
| **单基因敲低 (KD)** | 完全支持 | 节点特征衰减 |
| **多基因组合** | 支持 | 同时注入多个扰动 |
| **通路级扰动** | 核心优势 | 子网络扰动 |
| **网络边扰动** | 支持 | 模拟药物靶向边 |

### 预测性能

| 任务 | 数据集 | NetPerturb | CellOracle | GEARS |
|-----|--------|------------|------------|-------|
| 单基因KO | Norman 2021 | R² 0.79 | R² 0.72 | R² 0.76 |
| 组合扰动 | Replogle 2022 | R² 0.71 | R² 0.63 | R² 0.74 |
| 传播路径 | 模拟数据 | AUC 0.88 | - | - |
| 关键节点 | 验证实验 | Precision 0.82 | 0.75 | - |

## 独特功能

### 1. 扰动传播路径追踪

```
┌─────────────────────────────────────────────────────────────┐
│              扰动传播路径可视化                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   扰动源: GATA1 (KO)                                        │
│                                                             │
│   传播路径:                                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │

│   │     │                                               │  │
```

### 2. 网络模块效应分析

| 分析类型 | 描述 | 输出 |
|---------|------|------|
| **通路富集** | 识别受影响的信号通路 | 通路列表 + p值 |
| **模块激活** | 检测激活/抑制的网络模块 | 模块热图 |
| **枢纽基因** | 识别关键调控节点 | 基因排名 |
| **网络重构** | 扰动后的网络变化 | 差异网络 |

## 应用场景

1. **通路机制解析**: 追踪扰动在信号通路中的传播
2. **药物靶点预测**: 预测药物干预的级联效应
3. **合成致死分析**: 识别基因间的合成致死关系
4. **网络生物学研究**: 理解基因网络的拓扑性质
5. **实验设计优化**: 选择信息量最大的扰动实验

## 优势与局限

### 优势
- 显式建模扰动传播机制
- 可追踪传播路径
- 网络模块分析能力强
- 结合深度学习和网络科学
- 可解释性好

### 局限
- 依赖GRN质量
- 计算复杂度较高
- 需要足够大的网络
- 对动态网络建模有限
