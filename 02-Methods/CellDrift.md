### 核心创新

CellDrift 是一种专门用于分析**时序单细胞扰动数据**的方法。与现有模型独立测量每个时间点的扰动响应不同，CellDrift 通过广义线性模型显式建模基因程序在时间序列上的**时间一致性**，从而更准确地识别和量化扰动效应的动态变化。

## 4. 核心技术

### 架构设计

CellDrift 基于广义线性模型，显式建模时间一致性：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CellDrift 工作流程                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  输入: 时序单细胞数据                                                │
│  ├── 时间点: t₁, t₂, t₃, ..., tₙ                                   │
│  ├── 条件: 对照 vs 扰动                                              │
│  └── 表达矩阵: X in R^(cells × genes)                                │
│                                                                     │
│       ↓                                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  步骤1: 基因程序识别                                          │   │
│  │  ├── 降维 (PCA/UMAP)                                         │   │
│  │  ├── 聚类识别基因模块                                         │   │
│  │  └── 基因程序定义: GP₁, GP₂, ..., GPₘ                        │   │
│  └─────────────────────────────────────────────────────────────┘   │
│       ↓                                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  步骤2: 时间一致性GLM建模                                     │   │
│  │  ├── 对每个基因程序 GPⱼ:                                     │   │
│  │  │                                                            │   │
│  │  │   y ~ β₀ + β₁·Time + β₂·Condition + β₃·Time×Condition    │   │
│  │  │        + ε                                                 │   │
│  │  │                                                            │   │
│  │  │   约束: 时间系数 β₁ 在相邻时间点间平滑                      │   │
│  │  │                                                            │   │
│  │  └── 输出: 时序扰动效应曲线                                    │   │
│  └─────────────────────────────────────────────────────────────┘   │
│       ↓                                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  步骤3: 动态效应量化                                          │   │
│  │  ├── 早期响应基因程序识别                                     │   │
│  │  ├── 晚期响应基因程序识别                                     │   │
│  │  └── 动态变化率计算                                           │   │
│  └─────────────────────────────────────────────────────────────┘   │
│       ↓                                                             │
│  输出: 时序扰动效应图谱 + 动态基因程序 + 统计显著性                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 技术特点

1. **时间一致性约束**：确保基因程序效应在相邻时间点间平滑变化
2. **基因程序级别分析**：在模块层面而非单个基因层面分析
3. **显式时间交互**：建模时间与扰动条件的交互效应
4. **统计显著性**：提供严格的统计检验框架

## 6. 数学模型

### GLM 框架

CellDrift 使用广义线性模型建模基因程序表达：

```
对于基因程序 g 在细胞 i 中的表达 y_gi:

y_gi ~ GLM(μ_gi, φ)

其中：
μ_gi = g⁻¹(η_gi)  # 连接函数的逆

线性预测器:
η_gi = β_g0 + β_g1·Time_i + β_g2·Condition_i + β_g3·(Time×Condition)_i

约束条件（时间一致性）:
|beta_g1(t+1) - beta_g1(t)| < lambda  # 相邻时间点系数变化受限
```

### 时间一致性约束

```python
# 时间平滑约束的伪代码
def temporal_smoothness_constraint(beta_t, lambda_smooth):
    """
    确保时间系数在相邻时间点间平滑变化
    """
    smoothness_penalty = 0
    for t in range(1, T):
        smoothness_penalty += (beta_t[t] - beta_t[t-1])²
    return lambda_smooth * smoothness_penalty

# 添加到损失函数
loss = negative_log_likelihood + temporal_smoothness_constraint(beta_time, lambda)
```

## 7. 训练数据

### 数据来源

| 数据类型 | 描述 |
|----------|------|
| **时序scRNA-seq** | 药物处理或基因扰动的时间序列数据 |
| **最少时间点** | 建议 ≥3 个时间点 |
| **细胞数量** | 每个时间点建议 ≥100 个细胞 |

### 数据预处理

- **质量控制**：标准scRNA-seq质控流程
- **归一化**：log1p转换
- **批次校正**：如有多批次数据
- **基因程序识别**：PCA + 聚类或预定义的基因集

## 8. 分析流程

### 标准工作流程

```
1. 数据导入与质控
   └── 读取时序scRNA-seq数据

2. 基因程序识别
   ├── 计算高变异基因
   ├── PCA降维
   ├── 聚类识别基因模块
   └── 定义基因程序活性评分

3. GLM建模
   ├── 为每个基因程序拟合GLM
   ├── 应用时间一致性约束
   └── 计算统计显著性

4. 结果解释
   ├── 识别早期响应程序
   ├── 识别晚期响应程序
   ├── 计算动态变化率
   └── 通路富集分析

5. 可视化
   ├── 时序效应曲线
   ├── 热图展示
   └── 动态轨迹图
```
