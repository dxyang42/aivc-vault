## 三、修复建议

### 5.1 LaTeX 公式转换方案

**方案A: ASCII 格式（推荐）**
```
原: L_CFM = E_{t~U(0,1)}[ ||v_θ(x_t, t, c) - (x_1 - x_0)||² ]
改: L_CFM = E[ ||v_θ(x_t, t, c) - (x_1 - x_0)||² ]
    其中 t ~ Uniform(0,1)
```

**方案B: 纯文本描述**
```
原: P(X = k) = π · 1_{k=0} + (1-π) · NB(k; μ, θ)
改: 零膨胀负二项分布概率:
    - 以概率 π 取值为 0
    - 以概率 (1-π) 服从负二项分布 NB(μ, θ)
```
