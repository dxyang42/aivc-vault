## 核心技术

### 架构设计

STATE采用**双模块架构**：

```
┌─────────────────────────────────────────────────────────────────────┐
│                  STATE 双模块架构                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   模块1: STATE Transition (ST) - 状态转换模块                       │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              ST 模块 - 基于Transformer的扰动预测             │  │
│   │                                                             │  │
│   │   输入: 细胞集合 (最佳大小: 256个细胞)                      │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   细胞1 ──┐                                         │  │  │
│   │   │   细胞2 ──┤                                         │  │  │
│   │   │   细胞3 ──┼──→ 基因表达特征提取 ──→ 细胞嵌入        │  │  │
│   │   │    ...  ──┤   (每个细胞表示为向量)                  │  │  │
│   │   │  细胞256 ─┘                                         │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                            ↓                               │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │           Transformer 编码器                         │  │  │
│   │   │                                                     │  │  │
│   │   │   ┌─────────────────────────────────────────────┐  │  │  │
│   │   │   │        多头自注意力机制                      │  │  │  │
│   │   │   │                                             │  │  │  │
│   │   │   │   输入: 细胞嵌入序列 (256 × d_model)        │  │  │  │
│   │   │   │                                             │  │  │  │
│   │   │   │   对每个注意力头:                           │  │  │  │
│   │   │   │   - Query: 细胞查询向量                     │  │  │  │
│   │   │   │   - Key: 细胞键向量                         │  │  │  │
│   │   │   │   - Value: 细胞值向量                       │  │  │  │
│   │   │   │                                             │  │  │  │
│   │   │   │   注意力权重 = softmax(Q × K^T / sqrt(d_k)) │  │  │  │
│   │   │   │   输出 = 注意力权重 × V                     │  │  │  │
│   │   │   │                                             │  │  │  │
│   │   │   │   多头: 多个注意力头并行，结果拼接          │  │  │  │
│   │   │   │                                             │  │  │  │
│   │   │   └─────────────────────────────────────────────┘  │  │  │
│   │   │                        ↓                          │  │  │
│   │   │   ┌─────────────────────────────────────────────┐  │  │  │
│   │   │   │        前馈神经网络 (FFN)                    │  │  │  │
│   │   │   │                                             │  │  │  │
│   │   │   │   全连接层 ──→ 激活函数 ──→ 全连接层       │  │  │  │
│   │   │   │                                             │  │  │  │
│   │   │   └─────────────────────────────────────────────┘  │  │  │
│   │   │                        ↓                          │  │  │
│   │   │        Layer Normalization + 残差连接             │  │  │
│   │   │                                                     │  │  │
│   │   │   重复 N 层 Transformer Block                     │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                            ↓                               │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │           扰动条件编码                               │  │  │
│   │   │                                                     │  │  │
│   │   │   扰动标识 ──→ 嵌入层 ──→ 扰动嵌入向量             │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                            ↓                               │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │           预测头                                   │  │  │
│   │   │                                                     │  │  │
│   │   │   Transformer输出 + 扰动嵌入 ──→ MLP ──→ 预测状态  │  │  │
│   │   │                                                     │  │  │
│   │   │   输出: 预测的扰动后细胞状态                        │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   模块2: STATE Embedding (SE) - 状态嵌入模块                        │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              SE 模块 - 细胞嵌入学习                          │  │
│   │                                                             │  │
│   │   输入: 单细胞基因表达                                      │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   编码器网络 (类似Transformer或MLP)                 │  │  │
│   │   │                                                     │  │  │
│   │   │   基因表达 ──→ 神经网络 ──→ 细胞嵌入               │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                            ↓                               │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │           解码器 (可选)                              │  │  │
│   │   │                                                     │  │  │
│   │   │   细胞嵌入 ──→ 解码器 ──→ 重建基因表达             │  │  │
│   │   │                                                     │  │  │
│   │   │   训练目标: 最小化重建误差                          │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   作用: 学习高质量的细胞表示，提升ST模块预测准确性          │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   联合训练:                                                         │
│   - Transition损失: 预测细胞状态转换的准确性                      │
│   - Embedding损失: 细胞嵌入的重建质量                             │
│   - 正则化损失: 防止过拟合                                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 技术特点
- 基于细胞集合的自注意力机制
- 支持单细胞分辨率转录组预测
- 支持嵌入预测两种模式
- 多尺度机器学习架构

## 训练数据

| 数据属性 | 规模 |
|---------|------|
| **观测细胞** | 1.7亿个 |
| **扰动细胞** | 1亿个 |
| **细胞系** | 70种人类细胞系 |
| **数据来源** | Tahoe-100M, Parse-PMBC, Replogle-Nadig等 |

## 性能表现

| 指标 | 结果 |
|------|------|
| **Tahoe-100M准确率提升** | 50% |
| **DES指标提升** | 100% |
| **差异表达基因识别** | 准确率是现有模型的2倍 |
| **预测能力** | 化学、遗传、细胞因子扰动 |

## 功能特点

- 零样本预测: 支持对未见扰动的预测
- 多扰动类型: 化学、遗传、细胞因子
- 大规模训练: 基于1亿+细胞数据
- 开源: 支持非商业用途
