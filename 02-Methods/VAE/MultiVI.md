### 关键创新

1. **马赛克整合 (Mosaic Integration)**
   - 支持非配对数据 (不同细胞测不同组学)
   - 支持部分配对数据
   - 支持完全配对数据
   - 灵活处理任意组合的多组学数据

2. **模态特异性解码器**
   - 每个组学类型独立解码器
   - 适应不同数据分布特性
   - 共享潜在空间整合信息

3. **缺失模态推断**
   - 从其他模态预测缺失的组学数据
   - 跨模态插补
   - 构建完整的多组学图谱

## 支持的组学类型

| 组学类型 | 分布模型 | 说明 |
|---------|---------|------|
| scRNA-seq | ZINB | 单细胞转录组 |
| scATAC-seq | Bernoulli/ZINB | 染色质可及性 |
| CITE-seq | NB | 表面蛋白 |
| 空间转录组 | ZINB | 空间表达 |
| DNA甲基化 | Beta/Bernoulli | 表观遗传 |

## 核心能力

| 能力 | 描述 |
|------|------|
| **马赛克整合** | 非配对多组学数据 |
| **缺失模态预测** | 从部分数据推断完整图谱 |
| **批次校正** | 跨组学批次效应校正 |
| **联合嵌入** | 统一的多组学细胞表示 |
| **可扩展** | 支持大规模数据集 |

## 技术特点

- **灵活输入**: 接受任意组合的多组学数据
- **概率框架**: 不确定性量化
- **端到端**: 无需预处理对齐
- **GPU加速**: 大规模数据支持

## 应用场景

1. **多组学图谱项目**
   - Human Cell Atlas
   - Mouse Cell Atlas
   - 疾病多组学图谱

2. **缺失数据补全**
   - 从RNA预测ATAC
   - 从蛋白质推断RNA

3. **跨技术研究**
   - 整合不同技术平台数据
   - 跨实验室数据整合

## 优势与局限

### 优势
- 支持非配对数据 (马赛克整合)
- 灵活的组学组合
- 缺失模态推断
- 与scVI生态兼容

### 局限
- 需要足够的数据重叠
- 计算复杂度高
- 模态间关系学习依赖数据质量

## 与Perturbation的关联

MultiVI为扰动预测提供多组学视角：
- **多组学扰动响应**: 同时分析转录组、表观组、蛋白组变化
- **机制解析**: 从多层面理解扰动机制
- **数据整合**: 整合不同技术的扰动实验
