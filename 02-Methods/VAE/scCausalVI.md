# scCausalVI

## 基本信息

- **全称**: scCausalVI: disentangles single-cell perturbation responses with causality-aware generative modeling
- **发表时间**: 2025年
- **发表期刊**: Cell Systems
- **机构**: 宾夕法尼亚大学
- **作者**: Ran An, et al.

## 核心技术

scCausalVI 是一个因果感知的变分推断模型，用于单细胞扰动数据分析：

1. **因果解耦**: 分离固有细胞变异和扰动诱导效应
2. **反事实预测**: 在单细胞分辨率进行虚拟扰动
3. **因果操作**: 使用 do-算子进行因果推理
4. **细胞异质性**: 识别对扰动响应不同的细胞亚群

## 架构细节

### 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                  scCausalVI Architecture                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   核心思想: 结构因果模型 (SCM) + 变分自编码器               │
│                                                             │
│   因果图:                                                   │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │         Z_cell (细胞固有状态)                       │  │
│   │              ↓                                      │  │
│   │         Z_pert (扰动效应) ←── T (处理)             │  │
│   │              ↓              (扰动类型/剂量)         │  │
│   │         X (观测基因表达)                            │  │
│   │                                                     │  │
│   │   假设:                                             │  │
│   │   - 细胞固有状态与扰动独立                          │  │
│   │   - 扰动效应依赖于处理                              │  │
│   │   - 观测是两者共同作用的结果                        │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   生成模型:                                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   先验分布:                                         │  │
│   │   p(Z_cell) = N(0, I)                              │  │
│   │   p(Z_pert | T) = N(mu_pert(T), sigma_pert(T))     │  │
│   │                                                     │  │
│   │   解码器:                                           │  │
│   │   X = Decoder(Z_cell, Z_pert)                      │  │
│   │       = f(Z_cell + g(Z_pert))                      │  │
│   │                                                     │  │
│   │   其中:                                             │  │
│   │   - f: 神经网络解码器                               │  │
│   │   - g: 扰动效应变换                                 │  │
│   │   - +: 假设效应可叠加                               │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   推断模型 (编码器):                                        │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   输入: X (基因表达)                                │  │
│   │                                                     │  │
│   │   q(Z_cell | X) = N(mu_cell(X), sigma_cell(X))     │  │
│   │   q(Z_pert | X, T) = N(mu_pert(X,T), sigma_pert)   │  │
│   │                                                     │  │
│   │   编码器架构:                                       │  │
│   │   ┌─────────────────────────────────────────────┐  │  │
│   │   │                                             │  │  │
│   │   │  X ──→ MLP Encoder ──┬──→ mu_cell          │  │  │
│   │   │   (2048-1024-512)    ├──→ log_sigma_cell    │  │  │
│   │   │                      ├──→ mu_pert           │  │  │
│   │   │                      └──→ log_sigma_pert    │  │  │
│   │   │                                             │  │  │
│   │   └─────────────────────────────────────────────┘  │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 因果 do-算子实现

```
┌─────────────────────────────────────────────────────────────┐
│              Causal do-operator in scCausalVI               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   标准条件概率 vs 干预概率:                                 │
│                                                             │
│   条件概率 (观察):                                          │
│   P(X | T=t)                                                │
│   "观察到处理为t时，X的分布"                                │
│                                                             │
│   干预概率 (do-算子):                                       │
│   P(X | do(T=t))                                            │
│   "强制将处理设为t时，X的分布"                              │
│                                                             │
│   在scCausalVI中的实现:                                     │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   反事实预测流程:                                   │  │
│   │                                                     │  │
│   │   1. 编码观测细胞:                                  │  │
│   │      Z_cell ~ q(Z_cell | X_obs)                    │  │
│   │                                                     │  │
│   │   2. 设定干预:                                      │  │
│   │      T_counter = t_new  (新的扰动条件)             │  │
│   │                                                     │  │
│   │   3. 采样扰动效应:                                  │  │
│   │      Z_pert ~ p(Z_pert | T=t_new)                  │  │
│   │                                                     │  │
│   │   4. 生成反事实表达:                                │  │
│   │      X_cf = Decoder(Z_cell, Z_pert)                │  │
│   │                                                     │  │
│   │   结果: 同一细胞在不同扰动下的预测表达              │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   平均处理效应 (ATE) 估计:                                  │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   ATE = E[X | do(T=1)] - E[X | do(T=0)]            │  │
│   │                                                     │  │
│   │   实现:                                             │  │
│   │   - 对多个细胞进行反事实预测                        │  │
│   │   - 计算干预组和对照组的平均差异                    │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 训练目标

```
┌─────────────────────────────────────────────────────────────┐
│                scCausalVI Training Objective                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   证据下界 (ELBO):                                          │
│                                                             │
│   L = E_q[log p(X | Z_cell, Z_pert)]                       │
│       - KL(q(Z_cell|X) || p(Z_cell))                       │
│       - KL(q(Z_pert|X,T) || p(Z_pert|T))                   │
│                                                             │
│   其中:                                                     │
│   - 重建项: 确保生成数据与观测一致                          │
│   - KL_cell: 正则化细胞状态分布                             │
│   - KL_pert: 正则化扰动效应分布                             │
│                                                             │
│   额外约束:                                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   1. 解耦约束:                                      │  │
│   │      确保 Z_cell 和 Z_pert 信息不重叠               │  │
│   │      通过互信息最小化实现                           │  │
│   │                                                     │  │
│   │   2. 因果一致性约束:                                │  │
│   │      对于同一细胞，不同扰动的预测应合理             │  │
│   │                                                     │  │
│   │   3. 处理效应约束:                                  │  │
│   │      已知扰动的效应应与文献一致                     │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 模型配置

| 参数 | 值 |
|-----|-----|
| 细胞状态维度 | 64 |
| 扰动效应维度 | 32 |
| 编码器层数 | 3 |
| 解码器层数 | 3 |
| 隐藏维度 | 512-1024 |
| 激活函数 | ReLU |
| 优化器 | Adam |
| 学习率 | 1e-4 |

## 输入/输出

- **输入**:
  - 单细胞基因表达 (scRNA-seq)
  - 处理/扰动标签
  - 可选: 剂量信息
- **输出**:
  - 细胞固有状态 (Z_cell)
  - 扰动效应 (Z_pert)
  - 反事实预测
  - 处理效应估计

## 性能指标

scCausalVI 在扰动响应预测上表现优异：

| 任务 | 数据集 | scCausalVI | 次优方法 |
|-----|--------|------------|----------|
| 反事实预测 | Norman 2021 | R^2 0.78 | CPA (0.71) |
| 效应解耦 | Adamson 2016 | AUC 0.89 | Linear (0.82) |
| ATE估计 | Replogle 2022 | MAE 0.12 | scGen (0.18) |
| 细胞异质性 | Srivatsan 2020 | 识别率 85% | - |

## 功能特点

1. **因果解耦**: 分离固有变异和扰动效应
2. **单细胞反事实**: 个体水平的虚拟扰动
3. **因果可解释**: 基于 do-算子的因果推理
4. **异质性分析**: 识别响应不同的细胞亚群
5. **不确定性量化**: 变分框架提供置信度

## 应用场景

- 药物响应预测
- 扰动机制解析
- 细胞异质性研究
- 治疗靶点发现
- COVID-19 感染响应分析

## 优势与局限

**优势**:
- 显式因果建模
- 单细胞分辨率反事实预测
- 效应解耦可解释
- 不确定性量化

**局限**:
- 假设效应可叠加
- 需要足够的扰动数据
- 计算复杂度较高
