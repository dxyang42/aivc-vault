## 核心技术

### 完整架构设计

CPA是**组合扰动自编码器**，通过解耦表示学习分离扰动、细胞类型、剂量和时间效应：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        CPA 完整架构图                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                         输入层                                     │   │
│  │                                                                     │   │
│  │    基因表达向量 x (20,000+维)                                      │   │
│  │    扰动标识 p (One-hot编码或嵌入向量)                              │   │
│  │    细胞类型标识 c (One-hot编码或嵌入向量)                          │   │
│  │    剂量值 d (连续标量值)                                           │   │
│  │    时间值 t (可选，连续标量)                                       │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                      编码器网络                                    │   │
│  │                                                                     │   │
│  │    基因表达向量 x                                                   │   │
│  │         ↓                                                           │   │
│  │    ┌─────────────────────────────────────────────────────────┐     │   │
│  │    │  全连接层1: n_genes → 512  + ReLU                       │     │   │
│  │    │         ↓                                               │     │   │
│  │    │  全连接层2: 512 → 256  + ReLU                           │     │   │
│  │    │         ↓                                               │     │   │
│  │    │  全连接层3: 256 → 128  + ReLU                           │     │   │
│  │    │         ↓                                               │     │   │
│  │    │  全连接层4: 128 → 50-100 (基础潜在维度)                 │     │   │
│  │    └─────────────────────────────────────────────────────────┘     │   │
│  │                      ↓                                              │   │
│  │    输出: 基础潜在表示 z_base (50-100维)                             │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │              条件嵌入层 (CPA核心创新 - 解耦表示)                   │   │
│  │                                                                     │   │
│  │    四个独立的嵌入网络，分别编码不同因素:                            │   │
│  │                                                                     │   │
│  │    ┌─────────────────────────────────────────────────────────┐     │   │
│  │    │  1. 扰动嵌入 (z_p)                                      │     │   │
│  │    │     扰动ID → 嵌入层 → 扰动向量 (d维)                    │     │   │
│  │    │     (每个扰动学习一个固定维度的嵌入向量)                │     │   │
│  │    │                                                         │     │   │
│  │    │  2. 细胞类型嵌入 (z_c)                                  │     │   │
│  │    │     细胞类型ID → 嵌入层 → 细胞类型向量 (d维)            │     │   │
│  │    │     (每个细胞类型学习一个固定维度的嵌入向量)            │     │   │
│  │    │                                                         │     │   │
│  │    │  3. 剂量嵌入 (z_d)                                      │     │   │
│  │    │     剂量值 → 小型MLP → 剂量向量 (d维)                   │     │   │
│  │    │     (连续剂量值通过神经网络映射到嵌入空间)              │     │   │
│  │    │                                                         │     │   │
│  │    │  4. 时间嵌入 (z_t, 可选)                                │     │   │
│  │    │     时间值 → 小型MLP → 时间向量 (d维)                   │     │   │
│  │    │     (支持时间序列扰动数据)                              │     │   │
│  │    └─────────────────────────────────────────────────────────┘     │   │
│  │                                                                     │   │

│                                    ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │

│  │    │                                                         │     │   │

│  │    │                                                         │     │   │
│  │    │   关键特性:                                             │     │   │

│  │    │   - 加法组合使模型具有可解释性                          │     │   │
│  │    │   - 支持任意组合推断                                    │     │   │
│  │    │   - 可分离各因素效应                                    │     │   │
│  │    │                                                         │     │   │
│  │    └─────────────────────────────────────────────────────────┘     │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                      解码器网络                                    │   │
│  │                                                                     │   │
│  │    输入: 组合潜在表示 z_combined (50-100维)                         │   │
│  │         ↓                                                           │   │
│  │    ┌─────────────────────────────────────────────────────────┐     │   │
│  │    │  对称结构: 与编码器镜像                                 │     │   │
│  │    │         ↓                                               │     │   │
│  │    │  全连接层1: 50-100 → 128  + ReLU                        │     │   │
│  │    │         ↓                                               │     │   │
│  │    │  全连接层2: 128 → 256  + ReLU                           │     │   │
│  │    │         ↓                                               │     │   │
│  │    │  全连接层3: 256 → 512  + ReLU                           │     │   │
│  │    │         ↓                                               │     │   │
│  │    │  全连接层4: 512 → n_genes  + 适当激活函数               │     │   │
│  │    └─────────────────────────────────────────────────────────┘     │   │
│  │                      ↓                                              │   │
│  │    输出: 重建/预测基因表达 x'                                       │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    ↓                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                         输出层                                     │   │
│  │                                                                     │   │
│  │    输出类型:                                                        │   │
│  │    - 重建表达 (去噪后的基因表达)                                    │   │
│  │    - 预测扰动后表达 (给定扰动+细胞类型+剂量)                        │   │
│  │    - 解耦潜在表示 (各因素独立向量)                                  │   │
│  │    - 剂量-响应曲线 (连续剂量预测)                                   │   │
```

### 技术特点

- **解耦表示**: 显式分离扰动、细胞类型、剂量、时间四个因素
- **组合建模**: 支持多扰动组合（通过加法组合实现）
- **剂量-响应**: 连续剂量值建模，支持剂量外推

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单药物处理** | 完全支持 | 多种剂量水平 |
| **多药物组合** | 完全支持 | 加法组合模型 |
| **单基因扰动** | 支持 | KO/KD/OE |
| **基因-药物组合** | 支持 | 跨类型组合 |
| **跨细胞类型** | 核心优势 | 迁移扰动效应 |
| **剂量外推** | 支持 | 预测未测试剂量 |

### 预测准确率与评估指标

| 指标 | 结果 | 说明 |
|------|------|------|
| **分布外预测** | 支持 | 预测未见药物组合 |
| **跨细胞类型** | 良好 | 可转移扰动效应 |
| **剂量-响应** | 准确 | 剂量-响应曲线估计 |
| **组合效应** | 中等 | 预测遗传相互作用 |
| **R²分数** | 0.70-0.85 | 基因表达预测 |

### 评估数据集
- **药物筛选数据**: 100+药物，多剂量
- **细胞类型**: 10+细胞类型
- **组合扰动**: 多药物组合实验

## 与其他扰动预测方法的差异

| 特性 | CPA | GEARS | scPerb | CellFlow |
|------|-----|-------|--------|----------|
| **解耦表示** | 显式4因素 | 隐式 | 2因素 | 隐式 |
| **剂量建模** | 连续值 | 不支持 | 不支持 | 部分支持 |
| **细胞类型迁移** | 支持 | 有限 | 有限 | 支持 |
| **组合建模** | 加法组合 | 非加性 | 需参考 | 支持 |
| **可解释性** | 高 | 高 | 高 | 中等 |
| **训练数据要求** | 多条件 | 配对样本 | 参考细胞 | 大规模 |

## 训练策略

### 损失函数组成

训练损失由三部分组成：
- **重构损失**: 衡量重建/预测准确性
- **对抗损失**: 确保解耦表示的质量
- **KL散度**: 正则化潜在空间

### 优化器
- **类型**: Adam
- **学习率**: 1e-3
- **批次大小**: 128

## 功能特点

- 分布外预测: 预测未见药物组合
- 可解释潜在空间: 学习可解释的扰动/细胞嵌入
- 剂量-响应曲线: 估计每个扰动的剂量-响应
- 跨细胞类型: 将扰动效应转移到未见细胞类型
- 组合预测: 预测组合遗传相互作用
- 不确定性: 提供预测不确定性

## 应用场景

1. **药物组合发现**: 预测协同/拮抗效应，优化药物配比
2. **剂量优化**: 确定最佳剂量，避免毒性
3. **细胞类型特异性**: 预测不同细胞类型的响应，个性化医疗

## 优势与局限

### 优势
- 解耦表示，可解释性强
- 支持组合扰动
- 剂量-响应建模
- 跨细胞类型泛化
- 开源，FAIR支持

### 局限
- 需要多条件训练数据
- 对未见扰动的泛化有限
- 组合效应为加法假设（非加性效应有限）
- 计算成本较高
