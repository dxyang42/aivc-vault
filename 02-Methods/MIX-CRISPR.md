# MIX-CRISPR (Combinatorial Perturbation Analysis)

> [[方法地图]] | [[团队地图]]

## 1. 基本信息

| 属性 | 详情 |
|------|------|
| **模型名称** | MIX-CRISPR / Combinatorial Perturbation Analysis |
| **发布时间** | 2021年 |
| **机构** | Broad Institute / MIT |
| **论文** | Replogle et al., Nature Biotechnology 2021 |
| **类型** | 组合扰动筛选与分析方法 |
| **定位** | 大规模组合CRISPR筛选的标准方法 |

MIX-CRISPR 是一套用于大规模组合CRISPR筛选的计算分析方法，由 Replogle 等人在 2021 年开发。该方法能够系统性地分析双基因组合扰动的遗传相互作用，识别协同致死、合成致死等重要的基因对。

## 2. 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **双基因敲除** | 核心功能 | 系统性双基因组合筛选 |
| **单基因敲除** | 支持 | 作为组合分析的参照 |
| **三基因组合** | 扩展支持 | 高阶组合分析 |
| **遗传相互作用** | 核心优势 | 定量GI评分计算 |
| **协同/拮抗效应** | 支持 | 分类基因对关系 |

### 遗传相互作用评分

```
┌─────────────────────────────────────────────────────────────────────┐
│              遗传相互作用 (GI) 评分计算                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   基础概念:                                                         │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   观测值 (Observed): 双基因扰动的实际表型                  │  │
│   │   期望值 (Expected): 基于单基因效应的预期表型              │  │
│   │   GI 评分 = Observed - Expected                            │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   期望值计算模型:                                                   │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   1. 乘性模型 (Multiplicative):                            │  │
│   │      E = f(A) * f(B) / f(control)                          │  │
│   │                                                             │  │
│   │   2. 加性模型 (Additive):                                  │  │
│   │      E = f(A) + f(B) - f(control)                          │  │
│   │                                                             │  │
│   │   3. Bliss 模型:                                           │  │
│   │      E = f(A) + f(B) - f(A)*f(B)                           │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   GI 评分解释:                                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   GI > 0: 协同效应 (Synergistic)                           │  │
│   │   GI = 0: 加性效应 (Additive)                              │  │
│   │   GI < 0: 拮抗效应 (Antagonistic)                          │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 评估指标

| 指标 | 说明 |
|------|------|
| **GI 评分** | 遗传相互作用强度 |
| **p-value** | 统计显著性 |
| **FDR** | 校正后的假阳性率 |
| **效应大小** | 表型变化幅度 |

## 3. 分析流程

### 整体架构

```
┌─────────────────────────────────────────────────────────────────────┐
│              MIX-CRISPR 分析流程                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   阶段1: 数据预处理                                                 │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   输入: 单细胞组合扰动数据                                  │  │
│   │        ↓                                                    │  │
│   │   质量控制:                                                 │  │
│   │   - 细胞过滤 (低质量细胞去除)                               │  │
│   │   - 基因过滤 (低表达基因去除)                               │  │
│   │   - 批次校正 (去除技术变异)                                 │  │
│   │        ↓                                                    │  │
│   │   归一化: 对数变换、文库大小归一化                          │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   阶段2: sgRNA 分配                                                 │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   目标: 将每个细胞分配到对应的扰动组合                      │  │
│   │                                                             │  │
│   │   方法:                                                     │  │
│   │   - 捕获sgRNA序列识别                                       │  │
│   │   - 直接捕获测序 (CROP-seq)                                 │  │
│   │   - 表达特征反卷积                                          │  │
│   │                                                             │  │
│   │   输出: 细胞 × 扰动组合 分配矩阵                            │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   阶段3: 差异表达分析                                               │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   对每个扰动组合:                                           │  │
│   │   - 识别差异表达基因 (DEGs)                                 │  │
│   │   - 计算扰动特征签名                                        │  │
│   │   - 估计扰动效应大小                                        │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   阶段4: GI 评分计算                                                │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   对每个基因对 (A, B):                                      │  │
│   │   - 计算观测表型: f(A+B)                                    │  │
│   │   - 计算期望表型: E(A,B)                                    │  │
│   │   - GI = f(A+B) - E(A,B)                                    │  │
│   │   - 统计检验 (t-test / permutation)                         │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   阶段5: 结果解释                                                   │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   - 显著GI筛选 (FDR < 0.05)                                 │  │
│   │   - 功能富集分析                                            │  │
│   │   - 通路相互作用网络构建                                    │  │
│   │   - 可视化 (热图、网络图)                                   │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 核心组件

| 组件 | 功能 |
|------|------|
| **质量控制模块** | 过滤低质量细胞和基因 |
| **sgRNA分配器** | 将细胞映射到扰动组合 |
| **差异表达引擎** | 识别扰动响应基因 |
| **GI计算器** | 计算遗传相互作用评分 |
| **可视化工具** | 生成结果图表 |

## 4. 训练数据

### 数据来源

| 数据类型 | 描述 | 规模 |
|----------|------|------|
| **Perturb-seq** | 单细胞CRISPR筛选 | 大规模 |
| **CROP-seq** | 直接捕获sgRNA | 中等规模 |
| **ECCITE-seq** | 多模态扰动数据 | 多样化 |

### 数据集覆盖

- **Replogle 2021**: 系统性的双基因筛选数据
- **细胞系**: K562、RPE1、Jurkat 等
- **基因对数量**: 数千至数万对组合
- **细胞数量**: 数百万单细胞

## 5. 与其他扰动预测方法的比较

| 特性 | MIX-CRISPR | GEARS | Linear | X-Pert |
|------|------------|-------|--------|--------|
| **分析类型** | 后验分析 | 预测模型 | 基线 | 预测模型 |
| **组合GI** | 核心功能 | 预测 | 不支持 | 支持 |
| **统计检验** | 完整支持 | 不确定性 | 无 | 部分支持 |
| **实验设计** | 需要专门设计 | 任意数据 | 任意数据 | 任意数据 |
| **计算复杂度** | 中等 | 高 | 低 | 高 |

## 6. 功能特点

### 核心能力

| 功能 | 描述 |
|------|------|
| **系统性筛选** | 支持大规模组合扰动实验 |
| **GI评分** | 定量遗传相互作用强度 |
| **显著性检验** | 统计显著性评估 |
| **功能注释** | 结合通路数据库解释结果 |
| **网络分析** | 构建基因相互作用网络 |

### 技术特色

- **多模型支持**: 乘性、加性、Bliss等多种GI模型
- **质量控制**: 严格的数据质控流程
- **可扩展性**: 支持从双基因扩展到多基因
- **开源生态**: 与Scanpy、Seurat等工具兼容

## 7. 应用场景

### 主要应用

| 应用 | 描述 |
|------|------|
| **合成致死发现** | 识别癌症治疗的协同致死基因对 |
| **通路分析** | 揭示基因间的功能关系 |
| **药物靶点发现** | 发现新的药物组合靶点 |
| **基因功能注释** | 通过相互作用推断基因功能 |

### 应用案例

1. **癌症研究**: 发现肿瘤细胞特异性合成致死相互作用
2. **免疫学**: 解析免疫信号通路的基因网络
3. **发育生物学**: 理解发育过程中的基因调控网络

## 8. 优势与局限

### 核心优势

| 优势 | 说明 |
|------|------|
| **系统性** | 全面评估基因对组合 |
| **定量性** | 提供可比较的GI评分 |
| **统计严谨** | 完整的显著性检验 |
| **标准化** | 成为领域标准分析方法 |
| **可解释** | 结果易于生物学解释 |

### 当前局限

| 局限 | 说明 | 潜在改进方向 |
|------|------|-------------|
| **实验成本** | 大规模筛选成本高昂 | 计算预测辅助筛选 |
| **细胞类型** | 结果可能细胞类型特异 | 跨细胞类型整合 |
| **时间动态** | 主要关注稳态 | 时序扰动分析 |
| **空间信息** | 未整合空间数据 | 结合空间转录组 |
