# X-Cell

**X-Cell** 是由 Xaira Therapeutics 于 2026 年 3 月发布的扩散语言模型（Diffusion Language Model），是目前最大的因果扰动预测模型，拥有 49 亿参数（X-Cell-Ultra）。该模型通过整合多模态生物先验知识，实现了对跨多样细胞上下文的全基因组转录响应的精准预测。

| **下游效应预测** | 信号通路激活/抑制状态 |

## 3. 网络架构

### 3.1 核心架构

X-Cell 采用**扩散语言模型架构**，将单细胞转录组数据视为"基因词汇"的序列，通过扩散过程学习从噪声中生成真实的转录响应。

```
┌─────────────────────────────────────────────────────────────┐
│                    X-Cell 架构概览                           │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐     │
│  │  多模态输入  │───▶│  跨注意力层  │───▶│  扩散解码器  │     │
│  │  编码器      │    │  (Cross-Attn)│    │  (Diffusion)│     │
│  └─────────────┘    └─────────────┘    └─────────────┘     │
│         │                   │                   │          │
│         ▼                   ▼                   ▼          │
│  • 自然语言           • 整合生物先验        • 生成转录响应   │
│  • 蛋白质PLM          • 多模态融合          • 去噪过程     │
│  • 相互作用网络                                          │
│  • 遗传依赖图谱                                          │
│  • 形态学特征                                            │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 关键技术组件

| 组件 | 技术细节 |
|------|----------|
| **跨注意力机制** | 整合多模态生物先验知识到扩散过程 |
| **条件扩散** | 以细胞上下文和扰动信息为条件生成响应 |
| **Transformer 骨干** | 处理基因序列的长程依赖关系 |

## 4. 训练数据

### 4.1 数据集概览

X-Cell 在 **X-Atlas/Pisces** 数据集上进行训练，这是目前最大规模的扰动单细胞数据集之一。

| 数据属性 | 规模 |
|----------|------|
| **扰动单细胞数量** | 2560 万 |
| **CRISPRi Perturb-seq Screens** | 7 个 |
| **扰动类型** | CRISPRi（基因干扰） |
| **读数类型** | 单细胞转录组（scRNA-seq） |

### 4.2 数据来源

- **Perturb-seq 筛选实验**：7 个独立的 CRISPRi Perturb-seq 筛选
- **细胞类型多样性**：涵盖多种细胞上下文环境
- **全基因组覆盖**：系统性的基因扰动设计

---

## 5. 训练策略

### 5.1 训练目标

X-Cell 采用**条件扩散模型**的训练范式：

1. **前向过程**：逐步向真实转录响应添加高斯噪声
2. **反向过程**：学习从噪声中恢复真实的转录响应
3. **条件控制**：以细胞上下文和扰动信息为条件进行生成

### 5.2 多模态先验整合

通过**跨注意力机制**整合以下生物先验：

| 先验类型 | 整合方式 |
|----------|----------|
| **自然语言** | 基因功能描述文本编码 |
| **蛋白质语言模型** | 蛋白质序列嵌入（如 ESM） |
| **相互作用网络** | 蛋白质-蛋白质相互作用图 |
| **遗传依赖图谱** | 基因间功能依赖关系 |
| **形态学特征** | 细胞形态学测量数据 |

### 5.3 缩放定律发现

X-Cell 首次证明了**扰动预测遵循幂律缩放**：

- **发现**：模型性能随计算量、数据量、参数量的增加呈幂律提升
- **关键洞察**：扰动预测的缩放指数与大型语言模型（LLM）的缩放指数相匹配
- **意义**：为扰动预测模型的规模化发展提供了理论依据

## 10. 优势与局限

### 10.1 优势

| 优势 | 说明 |
|------|------|
| **规模领先** | 49 亿参数，目前最大的因果扰动模型 |
| **多模态整合** | 创新性地融合多种生物先验知识 |
| **缩放可预测** | 建立了扰动预测的幂律缩放规律 |
| **跨上下文泛化** | 在多样细胞类型中表现稳定 |
| **开源可用** | 代码和模型权重完全开源 |

### 10.2 局限

| 局限 | 说明 |
|------|------|
| **数据依赖性** | 性能受限于训练数据的覆盖范围 |
| **计算需求** | 49 亿参数模型需要大量计算资源 |
| **扰动类型限制** | 主要针对 CRISPRi 扰动训练，其他扰动类型泛化能力待验证 |
| **机制解释性** | 作为深度学习模型，内部决策过程的黑箱性 |

### 10.3 未来方向

- 扩展至更多扰动类型（CRISPRa、化合物处理等）
- 整合空间转录组信息
- 开发更轻量级的模型变体
- 增强模型的可解释性
