# scKGBERT

## 基本信息

- **全称**: scKGBERT: a knowledge-enhanced foundation model for single-cell analysis
- **发表时间**: 2025年
- **发表期刊**: Genome Biology
- **机构**: 中国科学院北京基因组研究所
- **作者**: Jing Gong, Minghao Hao, et al.

## 核心技术

scKGBERT 是首个将知识图谱（蛋白质-蛋白质相互作用网络）整合到单细胞基础模型中的方法：

1. **知识增强**: 整合 890万 蛋白质-蛋白质相互作用 (PPI)
2. **双模态学习**: 同时学习基因表示和细胞表示
3. **大规模数据**: 基于 4100万 单细胞 RNA-seq 数据预训练
4. **知识图谱嵌入**: 利用 PPI 网络结构增强基因表示

### 知识图谱编码

```
┌─────────────────────────────────────────────────────────────┐
│                 Knowledge Graph Encoding                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   PPI 网络表示学习:                                         │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   输入: PPI 网络 G = (V, E)                        │  │
│   │         V: 蛋白质节点 (~20,000)                    │  │
│   │         E: 相互作用边 (~8,900,000)                 │  │
│   │                                                     │  │
│   │   编码方法:                                         │  │
│   │   ┌─────────────────────────────────────────────┐  │  │
│   │   │ 1. Node2Vec 随机游走                        │  │  │
│   │   │    - 在PPI网络上进行随机游走                │  │  │
│   │   │    - 学习节点嵌入                           │  │  │
│   │   │                                             │  │  │
│   │   │ 2. Graph Attention Network (GAT)            │  │  │
│   │   │    - 考虑邻居节点的注意力权重               │  │  │
│   │   │    - 捕捉局部网络结构                       │  │  │
│   │   │                                             │  │  │
│   │   │ 3. 多关系编码                               │  │  │
│   │   │    - 区分不同类型的相互作用                 │  │  │
│   │   │    - 物理相互作用 vs 功能关联               │  │  │
│   │   └─────────────────────────────────────────────┘  │  │
│   │                                                     │  │
│   │   输出: 每个基因的知识嵌入向量 (256-dim)           │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   知识来源:                                                 │
│   - STRING 数据库 (蛋白相互作用)                            │
│   - BioGRID 数据库 (实验验证相互作用)                       │
│   - GO 注释 (功能关联)                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 预训练任务

### 任务1: 掩码基因建模 + 知识约束

```
标准 MLM:
输入:  [G1, G2, MASK, G4, G5]
预测:  G3 的表达值

知识增强 MLM:
输入:  [G1, G2, MASK, G4, G5]
知识:  G3 与 G1,G5 有PPI连接
预测:  G3 的表达值 (受PPI约束)
损失:  L_MLM + lambda * L_Knowledge
```

### 任务2: 基因-基因关系预测

```
输入:  基因对 (Gi, Gj)
知识:  PPI网络中是否有边
预测:  两个基因是否相互作用
损失:  二分类交叉熵
```

### 任务3: 细胞类型预测

```
输入:  细胞基因表达
知识:  细胞类型标志物基因
预测:  细胞类型
损失:  多分类交叉熵
```

## 模型配置

| 参数 | 值 |
|-----|-----|
| 参数量 | 110M |
| Transformer 层数 | 24 |
| 隐藏维度 | 1024 |
| 注意力头数 | 16 |
| 基因嵌入维度 | 256 (表达) + 256 (知识) |
| 细胞嵌入维度 | 1024 |
| 预训练数据 | 41M 细胞 |
| PPI 数据 | 8.9M 相互作用 |

## 输入/输出

- **输入**:
  - 单细胞基因表达数据
  - 可选: 查询基因对关系
- **输出**:
  - 1024 维细胞嵌入
  - 融合知识的基因嵌入
  - 细胞类型预测
  - 基因-基因关系预测

## 性能指标

scKGBERT 在多个任务上优于纯表达基础模型：

| 任务 | 数据集 | scKGBERT | scFoundation | 提升 |
|-----|--------|----------|--------------|------|
| 细胞注释 | Zheng68K | 95.8% | 94.2% | +1.6% |
| 细胞注释 | Human Liver | 94.5% | 93.2% | +1.3% |
| 基因关系预测 | STRING | AUC 0.89 | - | - |
| 通路富集 | KEGG | NES 2.8 | NES 2.3 | +22% |
| 标志物识别 | CellMarker | F1 0.91 | F1 0.85 | +7% |

## 功能特点

1. **知识增强表示**: 基因嵌入融合PPI网络信息
2. **可解释性**: 注意力权重反映基因间关系
3. **双模态学习**: 同时优化细胞表示和基因表示
4. **关系预测**: 可预测基因间相互作用
5. **通路分析**: 更好的生物学通路识别

## 应用场景

- 细胞类型精细注释
- 基因功能预测
- 通路富集分析
- 基因关系推断
- 药物靶点预测

## 优势与局限

**优势**:
- 首个知识增强的单细胞基础模型
- 融合大规模PPI网络信息
- 更好的基因功能预测
- 可解释的基因关系

**局限**:
- PPI网络质量依赖先验知识
- 计算复杂度高于纯表达模型
- 对未知基因功能预测有限
