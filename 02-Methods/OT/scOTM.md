### 核心创新

scOTM 是一种基于**深度学习和最优传输理论**的单细胞扰动响应预测框架。该方法专门针对**非配对数据**设计，能够从未配对的对照和扰动细胞中学习扰动效应，并泛化到未见的细胞类型。

## 6. 网络结构

### 最优传输层

```python
# 最优传输的伪代码实现
def optimal_transport_loss(z_control, z_perturb, epsilon=0.1):
    """
    计算熵正则化最优传输损失 (Sinkhorn算法)
    """
    # 计算成本矩阵
    C = pairwise_distances(z_control, z_perturb, metric='sqeuclidean')
    
    # 添加生物学成本
    C_bio = biological_cost_matrix(z_control, z_perturb)
    C = C + lambda_bio * C_bio
    
    # Sinkhorn迭代计算传输计划
    P = sinkhorn(C, epsilon, max_iter=100)
    
    # 计算传输成本
    transport_cost = torch.sum(P * C)
    
    return transport_cost, P

def sinkhorn(C, epsilon, max_iter):
    """Sinkhorn-Knopp算法"""
    K = torch.exp(-C / epsilon)
    u = torch.ones(C.shape[0])
    v = torch.ones(C.shape[1])
    
    for _ in range(max_iter):
        u = 1.0 / (K @ v)
        v = 1.0 / (K.T @ u)
    
    P = torch.diag(u) @ K @ torch.diag(v)
    return P
```

### 条件生成网络

```python
class ConditionalOTM(nn.Module):
    def __init__(self, input_dim, latent_dim, num_perturbations):
        super().__init__()
        # 编码器
        self.encoder = MLP(input_dim, [512, 256], latent_dim)
        
        # 扰动嵌入
        self.perturbation_embedding = nn.Embedding(num_perturbations, 64)
        
        # 条件传输网络
        self.transport_net = MLP(latent_dim + 64, [256, 256], latent_dim)
        
        # 解码器
        self.decoder = MLP(latent_dim, [256, 512], input_dim)
    
    def forward(self, x_control, perturbation_id):
        # 编码
        z = self.encoder(x_control)
        
        # 扰动嵌入
        p_emb = self.perturbation_embedding(perturbation_id)
        
        # 条件传输
        z_perturb = self.transport_net(torch.cat([z, p_emb], dim=-1))
        
        # 解码
        x_perturb = self.decoder(z_perturb)
        
        return x_perturb
```

## 8. 训练策略

### 损失函数

```
L_total = L_OT + L_recon + L_cell_type + L_kl

其中：
- L_OT: 最优传输损失 (Sinkhorn距离)
- L_recon: 重构损失 (MSE)
- L_cell_type: 细胞类型分类损失
- L_kl: KL散度 (正则化)
```

### 训练流程

```
1. 预训练编码器
   └── 使用所有数据训练自编码器

2. 最优传输学习
   ├── 计算对照和扰动分布
   ├── 学习最优传输映射
   └── 最小化Sinkhorn距离

3. 条件预测训练
   ├── 整合扰动信息
   ├── 训练条件传输网络
   └── 端到端微调

4. 细胞类型泛化优化
   └── 使用域适应技术
```
