## 核心技术

### 架构设计

CellOT采用**神经最优传输 (Neural Optimal Transport)** 框架，结合**输入凸神经网络 (Input Convex Neural Networks, ICNN)**：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    CellOT 神经最优传输架构                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   问题背景                                                               │
│   ├── 观测数据: 未配对分布                                              │
│   │   ├── 控制组细胞分布 μ_control (未扰动)                            │
│   │   └── 扰动组细胞分布 μ_perturbed (扰动后)                          │
│   └── 挑战: 无法追踪单个细胞的扰动前后变化 (细胞被销毁)                   │
│                              ↓                                          │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │              最优传输理论框架                                    │  │
│   │                                                                 │  │
│   │  目标: 学习最优传输映射 T: R^n → R^n                            │  │
│   │        使得 T#μ_control ≈ μ_perturbed                          │  │
│   │        (将控制组分布推前到扰动组分布)                            │  │
│   │                                                                 │  │
│   │  数学表述:                                                       │  │
│   │  ┌─────────────────────────────────────────────────────────┐   │  │
│   │  │  min_T ∫ c(x, T(x)) dμ_control(x)                      │   │  │
│   │  │                                                         │   │  │
│   │  │  s.t. T#μ_control = μ_perturbed                        │   │  │
│   │  │                                                         │   │  │
│   │  │  其中 c(x,y) 是传输代价函数 (如欧氏距离平方)            │   │  │
│   │  └─────────────────────────────────────────────────────────┘   │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              ↓                                          │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │              输入凸神经网络 (ICNN)                               │  │
│   │                                                                 │  │
│   │  为什么需要ICNN?                                                │  │
│   │  ├── 保证传输映射的凸性 → 存在唯一最优解                         │  │
│   │  ├── 保证传输是确定性的 (Monge映射)                              │  │
│   │  └── 通过Brenier定理与Kantorovich对偶关联                        │  │
│   │                                                                 │  │
│   │  ICNN架构:                                                     │  │
│   │  ┌─────────────────────────────────────────────────────────┐   │  │
│   │  │                                                         │   │  │
│   │  │   f(x; θ) = z_L                                        │   │  │
│   │  │                                                         │   │  │
│   │  │   z_1 = σ_0(W_0 x + b_0)                               │   │  │
│   │  │   z_{l+1} = σ_l(z_l, W_l x + b_l)  for l = 1,...,L-1  │   │  │
│   │  │                                                         │   │  │
│   │  │   约束: W_l ≥ 0 (非负权重) 保证凸性                      │   │  │
│   │  │                                                         │   │  │
│   │  │   σ_l 使用 ReLU 或 Softplus 激活函数                     │   │  │
│   │  │                                                         │   │  │
│   │  └─────────────────────────────────────────────────────────┘   │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              ↓                                          │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │              训练过程                                            │  │
│   │                                                                 │  │
│   │  1. 采样控制组细胞 x ~ μ_control                               │  │
│   │  2. 计算 f(x) 和 ∇f(x) (通过自动微分)                          │  │
│   │  3. ∇f(x) 即为预测的扰动后细胞状态                              │  │
│   │  4. 计算与真实扰动组分布的Wasserstein距离                       │  │
│   │  5. 最小化对偶目标函数                                          │  │
│   │                                                                 │  │
│   │  对偶形式 (Kantorovich):                                       │  │
│   │  ┌─────────────────────────────────────────────────────────┐   │  │
│   │  │  max_f min_g ∫ f(x)dμ_control + ∫ g(y)dμ_perturbed    │   │  │
│   │  │  s.t. f(x) + g(y) ≤ c(x,y)                            │   │  │
│   │  │                                                         │   │  │
│   │  │  对于二次代价: f = g = 凸函数, 且 T = ∇f               │   │  │
│   │  └─────────────────────────────────────────────────────────┘   │  │
│   │                                                                 │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              ↓                                          │
│   推理阶段                                                               │
│   ├── 输入: 新细胞的基因表达谱 x                                       │
│   ├── 计算: ∇f(x) (ICNN的梯度)                                        │
│   └── 输出: 预测的扰动后基因表达谱                                     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 技术特点

- **最优传输理论**: 基于Wasserstein距离的最优映射学习
- **输入凸神经网络**: 保证映射的凸性和唯一性
- **未配对数据学习**: 无需细胞级别的配对数据
- **单细胞分辨率**: 预测单个细胞的扰动响应

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **化学扰动** | 完全支持 | 药物处理响应预测 |
| **基因扰动** | 完全支持 | 敲除/过表达效应 |
| **机械扰动** | 支持 | 细胞力学环境变化 |
| **多条件预测** | 支持 | 不同扰动条件的响应 |

### 预测能力评估

| 指标 | 结果 | 对比基线 |
|------|------|---------|
| **细胞状态转换** | 准确建模分布转换 | vs VAE/GAN |
| **差异表达基因** | 高精度预测 | vs 线性模型 |
| **细胞类型特异性** | 捕捉细胞类型差异 | vs 统一模型 |
| **剂量响应** | 支持剂量-效应关系 | - |

## 技术优势

### 与生成模型对比

| 特性 | CellOT | VAE | GAN | 线性模型 |
|------|--------|-----|-----|---------|
| **理论基础** | 最优传输 | 变分推断 | 对抗训练 | 线性假设 |
| **未配对数据** | 原生支持 | 支持 | 支持 | 需配对 |
| **映射可逆性** | 是 | 否 | 否 | 是 |
| **训练稳定性** | 高 | 中 | 低 | 高 |
| **可解释性** | 中 | 中 | 低 | 高 |

### 核心创新

1. **ICNN架构**: 首次将输入凸神经网络系统应用于单细胞扰动预测

2. **Monge映射学习**: 学习确定性的细胞状态转换映射，而非条件分布

3. **Wasserstein几何**: 利用Wasserstein距离的自然几何结构建模细胞状态流形

4. **物理可解释性**: 传输映射可以解释为细胞状态的"最短路径"转换

## 应用场景

1. **药物响应预测**: 预测患者细胞对药物治疗的个体化响应
2. **基因功能研究**: 推断基因扰动的转录后果
3. **疾病机制解析**: 理解疾病状态下细胞状态的转换
4. **细胞重编程**: 预测分化/去分化过程中的状态变化
