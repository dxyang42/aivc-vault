## 核心技术

### 架构设计

UCE 是一个**跨物种通用细胞嵌入模型**，通过蛋白质编码基因的统一表示实现跨物种迁移：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    UCE 架构概览                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   核心思想: 以蛋白质为中心，而非基因为中心                           │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              输入层: 基因表达 → 蛋白质表示                   │  │
│   │                                                             │  │
│   │   基因表达 ──→ 按蛋白质家族聚合 ──→ 蛋白质表达矩阵         │  │
│   │                                                             │  │
│   │   关键: 使用蛋白质嵌入而非基因ID，实现跨物种统一             │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              染色体级 Transformer 编码器                     │  │
│   │                                                             │  │
│   │   输入: 按染色体位置排序的蛋白质表达                         │  │
│   │                                                             │  │
│   │   处理:                                                     │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   蛋白质1 ──┐                                       │  │  │
│   │   │   蛋白质2 ──┤                                       │  │  │
│   │   │   蛋白质3 ──┼──→ 染色体级自注意力 ──→ 上下文感知     │  │  │
│   │   │     ...   ──┤      表示                             │  │  │
│   │   │  蛋白质N ──┘                                       │  │  │
│   │   │                                                     │  │  │
│   │   │   位置编码: 染色体位置 + 物种标识                     │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   层数: 33层 Transformer                                    │  │
│   │   隐藏维度: 1280                                            │  │
│   │   注意力头数: 20                                            │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              蛋白质语言模型集成                              │  │
│   │                                                             │  │
│   │   使用预训练的蛋白质语言模型 (ESM-2) 嵌入作为输入            │  │
│   │   整合序列、结构和功能信息到细胞表示中                       │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              通用细胞嵌入输出                                │  │
│   │                                                             │  │
│   │   输出: 1280维通用细胞嵌入向量                               │  │
│   │   特性: 跨物种、跨组织、跨模态可比                           │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 扰动预测能力

### 跨物种扰动迁移

UCE 的核心优势在于**跨物种扰动效应预测**：

| 能力 | 描述 | 应用场景 |
|------|------|---------|
| **人类→小鼠迁移** | 将人类细胞扰动效应映射到小鼠模型 | 临床前研究 |
| **小鼠→人类迁移** | 将小鼠实验结果映射到人类细胞 | 转化医学 |
| **模式生物→人类** | 从斑马鱼/果蝇等推断人类响应 | 药物筛选 |
| **跨组织迁移** | 将一种组织的扰动效应迁移到另一种组织 | 组织特异性研究 |

### 扰动效应建模

```
┌─────────────────────────────────────────────────────────────┐
│              UCE 扰动效应建模流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   输入:                                                     │
│   ├── 对照细胞表达: X_ctrl                                  │
│   └── 扰动细胞表达: X_pert                                  │
│                                                             │
│   步骤1: 编码到通用嵌入空间                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   E_ctrl = UCE_Encoder(X_ctrl)   → 1280维         │  │
│   │   E_pert = UCE_Encoder(X_pert)   → 1280维         │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   步骤2: 计算扰动向量                                       │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   ΔE = E_pert - E_ctrl  (通用空间中的扰动效应)      │  │
│   │                                                     │  │
│   │   该向量在跨物种/跨组织间具有可比性                 │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   步骤3: 跨物种迁移                                         │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   目标物种对照: X_target_ctrl                       │  │
│   │   E_target = UCE_Encoder(X_target_ctrl)             │  │
│   │                                                     │  │
│   │   预测: E_target_pert = E_target + ΔE               │  │
│   │                                                     │  │
│   │   (假设: 扰动效应在通用空间中保守)                  │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   输出: 目标物种的预测扰动后嵌入                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 应用场景

### 1. 跨物种药物发现

```
流程:
人类疾病细胞 ──→ UCE嵌入 ──→ 计算扰动响应 ──→ 映射到小鼠模型
                                          ↓
                                    小鼠实验验证
                                          ↓
                                    结果映射回人类
```

### 2. 模式生物筛选

| 步骤 | 描述 |
|------|------|
| 1. 斑马鱼筛选 | 低成本大规模药物筛选 |
| 2. UCE嵌入 | 将斑马鱼响应映射到通用空间 |
| 3. 人类预测 | 推断人类细胞响应 |
| 4. 优先级排序 | 选择最有希望的候选药物 |

### 3. 疾病模型比较

- 比较人类疾病样本与小鼠疾病模型的相似性
- 评估模式生物对特定疾病的建模准确性
- 识别跨物种保守的疾病机制

## 优势与局限

### 优势
- 真正的跨物种通用表示
- 整合蛋白质语言模型先验
- 大规模训练数据（3600万细胞）
- 开源，社区支持

### 局限
- 主要关注细胞类型表示，非专门的扰动预测
- 需要下游模型进行具体的扰动效应预测
- 计算资源需求较高
- 对非编码区变异建模有限
