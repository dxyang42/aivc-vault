## 核心技术

### 架构设计

CellOracle整合**基因调控网络(GRN)推断**与**计算机模拟**：

```
┌─────────────────────────────────────────────────────────────────────┐
│                  CellOracle 基因调控网络模拟架构                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   整体流程: 数据输入 ──→ GRN推断 ──→ 扰动模拟 ──→ 命运预测          │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              步骤1: 数据预处理                               │  │
│   │                                                             │  │
│   │   输入数据:                                                 │  │
│   │   ┌──────────────────┐      ┌──────────────────┐           │  │
│   │   │   scRNA-seq      │      │   scATAC-seq     │           │  │
│   │   │   (必需)         │      │   (可选)         │           │  │
│   │   │                  │      │                  │           │  │
│   │   │  - 基因表达矩阵  │      │  - Peak矩阵      │           │  │
│   │   │  - 细胞类型标签  │      │  - 染色质可及性  │           │  │
│   │   │  - 质量控制      │      │  - TF结合位点    │           │  │
│   │   └────────┬─────────┘      └────────┬─────────┘           │  │
│   │            └─────────────────────────┘                     │  │
│   │                         ↓                                  │  │
│   │   预处理:                                                  │  │
│   │   - 质量控制 (QC)                                          │  │
│   │   - 归一化                                                 │  │
│   │   - 降维 (PCA/UMAP)                                        │  │
│   │   - 聚类/细胞类型识别                                      │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              步骤2: 基因调控网络(GRN)推断                    │  │
│   │                                                             │  │
│   │   目标: 构建细胞类型特异性的基因调控网络                    │  │
│   │                                                             │  │

│   │   │                                                     │  │  │
│   │   │   2a. 转录因子(TF)识别:                             │  │  │

│   │   │       - 结合基序数据库筛选                          │  │  │
│   │   │                                                     │  │  │
│   │   │   2b. 基序-靶基因映射:                              │  │  │
│   │   │       ┌─────────────────────────────────────────┐  │  │  │
│   │   │       │                                         │  │  │  │
│   │   │       │   参考基因组 ──→ 基序扫描 ──→         │  │  │  │
│   │   │       │   TF结合位点                            │  │  │  │
│   │   │       │                                         │  │  │  │
│   │   │       │   输入:                                 │  │  │  │
│   │   │       │   - 基序数据库 (JASPAR/CIS-BP)          │  │  │  │

│   │   │       │   - 基因注释 (GTF)                      │  │  │  │

│   │   │       │                                         │  │  │  │
│   │   │       │   对每个靶基因:                         │  │  │  │

│   │   │       │   - 每个细胞类型独立推断                │  │  │  │
│   │   │       │                                         │  │  │  │
│   │   │       │   输出: TF-靶基因调控边权重             │  │  │  │
│   │   │       │                                         │  │  │  │
│   │   │       └─────────────────────────────────────────┘  │  │  │
│   │   │                                                     │  │  │

│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   3a. 选择目标TF                                    │  │  │
│   │   │                                                     │  │  │
│   │   │   3b. 扰动操作:                                     │  │  │
│   │   │       ┌─────────────────────────────────────────┐  │  │  │
│   │   │       │                                         │  │  │  │

│   │   │       │   过表达 (OE): 增加TF表达               │  │  │  │

│   │   │       └─────────────────────────────────────────┘  │  │  │
│   │   │                                                     │  │  │
│   │   │   3c. 网络传播计算:                                 │  │  │
│   │   │       ┌─────────────────────────────────────────┐  │  │  │
│   │   │       │                                         │  │  │  │
│   │   │       │   迭代传播算法:                         │  │  │  │
│   │   │       │                                         │  │  │  │
```

### 技术特点
- **GRN推断**: 从单细胞数据推断基因调控网络
- **计算机模拟**: 模拟转录因子扰动
- **细胞命运预测**: 预测细胞身份变化

## 输入/输出

### 输入格式
| 输入类型 | 描述 | 格式 |
|---------|------|------|
| **scRNA-seq** | 单细胞转录组 | count矩阵 |
| **scATAC-seq** | 单细胞染色质可及性 | peak矩阵 |
| **基序注释** | TF结合基序 | 列表 |
| **参考基因组** | 基因组版本 | hg38/mm10 |

### 输出格式
| 输出类型 | 描述 | 格式 |
|---------|------|------|
| **GRN** | 基因调控网络 | 边列表 |
| **TF扰动效应** | 转录因子扰动后的表达变化 | 向量 |
| **细胞身份转换** | 细胞命运转换概率 | 矩阵 |
| **矢量图** | 2D可视化 | 向量场 |

## 训练数据

| 数据属性 | 要求 |
|---------|------|
| **scRNA-seq** | 必需 |
| **scATAC-seq** | 可选，提高准确性 |
| **细胞数量** | 大于1000 |
| **细胞类型** | 多种 |

## 性能指标

| 应用 | 验证 |
|------|------|
| **小鼠造血** | 正确预测分化轨迹 |
| **人造血** | 验证已知调控因子 |
| **斑马鱼胚胎** | 预测发育过程 |

## 功能特点

- GRN推断: 从单细胞数据推断基因调控网络
- TF扰动模拟: 模拟转录因子敲除/过表达
- 细胞命运预测: 预测细胞身份转换
- 可视化: 生成细胞命运矢量图
- 多物种: 支持人和小鼠

## 应用场景

1. **发育生物学**
   - 预测发育轨迹
   - 识别关键调控因子

2. **细胞重编程**
   - 预测重编程结果
   - 优化重编程因子

3. **疾病建模**
   - 模拟疾病相关TF扰动
   - 理解疾病机制

## 支持基因组版本

| 物种 | 版本 |
|------|------|
| **Human** | hg38, hg19 |
| **Mouse** | mm39, mm10, mm9 |

## 优势与局限

### 优势
- 机制驱动，可解释性强
- 整合多组学数据
- 可视化直观
- 开源，社区活跃
- 多物种支持

### 局限
- 依赖GRN推断准确性
- 线性假设可能不适用
- 需要多组学数据
- 计算资源需求高
