# AIVC Perturbation 方法调研 - 第1轮报告

**调研时间**: 2026-03-29
**调研方法**: arXiv 搜索 + agent-browser 网页抓取

## 完成情况

成功创建 **7** 个方法文档，保存于 `projects/aivc-vault/02-Methods/`:

### 已完整收录 (4个)

1. **scDFM** (Yu et al., 2026)
   - **核心创新**: 基于条件流匹配的分布级建模，引入 MMD 目标和 PAD-Transformer 骨干网络，在组合扰动上 MSE 降低 19.6%
   - **架构**: Flow Matching + PAD-Transformer
   - **论文**: https://arxiv.org/abs/2602.07103

2. **SCALE** (Chen et al., 2026)
   - **核心创新**: BioNeMo 基础的大规模训练框架（12.51x 加速）+ 集合感知流架构 + 生物学忠实评估，在 Tahoe-100M 上 PDCorr 提升 12.02%
   - **架构**: Flow Matching + LLaMA-based Cellular Encoding
   - **论文**: https://arxiv.org/abs/2603.17380

3. **CFM-GP** (Abir et al., 2025)
   - **核心创新**: 统一的条件流匹配框架，支持跨细胞类型的基因扰动学习
   - **架构**: Conditional Flow Matching
   - **论文**: https://arxiv.org/abs/2508.08312

4. **scPPDM** (Liang et al., 2025)
   - **核心创新**: 基于扩散模型的单细胞药物反应预测
   - **架构**: Diffusion Model
   - **论文**: https://arxiv.org/abs/2510.11726

## 技术趋势分析

### 主流架构
- **Flow Matching** 成为主流：scDFM、SCALE、CFM-GP 均采用
- **Diffusion Model**：scPPDM 等药物预测方法使用
- **Transformer-based**：SCALE 使用 LLaMA 细胞编码

### 关键创新点
1. **分布级建模**: 超越细胞级对应，捕捉群体变化 (scDFM)
2. **大规模可扩展**: 支持 1亿级数据训练 (SCALE)
3. **跨细胞类型**: 统一框架处理多种细胞类型 (CFM-GP)
4. **生物学评估**: 关注生物学忠实度而非仅重建精度 (SCALE)

### 数据集
- **Tahoe-100M**: 1亿扰动数据集，成为新的基准
- **Norman、Replogle 等**: 经典扰动数据集
