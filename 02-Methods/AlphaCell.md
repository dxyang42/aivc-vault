## 核心技术

### 架构设计
AlphaCell是**虚拟细胞世界模型**，由两个核心组件组成：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    AlphaCell 架构概览                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              基础模型 (Foundation Model)                     │   │
│  │                                                             │   │
│  │  功能: 学习细胞状态表征                                     │   │
│  │       └── 全基因组表征学习                                  │   │
│  │                                                             │   │
│  │  创新点 1: 潜流形校正 (Latent Manifold Correction)          │   │
│  │  ├── 掩码自监督学习 (Masked Self-Supervised Learning)      │   │
│  │  ├── 域对抗神经网络 (Domain Adversarial Network)           │   │
│  │  └── 弧面损失 (ArcFace Loss)                               │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              流模型 (Flow Model)                             │   │
│  │                                                             │   │
│  │  核心: 基于最优传输条件流匹配 (Optimal Transport CFM)       │   │
│  │                                                             │   │
│  │  创新点 2: 生物现实重建 (Biologically Realistic Reconstr.) │   │
│  │  └── 高保真全基因组表达谱重建                               │   │
│  │                                                             │   │
│  │  创新点 3: 通用状态转换 (Universal State Transition)       │   │
│  │  └── 基于最优传输条件流匹配模拟细胞状态转换                 │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  输出: 预测的扰动后细胞状态                                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 条件流匹配目标函数

AlphaCell基于条件流匹配(Conditional Flow Matching, CFM)框架：

```
┌─────────────────────────────────────────────────────────────────────┐
│              条件流匹配 (Conditional Flow Matching)                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  目标: 学习从对照分布到扰动分布的传输映射                           │
│                                                                     │
│  速度场: v_theta(x, t, c) = E[(x_1 - x_0) / (1 - 0) | x_t = x, c]      │
│                                                                     │
│  其中:                                                              │
│  - x_0 ~ p_0 : 对照状态分布                                         │
│  - x_1 ~ p_1 : 扰动后状态分布                                       │
│  - c = [c_pert; c_cell] : 扰动条件和细胞类型条件的拼接              │
│  - t in [0, 1] : 流时间参数                                          │
│                                                                     │
│  条件流匹配损失函数:                                                │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  L_CFM = E[ ||v_theta(x_t, t, c) - (x_1 - x_0)||^2 ]            │   │
│  │                                                             │   │
│  │  其中:                                                      │   │
│  │  - t 从 0 到 1 均匀采样                                     │   │
│  │  - x_0 来自对照分布 p_0                                     │   │
│  │  - x_1 来自扰动分布 p_1                                     │   │
│  │  - x_t = (1-t)·x_0 + t·x_1 为线性插值路径                   │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 最优传输目标

AlphaCell采用最优传输理论进行状态转换建模：

```
最小化传输成本:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  目标: 找到最优传输映射 T，最小化以下成本:                          │
│                                                                     │
│  Cost = integral c(x, T(x)) dp_0(x) + lambda * Reg(T)                           │
│                                                                     │
│  其中:                                                              │
│  - T: X → X 为传输映射                                              │
│  - c(x, y) = ||x - y||² : 传输成本（欧氏距离平方）                  │
│  - Reg(T): 正则化项                                                 │
│  - lambda: 正则化系数                                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 潜流形校正机制

### 1. 掩码自监督学习

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  掩码重建损失:                                                      │
│                                                                     │
│  L_mask = E[ sum_{i in M} ||x_i - x_hat_i||^2 ]                            │
│                                                                     │
│  其中:                                                              │
│  - M: 随机掩码的基因集合                                            │
│  - x̂_i = Decoder(Encoder(x_{非M})): 从非掩码基因重建              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 2. 域对抗训练

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  对抗损失确保潜流形的生物学一致性:                                  │
│                                                                     │
│  L_adv = E_{x~真实数据}[log D(x)] + E_{z~潜在空间}[log(1 - D(G(z)))]
  
  (注: E_{x~真实数据} 表示对真实数据分布求期望)│
│                                                                     │
│  训练目标: 最小化生成器G损失，最大化判别器D损失                     │
│                                                                     │
│  其中:                                                              │
│  - G: 生成器（解码器）                                              │
│  - D: 域判别器                                                      │
│  - 真实数据分布: 观测到的细胞表达                                   │
│  - 潜在空间分布: 编码器输出的潜在表示                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 3. 弧面损失 (ArcFace Loss)

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  用于优化细胞类型在潜流形上的分离:                                  │
│                                                                     │
│  L_arc = -(1/N) * sum_{i=1}^N log( P_i / (P_i + sum_{j!=y_i} N_j) )     │
│                                                                     │
│  其中:                                                              │
│  - P_i = exp(s * cos(theta_{y_i} + m)) : 正类概率项                   │
│  - N_j = exp(s * cos(theta_j)) : 负类概率项                           │
│  - s: 缩放因子                                                      │
│  - m: 边际参数                                                      │
│  - theta_{y_i}: 细胞嵌入与细胞类型中心的角度                            │
│  - N: 样本数量                                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 总损失函数

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  L_total = L_CFM + lambda_1 * L_mask + lambda_2 * L_adv + lambda_3 * L_arc + lambda_4 * L_recon    │
│                                                                     │
│  其中:                                                              │
│  - L_CFM: 条件流匹配损失                                            │
│  - L_mask: 掩码自监督损失                                           │
│  - L_adv: 域对抗损失                                                │
│  - L_arc: 弧面损失                                                  │
│  - L_recon: 全基因组重建损失                                        │
│  - lambda_1, lambda_2, lambda_3, lambda_4: 各损失的权重系数                                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 技术特点

- **世界模型 (World Model) 架构**: 构建虚拟细胞世界模型
- **全基因组表征**: 融合全基因组信息
- **连续状态转换建模**: 模拟细胞状态的连续变化
- **最优传输**: 使用最优传输理论进行状态转换
