# Celcomen (Spatial Causal Disentanglement)

> [[方法地图]] | [[团队地图]]

## 1. 基本信息

| 属性 | 内容 |
|------|------|
| **模型名称** | Celcomen (Spatial Causal Disentanglement) |
| **发布时间** | 2025年 |
| **发表期刊** | Nature Communications 2025 |
| **研究机构** | 剑桥大学 |
| **技术类型** | 图神经网络 (GNN) + 因果推断 |
| **应用领域** | 空间转录组学、因果基因调控分析 |

### 核心贡献

Celcomen 是一种创新的空间因果解耦方法，首次将数学因果关系与图神经网络相结合，实现了空间转录组数据中的因果推断可识别性。该方法为构建**虚拟组织 (Virtual Tissues)** 奠定了重要基础。

## 2. 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **基因敲除 (KO)** | 支持 | 空间位置特异的基因扰动 |
| **药物处理** | 支持 | 空间药物响应预测 |
| **Perturb-map** | 完全支持 | 空间分辨的CRISPR筛选 |
| **反事实预测** | 核心优势 | 预测扰动后的空间表达 |
| **组合扰动** | 部分支持 | 多基因空间扰动 |

### 预测准确率与评估指标

| 指标 | 描述 | 典型值 |
|------|------|--------|
| **因果推断准确性** | 因果边预测AUC-ROC | 0.85-0.92 |
| **AUPRC** | 精确率-召回率曲线下面积 | 0.78-0.88 |
| **因果效应误差** | 估计因果效应的RMSE | < 0.15 |
| **反事实MSE** | 扰动后表达预测误差 | < 0.20 |
| **Pearson r** | 预测与真实相关性 | > 0.75 |
| **Spearman ρ** | 秩相关性 | > 0.70 |

### 评估数据集
- **胶质母细胞瘤**: 人类Visium数据
- **胎儿脾脏**: 人类发育数据
- **肺癌**: 小鼠Perturb-map数据

## 3. 与其他扰动预测方法的差异

| 特性 | Celcomen | GEARS | CPA | X-Pert |
|------|----------|-------|-----|--------|
| **空间信息** | 核心 | 不支持 | 不支持 | 不支持 |
| **因果推断** | 数学保证 | 相关性 | 相关性 | 相关性 |
| **反事实预测** | 支持 | 支持 | 支持 | 支持 |
| **细胞内/间调控** | 双重解耦 | 不支持 | 不支持 | 不支持 |
| **可识别性** | 数学保证 | 无 | 无 | 无 |
| **虚拟组织** | 基础技术 | 不适用 | 不适用 | 不适用 |

## 4. 输入/输出

### 输入

| 输入类型 | 描述 | 格式 |
|---------|------|------|
| **空间转录组数据** | 基因表达矩阵与空间坐标 | Visium, Perturb-map |
| **空间邻接图** | 细胞/spot之间的空间邻接关系 | 邻接矩阵/边列表 |
| **扰动信息** (可选) | 基因扰动或药物处理条件 | 二进制掩码或连续值 |
| **细胞类型注释** (可选) | 细胞类型或状态标签 | 分类标签 |

### 输出

| 输出类型 | 描述 |
|---------|------|
| **因果效应估计** | 基因间的因果调控关系强度 |
| **细胞内调控程序** | 每个细胞内部基因的因果调控网络 |
| **细胞间调控程序** | 细胞间信号传递的因果效应 |
| **反事实预测** | 扰动后的基因表达预测 |
| **潜在表示** | 解耦后的因果潜在变量 |

### 5.2 关键组件

#### 5.2.1 因果图神经网络 (Causal GNN)

- **图消息传递**: 在空间邻接图上进行消息传递
- **因果注意力机制**: 区分因果关系与相关关系
- **结构方程建模**: 将基因调控建模为结构方程

#### 5.2.2 拉格朗日力学约束

- **物理启发的正则化**: 利用拉格朗日力学框架约束因果效应
- **能量最小化**: 确保因果关系的能量最优性
- **动态系统建模**: 将基因调控视为动态物理系统

#### 5.2.3 因果可识别性模块

- **前门/后门准则**: 应用因果推断理论确保可识别性
- **工具变量**: 利用空间信息作为工具变量
- **do-calculus**: 实现干预分布的精确计算

## 6. 训练数据

### 6.1 验证数据集

| 数据集 | 物种/组织 | 技术平台 | 样本特点 |
|--------|----------|----------|---------|
| **胶质母细胞瘤** | 人类 | Visium | 肿瘤微环境、异质性 |
| **胎儿脾脏** | 人类 | Visium | 发育生物学、器官发生 |
| **肺癌** | 小鼠 | Perturb-map | 基因扰动、药物筛选 |

### 6.2 数据预处理

1. **质量控制**: 过滤低质量细胞和基因
2. **归一化**: 对数转换和尺寸因子归一化
3. **高变基因选择**: 选择信息丰富的基因子集
4. **空间图构建**: 基于空间邻近性构建邻接图
5. **扰动编码**: 将扰动信息编码为特征向量

### 6.3 数据增强

- **空间扰动**: 模拟局部基因表达变化
- **图扰动**: 随机移除或添加边以增强鲁棒性
- **反事实采样**: 生成合成扰动数据

## 7. 训练策略

### 7.1 损失函数组成

总损失由四部分组成：
- **重构损失**: 基因表达重构损失
- **因果损失**: 因果效应估计损失
- **拉格朗日损失**: 拉格朗日约束损失
- **解耦损失**: 细胞内/细胞间解耦正则化

### 7.2 训练流程

```
阶段 1: 预训练 (Warm-up)
├── 仅使用重构损失训练编码器-解码器
├── 学习基础空间表示
└── Epochs: 50-100

阶段 2: 因果学习
├── 加入因果损失
├── 训练因果效应估计模块
└── Epochs: 100-200

阶段 3: 解耦优化
├── 加入解耦损失
├── 优化细胞内/细胞间分离
└── Epochs: 100-200

阶段 4: 微调
├── 联合优化所有损失
├── 学习率衰减
└── Epochs: 50-100
```

### 7.3 优化器配置

- **优化器**: AdamW
- **初始学习率**: 1e-3
- **学习率调度**: Cosine Annealing
- **批量大小**: 1 (整张组织切片)
- **梯度裁剪**: 最大范数 1.0

## 8. 功能特点

### 8.1 核心功能

| 功能 | 描述 |
|------|------|
| **因果发现** | 从空间数据中发现基因间的因果关系 |
| **反事实推理** | 预测基因扰动后的表达变化 |
| **双重解耦** | 分离细胞内和细胞间调控程序 |
| **空间感知** | 利用空间信息增强因果推断 |
| **可识别性保证** | 数学上保证因果效应的可识别性 |

### 8.2 独特优势

1. **首个空间因果方法**: 首个专门针对空间转录组数据的因果推断方法
2. **物理启发**: 引入拉格朗日力学提供生物学约束
3. **可解释性**: 提供明确的因果机制解释
4. **虚拟组织基础**: 为构建虚拟组织提供核心技术

### 8.3 输出可视化

- **因果网络图**: 基因调控网络可视化
- **空间因果热图**: 空间分布的因果效应
- **反事实对比**: 扰动前后表达变化对比
- **潜在空间**: UMAP/t-SNE降维可视化

### 10.3 使用示例

```python
import celcomen
from celcomen import CelcomenModel

# 加载空间转录组数据
adata = celcomen.load_visium("path/to/data")

# 初始化模型
model = CelcomenModel(
    n_genes=adata.n_vars,
    n_latent=32,
    causal_identifiable=True
)

# 训练模型
model.fit(adata, epochs=300)

# 因果发现
intra_programs, inter_programs = model.disentangle(adata)

# 反事实预测
counterfactual = model.predict_counterfactual(
    adata, 
    perturbation={"gene_A": 0.5}
)
```

## 11. 优势与局限

### 11.1 主要优势

| 优势 | 说明 |
|------|------|
| **因果可识别性** | 数学上保证因果效应的可识别性，而非仅相关性 |
| **空间感知** | 充分利用空间信息增强推断 |
| **双重解耦** | 同时建模细胞内和细胞间调控 |
| **物理约束** | 拉格朗日力学提供生物学合理的约束 |
| **反事实能力** | 支持扰动预测和虚拟实验 |

### 11.2 当前局限

| 局限 | 说明 | 潜在解决方案 |
|------|------|-------------|
| **计算成本** | 因果推断计算量大 | GPU优化、近似算法 |
| **数据需求** | 需要高质量空间数据 | 数据增强、迁移学习 |
| **可扩展性** | 大规模数据集处理 | 图采样、分块处理 |
| **验证难度** | 因果效应难以实验验证 | 与 Perturb-seq 结合 |
| **平台限制** | 主要针对 Visium/Perturb-map | 扩展到其他平台 |

### 11.3 未来发展方向

1. **多组学整合**: 整合蛋白质组、表观遗传学数据
2. **时序建模**: 扩展到时序空间转录组数据
3. **单细胞分辨率**: 支持亚细胞级因果推断
4. **虚拟组织**: 构建完整的虚拟组织模拟系统
5. **临床应用**: 开发临床决策支持工具
