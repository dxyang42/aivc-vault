## 核心技术

### 架构设计
GEARS结合**深度学习**与**基因-基因关系知识图谱**：

```
┌─────────────────────────────────────────────────────────────────────┐
│                     GEARS 架构概览                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  输入层                                                             │
│  ├── 未扰动细胞基因表达 (20,000+ 基因)                              │
│  └── 扰动基因标识 (单基因/多基因)                                   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                 扰动嵌入层 (Perturbation Embedding)          │   │
│  │  ├── 每个基因学习一个d维嵌入向量                              │   │
│  │  ├── 单基因扰动: 直接使用该基因嵌入                           │   │
│  │  └── 多基因组合: 组合嵌入 (非简单加和)                        │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              图神经网络层 (GNN - Graph Attention)            │   │
│  │  ├── 输入: 基因关系图 (STRING数据库 + 共表达网络)            │   │
│  │  ├── 节点: 基因 (20,000+)                                    │   │
│  │  ├── 边: 基因间关系 (蛋白互作、共表达等)                     │   │
│  │  └── 消息传递: 注意力机制聚合邻居信息                         │   │
│  │       ┌─────────────────────────────────────┐                │   │
│  │       │  注意力系数计算:                    │                │   │
│  │       │  e_ij = LeakyReLU(a^T [Wh_i || Wh_j])│               │   │
│  │       │  alpha_ij = softmax_j(e_ij)         │                │   │
│  │       │                                     │                │   │
│  │       │  消息聚合:                          │                │   │
│  │       │  h_i' = sigma(sum_j alpha_ij * Wh_j)│                │   │
│  │       └─────────────────────────────────────┘                │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │              非加性效应预测模块 (核心创新)                   │   │
│  │  ├── 加性效应: 各单基因扰动效应的线性叠加                   │   │
│  │  └── 非加性效应: GNN学习的基因交互项                        │   │
│  │       x_combo = x_control + sum(delta_i) + delta_interaction │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  输出层: 预测的扰动后基因表达谱                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 技术特点
- **图神经网络 (GNN)**: 融合基因关系图，使用Graph Attention Network
- **知识图谱整合**: STRING数据库、基因共表达网络、Gene Ontology
- **几何深度学习**: 捕捉基因间相互作用
- **非加性效应建模**: 预测多基因扰动的非加性效应（协同/拮抗）

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 预测单个基因敲除的转录响应 |
| **单基因过表达 (OE)** | 完全支持 | 预测基因过表达效应 |
| **双基因组合** | 核心优势 | 预测基因对的非加性相互作用 |
| **三基因及以上组合** | 支持 | 可扩展至多基因组合 |
| **未见基因预测** | 支持 | 利用知识图谱泛化到新基因 |

### 预测准确率与评估指标

| 指标 | 结果 | 对比基线 |
|------|------|---------|
| **整体预测精度** | 比现有方法高 **40%** | vs 传统线性模型 |
| **组合扰动预测** | 提升最显著 | vs 加性假设模型 |
| **遗传交互识别** | 能力是先前方法的 **2倍** | vs 统计检验方法 |
| **差异表达基因预测** | AUPRC > 0.45 | - |
| **Pearson相关性** | r > 0.85 (top基因) | - |

### 评估数据集
- **Norman Dataset**: 105单基因 + 131双基因扰动，91k观测值
- **Adamson Dataset**: 多个细胞系的扰动数据
- ** Dixit Dataset**: Perturb-seq数据

## 与其他扰动预测方法的差异

| 特性 | GEARS | CPA | scPerb | Linear Model |
|------|-------|-----|--------|--------------|
| **知识图谱** | 使用 | 不使用 | 不使用 | 不使用 |
| **非加性效应** | 显式建模 | 部分支持 | 隐式学习 | 不支持 |
| **未见基因泛化** | 强 | 中等 | 弱 | 无 |
| **组合预测** | 核心优势 | 支持 | 支持 | 加性假设 |
| **可解释性** | 高 | 中等 | 中等 | 高 |
| **不确定性量化** | 支持 | 支持 | 不支持 | 不支持 |

## 核心能力

| 能力 | 描述 |
|------|------|
| **单基因扰动预测** | 预测单基因扰动的转录结果 |
| **多基因扰动预测** | 预测多基因组合扰动，捕捉非加性效应 |
| **未见基因预测** | 利用知识图谱预测未见过的基因扰动 |
| **遗传交互分类** | 预测四种遗传交互亚型（协同、拮抗等） |
| **基因组合推荐** | 推荐有信息量的基因组合进行实验验证 |
| **不确定性量化** | 提供预测置信度估计 |

## 训练与验证

### 损失函数组成

GEARS的训练目标包含多个部分：

1. **重建损失**: 预测表达与真实表达的均方误差
2. **交互损失**: 约束非加性效应项的准确性
3. **正则化**: L2权重衰减防止过拟合

### 训练数据要求
- 需要配对的控制-扰动细胞样本
- 基因关系图（STRING等数据库）
- 建议至少数百个扰动样本用于训练

## 应用场景

1. **药物发现**: 预测药物组合的协同效应，识别新的药物靶点
2. **癌症生物学**: 预测癌细胞对治疗的响应，理解耐药机制
3. **再生医学**: 预测细胞重编程结果，优化分化协议
4. **个性化医疗**: 基于患者基因型预测治疗反应

## 优势与局限

### 优势
- 结合先验知识（知识图谱），可解释性强
- 组合扰动预测能力突出，非加性效应建模
- 支持未见基因的零样本预测
- 开源，社区活跃

### 局限
- 需要高质量的基因关系数据
- 对未见细胞类型的泛化能力有限
- 计算复杂度随基因数增加
