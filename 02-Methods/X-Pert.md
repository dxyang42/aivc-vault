# X-Pert

## 1. 基本信息

| 属性 | 详情 |
|------|------|
| **模型名称** | X-Pert |
| **发布时间** | 2025-2026年 |
| **机构** | Chen Li 等 |
| **GitHub** | [Chen-Li-17/X-Pert](https://github.com/Chen-Li-17/X-Pert) |
| **论文** | Nature Machine Intelligence 2025 |
| **类型** | Transformer-based 框架 |
| **文档** | [x-pert.readthedocs.io](https://x-pert.readthedocs.io/) |

X-Pert 是一个基于 Transformer 的深度学习框架，专为基因表达预测和扰动效应建模而设计。该框架通过创新的架构设计，能够联合建模遗传、化学和组合扰动，为单细胞基因组学研究提供了强大的预测工具。

---

## 2. 输入/输出

### 输入

| 输入类型 | 描述 |
|----------|------|
| **基因表达数据** | 单细胞或批量 RNA-seq 表达矩阵 |
| **扰动信息** | 遗传扰动（CRISPR、RNAi）、化学扰动（药物处理）、组合扰动 |
| **剂量信息** | 定量扰动强度（药物浓度、sgRNA 效率等） |
| **细胞元数据** | 细胞类型、批次、实验条件等 |

### 输出

| 输出类型 | 描述 |
|----------|------|
| **预测基因表达** | 扰动后的基因表达谱 |
| **扰动效应评分** | 各基因对扰动的响应程度 |
| **组合效应预测** | 多扰动联合作用的协同/拮抗效应 |
| **潜在空间表示** | 细胞和扰动的低维嵌入 |

## 3. 网络架构

### 3.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                        X-Pert 架构                           │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────┐    ┌─────────────────┐                 │
│  │ Perturbation    │    │   Cell Encoder  │                 │
│  │   Perceiver     │◄──►│                 │                 │
│  │  (扰动感知器)    │    │  (细胞编码器)    │                 │
│  └────────┬────────┘    └────────┬────────┘                 │
│           │                      │                          │
│           ▼                      ▼                          │
│    ┌─────────────────────────────────────┐                  │
│    │        Perturbverse                 │                  │
│    │    (共享潜在空间)                    │                  │
│    └─────────────────────────────────────┘                  │
│                      │                                      │
│                      ▼                                      │
│    ┌─────────────────────────────────────┐                  │
│    │      剂量与功效感知模块              │                  │
│    │   (Dose & Efficacy Awareness)       │                  │
│    └─────────────────────────────────────┘                  │
│                      │                                      │
│                      ▼                                      │
│    ┌─────────────────────────────────────┐                  │
│    │      预测输出层                      │                  │
│    │   (Predicted Gene Expression)       │                  │
│    └─────────────────────────────────────┘                  │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 核心组件

#### Perturbation Perceiver（扰动感知器）
- **功能**：将异质扰动（遗传、化学、组合）嵌入共享潜在空间 "Perturbverse"
- **特点**：
  - 统一处理不同类型的扰动输入
  - 学习扰动之间的语义关系
  - 支持未见扰动的泛化

#### Cell Encoder（细胞编码器）
- **功能**：融合多维度细胞信息
- **输入 Token**：
  - **基因身份 Token**：标识特定基因
  - **表达 Token**：基础基因表达水平
  - **扰动影响 Token**：扰动对基因的特定影响

#### 剂量与功效感知模块
- **功能**：整合定量扰动强度信息
- **作用**：
  - 建模剂量-响应关系
  - 考虑 sgRNA 效率差异
  - 提高预测准确性

## 4. 训练数据

### 4.1 数据来源

| 数据类型 | 描述 | 规模 |
|----------|------|------|
| **单细胞 Perturb-seq** | CRISPR 筛选后的单细胞表达数据 | 大规模 |
| **批量 RNA-seq** | 药物处理后的批量表达数据 | 大规模 |
| **组合扰动数据** | 多基因/药物联合处理数据 | 中等规模 |
| **剂量响应数据** | 不同浓度/效率的扰动数据 | 多样化 |

### 4.2 数据集覆盖

- **细胞系**：多种人类细胞系（K562、RPE1、A549 等）
- **扰动类型**：
  - 遗传：KO（敲除）、KD（敲低）、OE（过表达）
  - 化学：小分子药物、生物制剂
  - 组合：双基因、基因-药物组合
- **数据规模**：涵盖数千至数万个细胞的表达谱

## 5. 训练策略

### 5.1 预训练阶段

| 阶段 | 目标 | 数据 |
|------|------|------|
| **基础预训练** | 学习基因表达的基础分布 | 大规模未扰动细胞数据 |
| **扰动感知预训练** | 建立 Perturbverse 空间 | 单扰动数据 |
| **组合预训练** | 学习组合效应模式 | 组合扰动数据 |

### 5.2 微调策略

- **领域适应**：针对特定细胞类型或实验条件微调
- **少样本学习**：利用有限的实验数据快速适应新场景
- **持续学习**：支持增量更新，整合新扰动数据

### 5.3 损失函数

- **重建损失**：预测表达与真实表达的差异
- **对比损失**：学习有意义的 Perturbverse 表示
- **剂量感知损失**：约束剂量-响应关系的准确性

### 6.1 单细胞数据集表现

|--------|------|-------------|
| **Norman et al.** | 基因表达预测相关性 | 优于现有方法 |

| **Papalexi et al.** | 组合效应预测 | 准确识别协同效应 |

| 数据集 | 任务 | 性能 |
|--------|------|------|
| **LINCS L1000** | 药物响应预测 | 高准确率 |
| **Drug repurposing** | 药物组合预测 | 有效识别协同药物对 |

### 6.3 关键优势

- **未见扰动预测**：对训练时未见的扰动仍能保持高预测精度

- **大规模筛选支持**：可扩展至全基因组规模的扰动预测

## 7. 功能特点

### 7.1 核心能力

| 功能 | 描述 |
|------|------|
| **多类型扰动建模** | 统一处理遗传、化学、组合扰动 |
| **剂量感知预测** | 整合定量扰动强度，改善预测精度 |
| **未见扰动泛化** | 对新型扰动具有良好的预测能力 |
| **组合效应分析** | 识别协同、拮抗、加性效应 |
| **可解释性** | 提供扰动-基因关系的洞察 |

### 7.2 技术特色

- **Perturbverse 空间**：统一的扰动表示空间，支持异质扰动的比较和组合
- **Cell Encoder 设计**：有效融合基因身份、表达和扰动影响信息
- **Transformer 架构**：利用自注意力机制捕获基因间的复杂关系

## 10. 优势与局限

### 10.1 核心优势

| 优势 | 说明 |
|------|------|
| **统一框架** | 单一模型处理多种扰动类型，简化工作流程 |
| **强大泛化** | 对未见扰动和组合具有出色的预测能力 |
| **剂量感知** | 整合定量信息，提供更精细的预测 |
| **可扩展性** | 支持大规模筛选和全基因组分析 |
| **开源友好** | 完整的开源实现和文档支持 |

### 10.2 当前局限

| 局限 | 说明 | 潜在改进方向 |
|------|------|-------------|
| **数据依赖性** | 需要高质量的大规模扰动数据进行训练 | 开发数据增强和迁移学习策略 |
| **计算资源** | 大规模模型训练和推理需要 GPU 资源 | 模型压缩和高效推理优化 |
| **细胞类型特异性** | 跨细胞类型的泛化可能受限 | 整合更多细胞类型数据 |
| **时间动态** | 主要关注稳态表达，时间动态建模有限 | 引入时序建模组件 |
| **空间信息** | 未整合空间转录组信息 | 扩展支持空间组学数据 |

### 10.3 适用性评估

**推荐使用场景**：
- 需要统一建模多种扰动类型的研究
- 大规模药物筛选和功能基因组学项目
- 需要预测未见扰动效应的探索性研究
- 药物组合优化和重定位研究

**需谨慎考虑的场景**：
- 数据稀缺的罕见细胞类型
- 需要精确时间动态建模的研究
- 空间异质性至关重要的组织研究
