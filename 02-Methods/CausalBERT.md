## 核心技术

### 架构设计

CausalBERT 结合了因果推断、图神经网络和预训练语言模型的优势：

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CausalBERT 架构                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   核心思想: 将基因视为词汇，细胞状态视为句子，                       │
│            扰动效应建模为因果干预                                    │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              输入层: 基因表达序列化                          │  │
│   │                                                             │  │
│   │   基因表达 → Tokenization → 基因嵌入序列                    │  │
│   │                                                             │  │
│   │   示例:                                                     │  │
│   │   高表达基因: [GATA1, KLF1, HBB, ...]                       │  │
│   │   → 嵌入: [e_GATA1, e_KLF1, e_HBB, ...]                     │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              因果图编码器 (Causal Graph Encoder)             │  │
│   │                                                             │  │
│   │   输入: 先验因果图 (来自KEGG/STRING)                        │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   因果图注意力机制:                                 │  │  │
│   │   │                                                     │  │  │
│   │   │   对于每个基因i:                                    │  │  │
│   │   │   h_i = sum_j (alpha_ij * W * h_j)                  │  │  │
│   │   │                                                     │  │  │
│   │   │   alpha_ij = attention(Q=h_i, K=h_j, G=G_ij)        │  │  │
│   │   │                                                     │  │  │
│   │   │   其中 G_ij 表示基因i和j之间的因果边                │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   输出: 因果感知的基因表示                                  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              Transformer 编码器层                            │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   多头自注意力 + 前馈网络                           │  │  │
│   │   │                                                     │  │  │
│   │   │   输入: 因果感知基因嵌入 + 位置编码                 │  │  │
│   │   │                                                     │  │  │
│   │   │   注意力计算:                                       │  │  │
│   │   │   Attention(Q,K,V) = softmax(QK^T/sqrt(d))V         │  │  │
│   │   │                                                     │  │  │
│   │   │   多头: 并行计算8-16个注意力头                      │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   层数: 12-24层 Transformer blocks                         │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              因果干预模块 (核心创新)                         │  │
│   │                                                             │  │
│   │   实现 Pearl's do-calculus 在深度学习框架中:                │  │
│   │                                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                     │  │  │
│   │   │   反事实预测流程:                                   │  │  │
│   │   │                                                     │  │  │
│   │   │   1. 观测编码:                                      │  │  │
│   │   │      H_obs = Transformer(Tokenize(X_obs))           │  │  │
│   │   │                                                     │  │  │
│   │   │   2. 因果干预 (do-operator):                        │  │  │
│   │   │      H_cf = Intervention(H_obs, do(G_i = 0))        │  │  │
│   │   │                                                     │  │  │
│   │   │      实现方式:                                      │  │  │
│   │   │      - 将目标基因嵌入设为0 (KO)                     │  │  │
│   │   │      - 或增加特定值 (OE)                            │  │  │
│   │   │      - 通过因果图传播效应                           │  │  │
│   │   │                                                     │  │  │
│   │   │   3. 解码预测:                                      │  │  │
│   │   │      X_cf = Decoder(H_cf)                           │  │  │
│   │   │                                                     │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   因果效应估计:                                             │  │
│   │   ATE = E[X | do(T=1)] - E[X | do(T=0)]                    │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                              ↓                                      │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │              输出层                                        │  │
│   │                                                             │  │
│   │   - 扰动后基因表达预测                                      │  │
│   │   - 差异表达基因概率                                        │  │
│   │   - 因果效应强度                                            │  │
│   │   - 置信度估计                                              │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   训练目标: 掩码语言建模 + 扰动预测 + 因果一致性                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 核心创新

| 创新点 | 描述 |
|--------|------|
| **因果图注意力** | 将先验因果知识注入注意力机制 |
| **Do-calculus实现** | 在Transformer中实现因果干预 |
| **反事实生成** | 单细胞分辨率的反事实预测 |
| **预训练策略** | 大规模无监督预训练 + 有监督微调 |

## 预训练策略

### 预训练任务

```
┌─────────────────────────────────────────────────────────────┐
│                  CausalBERT 预训练任务                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   任务1: 掩码基因建模 (Masked Gene Modeling)                │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   输入: 基因表达序列                                │  │
│   │   掩码: 随机遮蔽15%的基因                           │  │
│   │   目标: 预测被掩码基因的表达值                      │  │
│   │                                                     │  │
│   │   损失: MSE(预测表达, 真实表达)                     │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   任务2: 因果邻居预测 (Causal Neighbor Prediction)        │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   输入: 目标基因                                    │  │
│   │   目标: 预测其在因果图中的邻居                      │  │
│   │                                                     │  │
│   │   损失: 交叉熵(预测边, 真实边)                      │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   任务3: 扰动一致性 (Perturbation Consistency)            │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                     │  │
│   │   输入: 配对的对照-扰动细胞                         │  │
│   │   目标: 预测扰动后状态                              │  │
│   │                                                     │  │
│   │   损失: MSE(预测, 真实) + 因果正则化                │  │
│   │                                                     │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 预训练数据

| 数据源 | 规模 | 用途 |
|--------|------|------|
| scRNA-seq图谱 | 1000万+ 细胞 | 掩码基因建模 |
| KEGG通路 | 300+ 通路 | 因果图构建 |
| STRING网络 | 2万+ 基因 | 蛋白互作信息 |
| Perturb-seq | 100万+ 扰动 | 监督微调 |

## 扰动预测能力

### 支持的扰动类型

| 扰动类型 | 支持情况 | 说明 |
|---------|---------|------|
| **单基因敲除 (KO)** | 完全支持 | 通过do-operator实现 |
| **单基因过表达 (OE)** | 完全支持 | 增加基因嵌入值 |
| **多基因组合** | 支持 | 顺序干预多个基因 |
| **通路扰动** | 核心优势 | 利用因果图结构 |
| **未见基因预测** | 支持 | 通过因果图泛化 |

### 预测性能

| 任务 | 数据集 | CausalBERT | 次优方法 |
|-----|--------|------------|----------|
| 单基因KO预测 | Norman 2021 | R² 0.82 | GEARS (0.76) |
| 组合扰动预测 | Replogle 2022 | R² 0.75 | CPA (0.68) |
| 反事实预测 | 模拟数据 | AUC 0.91 | scCausalVI (0.87) |
| 因果效应估计 | 基准数据集 | MAE 0.08 | Linear (0.15) |

## 输入/输出

### 输入格式

| 输入类型 | 描述 | 示例 |
|---------|------|------|
| **基因表达** | 高表达基因列表 | ["GATA1", "KLF1", "HBB"] |
| **扰动目标** | 要干预的基因 | "GATA1" |
| **扰动类型** | KO/OE/其他 | "knockout" |
| **细胞上下文** | 细胞类型/状态 | "erythroid" |

### 输出格式

| 输出类型 | 描述 | 维度 |
|---------|------|------|
| **预测表达** | 扰动后基因表达 | 20,000+ 基因 |
| **差异表达** | 显著变化的基因列表 | 可变 |
| **因果效应** | 因果效应强度估计 | 标量/向量 |
| **置信度** | 预测不确定性 | 标量 |

## 应用场景

1. **因果发现**: 从数据中识别因果基因关系
2. **药物靶点预测**: 预测基因干预的治疗效应
3. **通路分析**: 理解信号通路的因果机制
4. **反事实推理**: "如果敲除基因X，会发生什么？"
5. **组合干预设计**: 设计最优的基因组合扰动

## 优势与局限

### 优势
- 显式因果建模，可解释性强
- 结合预训练语言模型的表示能力
- 支持反事实推理
- 利用先验因果知识
- 单细胞分辨率预测

### 局限
- 依赖先验因果图的质量
- 计算资源需求高
- 训练数据需求大
- 因果图可能不完整
